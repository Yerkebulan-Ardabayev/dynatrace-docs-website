    Navigate the Dynatrace platform¶

    • Latest Dynatrace
    • Reference
    • 11-min read
    • Updated on Jan 28, 2026

    Learn your way around the Dynatrace web UI.

    If you haven't upgraded to the latest Dynatrace, go to the bottom of this page for help with the old interface.

    Dock¶

    The Dock is your starting point. It's the expandable/collapsable panel on the left of the Dynatrace display. This is your tool belt, which you can equip with links to applications that you need daily.

    Use the Dock to:

    • Quickly switch between your pinned or recently used apps.
    • Go to the Launcher, global platform search, all apps in your environment, additional support resources, and your user preferences.

    The upper part of the Dock includes:

    • Dynatrace is your home.
    • Search enables you to search the platform.
    • Apps
    • Pinned and active apps, depending on your configuration and activity. In this example, the Notebooks, Dashboards, and Segments apps are all listed.

    The lower part of the Dock includes:

    • and to hide or display the Dock
    • to access support links
    • [your initials] [your name] to access user-specific links

    Dynatrace¶

    Select Dynatrace in the upper-left corner to go to your Dynatrace home page.

    • The default Getting started with Dynatrace "launchpad" page gives you a quick overview of prominent features to start using the platform. It's focused on introducing you to key aspects of the platform and reducing your learning curve.
    • You can create additional launchpads (custom home pages) with handpicked lists of apps, documents, links, and informational text. For more information, see Launchpads.

    Search¶

    The Search option is always available in the Dock.

    • If you collapse the Dock, you'll only see the icon.
    • The keyboard shortcut for Search is CTRL+K or CMD+K.

    Dynatrace platform search provides you with customizable options and features to help you filter search results. For details on advanced search features, see Platform search.

    Apps¶

    Select Apps to list and launch your installed apps. (If you collapse the Dock, you'll only see the icon.)

    • Browse apps by category.
    • Use the filter bar to search by app name
    • Select Explore more to find more apps on Dynatrace Hub

    Pinned and active apps¶

    The middle section of the Dock lists default apps and any other apps that you have pinned to the Dock or have recently used. Pin your favorite apps here to customize the Dock for quick access.

    To pin an app to the Dock

    1. Use Search or Apps to find and select the app you want to pin. The app icon and name will be displayed in the Dock.
    2. In the Dock, hover over the app and select Pin to dock.
      A pinned app will remain in the Dock until you remove it (hover over the app and select Unpin from dock).

    Collapse and Expand¶

    • Collapse hides the labels and displays only a narrow column of icons.

    • This frees screen space for the display of your data.

    • Icons might be all you need for frequently used apps and functions.
    • Expand displays the full Dock again so you can read labels.

    User menu¶

    To open the user menu, select your name in the lower-left corner, at the bottom of the Dock.

    • [your name] displays User settings, where you can change settings such as the display language, timezone, and light mode/dark mode.
    • Environments lists links to your available environments.
    • Appearance offers a quick switch to light, dark, or browser theme mode.
    • Account Management opens Account Management in a new tab.
    • Latest Dynatrace toggles the latest Dynatrace on and off.
    • Sign out signs you out of Dynatrace.

    Previous Dynatrace¶

    If you haven't upgraded to the latest Dynatrace, the following sections describe basic navigation for the previous Dynatrace.

    Filter the menu

    Use the filter box above the menu to find and select Dynatrace pages.

    1. Enter a partial search string.

    2. Select the best match.

    Save your favorites

    • To add a Dynatrace page to your favorites, hover over the page name and select the star

    * To access a favorite, expand the Favorites section and select the page

    * To remove a page from your favorites, select the star again

    Expand and collapse menu sections

    • To show the contents of a menu section, select the expand button.

    * To hide the contents of a menu section, select the collapse button.

    Hide or show the menu

    • To hide the menu, select the arrow button above the menu.

    * To display the menu, select the menu button in the upper-left corner.

    User menu

    The contents of this user menu depend on your deployment type, configuration, and permissions.

    • Go to the user menu for your own account information, to sign out, to switch between environments, and to get the latest Dynatrace news.
    • If you have administration rights, select Account settings on the user menu to manage user, group, and account settings.
