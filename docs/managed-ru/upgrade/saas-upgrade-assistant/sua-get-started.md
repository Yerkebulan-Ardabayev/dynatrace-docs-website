# Начало работы с SaaS Upgrade Assistant

* Updated on Dec 19, 2024

Latest Dynatrace

## Как загрузить данные и начать обновление?

1. В Dynatrace Launcher выберите ![SaaS Upgrade Assistant](https://dt-cdn.net/images/saas-upgrade-assistant-61dc5b83c0.svg "SaaS Upgrade Assistant") **SaaS Upgrade Assistant**.
2. Выберите **Upload data**, чтобы импортировать конфигурацию вашей среды Dynatrace Managed.
3. Выберите загруженную среду для перехода на страницу **Upgrade overview**.
4. **Select deployment type** — выберите между полным развёртыванием (**Deploy all configurations**) и выборочным развёртыванием (**Choose configurations to deploy**).
5. **Choose configurations to include in the deployment** — подтвердите количество включённых конфигураций и перейдите на вкладку **Configurations** для добавления дополнительных.
   Примечание: этот шаг отображается только при выборочном развёртывании.
6. [Обновите владельцев дашбордов](/managed/upgrade/saas-upgrade-assistant/sua-update-dashboard-owners "Update dashboard owners automatically with SaaS Upgrade Assistant.").
7. **Prepare authorization token** — создайте временный токен, удалите старый или перейдите в ![Settings](https://dt-cdn.net/images/settings-icon-256-38e1321b51.webp "Settings") **Settings** для управления токенами.
8. **Run deployment** — проверьте итоговую сводку и запустите развёртывание.
