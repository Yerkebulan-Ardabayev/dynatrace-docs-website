# Локальная самомониторинг-среда

* Extension
* Published Nov 16, 2021

Кластер Dynatrace Managed включает выделенную локальную самомониторинг-среду, собирающую внутренние метрики работоспособности развёртывания. Эта среда является полноценной доступной средой Dynatrace. Она содержит все метрики самомониторинга, собранные и агрегированные из других сред вашего кластера Dynatrace Managed. Данные хранятся исключительно локально на вашем кластере. Подробности см. в разделе [Метрики самомониторинга](/managed/analyze-explore-automate/metrics-classic/self-monitoring-metrics "Explore the complete list of self-monitoring Dynatrace metrics.").

## Сведения

Расширение предназначено для пользователей, ответственных за мониторинг работоспособности и размера Managed Cluster Dynatrace.

**Расширение позволяет**:

* Получить обзор текущего использования кластера Dynatrace.
* Проверить, правильно ли масштабирован кластер Dynatrace для обработки текущей нагрузки.
* Просматривать текущий и прошлый ввод данных в кластер Dynatrace, например вызовы сервисов или пользовательские сеансы/действия.
* Просматривать количество подключённых OneAgent к кластеру Dynatrace в динамике.

### Самомониторинг-среда

Перейти в самомониторинг-среду можно из [меню пользователя](/managed/discover-dynatrace/get-started/dynatrace-ui#user "Navigate the Dynatrace Managed platform"), так же как и в любую другую среду. Локальная самомониторинг-среда называется **Local-Self-Monitoring**. Она также отображается в Cluster Management Console, где можно назначать ей разрешения пользователей.

### Изучение метрик самомониторинга

Для просмотра реализованных метрик самомониторинга выполните фильтрацию по метрикам с префиксом `dsfm:` в [браузере метрик](/managed/analyze-explore-automate/dashboards-classic/metrics-browser "Browse metrics with the Dynatrace metrics browser."). Описание метрик содержит подробности. Самомониторинг-среду можно использовать для настройки оповещений или дополнительных панелей управления. Среда защищена от удаления.

### Лицензирование

Без учёта в потреблении лицензии

Самомониторинг-среда исключена из потребления лицензии. Некоторые функции ограничены в этой среде. Например, к ней нельзя подключить OneAgent.

**Требуется лицензирование DDU**  
По техническим причинам самомониторинг-среда включена только для клиентов Dynatrace Managed, использующих лицензирование DDU, хотя самомониторинг-среда исключена из всего потребления лицензий.

### Панель управления работоспособностью и использованием развёртывания кластера

Панель управления показывает агрегированные данные для сред вашего Managed Cluster. Она даёт представление о текущем использовании кластера и о том, по-прежнему ли кластер правильно масштабирован для работы с текущей нагрузкой.

Панель управления предоставляет:

* Аналитические данные о текущем использовании вашего Managed Cluster и Dynatrace Server.
* Обзор различных каналов ввода данных и текущего показателя захвата.
* Обзор подключённых OneAgent к Dynatrace.

Самомониторинг-панель управления доступна для кластеров Dynatrace Managed для использования в самомониторинг-среде кластера. Хотя её можно использовать в других средах (Managed и SaaS), часть отображаемых данных может быть неточной для них.

## Начало работы

1. Перейдите в самомониторинг-среду Dynatrace Managed, в которую необходимо установить панель управления.
2. В Dynatrace Hub выберите **Dynatrace Self-Monitoring (Managed)**.
3. Выберите **Add to environment** в правом нижнем углу для установки расширения самомониторинга в вашей среде.
