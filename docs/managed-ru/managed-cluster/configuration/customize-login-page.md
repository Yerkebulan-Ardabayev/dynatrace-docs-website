# Настройка страницы входа

* Практическое руководство
* Чтение за 1 минуту
* Обновлено 18 июня 2026 г.

Настройка страницы входа позволяет отобразить пользователям Managed Cluster системную информацию, сведения об аутентификации, юридические примечания или контакты администратора перед входом в систему.

Чтобы настроить страницу входа, перейдите в **User Authentication** > **Login screen** и измените сообщения по умолчанию (все сообщения необязательны):

* **Title**, текст, отображаемый под надписью "Dynatrace Managed."
* **Introduction**, текст, отображаемый более мелким шрифтом под заголовком.
* **Details**, изначально свёрнутый текст, который можно развернуть. Это поле нужно задавать только если задано поле **Introduction**.
