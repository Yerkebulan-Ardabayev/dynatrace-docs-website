# Настройка экстренных контактов в Dynatrace Managed

* Published Jun 14, 2021

Для указания экстренных контактов в Dynatrace Managed

1. В Cluster Management Console перейдите в **Settings** > **Emails** > **Email notifications**.
2. На странице **Email notifications** в разделе **Emergency notifications recipients** введите один или несколько адресов электронной почты, разделённых запятыми.

   Эти адреса будут получать экстренные уведомления от экспертов Dynatrace в случае критических состояний кластера или серьёзных уязвимостей безопасности, требующих немедленного реагирования.
3. Сохраните изменения.
