# Удалённый доступ к кластеру

* Updated on Aug 11, 2022

Эксперты Dynatrace могут удалённо помочь вам с обновлением кластера Dynatrace Managed и устранением неполадок. Для этого необходимо предоставить им разрешение на удалённый доступ к кластеру. Вы можете настроить разрешения удалённого доступа для кластера Dynatrace Managed, чтобы авторизовать экспертов Dynatrace для предоставления обновлений и проактивной поддержки.

Требуются права администратора

Для доступа к Cluster Management Console необходимы права администратора кластера.

Для настройки уровня разрешений в кластере в разделе **Cluster Management Console** перейдите в **Settings** > **Remote access permissions**.

На этой странице можно разрешить экспертам Dynatrace удалённый доступ к кластеру. Если этот параметр включён и обнаружены события, эксперты Dynatrace могут удалённо скорректировать настройки кластера для обеспечения оптимальной производительности и стабильности.

Безопасность

Все обмены данными с Mission Control защищены и осуществляются по HTTPS с проверкой сертификатов, аналогичной браузерной. Все изменения конфигурации Dynatrace Managed полностью фиксируются в журнале аудита, каждый удалённый доступ регистрируется как отдельное событие (перейдите в **Events**, чтобы просмотреть список записанных событий). Команда Mission Control не имеет доступа к сертификатам или учётным данным пользователей, а также не может получить root-доступ к серверам.

После включения удалённого доступа Dynatrace можно задать разрешения удалённого доступа для экспертов Dynatrace, выбрав одну из следующих областей.

* **All**

  Вся команда экспертов Dynatrace может получить доступ к вашему кластеру и предоставить вам полный спектр проактивной поддержки с оптимизацией настроек кластера.
* **Read-only access to all**

  Вся команда экспертов Dynatrace может получить доступ к кластеру, но не может изменять настройки кластера. Данный вариант существенно ограничивает уровень проактивной поддержки. При выборе этого варианта для пользователя с удалённым доступом доступна только роль **Viewer**. Эксперты Dynatrace свяжутся с вами для внесения необходимых изменений.
* **Approved**

  Доступ к кластеру разрешён только одобренным экспертам Dynatrace. Администраторы кластера получают email-уведомления о запросах на удалённый доступ, ожидающих подтверждения. Администратор кластера должен одобрить каждый запрос для предоставления разрешений. Вы можете настроить срок действия и роль, которую предоставляете. Также можно заранее предоставить разрешения известным экспертам Dynatrace.

  Данная область предоставляет максимальный контроль над тем, кто может получить доступ к кластеру, однако существенно ограничивает возможности команды экспертов Dynatrace по предоставлению проактивной поддержки.

  Для пользователя с удалённым доступом можно назначить роль **Admin**, **User** или **Viewer**. Подробнее о разрешениях, назначаемых каждой роли, см. в таблице ниже.
