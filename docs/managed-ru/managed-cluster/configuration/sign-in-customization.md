# Настройка страницы входа

* Published Jan 28, 2021

Как администратор кластера Dynatrace Managed, вы можете столкнуться с ситуациями, когда необходимо передать информацию пользователям кластера перед входом в систему. Например, может потребоваться отобразить системную информацию, сведения об аутентификации, юридические уведомления или контактные данные администратора. Для этого можно настроить страницу входа для отображения конкретной информации.

Для настройки страницы входа перейдите в **User Authentication** > **Login screen** и измените сообщения по умолчанию (все сообщения являются необязательными):

* **Title** — текст, отображаемый под надписью «Dynatrace Managed».
* **Introduction** — текст, отображаемый меньшим шрифтом под заголовком.
* **Details** — изначально свёрнутый текст, который пользователь может развернуть. Может быть задан только при определённом значении **Introduction**.
