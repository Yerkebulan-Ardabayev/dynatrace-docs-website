# Переход с Managed на развёртывание SaaS

* Published Apr 24, 2023

Данная документация предназначена для владельцев платформы Dynatrace, архитекторов и менеджеров программ, планирующих переход с Dynatrace Managed на Dynatrace SaaS.

Воспользуйтесь последними инновациями Dynatrace, перейдя с развёртывания Dynatrace Managed на развёртывание Dynatrace SaaS. Спланируйте и выполните плавный переход с помощью этого подробного пошагового руководства, которое включает всю необходимую информацию, ресурсы и инструменты.
