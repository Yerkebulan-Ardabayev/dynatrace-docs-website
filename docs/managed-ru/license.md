Dynatrace Platform Subscription (DPS), это модель лицензирования на основе потребления, разработанная для гибкости.

Заключается соглашение (обычно на 1–3 года) с минимальным годовым обязательством.
По мере того как организация использует возможности Dynatrace, потребление засчитывается в счёт этого обязательства на основе прайс-листа, который определяет цену за единицу для каждой возможности.
После достижения годового обязательства можно продолжать использовать Dynatrace, оплачивая использование по требованию без надбавок за превышение.

Страницы этого раздела помогут узнать больше о том, как работает DPS.
