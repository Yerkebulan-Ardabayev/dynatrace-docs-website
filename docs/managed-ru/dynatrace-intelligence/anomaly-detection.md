# Обнаружение аномалий

# Обнаружение аномалий

* Overview
* 2-min read
* Updated on Dec 05, 2025

Dynatrace непрерывно отслеживает производительность каждого аспекта ваших приложений, сервисов и инфраструктуры, автоматически изучая все базисные метрики и общее состояние каждого компонента в вашей среде, включая время отклика приложений и сервисов. Такие переменные, как геолокация, тип браузера, операционная система, пропускная способность соединения и действия пользователей, учитываются автоматически. Это интеллектуальное [автоматическое построение базиса](/managed/dynatrace-intelligence/anomaly-detection/automated-multidimensional-baselining "Узнайте, как ИИ Dynatrace автоматически вычисляет базисы по многомерной схеме.") позволяет Dynatrace обнаруживать аномалии с высокой степенью детализации и уведомлять вас об обнаруженных проблемах в режиме реального времени. Вы можете настраивать пороги, сформированные автоматическим построением базиса, либо [регулируя чувствительность обнаружения проблем](/managed/dynatrace-intelligence/anomaly-detection/adjust-sensitivity-anomaly-detection "Узнайте, как настроить чувствительность обнаружения проблем в Dynatrace."), либо — при необходимости — задавая собственные статические пороги.

## Сценарии использования

* Используйте метрические события для получения оповещений о проблемах в вашей среде.
* Настраивайте конфигурацию вручную или используйте автоадаптивный порог для обнаружения аномального поведения.
* Настраивайте оповещения для пользовательских событий.
