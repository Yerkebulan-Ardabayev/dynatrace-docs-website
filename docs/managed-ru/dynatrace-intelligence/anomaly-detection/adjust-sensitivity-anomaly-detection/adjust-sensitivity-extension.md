# Настройка чувствительности обнаружения аномалий для событий расширений

* How-to guide
* 1-min read
* Published Sep 06, 2021

[Расширения](/managed/ingest-from/extend-dynatrace "Узнайте, какие механизмы расширений предлагает Dynatrace."), которые вы установили, предоставляют различные пороги событий, оцениваемые системой обнаружения аномалий. Конкретный список зависит от расширения.

Чтобы настроить пороги аномалий производительности для компонентов инфраструктуры, отслеживаемых с помощью расширений

1. Перейдите в раздел **Settings** > **Anomaly detection** > **Extension events**. Здесь вы найдёте список всех типов событий, предоставляемых установленными расширениями мониторинга.
2. Разверните нужное событие, чтобы просмотреть и настроить доступные пороги.
