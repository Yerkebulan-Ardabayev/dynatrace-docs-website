# Анализ первопричин

* Explanation
* 1-min read
* Updated on Jan 28, 2026

Анализ первопричин (RCA) Davis AI автоматически оценивает всю собранную и поступившую информацию и выделяет объекты в причинно-следственной топологии, определённые как первопричина сложной ситуации. Он предлагает контекстно-зависимый подход, использующий доступную контекстную информацию — например, топологию на уровне кода — для определения точной причины проблемы.

## Сценарии использования

* Экономьте время, включая все релевантные аномалии и ранжируя причинно-следственных участников для определения основной причины.
* Снизьте нагрузку оповещений, объединяя несколько связанных аномалий в одну проблему.
* Повышайте точность анализа первопричин, используя все доступные данные из множества источников.
