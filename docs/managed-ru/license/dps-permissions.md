# Разрешения DPS

* Explanation
* 2-min read
* Published Sep 03, 2025

Разрешения Dynatrace структурированы на трёх уровнях:

| Область | Роль | Возможности |
| --- | --- | --- |
| Уровень аккаунта | Account Admin / Viewer | Просмотр и управление выставлением счетов, пользователями, подписками и окружениями. |
| Уровень окружения | Environment Admin / Viewer | Доступ и настройка окружений мониторинга. |
| Уровень подписки | Subscription Admin / Viewer | Просмотр потребления, прогнозов и распределения затрат по аккаунтам. |

Разрешения накапливаются и назначаются в **Account Management** > **Identity & access management**.
Подробное описание см. в [Работа с политиками](/managed/manage/identity-access-management/permission-management/manage-user-permissions-policies "Working with policies").
