# Подписка Dynatrace Platform Subscription для гибридных конфигураций (DPS for Hybrid)

# Подписка Dynatrace Platform Subscription для гибридных конфигураций (DPS for Hybrid)

* Пояснение
* Чтение: 2 мин
* Опубликовано 28 января 2026 г.

С DPS for Hybrid можно использовать одну лицензию Dynatrace Platform Subscription (DPS) для любой комбинации аккаунтов Managed и SaaS.
Единственное требование: аккаунты должны принадлежать одному клиенту.

С DPS for Hybrid:

* Единое годовое обязательство распределяется между несколькими аккаунтами.
* Ваша форма заказа содержит все соответствующие возможности тарифной карты для ваших развёртываний Managed и/или SaaS.
* Потребление по запросу (ODC) всегда включено для всех аккаунтов.

Подробнее о DPS в целом см. в разделе [Лицензирование Dynatrace](/managed/license "О подписке Dynatrace Platform Subscription (DPS), модели лицензирования для всех возможностей Dynatrace.").

## Зачем использовать DPS for Hybrid

Объединяя несколько аккаунтов под одной лицензией DPS, DPS for Hybrid позволяет:

* Подписать единый контракт независимо от числа аккаунтов.
* Максимально повысить гибкость и упростить лицензирование для удобства использования.
* Получить мгновенный доступ к единому дашборду с отображением затрат по всем аккаунтам.
* Планировать стратегически с учётом прогнозируемого годового обязательства по всем аккаунтам для максимального эффекта.
* Балансировать потребление между несколькими аккаунтами, не допуская перерасхода в одном аккаунте при недоиспользовании в других.

## Когда DPS for Hybrid вам подходит

DPS for Hybrid полезен, если:

* Вы хотите перенести своё развёртывание Managed в SaaS.
* У вас есть развёртывания Managed и SaaS, и вы хотите объединить их под одной лицензией DPS.
* У вас несколько развёртываний SaaS, и вы хотите объединить их под одной лицензией DPS.

## Когда DPS for Hybrid вам не подходит

Вот несколько сценариев, в которых DPS for Hybrid не является подходящим решением:

| Сценарий | Причина |
| --- | --- |
| Вы хотите обмениваться данными мониторинга между разными аккаунтами. | DPS for Hybrid касается вашей лицензии, а не данных мониторинга. Хотя обмениваться данными между разными окружениями в одном аккаунте можно, обмен данными мониторинга между разными аккаунтами невозможен. |
| Вы хотите распределять бюджеты по нескольким аккаунтам или отчитываться по ним. | Бюджеты распределяются и отображаются в отчётах на уровне лицензии. С помощью DPS for Hybrid нельзя распределять бюджет по отдельным аккаунтам или отчитываться по ним. Для этого аккаунтам нужны отдельные лицензии DPS. |
| Одна из ваших лицензий DPS приобретена через маркетплейс, например:  * Microsoft Marketplace (Azure). * AWS Marketplace. * Google Cloud Marketplace. | DPS for Hybrid не может обмениваться информацией о лицензии между аккаунтами на разных маркетплейсах.  Вы по-прежнему можете использовать DPS for Hybrid при развёртывании через гиперскейлер. На DPS for Hybrid не влияет поставщик технической инфраструктуры. |
| Вы являетесь [партнёром по решениям Dynatrace](https://www.dynatrace.com/partners/solution-partners/). | DPS for Hybrid предназначен только для конечных клиентов, а не для партнёров по решениям Dynatrace. |

## Управление затратами с DPS for Hybrid

Для управления затратами можно использовать прогнозы, события затрат и оповещения о бюджете.

### Прогнозы и события затрат

Прогнозы и события затрат помогают управлять бюджетом DPS for Hybrid на уровне аккаунта.
Подробнее см. в разделе [Настройка оповещений о затратах](/managed/license/cost-monitors "Узнайте, как использовать функцию Cost Monitors для создания прогнозов и событий затрат.").

Обратите внимание, что для подписок DPS for Hybrid действуют следующие особенности:

* Прогнозы доступны только на уровне аккаунта.
  Получить прогнозы на уровне подписки невозможно.
* Прогнозы показывают общие затраты в абсолютном выражении.
  Показать прогнозы в процентах от общего обязательства DPS невозможно.

Прогнозы и события затрат являются лишь оценками и поэтому в некоторых случаях могут вводить в заблуждение.
Прогнозы основаны на статистических моделях, поэтому полагаться на них следует с осторожностью.

### Оповещения о бюджете

Используйте вкладку **Budgets** в Account Management для настройки бюджетов вашей подписки Dynatrace Platform Subscription (DPS).
Общую информацию о бюджетах см. в разделе [Настройка оповещений о бюджете](/managed/license/budget-alerts "Узнайте, как настраивать бюджеты в Dynatrace.").

Обратите внимание, что для подписок DPS for Hybrid действуют следующие особенности:

* Бюджеты задаются на уровне аккаунта.
  Задать бюджеты на уровне подписки невозможно.
* Оповещения о бюджете на основе процентов работать не будут.
  Для бюджетов потребуется задать фиксированные суммы.
* При переходе с DPS на DPS for Hybrid все процентные бюджеты будут удалены, а получатели уведомлены по электронной почте.

## Часто задаваемые вопросы (FAQ)

Что такое DPS for Hybrid?

DPS for Hybrid позволяет управлять более чем одним аккаунтом и потреблять из единого бюджета.
Отдельный контракт не нужен.

Что можно делать с DPS for Hybrid?

Вы можете управлять несколькими развёртываниями Managed и/или SaaS под одной лицензией и обязательством DPS.

Как начать работу с DPS for Hybrid?

Обратитесь к своему контактному лицу в Dynatrace.
Контактное лицо организует всё необходимое.

Где можно получить поддержку при необходимости?

Вы всегда можете обратиться к своему контактному лицу в Dynatrace.

Если вы не получили приглашение на форум, обратитесь к своему контактному лицу в DT.

Если я перейду с классической модели лицензирования на DPS for Hybrid, что для меня изменится?

Вы получите все преимущества мощной подписки Dynatrace Platform Subscription (DPS).
Подробнее о DPS см. в разделе [Лицензирование Dynatrace](/managed/license "О подписке Dynatrace Platform Subscription (DPS), модели лицензирования для всех возможностей Dynatrace.").

Можно ли добавлять аккаунты в DPS for Hybrid после его настройки?

Да. Вы можете добавлять аккаунты в любое время.

Где можно увидеть данные по всем аккаунтам?

После настройки DPS for Hybrid вы можете видеть всю информацию уровня лицензии и аккаунта в Account Management, см. [Обзор подписки (DPS for Hybrid)](/managed/manage/account-management/license-subscription/subscription-overview-dps-hybrid "Просмотр сводки бюджета и анализа затрат DPS for Hybrid.").
Здесь представлена аналитика и информация, аналогичная Subscription Overview для стандартных лицензий DPS.

То есть у каждого аккаунта свой бюджет, а сверху есть единое обязательство?

Нет.
У DPS for Hybrid единый бюджет на уровне лицензии.
Внутри DPS for Hybrid нет бюджета на уровне аккаунта.

Каждый аккаунт потребляет из одного единого бюджета.
Вы можете настроить оповещения о бюджете и средства контроля в соответствии со своими потребностями.

Можно ли настроить уведомления на уровне аккаунта?

Да.
Вы можете настроить оповещения о бюджете и средства контроля для отдельных аккаунтов в соответствии со своими потребностями.

Сколько тарифных карт у меня будет при конфигурации DPS for Hybrid?

У клиентов DPS for Hybrid одна единая тарифная карта.
Она содержит все соответствующие возможности Managed и/или SaaS.
