# DPS для гибридной среды

# DPS для гибридной среды

* Объяснение
* Чтение за 2 мин
* Опубликовано 28 января 2026 г.

С помощью DPS для гибридной среды можно использовать одну лицензию Dynatrace Platform Subscription (DPS) для любой комбинации аккаунтов Managed и SaaS.
Единственное требование, чтобы аккаунты принадлежали одному и тому же клиенту.

С DPS для гибридной среды:

* Действует одно годовое обязательство, которое распределяется между несколькими аккаунтами.
* Форма заказа содержит все актуальные возможности тарифных карт для развёртываний Managed и/или SaaS.
* On-demand consumption (ODC) всегда включено для всех аккаунтов.

Подробнее о DPS в целом см. в разделе [Лицензирование Dynatrace](/managed/license "Dynatrace Platform Subscription, тарифные карты возможностей, гибридное лицензирование и предыдущие модели лицензирования.").

## Зачем использовать DPS для гибридной среды

Объединяя несколько аккаунтов под одной лицензией DPS, DPS для гибридной среды позволяет:

* Подписать один контракт независимо от количества аккаунтов.
* Максимизировать гибкость и упростить лицензирование для удобного использования.
* Мгновенно получить доступ к единой панели, отображающей затраты по всем аккаунтам.
* Стратегически планировать с учётом прогнозируемого годового обязательства по всем аккаунтам для максимального эффекта.
* Балансировать потребление между несколькими аккаунтами, не допуская перерасхода в одном аккаунте при недорасходе в других.

## Когда DPS для гибридной среды подходит

DPS для гибридной среды полезен, если:

* Нужно перенести развёртывание Managed на SaaS.
* Есть развёртывания Managed и SaaS, и их нужно объединить на одной лицензии DPS.
* Есть несколько развёртываний SaaS, и их нужно объединить на одной лицензии DPS.

## Когда DPS для гибридной среды не подходит

Вот несколько сценариев использования, для которых DPS для гибридной среды не является правильным решением:

| Сценарий использования | Причина |
| --- | --- |
| Нужно делиться данными мониторинга между разными аккаунтами. | DPS для гибридной среды ориентирован на лицензию, а не на данные мониторинга. Данными можно делиться между разными окружениями в рамках одного аккаунта, но передавать данные мониторинга между разными аккаунтами нельзя. |
| Нужно распределять или отчитываться по бюджетам между несколькими аккаунтами. | Бюджеты распределяются и отражаются в отчётах на уровне лицензии. DPS для гибридной среды нельзя использовать для распределения или отчётности по бюджету для отдельных аккаунтов. Если это нужно, у аккаунтов должны быть отдельные лицензии DPS. |
| Одна из лицензий DPS куплена через маркетплейс, например:  * Microsoft Marketplace (Azure). * AWS Marketplace. * Google Cloud Marketplace. | DPS для гибридной среды не может передавать информацию о лицензии между аккаунтами на разных маркетплейсах.  DPS для гибридной среды всё равно можно использовать при развёртывании через hyperscaler. На DPS для гибридной среды не влияет поставщик технической инфраструктуры. |
| Является [Solution Partner Dynatrace﻿](https://www.dynatrace.com/partners/solution-partners/). | DPS для гибридной среды рассчитан только на конечных клиентов, а не на Solution Partner Dynatrace. |

## Управление затратами с DPS для гибридной среды

Для управления затратами можно использовать прогнозы, события затрат и оповещения о бюджете.

### Прогнозы и события затрат

Прогнозы и события затрат помогают управлять бюджетом DPS для гибридной среды на уровне аккаунта.
Подробнее см. в разделе [Мониторы затрат](/managed/manage-your-costs/control/cost-monitors "Узнайте, как использовать функцию Cost Monitors для построения прогнозов и отслеживания событий затрат.").

Обратите внимание, что для подписок DPS для гибридной среды действуют следующие особенности:

* Прогнозы доступны только на уровне аккаунта.
  Получить прогнозы на уровне подписки нельзя.
* Прогнозы показывают общие затраты в абсолютном выражении.
  Показать прогнозы в виде процента от общего обязательства DPS нельзя.

Прогнозы и события затрат являются лишь оценками и поэтому в некоторых случаях могут вводить в заблуждение.
Прогнозы основаны на статистических моделях, поэтому полагаться на них нужно с осторожностью.

### Оповещения о бюджете

Используйте вкладку **Budgets** в Account Management, чтобы настроить бюджеты Dynatrace Platform Subscription (DPS).
Общую информацию о бюджетах см. в разделе [Оповещения о бюджете](/managed/manage-your-costs/control/budgets "Узнайте, как настроить бюджеты в Dynatrace.").

Обратите внимание, что для подписок DPS для гибридной среды действуют следующие особенности:

* Бюджеты определяются на уровне аккаунта.
  Определить бюджеты на уровне подписки нельзя.
* Оповещения о бюджете на основе процентов работать не будут.
  Для бюджетов нужно настраивать фиксированные суммы.
* При переходе с DPS на DPS для гибридной среды все бюджеты на основе процентов будут удалены при переключении, а получатели будут уведомлены по электронной почте.

## Часто задаваемые вопросы (FAQ)

Что такое DPS для гибридной среды?

DPS для гибридной среды позволяет управлять более чем одним аккаунтом и расходовать средства из единого бюджета.
Отдельный контракт не требуется.

Что можно делать с DPS для гибридной среды?

Можно управлять несколькими развёртываниями Managed и/или SaaS в рамках одной лицензии DPS и одного обязательства.

Как начать работу с DPS для гибридной среды?

Обратитесь к контактному лицу Dynatrace.
Оно инициирует всё необходимое.

Где можно получить поддержку в случае необходимости?

Всегда можно обратиться к контактному лицу Dynatrace.

Если приглашение на форум не получено, обратитесь к контактному лицу DT.

Если перейти с классической модели лицензирования на DPS для гибридной среды, что изменится?

Будут доступны все преимущества мощной Dynatrace Platform Subscription (DPS).
Подробнее о DPS см. в разделе [Лицензирование Dynatrace](/managed/license "Dynatrace Platform Subscription, тарифные карты возможностей, гибридное лицензирование и предыдущие модели лицензирования.").

Можно ли добавить дополнительные аккаунты в DPS для гибридной среды после настройки?

Да. Дополнительные аккаунты можно добавить в любой момент.

Где можно увидеть данные по всем аккаунтам?

После настройки DPS для гибридной среды всю информацию на уровне лицензии и аккаунтов можно увидеть в Account Management, см. [Обзор (DPS для гибридной среды)](/managed/manage/account-management/license-subscription/subscription-overview-dps-hybrid "Просмотр сводки бюджета и анализа затрат DPS для гибридной среды.").
Это даёт представление и сведения, аналогичные Subscription Overview для стандартных лицензий DPS.

Получается, у каждого аккаунта есть свой бюджет, а сверху ещё одно общее обязательство?

Нет.
DPS для гибридной среды имеет единый бюджет на уровне лицензии.
Бюджета на уровне аккаунта внутри DPS для гибридной среды не существует.

Каждый аккаунт расходует средства из одного общего бюджета.
Можно настроить оповещения и контроль бюджета в соответствии со своими потребностями.

Можно ли настроить уведомления на уровне аккаунта?

Да.
Можно настроить оповещения и контроль бюджета для отдельных аккаунтов в соответствии со своими потребностями.

Сколько тарифных карт будет при настройке DPS для гибридной среды?

У клиентов DPS для гибридной среды одна единая тарифная карта.
Она содержит все актуальные возможности Managed и/или SaaS.
