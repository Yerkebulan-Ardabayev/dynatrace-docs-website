# Отчёт о выставлении счетов

* Explanation
* 1-min read
* Published Nov 15, 2023

Отчёт о выставлении счетов DPS содержит все детали о состоянии вашего бюджета DPS за каждый день текущего периода. В нём показано, какие затраты и в какой день были учтены по дате учёта. Существует разница между датой потребления и датой учёта: потребление всегда происходит при использовании возможностей Dynatrace. Учёт происходит, когда потребление распознаётся и вычитается из вашего годового обязательства DPS. Таким образом, дата учёта может отличаться от даты потребления.

Чтобы перейти к отчёту о выставлении счетов в веб-интерфейсе Dynatrace, перейдите в **Subscription** > **Billing report**.

![DPS billing reports](https://dt-cdn.net/images/billingreport-1913-1256f12b00.png)

Отчёты о выставлении счетов DPS

Отчёт о выставлении счетов состоит из следующих диаграмм:

1. **Booked costs to date:** На этой диаграмме отображаются накопленные затраты по дням учёта. Также показано минимальное годовое обязательство по вашей подписке.
2. **Booked costs per day:** На этой диаграмме показаны затраты, учтённые за каждый отдельный день. Она не содержит накопленного состояния бюджета. Диаграмма помогает выявлять отклонения в ежедневных затратах.

3. **Number of reporting clusters:** На этой диаграмме отображается количество отдельных кластеров, передающих данные об использовании в каждый день. По ней можно определить периоды, в которые некоторые кластеры были недоступны. Когда кластеры временно недоступны, ежедневные затраты, как правило, снижаются.

API для Dynatrace Platform Subscription предоставляет доступ к исходным данным этого представления. Подробнее см. в документации по API.
