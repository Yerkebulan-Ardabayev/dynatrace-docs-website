# Классическое лицензирование Dynatrace

* 1-min read
* Published Mar 30, 2021

Потребление мониторинга при классическом лицензировании Dynatrace основано на различных единицах мониторинга, которые потребляет ваше окружение Dynatrace по мере использования возможностей платформы. Если у вас есть Dynatrace Platform Subscription, обратитесь к [документации Dynatrace Platform Subscription (DPS)](/managed/license "About Dynatrace Platform Subscription (DPS), the licensing model for all Dynatrace capabilities.").

Чтобы начать работу с Dynatrace, [свяжитесь с отделом продаж Dynatrace](https://www.dynatrace.com/contact/). Ваш представитель по продажам предоставит дополнительные сведения.

Эта страница предоставляется исключительно в информационных целях. К любому использованию продуктов или услуг Dynatrace применяются условия бесплатной пробной версии Dynatrace и/или вашей лицензии Dynatrace.

[### Жизненный цикл лицензии Dynatrace

Ознакомьтесь с жизненным циклом лицензий Dynatrace Managed и узнайте, как лицензии влияют на потребление сервисов Dynatrace.](/managed/manage/account-management/accounts-environments/license-lifecycle "Understand your Dynatrace DPS or Classic license lifecycle, and how it affects your consumption of Dynatrace services.")[### Мониторинг приложений и инфраструктуры (хост-единицы)

Каждый хост, отслеживаемый OneAgent в вашем окружении, потребляет хост-единицы.](/managed/license/monitoring-consumption-classic/application-and-infrastructure-monitoring "Understand how Dynatrace application and infrastructure monitoring consumption is calculated based on host units.")[### Davis data units (DDU)

Davis data units (DDU) — простое средство лицензирования определённых возможностей (пользовательских метрик, трасс, мониторинга логов и пользовательских событий) на платформе Dynatrace.](/managed/license/monitoring-consumption-classic/davis-data-units "Understand how Dynatrace monitoring consumption is calculated based on Davis data units (DDU).")[### Digital Experience Monitoring (единицы DEM)

Dynatrace Synthetic Monitoring, Real User Monitoring и Session Replay потребляют единицы DEM.](/managed/license/monitoring-consumption-classic/digital-experience-monitoring-units "Understand how Dynatrace Digital Experience Monitoring consumption is calculated based on DEM units.")[### Мониторинг Application Security (ASU)

Dynatrace Application Security потребляет единицы Application Security (ASU).](/managed/license/monitoring-consumption-classic/application-security-units "Understand how Dynatrace Application Security and Runecast SPM consumption are calculated.")
