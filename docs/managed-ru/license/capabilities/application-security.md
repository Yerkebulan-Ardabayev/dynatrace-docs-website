# Обзор Application Security (DPS)

# Обзор Application Security (DPS)

* Overview
* 13-min read
* Updated on Jun 05, 2025

[Application Security](/managed/secure/application-security "Access the Dynatrace Application Security functionalities.") помогает защитить облачные и локальные приложения в режиме реального времени с помощью интеллектуальной автоматизации.
Это включает визуализацию, анализ и мониторинг уязвимостей безопасности в вашей среде, а также блокировку обнаруженных атак на приложения.

На этой странице описаны различные возможности Application Security и функции, предоставляемые с подпиской DPS.

Информацию о том, как использование конкретной возможности переводится в потребление лицензионного лимита DPS, см. на страницах:

* [Runtime Vulnerability Analytics](/managed/license/capabilities/application-security/runtime-vulnerability-analytics "Learn how your consumption of the Dynatrace Runtime Vulnerability Analytics (RVA) DPS capability is billed and charged.")
* [Runtime Application Protection](/managed/license/capabilities/application-security/runtime-application-protection "Learn how how your consumption of the Runtime Application Protection (RAP) DPS capability is billed and charged.")
* [Security Posture Management](/managed/license/capabilities/application-security/security-posture-management "Learn how your consumption of the Dynatrace Security Posture Management (SPM) DPS capability is billed and charged.")

## Runtime Vulnerability Analytics

[Dynatrace Runtime Vulnerability Analytics](/managed/secure/application-security/vulnerability-analytics "Monitor, visualize, analyze, and remediate third-party and code-level vulnerabilities, track the remediation progress, and create monitoring rules.") позволяет обнаруживать, визуализировать, анализировать и отслеживать устранение уязвимостей сторонних компонентов и уязвимостей на уровне кода в вашей среде.

Эти уязвимости включают:

* [Уязвимости сторонних компонентов](/managed/secure/application-security/vulnerability-analytics/third-party-vulnerabilities "Monitor, visualize, analyze, and remediate third-party vulnerabilities, track the remediation progress, and create monitoring rules.") — возникают, когда приложение использует библиотеку или среду выполнения языка, содержащую уязвимости.
* [Уязвимости на уровне кода](/managed/secure/application-security/vulnerability-analytics/code-level-vulnerabilities "Monitor, visualize, and analyze code-level vulnerabilities.") — возникают, когда Dynatrace Application Security обнаруживает уязвимость в коде на основе анализа запросов, проходящих через приложения.

## Runtime Application Protection (RAP)

[Dynatrace Runtime Application Protection](/managed/secure/application-security/application-protection "Set up and configure Dynatrace Runtime Application Protection to monitor attacks and attack-generated code-level vulnerabilities.") использует анализ на уровне кода и анализ транзакций для автоматического обнаружения и блокировки атак на приложения в режиме реального времени.

## Security Posture Management (SPM)

[Security Posture Management (SPM)](/managed/upgrade/unavailable-in-managed "Your selection is unavailable in Dynatrace Managed.") обеспечивает непрерывный мониторинг и автоматизированную оценку сред Kubernetes®.
Благодаря аналитике конфигураций, проблем соответствия требованиям и управления рисками клиенты могут поддерживать высокий уровень защищённости.

Включает [Kubernetes Security Posture Management (KSPM)](/managed/upgrade/unavailable-in-managed "Your selection is unavailable in Dynatrace Managed.") — возможность, направленную на выявление и устранение рисков безопасности в кластерах Kubernetes.
