# Overview of all technologies running in your environment

# Обзор всех технологий в среде

* How-to guide
* Published Jul 19, 2017

Страница **Technology overview** предоставляет консолидированный обзор работоспособности и производительности всех отслеживаемых технологий в вашей среде. Это избавляет от необходимости просматривать несколько технологически-специфичных представлений анализа для получения нужной информации. Для доступа к странице **Technology overview** перейдите в **Technologies & Processes**.

Страница Technology overview организована в виде плиток — по одной для каждой технологически-специфичной группы процессов в вашей среде. Каждая группа процессов представляет набор процессов, выполняющих одну и ту же функцию на нескольких хостах, например кластеры Cassandra или ElasticSearch.

Для каждой группы процессов отображается количество экземпляров и распределение их использования ресурсов, включая минимальные и максимальные значения метрик.

В примере ниже показана группа процессов Java, состоящая из 2 экземпляров. В этой группе наблюдаются периодические пики потребления CPU: один из экземпляров потребляет 39% CPU, тогда как те же процессы на другой машине потребляют лишь 1,68% доступного CPU.

Выберите **Process group details** для доступа к представлению группы процессов, включающему комплексные категории метрик:

* **System performance** (память и CPU);
* **Networking** (связность и повторные передачи);
* **Technology-specific metrics** (время GC, приостановка);
* **Log files**.

Можно также выполнять анализ журналов для процессов, включённых в каждую группу процессов.
