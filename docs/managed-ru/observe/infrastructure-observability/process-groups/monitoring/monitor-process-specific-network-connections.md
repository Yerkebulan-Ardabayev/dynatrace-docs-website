# Monitor process-specific network connections

# Мониторинг сетевых соединений процессов

* How-to guide
* Updated on Apr 14, 2025

Для сетевого мониторинга Dynatrace предоставляет технологически-специфичные счётчики и комплексные сведения о сети. Dynatrace измеряет многочисленные ценные метрики и оповещает при обнаружении проблем в процессах.

При работе с сетевыми метриками, такими как повторные передачи и трафик, особенно важно понимать объём и качество взаимодействия между процессами. Только так можно правильно диагностировать и устранять проблемы, возникающие в соединениях между ключевыми компонентами инфраструктуры приложений. Когда Dynatrace обнаруживает сетевую проблему, например TCP-повторные передачи, можно точно определить затронутые сетевые каналы путём анализа соединений процессов, в которых возникла проблема.

Dynatrace отслеживает ключевые счётчики производительности на уровне соединений процессов, что позволяет легко определять пары процессов с высоким трафиком и проблемами связности. Метрики объёма и качества сети для пар процессов визуализируются в контексте выбранного хоста, а также в контексте соединений с другими процессами на других хостах.

Для анализа соединений процессов между хостами:

1. Перейдите в **Hosts**.
2. Выберите хост для анализа.
3. Выберите один из блоков **Processes** (входящих или исходящих) по обеим сторонам инфографики **Hosts** для просмотра:

   * количества процессов, участвующих в каждом взаимодействии;
   * скорости передачи данных;
   * количества новых соединений в минуту.
4. Выберите **Analyze process connections** для получения дополнительной информации о соединениях процессов и связанных метриках.

   Вы увидите все TCP-соединения, длящиеся более 10 секунд, установленные и принятые хостом. Соединения отображаются в формате, схожем с представлением [Smartscape topology](/managed/analyze-explore-automate/smartscape-classic "Learn how Smartscape visualizes all the entities and dependencies in your environment."). Средний столбец представляет анализируемый хост. Левая сторона показывает хосты и процессы, подключающиеся к анализируемому хосту. Правая сторона отображает исходящие соединения анализируемого хоста.

   Выберите любой узел процесса (средний столбец) для просмотра соответствующих сетевых метрик для соединений этого процесса (отображаются на правой панели). Для каждого соединения отображаются показатели **Transfer**, **Connectivity** и **Retransmissions**.
