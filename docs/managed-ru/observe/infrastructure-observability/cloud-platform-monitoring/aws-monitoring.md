# Мониторинг Amazon Web Services

* Explanation
* 2-min read
* Updated on Jan 28, 2026

Dynatrace автоматически обнаруживает, устанавливает базовые показатели и интеллектуально мониторит облачные среды Amazon Web Services.

## Мониторинг и обнаружение проблем

Мониторинг AWS в Dynatrace выполняет основанный на ИИ автоматический анализ влияния на бизнес, обнаружение проблем и анализ первопричин для всех сервисов AWS на основе всех метрик, публикуемых в Amazon CloudWatch. Этот анализ может быть дополнительно расширен полным стековым мониторингом, обеспечиваемым OneAgent, развёрнутым на вычислительных ресурсах, таких как Amazon EC2, AWS Lambda и Kubernetes. Все объекты также могут быть подвергнуты комплексному анализу журналов.

Помимо вышеперечисленного, Dynatrace обеспечивает понимание рабочей нагрузки применяемых ресурсов AWS и динамики среды по регионам.

Мониторинг AWS доступен всем клиентам Dynatrace независимо от того, является ли их кластер Dynatrace облачным (SaaS) или Managed.

## Простота использования и удобная визуализация

Мониторинг Amazon Web Services предоставляет согласованные готовые метрики, дашборды и оповещения сразу после включения мониторинга.

Измерения и анализ основных сервисов AWS визуализируются на специально созданном инфографическом дашборде (см. галерею). Для анализа влияния на бизнес и карт топологии также доступны удобные отчёты и дашборды.

## Гибкость

Пользователи Dynatrace могут выбирать, какие сервисы и экземпляры мониторятся и как выполняется этот процесс. Этот выбор можно сделать в рамках начальной настройки мониторинга или в любой последующий момент. Это также относится к новым сервисам, включённым в облаке AWS или добавленным компанией Amazon.

Можно выбирать метрики мониторинга для каждого из поддерживаемых сервисов. Аналогично, все дашборды для мониторинга AWS можно клонировать и легко настраивать по мере необходимости.

## Гибридные среды

Отслеживание сервисов и зависимостей не ограничивается AWS. Благодаря Davis® и его способности обрабатывать данные из всех типов сред, Dynatrace оптимизирован для мониторинга гибридных сред, включая бизнес-приложения, охватывающие несколько облачных платформ и платформ виртуализации, таких как Microsoft Azure, Google Cloud, VMware, Kubernetes, Openshift и локальную инфраструктуру.
