# Мониторинг Microsoft Azure

* Explanation
* 2-min read
* Updated on Jan 28, 2026

Dynatrace автоматически обнаруживает, устанавливает базовые показатели и интеллектуально мониторит облачные среды Microsoft Azure.

## Мониторинг и обнаружение проблем

Мониторинг Azure в Dynatrace выполняет основанный на ИИ автоматический анализ влияния на бизнес, обнаружение проблем и анализ первопричин для всех сервисов Azure на основе всех метрик, публикуемых в Azure Monitor. Этот анализ может быть дополнительно расширен полным стековым мониторингом, обеспечиваемым OneAgent, развёрнутым на вычислительных ресурсах, таких как Azure Virtual Machines, Azure App Service и Azure Kubernetes Service. Все объекты также могут быть подвергнуты комплексному анализу журналов.

Помимо вышеперечисленного, Dynatrace обеспечивает понимание рабочей нагрузки применяемых ресурсов Azure и динамики среды по регионам.

Мониторинг Azure доступен всем клиентам Dynatrace независимо от того, является ли их кластер Dynatrace облачным (SaaS) или Managed.

## Простота использования и удобная визуализация

Мониторинг Azure предоставляет согласованные готовые метрики, дашборды и оповещения сразу после включения мониторинга.

Измерения и анализ основных сервисов Azure визуализируются на специально созданном инфографическом дашборде (см. галерею). Для анализа влияния на бизнес и карт топологии также доступны удобные отчёты и дашборды.

## Гибкость

Пользователи Dynatrace могут выбирать, какие сервисы и экземпляры мониторятся и как выполняется этот процесс. Этот выбор можно сделать в рамках начальной настройки мониторинга или в любой последующий момент. Это также относится к новым сервисам, включённым в облаке Azure или добавленным компанией Microsoft.

Можно выбирать метрики мониторинга для каждого из поддерживаемых сервисов. Аналогично, все дашборды для мониторинга Azure можно клонировать и легко настраивать по мере необходимости.

## Гибридные среды

Отслеживание сервисов и зависимостей не ограничивается Azure. Благодаря Davis® и его способности обрабатывать данные из всех типов сред, Dynatrace оптимизирован для мониторинга гибридных сред, включая бизнес-приложения, охватывающие несколько облачных платформ и платформ виртуализации, таких как AWS, Google Cloud, VMware, Kubernetes, Openshift и локальную инфраструктуру.
