# Мониторинг Google Cloud

* Explanation
* 1-min read
* Published Jan 27, 2023

Dynatrace автоматически обнаруживает, устанавливает базовые показатели и интеллектуально мониторит динамические рабочие нагрузки Google Cloud.

## Полный стековый мониторинг

Dynatrace OneAgent обеспечивает полный стековый мониторинг основных вычислительных ресурсов, таких как Google Kubernetes Engine (GKE), Google Compute Engine (GCE) и Google Application Engine (GAE). Это даёт глубокую видимость на уровне кода и сквозные трассировки для всего, что работает на вычислительных сервисах.

## Мониторинг сервисов Google Cloud

Все метрики, публикуемые в Operations API (ранее Stackdriver), могут быть автоматически приняты в Dynatrace для предоставления данных для обнаружения проблем на основе ИИ и автоматического анализа первопричин. Наша интеграция с Google Cloud помогает отслеживать динамику гибридного облака, предоставляя высокоуровневый обзор сервисов Google Cloud в вашей учётной записи и разграничивая работоспособные и неработоспособные сервисы.
