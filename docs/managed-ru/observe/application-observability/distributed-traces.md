# Распределённые трассировки

* Overview
* 1-min read
* Updated on May 17, 2024

Распределённая трассировка — важный инструмент мониторинга, отладки и оптимизации распределённых программных архитектур. Визуализируя взаимосвязи сервисов и то, как запросы проходят через различные уровни приложения, этот инструмент помогает командам быстро понять производительность каждого микросервиса.
