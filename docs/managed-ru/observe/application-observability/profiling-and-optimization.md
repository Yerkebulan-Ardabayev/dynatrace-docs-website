# Профилирование и оптимизация

* Overview
* 1-min read
* Published Sep 25, 2018

Помимо автоматического обнаружения проблем, Dynatrace предлагает набор инструментов анализа для ручного выявления проблем.

* **CPU profiling** выделяет наибольших потребителей CPU в вашем окружении и позволяет детализировать до уровня метода при проблемах с CPU.
* **Memory dump analysis** и **Process crashes** позволяют обнаруживать сбои приложений в Windows и Linux и анализировать дампы памяти этих сбоев.

Чтобы получить доступ к параметрам профилирования и оптимизации

1. Перейдите в **Profiling & Optimization**.
2. Выберите нужный вариант анализа среди **CPU profiling**, **Memory dumps** и **Process crashes**.
