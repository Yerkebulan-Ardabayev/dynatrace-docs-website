# Digital Experience

* Обзор
* 1 мин чтения
* Опубликовано Feb 06, 2023

Мониторинг цифрового опыта (Digital Experience Monitoring, DEM) Gartner[1](#fn-1-1-def) определяет как дисциплину мониторинга доступности и производительности, которая поддерживает оптимизацию операционного опыта и поведения цифрового агента (человека или машины) при его взаимодействии с корпоративными приложениями и сервисами.

[### Концепции RUM

Узнайте о приложениях, действиях пользователей, сессиях, метриках и других понятиях.](/managed/observe/digital-experience/rum-concepts "Базовые концепции Dynatrace Real User Monitoring.")[### Web applications

Традиционные веб-приложения и single-page приложения, работающие в браузере](/managed/observe/digital-experience/web-applications "Настройка и использование Real User Monitoring для веб-приложений.")[### Mobile applications

Нативные мобильные приложения на iOS и Android, а также гибридные приложения, открываемые через браузер](/managed/observe/digital-experience/mobile-applications "Настройка и использование Real User Monitoring для мобильных приложений.")[### Custom applications

Традиционные rich-client приложения, IoT-приложения и даже Alexa-навыки](/managed/observe/digital-experience/custom-applications "Настройка и использование Real User Monitoring для пользовательских приложений.")[### Сегментация сессий

Разделяйте, комбинируйте и группируйте пользовательские сессии в осмысленные сегменты по общим признакам.](/managed/observe/digital-experience/session-segmentation "Анализ пользовательских сессий по разным приложениям с помощью Real User Monitoring.")[### Session Replay

Захват и визуальное воспроизведение полного цифрового опыта пользователей вашего приложения.](/managed/observe/digital-experience/session-replay "Использование Session Replay для понимания и устранения ошибок, с которыми сталкиваются ваши клиенты.")[### Synthetic Monitoring

Отслеживайте доступность и производительность приложения с точки зрения пользователей по всему миру и в круглосуточном режиме.](/managed/observe/digital-experience/synthetic-monitoring "Synthetic Monitoring: создание single-URL browser-монитора, browser clickpath или HTTP-монитора.")[![Сценарии использования](https://dt-cdn.net/images/icon-use-cases-9ac91e0c53.svg "Сценарии использования")
