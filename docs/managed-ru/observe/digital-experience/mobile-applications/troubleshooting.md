## Передача журналов для диагностики

Для мобильных iOS-приложений Dynatrace предоставляет API `shareLogsFile`, обеспечивающий удобную передачу локально хранимых файлов журналов через панель обмена iOS (`UIActivityViewController`). Подробнее см. в разделе [Передача журналов](/managed/manage/data-privacy-and-security/data-privacy/personal-data-captured-by-dynatrace#log-sharing "Узнайте, какие типы данных конечных пользователей могут быть записаны в ходе мониторинга Dynatrace и какие методы доступны для маскировки персональных данных конечных пользователей.").
