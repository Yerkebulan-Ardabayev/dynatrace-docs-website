# Удаление мобильного приложения

* How-to guide
* 1-min read
* Published Jan 30, 2023

Если приложение вам больше не нужно, вы можете удалить его через веб-интерфейс Dynatrace или через API.

## Удаление приложения через пользовательский интерфейс

Чтобы удалить приложение через веб-интерфейс Dynatrace:

1. Перейдите в **Settings** > **Monitoring** > **Monitoring overview**.
2. Перейдите на вкладку **Applications** и найдите приложение, которое нужно удалить.
3. Рядом с именем приложения нажмите **Expand** ![Expand row](https://dt-cdn.net/images/expand-row-icon-9c4950fc2e.svg "Expand row"), затем выберите **Remove** > **Yes**.

## Удаление приложения через API

Вы также можете удалять приложения из Dynatrace через Configuration API. Подробнее см. в разделе [Удаление мобильного или пользовательского приложения](/managed/dynatrace-api/configuration-api/rum/mobile-custom-app-configuration/apps/delete-app "Удалите мобильное или пользовательское приложение через Dynatrace API.").
