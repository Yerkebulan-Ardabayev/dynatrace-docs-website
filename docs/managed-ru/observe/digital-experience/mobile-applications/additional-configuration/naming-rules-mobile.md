# Создание пользовательских имён действий для мобильных приложений

* How-to guide
* 1-min read
* Published Jan 27, 2023

Dynatrace автоматически генерирует имена для большинства пользовательских действий. Вы можете определить правила именования действий, чтобы заменить автоматически созданные имена на предопределённые.

Чтобы создать правило именования пользовательских действий:

1. Перейдите в **Frontend**.
2. Выберите приложение, которое нужно настроить.
3. Нажмите **More** (**…**) > **Edit** в правом верхнем углу плитки с именем вашего приложения.
4. В настройках приложения выберите **Naming Rules**.
5. Разверните раздел **x action naming rules** и нажмите **Add naming rule**.
6. Введите **Resulting user action name** и задайте условия правила именования.
