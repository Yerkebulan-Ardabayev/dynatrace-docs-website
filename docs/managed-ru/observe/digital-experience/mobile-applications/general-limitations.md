# Общие ограничения RUM для мобильных приложений

* Troubleshooting
* 1-min read
* Updated on Jan 13, 2026

На этой странице описаны важные ограничения, связанные с мониторингом мобильных приложений.

## Ограничения по событиям

Эти ограничения не распространяются на данные мониторинга, собранные в веб-представлениях гибридных приложений.

Учитывайте следующие ограничения:

* **Событий в минуту**: существует ограничение на количество событий, которые могут быть сгенерированы в течение одной минуты. При превышении этого лимита события могут быть отброшены или не обработаны. По умолчанию лимит составляет 1 000 событий в минуту.
* **Дочерних действий на одно действие**: каждое действие может содержать ограниченное количество дочерних действий. При превышении этого лимита дополнительные дочерние действия могут не записываться. По умолчанию лимит составляет 200.
