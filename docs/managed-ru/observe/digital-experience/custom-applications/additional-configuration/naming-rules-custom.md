# Создание пользовательских имён действий в пользовательских приложениях

# Создание пользовательских имён действий в пользовательских приложениях

* How-to guide
* 1-min read
* Published Jan 30, 2023

Dynatrace автоматически генерирует имена для большинства пользовательских действий. Вы можете определить правила именования действий пользователей, чтобы заменить автоматически генерируемые имена на заданные вручную.

Чтобы создать правило именования действий пользователей:

1. Перейдите в **Frontend**.
2. Выберите приложение, которое нужно настроить.
3. Нажмите **More** (**…**) > **Edit** в правом верхнем углу плитки с именем вашего приложения.
4. В настройках приложения выберите **Naming Rules**.
5. Разверните раздел **x action naming rules** и нажмите **Add naming rule**.
6. Введите **Resulting user action name** и задайте условия правила именования.
