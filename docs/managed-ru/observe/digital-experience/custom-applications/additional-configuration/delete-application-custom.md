# Удаление пользовательского приложения

* How-to guide
* 1-min read
* Published Jan 30, 2023

Если приложение больше не нужно, вы можете удалить его через веб-интерфейс Dynatrace или через API.

## Удаление приложения через интерфейс

Чтобы удалить приложение через веб-интерфейс Dynatrace:

1. Перейдите в **Settings** > **Monitoring** > **Monitoring overview**.
2. Перейдите на вкладку **Applications** и найдите приложение, которое нужно удалить.
3. Рядом с именем приложения нажмите **Expand** ![Expand row](https://dt-cdn.net/images/expand-row-icon-9c4950fc2e.svg "Expand row"), затем выберите **Remove** > **Yes**.

## Удаление приложения через API

Приложения также можно удалять через Configuration API. Подробнее см. [Удаление мобильного или пользовательского приложения](/managed/dynatrace-api/configuration-api/rum/mobile-custom-app-configuration/apps/delete-app "Удаляйте мобильные и пользовательские приложения через Dynatrace API.").
