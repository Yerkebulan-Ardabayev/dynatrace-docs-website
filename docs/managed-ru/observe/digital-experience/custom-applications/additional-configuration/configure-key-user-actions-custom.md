# Настройка ключевых действий пользователей в пользовательских приложениях

# Настройка ключевых действий пользователей в пользовательских приложениях

* How-to guide
* 1-min read
* Published Jan 30, 2023

Большинство приложений содержат пользовательские действия, особенно важные для успеха вашего цифрового бизнеса. Примеры таких действий: регистрация, оформление заказа, поиск товаров. Подобные [ключевые действия пользователей](/managed/observe/digital-experience/rum-concepts/user-actions#key-user-actions "Узнайте, что такое пользовательские действия и как они помогают понять, что пользователи делают в вашем приложении.") могут выполняться дольше других или требовать завершения за время меньше среднего.

Функция ключевых действий пользователей позволяет [настроить пороги Apdex](/managed/observe/digital-experience/rum-concepts/scores-and-ratings/apdex-ratings#key-user-actions "Узнайте, как Dynatrace использует Apdex для измерения удовлетворённости пользователей производительностью приложения.") для каждого из таких действий. Вы можете использовать эту функцию для мониторинга ключевых действий с выделенной плиткой на дашборде и отслеживания исторических тенденций.

## Пометка действия пользователя как ключевого

1. Перейдите в **Frontend**.
2. Выберите приложение и прокрутите вниз до раздела **Top 3 user actions** или **Top 3 actions**.
3. Нажмите **View full details** или **Analyze performance**.
4. Прокрутите вниз до списка пользовательских действий и выберите нужное действие.  
   Откроется страница с подробной информацией о пользовательском действии.
5. В правом верхнем углу страницы нажмите **Mark as key user action**.

Вы можете определить до 500 ключевых действий пользователей на окружение по всем приложениям и до 100 ключевых действий на одно приложение.

Если вы достигли максимального лимита ключевых действий, рассмотрите использование вычисляемых метрик (доступно для [веб-](/managed/observe/digital-experience/web-applications/additional-configuration/rum-calculated-metrics-web "Создавайте вычисляемые метрики и пользовательские диаграммы на их основе для веб-приложений."), [мобильных](/managed/observe/digital-experience/mobile-applications/additional-configuration/rum-calculated-metrics-mobile "Создавайте вычисляемые метрики и пользовательские диаграммы на их основе для мобильных приложений.") и [пользовательских приложений](/managed/observe/digital-experience/custom-applications/additional-configuration/rum-calculated-metrics-custom "Создавайте вычисляемые метрики и пользовательские диаграммы на их основе для пользовательских приложений.")), которые предлагают аналогичные возможности.

## Закрепление ключевого действия на дашборде

[Dashboards Classic](/managed/analyze-explore-automate/dashboards-classic "Узнайте, как создавать дашборды Dynatrace Dashboards Classic, управлять ими и использовать их.")

1. Перейдите в **Frontend**.
2. Выберите приложение и прокрутите вниз до раздела **Top 3 user actions** или **Top 3 actions**.
3. Нажмите **View full details** или **Analyze performance**.
4. Найдите нужное ключевое действие пользователя и выберите его.  
   Откроется страница с подробной информацией о пользовательском действии.
5. В верхней части страницы нажмите **Pin to dashboard**. Подробнее см. [Закрепление плиток на дашборде](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/pin-tiles-to-your-dashboard "Узнайте, как закреплять плитки на дашбордах.").

## Настройка рейтинга Apdex для ключевого действия

Чтобы изменить пороги Apdex для ключевого пользовательского действия:

1. Перейдите в **Frontend**.
2. Выберите приложение и прокрутите вниз до раздела **Top 3 user actions** или **Top 3 actions**.
3. Нажмите **View full details** или **Analyze performance**.
4. Найдите нужное ключевое действие пользователя и выберите его.  
   Откроется страница с подробной информацией о пользовательском действии.
5. В правом верхнем углу страницы выполните одно из следующих действий:

   * Для веб-приложений: нажмите **More** (**…**) > **Edit** > **Key performance metric**.
   * Для мобильных и пользовательских приложений: нажмите ![Expand](https://dt-cdn.net/images/expandbutton-40-e1f11ff81d.png "Expand") (**Expand**) > **Edit** > **Key performance metric**.
6. Используйте ползунок для регулировки порогов Apdex.
