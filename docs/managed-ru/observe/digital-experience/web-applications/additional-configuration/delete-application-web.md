# Удаление веб-приложения

* How-to guide
* 1-min read
* Published Aug 09, 2022

Если приложение больше не нужно, его можно удалить через веб-интерфейс Dynatrace или через API.

## Удаление приложения через UI

Чтобы удалить приложение через веб-интерфейс Dynatrace:

1. Перейдите в **Settings** > **Monitoring** > **Monitoring overview**.
2. Перейдите на вкладку **Applications** и найдите приложение, которое нужно удалить.
3. Рядом с именем приложения выберите **Expand** ![Expand row](https://dt-cdn.net/images/expand-row-icon-9c4950fc2e.svg "Expand row"), затем выберите **Remove** > **Yes**.

## Удаление приложения через API

Приложение также можно удалить из Dynatrace через Configuration API. Подробнее см. [Удаление веб-приложения](/managed/dynatrace-api/configuration-api/rum/web-application-configuration-api/web-application/del-web-application "Delete a web application via the Dynatrace API.").
