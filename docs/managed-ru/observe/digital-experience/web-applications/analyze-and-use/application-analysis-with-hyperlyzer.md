# Анализ приложения с помощью Hyperlyzer

* How-to guide
* 1-min read
* Published Nov 06, 2017

Hyperlyzer позволяет анализировать использование приложения по нескольким измерениям. С его помощью можно видеть, где находятся пользователи, какую версию браузера и операционную систему они используют, а также количество пользовательских действий в приложении.

Чтобы проанализировать приложение с помощью Hyperlyzer:

1. Перейдите в **Web**.
2. Выберите приложение для анализа.
3. На странице приложения выберите **More** (**…**) > **Show top findings**.

   ![Hyperlyzer 1](https://dt-cdn.net/images/hyperlyzer-1-2199-49993e59fc.png)

   Hyperlyzer 1
4. Выберите значения, visualized в Hyperlyzer, чтобы отобразить данные в табличном формате.

   ![Hyperlyzer 2](https://dt-cdn.net/images/hyperlyzer-2-1859-80810e6312.png)

   Hyperlyzer 2
5. Для любого конкретного измерения (например, OS: `iOS` и Geolocation: `United States`) выберите **Filter selection**, чтобы сфокусировать анализ Hyperlyzer на этих измерениях. Таким образом можно детализировать результаты мониторинга в Hyperlyzer для лучшего понимания использования приложения.

   ![Hyperlyzer 3](https://dt-cdn.net/images/hyperlyzer-3-1876-2c73d28d66.png)

   Hyperlyzer 3
