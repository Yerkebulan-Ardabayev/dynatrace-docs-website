# Удаление веб-приложения в RUM Classic

* Практическое руководство
* Чтение за 1 мин
* Опубликовано 09 авг. 2022 г.

Когда приложение больше не нужно, его можно удалить через веб-интерфейс Dynatrace или через API.

## Удаление приложения через интерфейс

Чтобы удалить приложение через веб-интерфейс Dynatrace

1. Перейти в **Settings** > **Monitoring** > **Monitoring overview**.
2. Переключиться на вкладку **Applications** и найти приложение, которое нужно удалить.
3. Рядом с названием приложения выбрать **Expand** ![Expand row](https://dt-cdn.net/images/expand-row-icon-9c4950fc2e.svg "Expand row"), затем выбрать **Remove** > **Yes**.

## Удаление приложения через API

Приложения также можно удалять из Dynatrace через Configuration API. Подробности см. в разделе [Delete a web application](/managed/dynatrace-api/configuration-api/rum/web-application-configuration-api/web-application/del-web-application "Delete a web application via the Dynatrace API.").
