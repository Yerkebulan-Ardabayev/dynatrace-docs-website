# Вид карты мира в RUM Classic

* Практическое руководство
* Чтение за 1 минуту
* Опубликовано 17 июля 2019 г.

Вид **World map** позволяет легко переключаться между различными измерениями мониторинга с помощью вкладок в верхней части экрана.

![Worldmap](https://dt-cdn.net/images/map-2313-3f7868e353.png)

Worldmap

Чтобы открыть вид **World map**

1. Перейти в раздел **Web**.
2. Выбрать нужное приложение.
3. Выбрать плитку **View geolocation breakdown** на инфографике.
4. В разделе **Geolocation breakdown**, который появится слева под инфографикой, выбрать **View full world map**.

Вид карты мира можно закрепить напрямую на классической дашборде по своему выбору, выбрав **Pin to dashboard** и указав нужную дашборду. Подробнее см. [Закрепление плиток на дашборде](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/pin-tiles-to-your-dashboard "Learn to pin tiles to your dashboards.").

Если данные на карте мира не отображаются, может понадобиться [сопоставить внутренние IP-адреса с местоположениями](/managed/observe/digital-experience/rum-classic/web-applications/additional-configuration/map-internal-ip-addresses-to-locations-web "Configure Dynatrace to use local addresses to understand where the users of your web applications are.").
