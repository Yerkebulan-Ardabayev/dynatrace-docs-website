# Анализ приложений с помощью Hyperlyzer в RUM Classic

* Практическое руководство
* 1 минута на чтение
* Опубликовано 06 нояб. 2017 г.

Hyperlyzer позволяет анализировать использование приложения в нескольких измерениях. С помощью Hyperlyzer можно увидеть, где расположены пользователи, какую версию браузера они используют, их операционную систему и количество пользовательских действий, полученных приложением.

Чтобы проанализировать приложение с помощью Hyperlyzer

1. Перейти в **Web**.
2. Выбрать приложение, которое нужно проанализировать.
3. На странице приложения выбрать **More** (**…**) > **Show top findings**.

   ![Hyperlyzer 1](https://dt-cdn.net/images/hyperlyzer-1-2199-49993e59fc.png)

   Hyperlyzer 1
4. Выбрать значения, отображённые в Hyperlyzer, чтобы вывести эти результаты в табличном формате.

   ![Hyperlyzer 2](https://dt-cdn.net/images/hyperlyzer-2-1859-80810e6312.png)

   Hyperlyzer 2
5. Выбрать **Filter selection** для любого конкретного измерения (например, OS: `iOS` и Geolocation: `United States`), чтобы сфокусировать анализ Hyperlyzer на этих измерениях. Таким образом можно детализировать результаты мониторинга, предоставляемые Hyperlyzer, чтобы лучше понять использование приложения.

   ![Hyperlyzer 3](https://dt-cdn.net/images/hyperlyzer-3-1876-2c73d28d66.png)

   Hyperlyzer 3
