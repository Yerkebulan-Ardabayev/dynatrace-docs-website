## Совместное использование логов для устранения неполадок

Для мобильных приложений iOS Dynatrace предоставляет `shareLogsFile` API, который позволяет легко делиться локально сохранёнными файлами журналов через лист общего доступа iOS (`UIActivityViewController`). Подробнее см. [Совместное использование логов](/managed/manage/data-privacy-and-security/data-privacy/personal-data-captured-by-dynatrace#log-sharing "Узнайте, какие типы данных конечных пользователей могут собираться при мониторинге Dynatrace, и какие методы доступны для маскирования персональных данных конечных пользователей.").
