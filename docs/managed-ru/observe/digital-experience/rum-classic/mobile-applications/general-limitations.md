# Общие ограничения для мобильных приложений RUM Classic

* Устранение неполадок
* 1 мин чтения
* Обновлено 13 января 2026 г.

На этой странице описаны важные ограничения, связанные с мониторингом мобильных приложений.

## Ограничения событий

Эти ограничения не применяются к данным мониторинга, полученным в веб-представлениях (web views) гибридных приложений.

Следует учитывать следующие ограничения:

* **События в минуту**: существует ограничение на количество событий, которые могут быть сгенерированы в течение одной минуты. Превышение этого лимита может привести к отбрасыванию или необработке событий. Лимит по умолчанию составляет 1000 событий в минуту.
* **Дочерние элементы на действие**: каждое действие может содержать ограниченное количество дочерних действий. При превышении этого лимита дополнительные дочерние действия могут не записываться. Лимит по умолчанию составляет 200.
