# Удаление мобильного приложения в RUM Classic

* Практическое руководство
* Чтение за 1 минуту
* Опубликовано 30 января 2023 г.

Если приложение больше не нужно, его можно удалить либо через Dynatrace web UI, либо через API.

## Удаление приложения через UI

Чтобы удалить приложение через Dynatrace web UI:

1. Перейти в **Settings** > **Monitoring** > **Monitoring overview**.
2. Переключиться на вкладку **Applications** и найти приложение, которое нужно удалить.
3. Рядом с названием приложения выбрать **Expand** ![Expand row](https://dt-cdn.net/images/expand-row-icon-9c4950fc2e.svg "Expand row"), затем выбрать **Remove** > **Yes**.

## Удаление приложения через API

Приложения также можно удалять из Dynatrace через Configuration API. Подробнее см. [Delete a mobile or custom application](/managed/dynatrace-api/configuration-api/rum/mobile-custom-app-configuration/apps/delete-app "Delete a mobile or custom app via the Dynatrace API.").
