# Создание пользовательских названий действий пользователя для мобильных приложений в RUM Classic

* Практическое руководство
* Чтение 1 мин
* Опубликовано 27 янв. 2023 г.

Dynatrace автоматически генерирует названия для большинства действий пользователя. Можно определить правила именования действий пользователя, чтобы заменить автоматически сгенерированные названия на предопределённое название.

Чтобы создать правило именования действия пользователя

1. Перейти в **Frontend**.
2. Выбрать приложение, которое нужно настроить.
3. Выбрать **More** (**…**) > **Edit** в правом верхнем углу плитки с названием приложения.
4. В настройках приложения выбрать **Naming Rules**.
5. Развернуть раздел **x action naming rules**, затем выбрать **Add naming rule**.
6. Ввести **Resulting user action name** и указать условия правила именования.
