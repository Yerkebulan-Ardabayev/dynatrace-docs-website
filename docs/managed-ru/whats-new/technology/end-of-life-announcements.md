## Общие правила

Dynatrace развивается в ногу с технологическими стеками, на которых работает, чтобы обеспечивать высококачественный опыт. Когда партнёры-поставщики выводят из эксплуатации устаревшие компоненты, мы также прекращаем их поддержку, чтобы вы получали наиболее безопасные и полноценно поддерживаемые инновации.

Мы сообщаем об этих изменениях через два ключевых события, которые помогут вам планировать заранее:

### Конец поддержки (End of Support, EoS)

* Последняя дата, до которой Dynatrace предоставляет исправления ошибок или обновления безопасности.
* После EoS компонент продолжает работать, однако техническое обслуживание прекращается, а поддержка предоставляется по мере возможности. Функции больше не поддерживаются, и критические исправления не выпускаются.
* Ответственность за продолжение эксплуатации и сопутствующие риски переходит к клиенту.

### Конец жизненного цикла (End of Life, EoL)

* Самая поздняя дата, когда Dynatrace отключает, удаляет или блокирует компонент, прекращая его полезную работу.
* Пути обновления остаются открытыми, чтобы вы могли выполнить миграцию и продолжить получать полную ценность от Dynatrace.
* Обращения в службу поддержки, связанные с компонентом, больше не принимаются.

## Python Extension Framework 1.0

Мы переходим со снятых с поддержки сред выполнения Python 3.5/3.8 на текущую, полностью поддерживаемую версию Python, чтобы ваши расширения оставались безопасными и актуальными. Обновление фреймворка расширений закрывает незакрытые уязвимости безопасности, даёт доступ к последним библиотекам и улучшениям производительности, а также обеспечивает совместимость с будущими функциями Dynatrace.

### Расширения OneAgent

OneAgent версии 1.299 — последний выпуск, поддерживающий фреймворк Python Extensions 1.0.

* EoS: Поддержка (то есть исправления ошибок, обновления безопасности) завершилась Sep 30, 2025.
* Python Extension Framework 1.0 имеет статус EoS с Sep 1, 2024.
* Для продолжения использования расширений после даты EoL OneAgent версии 1.299 выполните миграцию на новый [фреймворк расширений](/managed/ingest-from/extensions "Learn how to create and manage Dynatrace Extensions.") и [обновите OneAgent](/managed/ingest-from/dynatrace-oneagent/oneagent-update "Learn how to update OneAgent.") до поддерживаемой версии. См. [Руководства по миграции для пользовательских расширений 1.0](#migration).

### Расширения ActiveGate

ActiveGate версии 1.299 — последний выпуск, поддерживающий Python Extensions 1.0.

* EoS: Поддержка (то есть исправления ошибок, обновления безопасности) завершилась Sep 30, 2025.
* Python Extension framework 1.0 имеет статус EoS с Sep 1, 2024.

### Руководства по миграции для пользовательских расширений 1.0

Выполните миграцию всех расширений Python 1.0 на новые расширения для поддержания безопасной и поддерживаемой среды.

* [Миграция расширений Python](https://developer.dynatrace.com/develop/extensions/dynatrace-extensions-vscode/guides/migrate/python-migrate/)
* [Видео: Практическое руководство по созданию расширений на основе Python с Dynatrace](https://www.youtube.com/watch?v=g-lPIZx66BA)

Для сохранения поддержки [обновите OneAgent](/managed/ingest-from/dynatrace-oneagent/oneagent-update "Learn how to update OneAgent.") до последней версии.

## Нативные мобильные приложения для iOS и Android

В рамках непрерывного стремления обеспечить оптимизированный и последовательный пользовательский опыт Dynatrace прекращает поддержку [нативных мобильных приложений для iOS и Android](/managed/analyze-explore-automate/notifications-and-alerting/push-notifications-via-the-dynatrace-mobile-app "Learn how you can connect your Dynatrace environments with the Dynatrace mobile app to receive problem alerts."). Это изменение соответствует стратегии предоставления единого адаптивного веб-интерфейса и использования современных гибких каналов уведомлений.

### Ключевые даты

* **Последнее обновление приложения**: конец 2025 года
* **Дата прекращения поддержки**: June 30, 2026
* **Завершение поддержки**: June 30, 2026

### Что изменится?

Нативные мобильные приложения Dynatrace перестанут быть доступны для загрузки и поддерживаться после даты прекращения поддержки.

Рекомендуется использовать Dynatrace через адаптивный веб-интерфейс, который предлагает полную функциональность на всех типах устройств и размерах экрана.

Push-уведомления теперь будут доставляться через сторонние интеграции: Slack, Teams, PagerDuty, электронную почту и ntfy.

### Причины изменения

* **Единый опыт**: единая адаптивная веб-платформа обеспечивает согласованную функциональность на всех устройствах.
* **Гибкость**: вы можете выбрать предпочтительные инструменты коммуникации для оповещений и обновлений.

### Рекомендуемые действия

* Начните использовать адаптивный веб-интерфейс.
* Настройте предпочтительные интеграции уведомлений через:

  + Dynatrace SaaS: [Недоступно в Dynatrace Managed](/managed/upgrade/unavailable-in-managed "Your selection is unavailable in Dynatrace Managed.")
  + Dynatrace Managed: [Уведомления о проблемах](/managed/analyze-explore-automate/notifications-and-alerting/problem-notifications "Learn how to integrate third-party problem notification systems with Dynatrace.")
* Удалите нативное приложение с устройств после даты прекращения поддержки.

### Нужна помощь?

Для получения помощи в переходе или настройке интеграций см. ссылки на документацию выше или обратитесь в службу поддержки.
