# Куки веб-интерфейса Dynatrace

* Справочник
* Published Apr 02, 2026

В следующих таблицах перечислены куки, используемые в веб-интерфейсе Dynatrace. Вследствие самомониторинга могут также добавляться [куки RUM](/managed/manage/data-privacy-and-security/data-privacy/rum-cookies-and-web-storage "Узнайте, как RUM и Session Replay используют куки, веб-хранилище и IndexedDB."). Все перечисленные куки являются куки первой стороны.

### Куки веб-интерфейса Dynatrace

В таблице ниже перечислены куки, устанавливаемые в веб-интерфейсе Dynatrace для единого входа (SSO).

| Куки | Истекает | Назначение |
| --- | --- | --- |
| `b925d32c` | Сессия | Указывает, вошёл ли пользователь в систему. |
| `ssoCSRFCookie` | Сессия | Обеспечивает защиту от подделки межсайтовых запросов (CSRF) при переключении между сервлетами в SSO. |
| `p23mn32t` | 5 лет | Содержит уникальный идентификатор в кодировке Base32, который указывает SSO, что пользователь входит с нового устройства. Идентификатор создаётся на основе логина пользователя, браузера и user agent. |
| `l34kn6no` | Сессия | Хранит `state` OpenID, когда SSO выступает в роли зависимой стороны, например при входе с помощью Microsoft через OpenID. |
| `iu2g34bw` | Сессия | Хранит `code_verifier` OpenID, когда SSO выступает в роли зависимой стороны, например при входе с помощью Microsoft через OpenID. |
| `a69k21bb` | Сессия | Хранит `redirect_uri` после успешного входа, когда SSO выступает в роли зависимой стороны, например при входе с помощью Microsoft через OpenID. |
| `cgq80xhu` | Сессия | Содержит SHA-256-хеш случайного UUID. При входе пользователя через OpenID этот куки используется для отслеживания состояния сессии через SSO OpenID iFrame и выполнения выхода на стороне клиента при необходимости. |
| `72ddbc27` | 3 месяца | Добавляется, когда пользователь выбирает параметр **Remember me** для сохранения учётных данных. Благодаря этому параметру пользователю не нужно вводить учётные данные повторно при истечении сессии — вход выполняется автоматически. |
| `kj76fg4h` | 5 минут | Предотвращает блокировку пользователя после неудачного федеративного входа при выбранном параметре **Remember me**. Если пользователь вошёл в систему, этот куки удаляется. |

### Куки веб-сервера Dynatrace

| Куки | Истекает | Назначение |
| --- | --- | --- |
| `SRV` | 1 час | Обеспечивает привязку сессии балансировщика нагрузки (HA Proxy). |
| `apmsessionid` | Сессия | Куки сессии веб-сервера. |
