# Конфиденциальность пользователей для iOS

* Чтение 3 мин
* Updated on Nov 21, 2024

Начиная с 8 декабря 2020 года Apple обязывает предоставлять информацию о практиках обеспечения конфиденциальности вашего приложения, включая практики сторонних партнёров, таких как Dynatrace.

На этой странице содержится информация о том, какие данные [Dynatrace OneAgent для iOS](/managed/observe/digital-experience/mobile-applications/instrument-ios-app "Инструментируйте мониторинг мобильных приложений для iOS, настройте автоматическую инструментацию и захватывайте дополнительные данные через ручную инструментацию.") собирает по умолчанию. Категории и типы данных соответствуют опроснику Apple, однако ответы отражают исходное состояние по умолчанию. Подробное описание отдельных типов данных см. в разделе [Сведения о конфиденциальности приложения в App Store](https://developer.apple.com/app-store/app-privacy-details/) на портале разработчиков Apple.

OneAgent может собирать дополнительные данные через вашу [ручную инструментацию](/managed/observe/digital-experience/mobile-applications/instrument-ios-app/customization/oneagent-sdk-for-ios "Обогатите мониторинг пользовательского опыта в мобильных приложениях с помощью OneAgent SDK для iOS."). Если вы настроили инструментацию для сбора дополнительных данных, обязательно отразите это в опроснике о конфиденциальности вашего приложения.
