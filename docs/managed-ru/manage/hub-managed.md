# Dynatrace Hub

* Обзор
* Чтение 7 мин
* Опубликовано 26 мая 2025 г.

Раскройте весь потенциал Dynatrace, найдя, активировав и запустив расширения, соответствующие вашим конкретным потребностям в области наблюдаемости.

Dynatrace Hub ![Hub](https://dt-cdn.net/images/hub-512-82db3c583e.png "Hub") — это центральное место для изучения и активации всех возможностей Dynatrace. Здесь можно найти решения для мониторинга окружений, анализа данных наблюдаемости с целью получения аналитики и автоматизации устранения проблем на основе полученных данных. Вы остаётесь в курсе последних дополнений к Dynatrace Managed. Расширения можно легко устанавливать одним щелчком через Dynatrace Hub и пользоваться постоянными обновлениями. Управляйте своими расширениями для полного контроля.

* Просматривайте все возможности Dynatrace Managed
* Будьте в курсе последних дополнений к Dynatrace Managed
* Находите решения для задач мониторинга
* Простая установка расширений одним щелчком
* Получайте постоянные обновления с новейшими функциями
* Управляйте своими расширениями

## Где найти Dynatrace Hub

Чтобы открыть Hub, в меню перейдите в раздел **Manage** и выберите **Dynatrace Hub**.

## Что представлено в Dynatrace Hub

В целом Dynatrace Hub перечисляет все технологии, которые Dynatrace официально поддерживает, в том числе:

* Технологии, поддерживаемые OneAgent «из коробки».
* Поддерживаемые фреймворки Open Observability.
* Расширения, разработанные Dynatrace.

## Extensions

С помощью [Extensions](/managed/ingest-from/extensions/develop-your-extensions "Разрабатывайте собственные Extensions в Dynatrace.") можно декларативно передавать метрики в Dynatrace, обеспечивая платформенную аналитику и возможности мониторинга. Dynatrace связывает данные осмысленным образом, позволяя изучать их, создавать инструментацию и настраивать оповещения.

## Обнаружение

Просматривайте все элементы Hub, используя предоставленные категории.

Если есть конкретная задача, можно выполнить поиск по элементам Hub и фильтровать результаты.

### Получение информации

После того как вы нашли интересующий элемент, выберите значок для открытия страницы обзора со следующей информацией:

* **Product information**
* **Technical information**
* **Content**
* **Release notes**

### Фильтрация

На вкладке **Browse** в Dynatrace Hub можно использовать строку поиска для фильтрации списка доступных расширений и технологий.
