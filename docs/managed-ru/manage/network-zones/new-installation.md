# Новая установка

* Практическое руководство
* Чтение 1 мин
* Опубликовано 28 янв 2020 г.

Если вы впервые внедряете Dynatrace в свою систему, рекомендуется сразу добавить сетевые зоны.

Если вы уже используете Dynatrace, обратитесь к разделу [Миграция](/managed/manage/network-zones/migration "Начните использовать сетевые зоны в существующем окружении Dynatrace."), поскольку это другая процедура.

1. [Анализ](/managed/manage/network-zones/new-installation/analyze "Узнайте, как проанализировать вашу сеть для подготовки к внедрению сетевых зон.")
2. [Планирование](/managed/manage/network-zones/new-installation/plan "Планирование сетевых зон, отражающих топологию вашей сети.")
3. [Развёртывание](/managed/manage/network-zones/new-installation/deploy "Узнайте, как развернуть сетевые зоны для существующего окружения Dynatrace.")
4. [Проверка](/managed/manage/network-zones/new-installation/verify "Узнайте, как убедиться в успешности внедрения сетевых зон.")
