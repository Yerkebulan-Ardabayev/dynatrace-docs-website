# Планирование сетевых зон (миграция)

* Чтение 1 мин
* Опубликовано 28 янв 2020 г.

Когда [анализ](/managed/manage/network-zones/migration/analyze "Узнайте, как проанализировать вашу сеть для подготовки к развёртыванию сетевых зон.") завершён, переходите к следующему шагу — планированию сетевых зон.

На этапе планирования необходимо проверить, сколько трафика генерируют ваши OneAgent. Это поможет спланировать необходимое количество ActiveGate. Смотрите [Требования к оборудованию и системные требования ActiveGate](/managed/ingest-from/dynatrace-activegate/installation "Узнайте, как настроить ActiveGate").
