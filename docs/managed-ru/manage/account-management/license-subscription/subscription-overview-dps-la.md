# Обзор подписки (Dynatrace Platform Subscription до 2023 года)

* Пояснение
* Чтение 2 мин
* Опубликовано 24 ноя 2025 г.

Представление **Overview** в Управлении аккаунтом предоставляет три ключевых анализа вашей подписки DPS.

* Использование аккаунта: сводка вашего использования, выраженная как доля от годового обязательства.
* Разбивка использования по кластерам: разбивка использования годового обязательства по кластерам и возможностям.
* Сведения об использовании: историческое представление в виде диаграммы и таблицы с данными использования по окружению и возможностям.

Доступ к этой информации осуществляется через **Account Management** > **Subscription** > **Overview**.

## Использование аккаунта

Раздел «Использование аккаунта» предоставляет сводку о том, какая часть вашего годового обязательства была использована.

* **Annual commit**: текстовая сводка текущего годового обязательства.
* **Commit period**: продолжительность подписки.

  Подписка Dynatrace Platform Subscription может охватывать несколько лет, разделённых на годовые периоды обязательства, каждый со своим годовым обязательством.

  + Даты основаны на UTC и охватывают период с `0:00` начальной даты до `23:59:59` конечной даты.
* Шкала лицензии: фактическое общее потребление, выраженное как доля от годового обязательства.

  Наведите курсор на шкалу лицензии, чтобы увидеть разбивку использования обязательства по возможностям.

## Разбивка использования по окружениям

Таблица **Environment usage breakdown** предоставляет снимок накопленного использования по окружению и функциям платформы.
Значения представляют итоги за текущий период обязательства, отображаемые в виде списка кластеров с детализацией по каждому окружению. Вы можете просматривать использование DPS по функции и общее использование DPS по кластеру и окружению.

Окружения отображаются с использованием понятного имени (если задано) и идентификатора окружения.

Вы можете использовать эту информацию для быстрого определения окружений, расходующих наибольшую долю годового обязательства DPS.

## Сведения об использовании

В то время как разделы **Account usage** и **Cluster usage** предоставляют сводку затрат на текущий момент, **Usage details** позволяют анализировать использование DPS с различными представлениями и фильтрами.

### Период

Ваша подписка может охватывать несколько годовых периодов.
По умолчанию Управление аккаунтом отображает использование за текущий период подписки.
Для просмотра использования за более ранний период выберите другой **Period**.

### Продукты

Таблица показывает накопленное использование функции, выбранной в списке **Products**.

* При выборе `All` в таблице отображается использование на верхнем уровне, например, host unit hours и Davis data units.
* При выборе конкретного продукта, например `Davis data units`, таблица обновляется для отображения разбивки (метрики, логи и т.д.) с общей суммой строки справа. Выбор временного диапазона также обновляет таблицу с накопленным использованием за определённый пользователем период.

Выберите **Product** для просмотра использования дополнительных функций. Например, выберите `Davis data units` для отображения использования пользовательских метрик, аналитики логов, бессерверных функций, событий и трейсов.

### Разрешение

* По умолчанию диаграмма отображает разрешение `Forecast`.
  При этом временной период устанавливается на годовой период обязательства, отображая накопленные затраты на текущий момент и прогнозируемые затраты до конца периода.
  Модель прогнозирования является линейной по всем точкам данных годового периода.
* Для фильтрации по продукту, окружению или временному периоду выберите разрешение `Daily` или `Hourly`.

### Фильтр

Используйте параметр **Filter** над диаграммой для просмотра информации о потреблении по выбранным окружениям.

### Легенда

Каждый элемент в легенде диаграммы является переключателем: выберите его, чтобы скрыть или показать соответствующий элемент на диаграмме.
Это может быть полезно, например, если вы хотите скрыть уровень годового обязательства при просмотре использования в окружении с низким потреблением.

### Тип диаграммы

Используйте кнопки диаграммы в правом верхнем углу для переключения типов диаграмм.

### Временной период

Чтобы изменить временной период диаграммы:

1. Разверните текущую настройку временного периода.
2. Укажите новый диапазон.
3. Нажмите **Apply**.
