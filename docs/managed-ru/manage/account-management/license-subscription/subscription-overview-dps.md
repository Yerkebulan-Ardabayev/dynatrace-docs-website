# Обзор подписки (Dynatrace Platform Subscription)

* Пояснение
* Чтение 3 мин
* Опубликовано 24 ноя 2025 г.

Представление **Overview** в Управлении аккаунтом предоставляет три ключевых анализа вашей подписки DPS.

* Сводка бюджета: сводка общих затрат и прогноза за период подписки с разбивкой по окружениям.
* Разбивка затрат и использования: детальная разбивка затрат и использования за последние 30 дней.
* Анализ затрат и использования: детальное представление с гибкими параметрами фильтрации для просмотра и сравнения затрат и использования.

Доступ к этой информации осуществляется через **Account Management** > **Subscription** > **Overview**.

Вся информация о бюджете и затратах также доступна через [Dynatrace Platform Subscription API](/managed/dynatrace-api/account-management-api/dynatrace-platform-subscription-api "Запрос данных о Dynatrace Platform Subscription через Account Management API.").

Если вы видите **Account Management** > **License**, обратитесь к [документации классического лицензирования Dynatrace](/managed/manage/account-management/license-subscription/license-overview-classic "Просмотр использования и сведений о классической лицензии Dynatrace.").

## Сводка бюджета

Раздел **Budget summary** отображает сведения о годовых затратах и прогнозе с разбивкой по окружениям.

![Экран сводки бюджета в Управлении аккаунтом для модели лицензирования Dynatrace Platform Subscription.](https://dt-cdn.net/images/account-management-dps-budget-summary-1826-6227bdd871.png)

Экран сводки бюджета в Управлении аккаунтом для модели лицензирования Dynatrace Platform Subscription.

* Общая информация о подписке включает:

  + **Annual commitment period**: текущий период подписки.
    (При многолетней подписке отображается только текущий период.)
  + **Annual commitment amount**: запланированный бюджет на текущий период подписки в валюте, указанной в вашем соглашении о подписке.
  + **Days remaining**: количество оставшихся дней текущего периода подписки.
* Кольцевая диаграмма **Total used budget** показывает, какой процент годового бюджета уже израсходован, а также фактическую сумму и накопленное использование.
  Наведите курсор на диаграмму, чтобы увидеть детальную разбивку по каждому окружению.
  (Обратите внимание, что диаграмма отображает только первые четыре окружения; все остальные группируются в **Other**.)
* Линейная диаграмма **Budget forecast** показывает накопленные затраты.
  Данные отображаются с начала периода подписки и включают прогноз затрат до конца периода.
  Наведите курсор на диаграмму, чтобы увидеть фактические и прогнозируемые затраты.

  Управление аккаунтом использует продвинутые модели прогнозирования для предсказания будущих затрат.
  Прогнозы доступны только для аккаунтов с данными о затратах за 15 и более дней.

  Прогнозные данные являются лишь оценкой.
  Они не гарантируют будущие затраты.
* **Forecast events**: прогнозируемое увеличение использования по требованию (в процентах и стоимости) к концу годового периода обязательства.
  Когда прогнозируется превышение годового обязательства, панель отображает конкретную дату, на которую ожидается это превышение.
* **Environment**: сведения о затратах по каждому окружению.

  + **Annual budget-to-date cost**: накопленные затраты для данного окружения, измеренные с начала периода обязательства.
  + **Last 0–30 days cost** и **Last 0–30 days change**: накопленные затраты и процентное изменение по окружению за последние 30 дней.
    Поскольку использование окружения может меняться в течение периода обязательства, эти столбцы помогают определить окружения с наибольшими затратами.
* Для просмотра разбивки затрат окружения по возможностям выберите **Actions** > **View details**.

## Затраты и использование

Панель **Cost and usage** предоставляет детальный анализ затрат и использования за последние 30 дней по вашей подписке.
Для просмотра этой информации выберите **Cost summary** (для затрат) или **Usage summary** (для использования).

### Сводка затрат

Вкладка **Cost Summary** отображает затраты за предыдущие 30 дней по всем окружениям и возможностям.

![Экран сводки затрат в Управлении аккаунтом для модели лицензирования Dynatrace Platform Subscription.](https://dt-cdn.net/images/account-management-dps-cost-summary-1818-cdf2edb006.png)

Экран сводки затрат в Управлении аккаунтом для модели лицензирования Dynatrace Platform Subscription.

Гистограмма вверху показывает затраты по окружению с течением времени.

* Наведите курсор на значение диаграммы, чтобы увидеть разбивку затрат по окружению.
* Выберите имя окружения, чтобы показать/скрыть его на диаграмме.
* Для фильтрации данных о затратах по конкретным окружениям или возможностям используйте фильтры **All environments** и **All capabilities**, которые видны над диаграммой.
  (Обратите внимание, что только первые четыре окружения с наибольшими затратами отображаются по отдельности. Все остальные группируются в **Other**.)

Табличные данные под диаграммой разбивают затраты на уровне возможностей.

* **Annual budget-to-date cost** — это накопленные затраты для данного окружения, измеренные с начала периода обязательства.
* **Last 0–30 days cost** и **Last 0–30 days change** показывают накопленные затраты и процентное изменение по окружению за предыдущие 30 дней.
  Поскольку использование окружения может меняться в течение периода обязательства, эти столбцы помогают определить окружения с наибольшими затратами.
* Для просмотра разбивки затрат по возможности выберите **Actions** > **View details**.

### Сводка использования

Вкладка **Usage summary** отображает диаграмму, показывающую использование отдельных возможностей за предыдущие 30 дней.

![Экран сводки использования в Управлении аккаунтом для модели лицензирования Dynatrace Platform Subscription.](https://dt-cdn.net/images/account-management-dps-usage-summary-1822-4917e05426.png)

Экран сводки использования в Управлении аккаунтом для модели лицензирования Dynatrace Platform Subscription.

* Общее использование каждой возможности отображается в единицах измерения этой возможности.
* Используйте фильтр над диаграммой для просмотра использования по различным окружениям и возможностям.

## Анализ затрат и использования по возможностям

Страница **Capability cost and usage analysis** позволяет просматривать затраты и использование подписки рядом и предоставляет дополнительные фильтры по временному периоду.

![Экран анализа затрат и использования по возможностям в Управлении аккаунтом для модели лицензирования Dynatrace Platform Subscription.](https://dt-cdn.net/images/account-management-dps-capability-cost-and-usage-2-1843-ab841d53ae.png)

Экран анализа затрат и использования по возможностям в Управлении аккаунтом для модели лицензирования Dynatrace Platform Subscription.

* Диаграмма **Environment cost** отображает ежедневные затраты в настроенной валюте вашей подписки.
  Диаграмма **Environment usage** отображает ежедневное использование в единицах измерения данной возможности.
* Используйте фильтр для изменения временного периода.
  По умолчанию диаграмма отображает данные за предыдущие 30 дней.
* Таблица отображает общие затраты и общее использование на основе выбранных фильтров.

## Анализ затрат и использования по окружению

Страница **Environment cost and usage analysis** позволяет просматривать затраты и использование возможностей для конкретного окружения.

![Экран анализа затрат и использования по окружению в Управлении аккаунтом для модели лицензирования Dynatrace Platform Subscription.](https://dt-cdn.net/images/account-management-dps-environment-cost-and-usage-1852-042963b02d.png)

Экран анализа затрат и использования по окружению в Управлении аккаунтом для модели лицензирования Dynatrace Platform Subscription.

Для просмотра анализа затрат и использования по любому окружению выберите **Actions** > **View details**.
Эта страница является вариантом страницы **Capability cost and usage analysis** и отображает накопленные затраты и использование по возможностям.

Выберите возможность из фильтра над диаграммой для просмотра использования за выбранный временной период.
