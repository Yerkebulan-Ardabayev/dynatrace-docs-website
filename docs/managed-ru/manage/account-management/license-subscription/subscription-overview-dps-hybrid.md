# Обзор подписки (DPS для гибридного развёртывания)

* Пояснение
* Чтение 3 мин
* Опубликовано 24 ноя 2025 г.
* Предварительная версия

Эта страница предназначена специально для клиентов с конфигурацией DPS для гибридного развёртывания — см. [DPS для гибридного развёртывания](/managed/license/dps-for-hybrid "DPS для гибридного развёртывания позволяет использовать одну подписку для нескольких аккаунтов.").

Если вы не используете общую лицензию DPS с другими аккаунтами, обратитесь к документации [Подписка (DPS)](/managed/manage/account-management/license-subscription/subscription-overview-dps "Просмотр сводки бюджета и анализа стоимости Dynatrace Platform Subscription.") для описания вашего представления **Overview**.

Доступ к обзору подписки

При создании новой конфигурации DPS для гибридного развёртывания все администраторы всех связанных аккаунтов по умолчанию получают доступ к **Subscription overview** для соответствующей подписки DPS.
В частности, это пользователи с разрешениями «View account» (`account-viewer`) или «View and manage account and billing information» (`account-company-info`) хотя бы для одного из аккаунтов Dynatrace, использующих подписку DPS.

Если администраторы конкретного аккаунта не должны иметь доступ к **Subscription overview**, обратитесь в Dynatrace для настройки прав доступа.

Представление обзора подписки в Управлении аккаунтом предоставляет ключевые анализы вашей подписки DPS для гибридного развёртывания.
Вы можете просматривать общую информацию о лицензии и детализацию на уровне аккаунтов.

Для просмотра этой информации перейдите в **Account Management** > **Subscription** > **Subscription Overview**.

## Уровень лицензии

При первом открытии страницы **Subscription** > **Subscription Overview** вы увидите информацию о вашей лицензии DPS.

* **Dynatrace Platform Subscription**: общая информация о вашей лицензии DPS.
* **Cost distribution - Accounts**: общие затраты на текущий момент по всем аккаунтам в вашей конфигурации DPS для гибридного развёртывания с разбивкой по аккаунтам.
* **Cost Trend**: как менялись затраты каждого аккаунта с течением времени.

Для просмотра потребления всех аккаунтов, привязанных к лицензии, убедитесь, что в выпадающем меню вверху справа выбрано **All accounts**.
Для детализации и просмотра затрат на уровне аккаунта выберите выпадающее меню, а затем имя нужного аккаунта.

Обзор подписки для конфигурации DPS для гибридного развёртывания в Управлении аккаунтом

### Панель Dynatrace Platform Subscription

Панель **Dynatrace Platform Subscription** предоставляет обзор лицензии DPS, к которой привязаны все аккаунты.
Включает:

* Годовую сумму обязательства.
* Годовой период обязательства.
* Оставшиеся дни.
  Отсчитывается от текущего дня до последнего полного дня годового периода обязательства (включительно).

Дополнительная информация о ваших лицензиях доступна при выборе:

* **History** — для просмотра всех лицензий, связанных с аккаунтами, включая активные и истёкшие.
* **Pricing** — для просмотра сведений о ценах на единицы возможностей.

### Распределение затрат — аккаунты

Панель **Cost distribution - Accounts** показывает, как каждый отдельный аккаунт вносит вклад в общее потребление лицензии.

* Круговая диаграмма слева показывает, какую долю общих затрат на текущий момент обеспечил каждый аккаунт.
* Гистограммы справа показывают затраты каждого аккаунта как долю от общего обязательства по лицензии.

### Тренд затрат

Панель **Cost Trend** показывает, как менялись затраты каждого аккаунта с течением времени.
Аккаунты суммируются, поэтому общее потребление всех аккаунтов также видно.

## Уровень аккаунта

Для просмотра потребления лицензии на уровне аккаунта выберите выпадающее меню в правом верхнем углу.
(По умолчанию отображается **All accounts**.)
При открытии меню отображается список всех аккаунтов, связанных с вашей лицензией DPS.
Выберите нужный аккаунт.

Представление на уровне аккаунта предоставляет следующую информацию:

* **Dynatrace Platform Subscription**: как и в представлении уровня лицензии, здесь отображается общая информация о вашей лицензии DPS.
* **Cost distribution**: общие затраты аккаунта на текущий момент.
* **Cost Forecast**: как менялись затраты аккаунта с течением времени, включая прогноз будущих затрат.
* **Events**: все применимые прогнозные события, связанные с аккаунтом.

![Обзор аккаунта для конфигурации DPS для гибридного развёртывания в Управлении аккаунтом](https://dt-cdn.net/images/account-management-hybrid-account-overview-5760-ac81f83d69.png)

Обзор аккаунта для конфигурации DPS для гибридного развёртывания в Управлении аккаунтом

### Панель Dynatrace Platform Subscription

Панель **Dynatrace Platform Subscription** предоставляет обзор лицензии DPS, к которой привязан данный аккаунт.
Это та же информация, что отображается в представлении обзора подписки — см. [Панель Dynatrace Platform Subscription](#subscription-overview-dps-pane).

### Распределение затрат

Панель **Cost distribution** предоставляет сведения о потреблении аккаунтом различных возможностей тарифного плана Dynatrace.

Выберите **View details** для просмотра страницы **Cost and usage details** — см. [Затраты и использование](/managed/manage/account-management/license-subscription/subscription-overview-dps#dps-cost-and-usage "Просмотр сводки бюджета и анализа стоимости Dynatrace Platform Subscription.").

### Прогноз затрат

Панель **Cost Forecast** показывает накопленные затраты.
Данные отображаются с начала периода подписки и включают прогноз затрат до конца периода.
Наведите курсор на диаграмму, чтобы увидеть фактические и прогнозируемые затраты.

Управление аккаунтом использует продвинутые модели прогнозирования для предсказания будущих затрат.
Прогнозы доступны только для аккаунтов с данными о затратах за 15 и более дней.

Прогнозные данные являются лишь оценкой.
Они не гарантируют будущие затраты.

### События

Панель **Events** показывает прогнозируемое увеличение использования по требованию (в процентах и стоимости) к концу годового периода обязательства.
Когда прогнозируется превышение годового обязательства, панель отображает конкретную дату, на которую ожидается это превышение.
