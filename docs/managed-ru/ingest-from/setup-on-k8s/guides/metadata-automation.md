# Метаданные и автоматизация

* Чтение: 1 мин
* Опубликовано 28 июля 2023 г.

Изучите различные способы автоматизации и управления метаданными в вашей системе.

[### Настройка распространения меток сборки

Настройте распространение меток сборки, чтобы передавать внедрённому OneAgent информацию о сборке и версии для вновь развёртываемых подов.](/managed/ingest-from/setup-on-k8s/guides/metadata-automation/build-label-propagation "Настройка распространения меток сборки")[### Добавление файла пользовательских свойств

Добавьте файл пользовательских свойств в вашу систему, что позволит определять пользовательские свойства метаданных и управлять ими.](/managed/ingest-from/setup-on-k8s/guides/metadata-automation/custom-properties-file "Добавление файла пользовательских свойств")[### Обогащение метрик метаданными

Обогатите ваши метрики дополнительными метаданными, повышая глубину и полезность анализа данных.](/managed/ingest-from/setup-on-k8s/guides/metadata-automation/metadata-enrichment "Обогащение метаданными в Dynatrace Operator добавляет контекст подам Kubernetes, прикрепляя соответствующие метаданные к сущностям (подам, хостам, процессам) для лучшей наблюдаемости.")
