# Оркестрация развёртывания OneAgent

* 1-min read
* Published Sep 25, 2020

Вы можете масштабно развёртывать OneAgent с помощью предоставляемых Dynatrace скриптов оркестрации. В настоящее время Dynatrace предоставляет коллекцию Ansible. Мы работаем над поддержкой нескольких технологий оркестрации, но вы можете помочь нам расставить приоритеты. Выберите **I use another type of script** на странице **Install OneAgent**, чтобы отправить отзыв.
