# Обзор работоспособности OneAgent

* 4-min read
* Updated on Jan 15, 2026

Обзор работоспособности OneAgent позволяет обнаруживать все развёрнутые [модули OneAgent](/managed/ingest-from/technology-support/oneagent-platform-and-capability-support-matrix "Узнайте, какие возможности поддерживает OneAgent на различных операционных системах и платформах.") в масштабе и выявлять аномалии до того, как они станут проблемами. С помощью обзора работоспособности OneAgent можно, например, обнаружить:

* Устаревшие модули OneAgent, требующие обновления для продолжения полной поддержки Dynatrace.
* Проблемы подключения между модулями OneAgent и кластером Dynatrace.
* Несовместимые модули OneAgent, требующие корректирующих действий.
* Процессы, требующие перезапуска для продолжения мониторинга.
* Модули OneAgent, у которых отключено автоматическое обновление.

## Начало работы

Чтобы начать работу с обзором работоспособности OneAgent, перейдите в **OneAgent Health**.

![OneAgent Health](https://dt-cdn.net/images/oneagent-health-1744-afe02d9e3b.png)

OneAgent Health

**Обзор работоспособности OneAgent** включает:

* Площадную диаграмму, ориентированную на [метрику состояния работоспособности](#health-state-metric).

  Площадная диаграмма включает данные о модулях OneAgent за выбранный период времени. Диаграмму можно разделить по состоянию работоспособности, состоянию мониторинга, типу модуля, версии модуля, типу процесса и типу операционной системы.
* Таблицу данных с различными атрибутами вносящих вклад модулей OneAgent.

  Таблица данных содержит данные в реальном времени о модулях OneAgent, наблюдавшихся в вашем окружении в последние 5 минут. Её можно фильтровать по определённым атрибутам. Для отображения дополнительных атрибутов выберите **Configure columns**.

  Над таблицей данных отображаются минимальная и максимальная совместимые версии OneAgent для данного кластера Dynatrace.

  Для отображения данных в этой таблице требуется разрешение окружения [Install OneAgent](/managed/manage/identity-access-management/permission-management/role-based-permissions#environment "Разрешения на основе ролей").

Выберите конкретную точку данных на площадной диаграмме для просмотра вносящих вклад модулей OneAgent в таблице данных.

## Метрика состояния работоспособности

Метрика **Health state** (`dsfm:cluster.oneagent.agent_modules`) показывает состояние работоспособности развёрнутых модулей OneAgent. Она делится на четыре класса: Critical, Warning, Info и Healthy. Каждый класс имеет разные последствия.

| Состояние работоспособности | Последствия |
| --- | --- |
| Critical | Модуль OneAgent испытывает проблемы, которые могут привести к перебою мониторинга. Требуются корректирующие действия. |
| Warning | Модуль OneAgent испытывает аномалии, которые могут повлиять на доступность мониторинга. Корректирующие действия рекомендуются. |
| Info | Работоспособность модуля OneAgent можно улучшить для обеспечения доступности мониторинга. Корректирующие действия не обязательны. |
| Healthy | Модуль OneAgent работает без сбоев мониторинга. Никаких действий не требуется. |

### Примеры состояний работоспособности

Следующие условия применяются к модулям OneAgent и приводят к конкретным рекомендациям.

| Состояние | Условие | Рекомендация |
| --- | --- | --- |
| Critical | Дата версии установщика старше 11 месяцев | Обновите OneAgent до новой версии для поддержания полной поддержки Dynatrace. |
| Critical | Контрольный сигнал модуля просрочен на 180 секунд | Dynatrace не получал данные от этого модуля в течение 180 секунд. Возможны проблемы со связью, или процесс неожиданно завершился. |
| Critical | Конфликт идентификатора хоста | Несколько мониторируемых хостов сообщают об использовании одного идентификатора хоста. Эта нештатная ситуация может быть результатом клонирования машины. Обратитесь к специалисту Dynatrace через онлайн-чат в вашем окружении. |
| Critical | Превышена квота хоста | Мониторинг в данный момент отключён. Обратитесь к специалисту Dynatrace через онлайн-чат в вашем окружении. |
| Critical | Версия установщика помечена Dynatrace как дефектная | Обновите OneAgent до новой версии. Процессы также могут потребовать перезапуска. |
| Critical | Версия модуля помечена Dynatrace как дефектная | Обновите OneAgent до новой версии. Может также потребоваться перезапуск процесса. |
| Warning | Версия установщика от 9 до 11 месяцев | Обновитесь до более новой версии OneAgent, чтобы оставаться актуальным. |
| Warning | Несовместима минимальная версия OneAgent | Обновите OneAgent до новой версии. Может также потребоваться перезапуск процесса, даже если вы уже обновили модуль. |
| Warning | Несовместима максимальная версия OneAgent | Подключать можно только версии OneAgent, равные или ниже версии кластера Dynatrace. Загрузите совместимую версию OneAgent из этого кластера. |
| Info | Требуется перезапуск процесса | Версия вашего модуля устарела. Перезапустите процесс для обновления модуля до новой версии OneAgent. |
| Info | Автообновление отключено | Включите автообновление, чтобы оставаться актуальным. |

### Ограничения

* Только метрика **Health state** включает модуль z/OS Java и интеграции AWS Lambda, Azure и Google Cloud.
* В таблице данных отображаются только модули OneAgent, наблюдавшиеся в последние 5 минут.
* Только таблицу данных можно фильтровать по зоне управления.
