# Сообщения модулей z/OS - системные сообщения модуля z/OS Java

* Чтение: 1 мин
* Опубликовано 22 марта 2019 г.

### ZDZ079D

|  |  |
| --- | --- |
| Пояснение | Заглушка ODIN zOS возвращает диагностический код. |
| Действие системы | Обработка продолжается. |
| Реакция пользователя | Обратитесь к специалисту по продуктам Dynatrace через чат в вашем окружении. |

### ZDZ090E

|  |  |
| --- | --- |
| Пояснение | Заглушка ODIN zOS не может найти таблицу поставщика. |
| Действие системы | Обработка завершается ошибкой. |
| Реакция пользователя | Перезапустите zDC с актуальным уровнем обслуживания (не ниже уровня Java-ODIN), затем перезапустите процесс Java. |
