# Сообщения модулей z/OS - системные сообщения zDC

* Чтение: 63 мин
* Обновлено 10 февраля 2026 г.

## ZDC000I

* **Полное сообщение** - ZDC000I INITIALIZATION @@ FOR zDC @@ VER @@.@@.@@
* **Пояснение** - Это информационное сообщение указывает, что инициализация zDC началась или завершилась. Отображается идентификатор подсистемы и версия продукта.
* **Действие системы** - Обработка продолжается.
* **Реакция пользователя** - Нет

## ZDC001E

* **Полное сообщение** - ZDC001E AN ERROR HAS OCCURRED DURING LOAD FOR MODULE @@
* **Пояснение** - При запуске системы zDC была предпринята попытка загрузить все модули обработки в память. Загрузка модуля, указанного в этом сообщении (`@@`), завершилась неудачей.
* **Действие системы** - zDC завершает работу.
* **Реакция пользователя** - Убедитесь, что операторы `//STEPLIB` или `//JOBLIB DD` в JCL, запускающем zDC, указывают на правильные наборы данных, и что эти наборы данных содержат нужный выпуск zDC.

## ZDC002E

* **Полное сообщение** - ZDC002E MODULE=@@ R15=@@ R1=@@
* **Пояснение** - Это сообщение следует за `ZDC001` и содержит имя модуля, а также содержимое регистров `R15` и `R1` по завершении операции LPAD.
* **Действие системы** - zDC завершает работу.
* **Реакция пользователя** - Сохраните информацию из этого сообщения, если потребуется обратиться к системному программисту или специалисту по продуктам Dynatrace через чат. Обратитесь к специалисту по продуктам Dynatrace через чат в вашем окружении.

## ZDC003E

* **Полное сообщение** - ZDC003E COMMON AREA STORAGE NOT AVAILABLE FOR MODULE @@
* **Пояснение** - Это сообщение указывает, что попытка загрузить модуль, указанный в сообщении, в расширенную общую память завершилась ошибкой из-за нехватки доступной памяти для размещения модуля.
* **Действие системы** - zDC завершает работу.
* **Реакция пользователя** - Увеличьте объём общей памяти, в частности субпула 241, и снова запустите zDC.

## ZDC004E

* **Полное сообщение** - ZDC004E AN ERROR HAS OCCURRED DURING BLDL FOR MODULE @@
* **Пояснение** - При запуске системы zDC была предпринята попытка загрузить все модули обработки в память. Операция `BLDL` для загрузки модуля, указанного в сообщении, завершилась неудачей.
* **Действие системы** - zDC завершает работу.
* **Реакция пользователя** - Убедитесь, что операторы `//STEPLIB` или `//JOBLIB DD` в JCL, запускающем zDC, указывают на правильные наборы данных, и что эти наборы данных содержат нужный выпуск zDC.

## ZDC006E

* **Полное сообщение** - ZDC006E SUBSYSTEM @@ ALREADY EXISTS AND IS ACTIVE
* **Пояснение** - Это сообщение указывает, что 4-символьный идентификатор подсистемы, указанный в наборе данных параметров `//SYSIN DD`, уже существует в системе и является активной подсистемой zDC.
* **Действие системы** - zDC завершает работу.
* **Реакция пользователя** - Убедитесь, что указанный идентификатор подсистемы уникален в вашей системе. Затем перезапустите zDC или остановите экземпляр zDC с этим именем.

## ZDC007E

* **Полное сообщение** - ZDC007E ECSA STORAGE NOT AVAILABLE FOR SUBSYSTEM @@, R15=@@
* **Пояснение** - Это сообщение указывает, что попытка выделить память для управляющего блока подсистемы завершилась ошибкой. В расширенном субпуле CSA 241 недостаточно памяти выше граничной линии для этих блоков.
* **Действие системы** - zDC завершает работу.
* **Реакция пользователя** - Увеличьте объём памяти расширенного субпула CSA 241 и перезапустите zDC.

## ZDC008I

* **Полное сообщение** - ZDC008I MODULE P/T DATE TIME VERSION ADDRESS LENGTH
* **Пояснение** - Это сообщение отображается только при запуске zDC. В нём указаны столбцы, отображаемые далее в нескольких сообщениях `ZDC009`: имя модуля, наибольший номер отслеживания проблем, дата и время последней сборки модуля, его версия и адрес основной памяти, по которому он загружен.
* **Действие системы** - Обработка продолжается.
* **Реакция пользователя** - Нет

## ZDC009I

* **Пояснение** - В этом сообщении перечислен один модуль, загруженный zDC. Описание столбцов см. в сообщении `ZDC008`.

## ZDC010E

* **Полное сообщение** - ZDC010E SDUMPX TAKEN SUCCESSFULLY FOR zDC @@ ABEND
* **Пояснение** - Это сообщение выводится на системную консоль подпрограммами дампа zDC. В нём указан код аварийного завершения.
* **Действие системы** - Выполнение завершается.
* **Реакция пользователя** - Подготовьте дамп для изучения системным персоналом или передачи специалисту по продуктам Dynatrace через чат. Обратитесь к специалисту по продуктам Dynatrace через чат в вашем окружении. Повторно запустите zDC.

## ZDC011E

* **Полное сообщение** - ZDC011E SDUMPX FAILED FOR zDC @@ ABEND. RETCODE=@@/@@
* **Пояснение** - Это сообщение выводится на системную консоль подпрограммами дампа zDC. Оно указывает, что попытка получить системный дамп завершилась ошибкой. В сообщении отображены код возврата и код причины сбоя.
* **Действие системы** - Выполнение завершается.
* **Реакция пользователя** - Изучите коды возврата и причины или передайте это сообщение специалисту по продуктам Dynatrace через чат. Обратитесь к специалисту по продуктам Dynatrace через чат в вашем окружении. Повторно запустите zDC.

## ZDC012E

* **Полное сообщение** - ZDC012E @@ ABEND DUMP IN DYNATRACE zDC @@
* **Пояснение** - Это сообщение является заголовком для всех макросов SDUMPX, выдаваемых zDC. Заголовок указывает код аварийного завершения, ставший причиной создания дампа.
* **Действие системы** - Выполнение завершается.
* **Реакция пользователя** - Убедитесь, что дамп сохранён и может быть передан системным программистам или специалисту по продуктам Dynatrace через чат. Обратитесь к специалисту по продуктам Dynatrace через чат в вашем окружении. Повторно запустите zDC.

## ZDC014E

* **Действие системы** - Задача журналирования завершается, но выполнение zDC продолжается.
* **Реакция пользователя** - Изучите последующие сообщения, содержащие подробную информацию о характере ошибки. Сохраните эту информацию для системных программистов или специалиста по продуктам Dynatrace через чат. Попробуйте повторно подключить и открыть журнал (команда MVS: `Modify ZdcStcName,LOG OPEN`). Обратитесь к специалисту по продуктам Dynatrace через чат в вашем окружении.
