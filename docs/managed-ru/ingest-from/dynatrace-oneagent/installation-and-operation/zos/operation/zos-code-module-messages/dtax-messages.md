# Сообщения модулей z/OS - сообщения DTAX

* Чтение: 43 мин
* Опубликовано 22 марта 2019 г.

## ZDTP001S

* **Полное сообщение** - ZDTP001S Persistent Storage allocation failed.
* **Пояснение** - Программа CICS PLT не смогла выделить необходимую область постоянной памяти.
* **Действие системы** - Программа PLT завершается, и кодовый модуль CICS не включается.
* **Реакция пользователя** - Выясните, по какой причине регион CICS не может предоставить память для транзакции DTAX.

  Если проблема не устраняется, обратитесь к специалисту по продуктам Dynatrace через чат в вашем окружении Dynatrace.

## ZDTP003S

* **Реакция пользователя** - Выясните, по какой причине регион CICS не может загрузить один из модулей DTAX: ZDTSOAPH.

## ZDTP004S

* **Полное сообщение** - ZDTP004S ZDTDCCAL final block send failed. rc:<xx>.
* **Пояснение** - Передача сообщения от кодового модуля CICS к zDC завершилась ошибкой.
* **Действие системы** - Кодовый модуль CICS продолжает работу, однако в распределённой трассировке отсутствуют некоторые узлы.

## ZDTP005S

* **Полное сообщение** - ZDTP005S START DTAX ICE failed.
* **Пояснение** - Команда `EXEC CICS START`, используемая для запуска транзакции `DTAX ICE` с интервалом 5 минут, завершилась ошибкой.
* **Действие системы** - Нет
* **Реакция пользователя** - Изучите подробное сообщение в соответствующем журнале задания CICS, чтобы получить точный код EIBRESP, указывающий причину сбоя команды EXEC CICS START.

  Если проблема не устраняется, обратитесь к специалисту по продуктам Dynatrace через чат в вашем окружении Dynatrace.

## ZDTP006S

## ZDTP007S

* **Полное сообщение** - ZDTP007S Enable of temporary Language Environment sensor failed no zDC.
* **Пояснение** - Указывает, что датчик Language Environment не удалось включить из-за недоступности zDC.
* **Действие системы** - Датчик Language Environment не включён. Кодовый модуль CICS не отслеживает динамические вызовы LE.
* **Реакция пользователя** - Запустите zDC, чтобы кодовый модуль CICS мог восстановить соединение.
