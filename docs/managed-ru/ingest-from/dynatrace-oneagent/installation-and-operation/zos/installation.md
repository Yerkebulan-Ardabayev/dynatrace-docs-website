# Обзор установки z/OS

* Чтение: 1 мин
* Опубликовано 22 июля 2016 г.

Поскольку модули z/OS требуют наличия нескольких компонентов как в окружении мейнфрейма, так и в открытых системах, процесс установки сложнее, чем стандартная установка OneAgent. Однако при небольшом планировании и координации с разными архитектурными группами установка кодовых модулей может пройти гладко.

Эффективная установка модулей z/OS обычно представляет собой командную работу, в которую входят:

* **Open Systems Administrator** устанавливает модуль zRemote.
* **Mainframe Systems Programmer** скачивает и устанавливает продуктовые наборы данных Dynatrace для z/OS.
* **Mainframe Security Administrator** настраивает безопасность для подсистемы zDC.
* **Mainframe Systems Programmer** устанавливает модули для каждой технологии, которую вы хотите мониторить.

В зависимости от состава вашей команды один и тот же человек может выполнять несколько необходимых задач.
