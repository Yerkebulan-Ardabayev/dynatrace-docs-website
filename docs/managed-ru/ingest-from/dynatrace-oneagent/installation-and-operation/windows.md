# OneAgent на Windows

# OneAgent на Windows

* Чтение: 1 мин
* Опубликовано 19 сентября 2018 г.

OneAgent можно установить на Windows с помощью установщика, предоставляемого в виде самораспаковывающегося EXE-файла (для установки на одном сервере), а также в виде MSI-пакета для развёртываний через групповые политики. Аналитические сведения о поддерживаемых возможностях OneAgent для Windows см. в [матрице поддержки платформ и возможностей OneAgent](/managed/ingest-from/technology-support/oneagent-platform-and-capability-support-matrix "Узнайте, какие возможности поддерживаются OneAgent в различных операционных системах и на разных платформах."). Поддерживаемые версии Windows см. в [Поддерживаемые технологии и версии OneAgent](/managed/ingest-from/technology-support "Найдите технические подробности о поддержке Dynatrace конкретных платформ и фреймворков разработки.").
