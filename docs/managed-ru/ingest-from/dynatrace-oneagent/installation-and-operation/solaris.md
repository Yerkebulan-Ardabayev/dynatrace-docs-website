# Solaris

# Solaris

* Чтение: 1 мин
* Обновлено 18 ноября 2025 г.

Dynatrace поддерживает Oracle Solaris (x86 и SPARC) для Java, Apache HTTP server и IBM HTTP server. Если ваша инфраструктура построена на этих технологиях, можно использовать Dynatrace OneAgent для эффективного мониторинга ваших хостов, сервисов и пользовательского опыта. Аналитические сведения о поддерживаемых возможностях OneAgent для Solaris см. в [матрице поддержки платформ и возможностей OneAgent](/managed/ingest-from/technology-support/oneagent-platform-and-capability-support-matrix "Узнайте, какие возможности поддерживаются OneAgent в различных операционных системах и на разных платформах.").
