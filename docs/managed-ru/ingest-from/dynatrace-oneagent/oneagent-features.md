# Функции OneAgent

# Функции OneAgent

* 4-min read
* Updated on Feb 11, 2025

При создании окружения OneAgent поставляется с большим набором функций, активированных по умолчанию. Тем не менее функции, добавленные в новых версиях OneAgent, а также функции с возможностью подключения (opt-in), такие как автоматическое обогащение записей лога идентификатором трассировки, всегда необходимо активировать явно.

Новые функции OneAgent полностью поддерживаются и протестированы с момента их появления.

* Для новых окружений вы можете активировать или деактивировать функции OneAgent (opt-in) в зависимости от конкретного сценария использования.
* Для существующих окружений вновь добавленные функции OneAgent по-прежнему требуют явной активации пользователями с разрешением **Write settings** (`settings.write`) в [таблице схемы настроек Settings API — OneAgent features](/managed/dynatrace-api/environment-api/settings/schemas/builtin-oneagent-features "Просмотрите таблицу схемы настроек builtin:oneagent.features вашего окружения мониторинга через Dynatrace API.").

## Область действия

Новые функции OneAgent обычно включаются глобально (для всего окружения). Однако вы можете включить или отключить их в рамках области действия группы процессов. Настройки группы процессов переопределяют глобальные настройки для той же функции OneAgent.

## Сценарии использования

* По мере постоянного добавления новых функций в OneAgent их необходимо явно включать в существующих окружениях мониторинга во избежание неожиданных изменений.
* При устранении проблем вы можете выборочно отключить определённые функции OneAgent на детальном уровне. Это может быть полезно для определения первопричины проблемы вплоть до конкретной функции. Подробнее смотрите в разделе [Устранение проблем глубокого мониторинга OneAgent](/managed/ingest-from/dynatrace-oneagent/oneagent-troubleshooting/troubleshoot-oneagent-deep-monitoring-issues "Работайте с командой поддержки Dynatrace для устранения проблем глубокого мониторинга OneAgent.").

## Веб-интерфейс или API

* Вы можете включать или отключать функции OneAgent через веб-интерфейс Dynatrace на странице **OneAgent features** на соответствующем уровне (глобальном или группы процессов).
* Вы можете использовать [Settings API](/managed/dynatrace-api/environment-api/settings "Узнайте, что предлагает Dynatrace Settings API.") Dynatrace для выполнения тех же настроек через вызовы API.

### Настройка через веб-интерфейс

Для включения или отключения функций OneAgent через веб-интерфейс Dynatrace используйте страницу **OneAgent features** на соответствующем уровне.

На каждой функции на странице **OneAgent features** отображается список требований:

* **Min. OneAgent version** — минимальная требуемая версия OneAgent. Функцию можно включить только при соответствии версии OneAgent этому требованию.
* **Requires restart** — указывает, требует ли функция перезапуска процесса перед вступлением в силу.

### Глобальная настройка

Чтобы включить или отключить функцию OneAgent глобально:

1. Перейдите в **Settings** и выберите **Preferences** > **OneAgent features**.
2. Найдите функцию в списке и включите или отключите переключатель **Enabled**.

### Настройка группы процессов

Чтобы включить или отключить функцию OneAgent для группы процессов:

1. Перейдите в **Smartscape Topology**.
2. Выберите **Processes**.
3. На карте топологии наведите курсор на процесс и выберите значок ссылки для перехода на страницу сведений о процессе. Например:

   ![Ссылка из топологии Smartscape на страницу сведений о процессе](https://dt-cdn.net/images/link-to-process-page-377-da0282e435.png)

   Ссылка из топологии Smartscape на страницу сведений о процессе
4. На странице сведений о процессе выберите > **Settings**.
5. На странице **Process group settings** выберите вкладку **OneAgent features**.
6. Выберите **Add override**, чтобы добавить специфичную для процесса настройку, которая переопределяет настройку окружения.
7. Введите строку поиска в поле **Feature**, чтобы найти и выбрать функцию.
8. Установите переключатели по мере необходимости для переопределения, затем выберите **Save changes**.

### Настройка через API

С помощью [Settings API](/managed/dynatrace-api/environment-api/settings "Узнайте, что предлагает Dynatrace Settings API.") вы можете:

* **Включать/отключать** настройки функций OneAgent.
* **Переопределять область действия** функции OneAgent для отдельной группы процессов или процесса.
* **Экспортировать** текущую конфигурацию окружения.

Для использования API необходим токен доступа с областями **Read settings** (`settings.read`) и **Write settings** (`settings.write`). Инструкции по получению токена смотрите в разделе [Создание токена доступа](/managed/dynatrace-api/basics/dynatrace-api-authentication#create-token "Узнайте, как пройти аутентификацию для использования Dynatrace API.").

Чтобы использовать Settings API:

1. Для получения формата JSON, необходимого для отправки конфигурации, используйте endpoint [Get a schema](/managed/dynatrace-api/environment-api/settings/schemas/get-schema "Просмотрите схему настроек через Dynatrace API."). Идентификатор схемы: `builtin:oneagent.features`.

   Пример: JSON-payload для отключения обогащения контекста лога Java для неструктурированных логов для конкретной группы процессов.

   ```
   [



   {



   "schemaId":  "builtin:oneagent.features",



   "scope": "PROCESS_GROUP-1",



   "value": {



   "enabled": false,



   "key": "JAVA_LOG_ENRICHMENT_UNSTRUCTURED"



   }



   }



   ]
   ```

   Вы также можете изменить настройки, применяемые ко всему окружению, используя область действия environment.

   Пример: JSON-payload для глобального включения поддержки Kafka Streams

   Dynatrace версии 1.244+

   ```
   [



   {



   "schemaId":  "builtin:oneagent.features",



   "scope": "Environment",



   "value": {



   "enabled": true,



   "key": "JAVA_KAFKA_STREAMS"



   }



   }



   ]
   ```
2. Для отправки конфигурации используйте endpoint [Post an object](/managed/dynatrace-api/environment-api/settings/objects/post-object "Создайте или проверьте объект настроек через Dynatrace API.").
