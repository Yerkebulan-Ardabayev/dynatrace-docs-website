# OpenTelemetry и Dynatrace

# OpenTelemetry и Dynatrace

* Overview
* 2-min read
* Updated on May 04, 2026

[OpenTelemetry (OTel)](https://www.opentelemetry.io) предоставляет стандартизированный способ сбора и экспорта телеметрических данных. Он позволяет приложениям и инфраструктуре передавать телеметрию в бэкенды (например, Dynatrace) в форматах, не зависящих от поставщика. Такие бэкенды могут агрегировать и анализировать данные.

Этот подход можно реализовать сервис за сервисом, применяя открытые стандарты OpenTelemetry там, где важна открытость, и используя расширенные возможности OneAgent там, где они необходимы. Dynatrace поддерживает такой гибкий подход, обеспечивая ведущую в отрасли аналитику независимо от сочетания вариантов инструментирования.

## Зачем использовать OpenTelemetry с Dynatrace?

Dynatrace обеспечивает сквозную наблюдаемость для всей телеметрии приложений, поддерживая стандарты OpenTelemetry. Благодаря унифицированному подходу к наблюдаемости Dynatrace предлагает комплексную платформу для просмотра, хранения и корреляции всех сигналов OpenTelemetry в одном месте. Это решение позволяет упростить сложность и ускорить инновации за счёт извлечения максимальной ценности из ваших данных независимо от их источника.

Dynatrace предоставляет:

* Мощную аналитику в масштабе. Выполнение масштабных запросов на высокой скорости, превращающее телеметрические данные в actionable intelligent insights для более быстрого решения проблем.
* Интеллект за пределами инструментирования. Dynatrace обогащает данные OpenTelemetry бизнес-контекстом и расширенной аналитикой для получения actionable intelligence, а не просто сбора данных.
* Ускорение инноваций. Работа с данными OpenTelemetry с помощью готового анализа, динамических визуализаций и мгновенных аналитических сведений о производительности, способствующих сотрудничеству между командами разработки и эксплуатации.
* Бесшовную интеграцию. Упрощение интеграции данных в платформу Dynatrace.

## Участие Dynatrace в OpenTelemetry

Dynatrace глубоко вовлечён в экосистему OpenTelemetry через несколько направлений:

* Dynatrace является одним из ведущих участников проекта OpenTelemetry с ключевыми ролями мейнтейнера, помогая формировать его основную функциональность и направление развития.
* Dynatrace активно участвует через руководящие позиции в нескольких специальных группах интересов (SIG) OpenTelemetry и в техническом комитете OpenTelemetry. Это помогает формировать будущее направление стандартов и практик OpenTelemetry.
* Dynatrace поддерживает более 30 проектов с открытым исходным кодом и вносит вклад в множество решений с открытым исходным кодом, отражая глубокую приверженность инновациям, движимым сообществом.

## Несколько путей использования данных OpenTelemetry

Dynatrace встречает клиентов там, где они находятся на своём пути к наблюдаемости, предлагая несколько вариантов приёма сигналов OpenTelemetry в платформу, включая:

* OTLP API
* OTel Collector
* Dynatrace OTel Collector

## Лицензирование

Лицензирование данных OpenTelemetry полностью интегрировано в модель подписки на платформу Dynatrace, обеспечивая прозрачное и единообразное выставление счетов для всех типов телеметрии (трассировки и span-ы, метрики, логи).
