# Amazon EKS (Elastic Kubernetes Service)

* Практическое руководство
* Чтение: 1 мин
* Опубликовано 04 мая 2020 г.

Dynatrace OneAgent обеспечивает комплексный мониторинг подов, узлов и кластеров Elastic Kubernetes Service. Процесс развёртывания OneAgent такой же, как и для других дистрибутивов. Подробнее см. [What is Amazon EKS?](https://dt-url.net/zn0388b).

Для Amazon Elastic Kubernetes Service потребление мониторинга рассчитывается по единицам хостов (host units). Подробнее см. [Application and Infrastructure Monitoring (Host Units)](/managed/license/monitoring-consumption-classic/application-and-infrastructure-monitoring "Узнайте, как рассчитывается потребление Dynatrace для мониторинга приложений и инфраструктуры на основе единиц хостов.").
