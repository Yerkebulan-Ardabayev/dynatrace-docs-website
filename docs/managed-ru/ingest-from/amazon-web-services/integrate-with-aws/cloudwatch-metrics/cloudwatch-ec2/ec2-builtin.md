# Amazon EC2 (встроенный)

# Amazon EC2 (встроенный)

* Практическое руководство
* Чтение: 1 мин
* Опубликовано 27 июля 2022 г.

Dynatrace собирает метрики для множества предварительно выбранных пространств имён, включая Amazon EC2. Можно просматривать метрики для каждого экземпляра сервиса, разделять метрики по нескольким измерениям и создавать пользовательские диаграммы, которые можно закрепить на дашбордах.

## Предварительные требования

Для включения мониторинга этого сервиса необходимо:

* Любая версия ActiveGate для развёртываний Dynatrace SaaS и Managed.

  Для доступа на основе ролей (будь то развёртывание [SaaS](/managed/ingest-from/amazon-web-services/integrate-with-aws/cloudwatch-metrics#role-based-access "Интеграция метрик из Amazon CloudWatch.") или [Managed](/managed/ingest-from/amazon-web-services/set-up-aws-monitoring-with-managed#role-based-access "Подключение аккаунта Amazon к Dynatrace Managed и начало мониторинга.")) требуется ActiveGate, установленный на хосте Amazon EC2.
* Обновлённая [политика мониторинга AWS](/managed/ingest-from/amazon-web-services/integrate-with-aws/cloudwatch-metrics#aws-policy-and-authentication "Интеграция метрик из Amazon CloudWatch.") для включения дополнительных сервисов AWS.
  Чтобы [обновить политику AWS IAM](https://docs.aws.amazon.com/IAM/latest/UserGuide/access_policies_manage-edit.html#edit-managed-policy-console), используйте приведённый ниже JSON, содержащий политику мониторинга (разрешения) для всех облачных сервисов.

## Просмотр метрик сервиса

Метрики сервиса можно просматривать в среде Dynatrace на **странице аккаунта AWS** или на странице **Dashboards**.

### Просмотр метрик на странице аккаунта AWS

Чтобы просмотреть метрики на странице аккаунта AWS

1. Перейдите в **AWS**.
2. Выберите аккаунт AWS, для которого нужно проверить метрики.
3. Выберите блок **Service**. Метрики выбранного сервиса отображаются под инфографикой в разделе сервиса.

### Просмотр метрик на дашборде

Также можно создать собственный дашборд. Дополнительные сведения о создании дашбордов см. в разделе [Создание и редактирование дашбордов Dynatrace](/managed/analyze-explore-automate/dashboards-classic/dashboards/create-dashboards "Узнайте, как создавать и редактировать дашборды Dynatrace.").

Особенности встроенных сервисов

Это встроенный сервис. Он мониторируется автоматически после создания нового экземпляра интеграции AWS. Для встроенных сервисов все метрики рекомендованы (изменить конфигурацию невозможно).
