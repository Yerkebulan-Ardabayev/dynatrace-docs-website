## Интеграция с Azure Monitor

Помимо мониторинга рабочих нагрузок Azure через OneAgent, можно интегрировать Dynatrace с Azure Monitor, чтобы мониторить инфраструктуру и получать аналитику даже в сценариях бессерверных приложений.

Подробнее см. в разделе [Мониторинг сервисов Azure через метрики Azure Monitor](/managed/ingest-from/microsoft-azure-services/azure-integrations/azure-monitoring-guide "Настройка и конфигурация мониторинга Azure в Dynatrace.")
