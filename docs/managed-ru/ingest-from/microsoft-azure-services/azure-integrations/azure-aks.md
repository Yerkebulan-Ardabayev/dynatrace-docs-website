# Azure Kubernetes Service (AKS)

* Обзор
* Чтение: 1 мин
* Опубликовано 04 мая 2020 г.

Dynatrace OneAgent обеспечивает расширенный мониторинг подов, узлов и кластеров **Azure Kubernetes Service**. Процесс развёртывания OneAgent аналогичен другим дистрибутивам.

Кроме того, вы можете включить сбор специфичных для AKS метрик из Azure Monitor
