# Интеграции с Google Cloud

* Обзор
* Чтение: 1 мин

Dynatrace OneAgent обеспечивает полностековый мониторинг на Google Compute Engine и Google Kubernetes Engine, а также мониторинг на уровне контейнеров и приложений. Dynatrace поддерживает Google App Engine в режиме application-only.
