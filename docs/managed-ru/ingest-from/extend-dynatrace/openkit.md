# Dynatrace OpenKit

* 3-min read
* Updated on Mar 16, 2023

Dynatrace OpenKit — это open source библиотека для инструментирования нативных мобильных и настольных приложений.

## Возможности

* Отслеживание сессий и действий пользователя.
* Отчёты о событиях, ошибках и сбоях.
* Трассировка веб-запросов.

## Ограничения

* Серверная трассировка (server-side traces) не поддерживается.
* Метрики не поддерживаются.
* Для передачи данных требуется соединение TLS 1.2+.
