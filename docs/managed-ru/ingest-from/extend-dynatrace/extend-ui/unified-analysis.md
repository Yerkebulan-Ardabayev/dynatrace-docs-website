# Страницы унифицированного анализа

* 4-min read
* Updated on Mar 16, 2023

Страницы унифицированного анализа (unified analysis pages) предоставляют контекстные представления для пользовательских типов сущностей.

## Типы страниц

* **Страница списка (list screen)** — отображает список всех экземпляров сущности определённого типа.
* **Страница деталей (details screen)** — отображает подробную информацию о конкретной сущности.

## Типы карточек

| Тип карточки | Описание |
| --- | --- |
| chart group | Группа графиков для отображения метрик |
| entity list | Список связанных сущностей |
| metric table | Таблица метрик для сравнения сущностей |
| properties | Свойства сущности |
| logs | Записи логов сущности |
| messages | Сообщения |
| events | События |
| health | Показатели работоспособности |

## Концепции

* **Действия (actions)** — действия, доступные для сущностей на странице.
* **Фильтрация** — фильтрация списка сущностей по атрибутам.
* **Инъекции** — добавление пользовательских карточек на страницы.
* **Исследовательский анализ** — инструменты для исследования данных.
