# Расширение трассировки

* 1-min read
* Published Mar 05, 2018

Dynatrace предоставляет несколько способов расширить возможности распределённой трассировки.

## Способы расширения

* [OpenTelemetry](/managed/ingest-from/dynatrace-oneagent/oneagent-and-opentelemetry/oneagent-otel "Узнайте, как отправлять данные OpenTelemetry в Dynatrace OneAgent.") — стандартный фреймворк для трассировки и метрик.
* [OpenTracing](/managed/ingest-from/extend-dynatrace/extend-tracing/opentracing "Узнайте, как интегрировать OpenTracing с Dynatrace.") — интеграция с OpenTracing API.
* [OneAgent SDK](/managed/ingest-from/extend-dynatrace/extend-tracing/oneagent-sdk "Dynatrace OneAgent SDK позволяет вручную инструментировать приложения.") — ручное инструментирование через OneAgent SDK.
