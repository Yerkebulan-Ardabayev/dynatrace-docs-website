# Расширение топологии

# Расширение топологии

* 3-min read
* Updated on Mar 16, 2023

Dynatrace предоставляет контекстно-насыщенную наблюдаемость благодаря встроенной модели топологии. При необходимости можно расширить модель топологии для специфических сценариев использования.

## Встроенная модель топологии (Smartscape)

Smartscape автоматически обнаруживает и отображает зависимости между сервисами, процессами, хостами и облачными инфраструктурами. Это базовая модель для большинства сценариев мониторинга.

## Пользовательская модель топологии

Пользовательская топология применяется для IoT и других специфических сценариев, когда встроенных типов сущностей недостаточно.

### Пример: Easy Shipping LTD

Представьте компанию Easy Shipping LTD с парком грузовиков. Каждый грузовик является сущностью (entity) пользовательской топологии с прицепами и контейнерами как дочерними сущностями.

## Преимущества пользовательской топологии

* Мониторинг специфических для бизнеса сущностей.
* Автоматическое обнаружение зависимостей между пользовательскими сущностями.
* Контекстное отображение метрик и событий в разрезе бизнес-сущностей.
