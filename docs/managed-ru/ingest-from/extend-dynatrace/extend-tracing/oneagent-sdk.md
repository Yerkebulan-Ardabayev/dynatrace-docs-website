# OneAgent SDK

# OneAgent SDK

* 3-min read
* Updated on Mar 16, 2023

OneAgent SDK позволяет вручную инструментировать приложения для расширения сквозной видимости в тех случаях, когда автоматическое инструментирование недоступно.

## Возможности SDK

* Трассировка удалённых вызовов (remote calls).
* Трассировка запросов к базам данных.
* Трассировка входящих веб-запросов.
* Поддержка асинхронного кода.
* Мониторинг очередей сообщений.
* Атрибуты запросов (request attributes).
