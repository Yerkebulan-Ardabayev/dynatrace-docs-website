# Расширение аналитики логов

* 1-min read
* Published Mar 05, 2018

Dynatrace предоставляет несколько способов расширить аналитику логов.

## Возможности

* [Log Analytics](/managed/analyze-explore-automate/log-monitoring "Узнайте, как использовать Log Analytics в Dynatrace.") — мощный инструмент для анализа логов.
* [API приёма логов](/managed/analyze-explore-automate/log-monitoring/acquire-log-data/logs-classic-ingestion-api "Узнайте, как Dynatrace принимает данные логов и каковы возможные ограничения.") — отправка логов через API.
* [API приёма логов OneAgent](/managed/ingest-from/extend-dynatrace/extend-logs/oneagent-log-ingest-api "Отправляйте логи приложений в Dynatrace с помощью локального API OneAgent.") — локальный API OneAgent для приёма логов.
