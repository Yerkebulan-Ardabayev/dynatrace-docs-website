## IBM JVM

У IBM JVM собственный [график поддержки](https://developer.ibm.com/javasdk/support/lifecycle/), но фактическая поддержка зависит от поддержки других продуктов IBM. Для Dynatrace самый важный из них, это WebSphere Application Server (см. [график поддержки](https://en.wikipedia.org/wiki/IBM_WebSphere_Application_Server)).

WebSphere Application Server находится на [политике Enhanced Lifecycle](https://www-01.ibm.com/software/support/lifecycle/lc-policy.html). Это означает, что хотя [WebSphere 8](https://www-01.ibm.com/software/support/lifecycleapp/PLCDetail.wss?q45=D704800I08089R72) достиг EOS в апреле 2018 года, IBM продолжит поддерживать его до 2021 года.

Dynatrace будет поддерживать WebSphere и лежащие в его основе версии JVM до тех пор, пока их поддерживает IBM. Подробности см. в [матрице поддерживаемых технологий](/managed/ingest-from/technology-support#java "Технические подробности о поддержке Dynatrace для конкретных платформ и фреймворков разработки.").

То есть на данный момент IBM JVM 6, 7 и 8 имеют бессрочную поддержку.

## Поддержка и прекращение поддержки

У JVM разные сроки поддержки в зависимости от вендора. Dynatrace берёт на себя обязательство поддерживать каждую JVM и версию не меньше, чем её вендор, и в большинстве случаев хотя бы на 1 год дольше.

Подробности о поддерживаемых JVM см. в [нашей матрице поддерживаемых технологий](/managed/ingest-from/technology-support#java "Технические подробности о поддержке Dynatrace для конкретных платформ и фреймворков разработки.").
