# Поддерживаемые версии Go

* Время чтения: 7 мин
* Обновлено 09 июля 2026 г.

При выходе новой мажорной версии Go Dynatrace добавляет поддержку этой версии. Поддержка добавляется и для каждой минорной версии, подробнее см. [Матрица версий](#go-version-matrix).

[Политика выпуска Go﻿](https://dt-url.net/uos3rmi) поддерживает две последние мажорные версии Go. Однако Dynatrace решил
поддерживать каждую версию Go как минимум на полгода дольше, чтобы оставалось достаточно времени на обновление.

## Внешние метаданные

OneAgent может поддерживать более новые версии Go, запрашивая внешние метаданные у кластера Dynatrace. После получения OneAgent
внешних метаданных он может инструментировать приложение, основанное на версии Go, для которой у OneAgent
нет встроенной поддержки. Требуется перезапуск процесса Go.

Поэтому [матрица версий](#go-version-matrix) содержит две разные верхние границы в каждом диапазоне версий. Встроенные
версии Go поддерживаются всегда, а верхняя граница в скобках может поддерживаться, если OneAgent способен получить требуемые
внешние метаданные от кластера Dynatrace.

Внешние метаданные публикуются независимо от цикла выпуска OneAgent, и OneAgent всегда будет получать самые свежие
доступные внешние метаданные от кластера Dynatrace. Это также означает, что в поэтапной среде приложения могут
быть инструментированы на более позднем этапе, хотя на более раннем этапе они не были инструментированы, поскольку тем временем стали доступны
новые метаданные. Если такое поведение нежелательно, отключить функцию OneAgent **Go external metadata**, чтобы внешние
метаданные не использовались. Функции OneAgent можно найти в **Settings** > **Preferences** > **OneAgent features**.
