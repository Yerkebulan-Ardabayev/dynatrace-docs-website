# Настройка Dynatrace на Microsoft Azure

# Настройка Dynatrace на Microsoft Azure

* Overview
* 1-min read
* Updated on Feb 20, 2026

Dynatrace автоматически обнаруживает, задаёт базовые значения и интеллектуально мониторит облачные среды Microsoft Azure, обеспечивая изоляцию доменов сбоев и видимость инфраструктуры и конечных пользователей в режиме реального времени.

Подробнее о причинах использования Dynatrace для мониторинга Azure смотрите [Microsoft Azure](https://www.dynatrace.com/hub/detail/microsoft-azure/) в Dynatrace Hub.
