# Настройка Dynatrace на Amazon Web Services

# Настройка Dynatrace на Amazon Web Services

* Практическое руководство
* 2 минуты на чтение
* Опубликовано 12 авг. 2021 г.

Dynatrace предлагает как инфраструктурный мониторинг всех сервисов AWS, так и полный мониторинг стека для сервисов, которые можно инструментировать с помощью технологии OneAgent. Кроме того, можно выбрать мониторинг логов, генерируемых облачными сервисами, а также собственных сервисов, размещённых в виртуальных машинах или контейнерах.

Больше причин для мониторинга AWS с помощью Dynatrace см. в разделе [AWS﻿](https://www.dynatrace.com/hub/detail/aws/?query=aws) в Dynatrace Hub.
