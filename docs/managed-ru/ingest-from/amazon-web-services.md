# Настройка Dynatrace на Amazon Web Services

# Настройка Dynatrace на Amazon Web Services

* How-to guide
* 2-min read
* Published Aug 12, 2021

Dynatrace предлагает как мониторинг инфраструктуры всех сервисов AWS, так и полный мониторинг стека для сервисов, которые можно инструментировать с помощью технологии OneAgent. Помимо этого, вы можете настроить мониторинг логов, генерируемых облачными сервисами, а также вашими собственными сервисами, размещёнными в виртуальных машинах или контейнерах.

Подробнее о причинах использования Dynatrace для мониторинга AWS смотрите [AWS](https://www.dynatrace.com/hub/detail/aws/?query=aws) в Dynatrace Hub.
