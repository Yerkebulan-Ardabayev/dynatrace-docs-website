# Установка

* 1-min read
* Published Apr 09, 2021

Dynatrace предлагает два типа ActiveGate: Environment ActiveGate и Cluster ActiveGate. Развёртывания Dynatrace Managed обычно требуют обоих типов, хотя наиболее важным является [Cluster ActiveGate](/managed/managed-cluster/installation/install-cluster-activegate "Установите Cluster ActiveGate на Linux или Windows для маршрутизации трафика OneAgent или запуска синтетических мониторов и подключите его к Managed Cluster."). Для развёртываний Dynatrace Managed смотрите [дополнительные сценарии использования ActiveGate для Dynatrace Managed](/managed/managed-cluster/basics/managed-deployments#scenario-2-pure-dynatrace-managed-setup "Поймите, как развёртывания Dynatrace Managed развиваются от базовой внутренней конфигурации до глобально распределённой высокодоступной архитектуры.").

Environment ActiveGate можно установить:

* с помощью установщика на виртуальный или физический хост — это называется **развёртывание на основе хоста**
* в контейнер — это называется **контейнерное развёртывание**
