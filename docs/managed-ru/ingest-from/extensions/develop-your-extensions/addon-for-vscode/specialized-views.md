# Специализированные представления

* Справочник
* Чтение: 2 мин
* Опубликовано 16 июня 2025 г.

**Dynatrace Extensions** создаёт свою запись на панели действий VS Code. Эта запись предоставляет два специализированных представления для эффективного управления проектами расширений в масштабе.

## Рабочие пространства Extension 2.0

Это представление помогает отслеживать все проекты Extension 2.0 и их рабочие пространства независимо от места хранения в файловой системе. Каждое зарегистрированное рабочее пространство отображается по имени корневой папки: текущее открытое рабочее пространство выделено зелёным, остальные синим.

В этом представлении доступны следующие действия:

* Нажмите кнопку «плюс», чтобы зарегистрировать новое рабочее пространство.
* Нажмите кнопку обновления, чтобы обновить список.

Элементы верхнего уровня этого списка представляют проекты расширений. Для каждого из них доступны следующие действия:

* Нажмите кнопку папки, чтобы открыть связанное рабочее пространство в редакторе VS Code.
* Нажмите кнопку корзины, чтобы отменить регистрацию проекта. Рабочее пространство не будет удалено из файловой системы.
* Выберите метку, чтобы увидеть имя расширения в данном рабочем пространстве и его версию.

  + Нажмите кнопку файла, чтобы быстро открыть манифест расширения. Он откроется в том же окне.

Это представление позволяет легко обновлять некоторые настройки поведения Dynatrace Extensions, связанные с каждым рабочим пространством. Для этого щёлкните правой кнопкой мыши на метке любого зарегистрированного рабочего пространства.

## Окружения

Как следует из названия, это представление сосредоточено на окружениях Dynatrace и развёрнутых расширениях. Текущее подключённое окружение Dynatrace выделено зелёным, остальные синим.

В этом представлении доступны следующие действия:

* Нажмите кнопку «плюс», чтобы зарегистрировать новое окружение.
* Нажмите кнопку обновления, чтобы обновить список.

Элементы верхнего уровня этого списка: зарегистрированные окружения Dynatrace. Для каждого из них доступны следующие действия:

* Нажмите кнопку карандаша, чтобы изменить любые сведения.
* Нажмите кнопку корзины, чтобы удалить это окружение.
* Выберите метку, чтобы развернуть список развёрнутых расширений.

Дочерние элементы окружения: его развёрнутые расширения. Выберите любое расширение, чтобы развернуть список дальше и отобразить конфигурации мониторинга расширения (статус указан рядом с именем). Доступны следующие действия:

* Нажмите кнопку «плюс», чтобы создать новую конфигурацию мониторинга.
* Нажмите кнопку карандаша, чтобы внести изменения в конфигурацию.
* Нажмите кнопку корзины, чтобы удалить конфигурацию.
* Нажмите кнопку файла, чтобы сохранить эту конфигурацию в файл. Файл будет сохранён в папке `./config` рабочего пространства.
