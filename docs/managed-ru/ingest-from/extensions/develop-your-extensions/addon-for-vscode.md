# Дополнение для VS Code

# Дополнение для VS Code

* Пояснение
* Чтение: 2 мин
* Опубликовано 16 июня 2025 г.

**Dynatrace Extensions** дополнение для Visual Studio Code, поддерживающее все аспекты разработки Extensions для Dynatrace. Найти его можно в [магазине расширений VS Code](https://marketplace.visualstudio.com/items?itemName=DynatracePlatformExtensions.dynatrace-extensions).

Оно предоставляет доступ к специализированному набору инструментов, готовому к использованию, и помогает решать следующие задачи:

1. Операционная эффективность
2. Создание контента
3. Проверка контента

Ознакомьтесь с инструкциями по [началу работы](/managed/ingest-from/extensions/develop-your-extensions/addon-for-vscode/getting-started "Настройте редактор Visual Studio Code и соберите и загрузите первое расширение в Dynatrace за 5 минут.") или продолжайте читать, чтобы узнать о некоторых функциях.

## Функции

Ниже перечислены некоторые функции **Dynatrace Extensions**.

### Операционная эффективность

**Dynatrace Extensions** повышает операционную эффективность при разработке Extensions. В него входят следующие функции:

* Управление проектами Extension 2.0 и их организация в масштабе.
* Обзор развёртываний в нескольких окружениях.
* Все операции, связанные с расширениями, доступны прямо в редакторе:

  + Создание, сборка, подпись, загрузка и активация расширений.
  + Создание конфигураций мониторинга и управление ими.
  + Создание учётных данных для подписи расширений и управление ими.

### Создание контента

Также возможно автоматически генерировать значительные части манифеста расширения и другого контента. Ниже перечислены доступные функции:

* Генерация [экранов единого анализа](/managed/ingest-from/extend-dynatrace/extend-ui/unified-analysis "Расширение веб-интерфейса Dynatrace с помощью настроенных под сущности страниц единого анализа.") для ваших сущностей.
* Генерация документации, дашбордов и оповещений.
* Выполнение запросов Windows Management Instrumentation (WMI) для автоматического извлечения метрик и измерений.
* Подключение к экспортёрам Prometheus для автоматического извлечения метрик, измерений и метаданных.
* Использование автодополнения кода, когда значения зависят от данных вашего окружения.

### Проверка контента

С помощью Dynatrace Extensions можно проверять манифест расширения на раннем этапе. Ниже перечислены доступные функции:

* Проверка соответствия целевым версиям схемы YAML.
* Получение пользовательской диагностики, выходящей за рамки схемы.
* Проверка селекторов метрик и сущностей по данным окружения.

## Поддержка

Этот проект с открытым исходным кодом основан на обратной связи и вкладе сообщества и не поддерживается Dynatrace официально.
