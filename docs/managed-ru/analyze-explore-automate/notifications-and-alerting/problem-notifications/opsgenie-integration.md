# Отправка уведомлений Dynatrace в Opsgenie

* Чтение: 2 мин
* Обновлено 10 октября 2022 г.

Dynatrace предлагает готовую интеграцию с Opsgenie, которая автоматически передаёт уведомления Dynatrace о проблемах в вашу среду Opsgenie в нескольких регионах.

Чтобы настроить интеграцию уведомлений о проблемах с Opsgenie:

1. Перейдите в **Settings** > **Integration** > **Problem notifications**.
2. Выберите **Add notification**.
3. Выберите **Opsgenie** из доступных типов уведомлений.
4. Настройте уведомление:

   * Введите **Display name** (отображаемое имя) для этой интеграции — произвольное имя, которое будет отображаться в Dynatrace в разделе **Settings** > **Integration** > **Problem notifications** после завершения настройки.
   * **Opsgenie API key**
     Перейдите в свою учётную запись Opsgenie и настройте интеграцию с Dynatrace. Вставьте сгенерированный ключ API Opsgenie в это поле.
   * **Opsgenie region domain**
     Выберите регион Opsgenie, начав вводить конечную точку региона. Если поле оставить пустым, по умолчанию выбирается регион US.
   * **Message**
     Введите текст и заполнители, связанные с проблемой.

     Заполнители

     В разделе **Available placeholders** на странице настройки перечислены заполнители, доступные для этой интеграции. Уведомления о проблемах автоматически заполняют значения заполнителей и включают их в соответствующие инциденты Opsgenie. Инциденты Opsgenie автоматически создаются при обнаружении проблем и закрываются при их устранении. Сведения об инциденте содержат подробную информацию о проблеме, а также прямую ссылку на проблему в Dynatrace.
5. Назначьте [профиль оповещений](/managed/analyze-explore-automate/notifications-and-alerting/alerting-profiles "Узнайте, как создавать профили оповещений и управлять ими.").
6. Выберите **Send test notification**, чтобы убедиться, что интеграция с Opsgenie работает.
7. **Save changes**.

## Уровни приоритета

Проблемы с доступностью Dynatrace сопоставляются с инцидентами Opsgenie P1, проблемы с ошибками — с P2, замедления — с P3 и т.д. Dynatrace также передаёт теги всех компонентов, затронутых проблемой, вместе с каждым инцидентом Opsgenie.

URL проблемы содержит прямую ссылку на среду Dynatrace, отправившую инцидент.

На следующем изображении показана проблема, обнаруженная Dynatrace и переданная в Opsgenie. Оповещение включает:

* теги всех сущностей, затронутых проблемой;
* ссылку на проблему в Dynatrace;
* серьёзность проблемы, обнаруженной Dynatrace, сопоставленную с соответствующим значением приоритета Opsgenie.
