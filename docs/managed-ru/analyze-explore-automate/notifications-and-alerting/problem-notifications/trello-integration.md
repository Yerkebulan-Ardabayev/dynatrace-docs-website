# Отправка уведомлений Dynatrace в Trello

* Чтение: 2 мин
* Обновлено 10 октября 2022 г.

Atlassian Trello предлагает удобный визуальный способ организации проектов независимо от применяемого гибкого процесса. Всё в Trello организовано в виде карточек. Карточки отображают детали задач, историй или ошибок, которые должна решить команда.

Dynatrace предлагает удобную интеграцию с Trello, позволяющую визуально организовать все автоматически обнаруженные инциденты непосредственно на досках Trello. Подключите свою среду мониторинга Dynatrace к доске Trello и напрямую передавайте проблемы, обнаруженные Dynatrace, в указанный список.

Чтобы настроить интеграцию уведомлений о проблемах с Trello:

1. Перейдите в **Settings** > **Integration** > **Problem notifications**.
2. Выберите **Add notification**.
3. Выберите **Trello** из доступных типов уведомлений.
4. Настройте интеграцию:

   * **Display name** — произвольное имя интеграции, которое будет отображаться в Dynatrace в разделе **Settings** > **Integration** > **Problem notifications** после завершения настройки.
   * Введите ключ приложения Trello.

     Ключ приложения Trello

     Войдите в Trello и позвольте ему автоматически сгенерировать [ключ приложения](https://trello.com/app-key), который следует вставить сюда.
   * Введите токен авторизации Trello.
   * Укажите доску Trello, в которую следует помещать карточки с проблемами.
   * Укажите список Trello, в который следует добавлять карточки с новыми проблемами.
   * Укажите список Trello, в который следует перемещать карточки с разрешёнными проблемами.
   * Введите текст карточки и заполнители проблемы, которые должны отображаться на новых карточках.

     Заполнители

     В разделе **Available placeholders** на странице настройки перечислены заполнители, доступные для этой интеграции. Заполнители автоматически заменяются реальными значениями в тексте карточки.
   * Введите описание карточки.
   * Назначьте [профиль оповещений](/managed/analyze-explore-automate/notifications-and-alerting/alerting-profiles "Узнайте, как создавать профили оповещений и управлять ими.").
5. Выберите **Send test notification**, чтобы убедиться, что интеграция с Trello работает. Должно быть отправлено тестовое уведомление в Trello.
6. Если тест прошёл успешно, нажмите **Save changes**.
