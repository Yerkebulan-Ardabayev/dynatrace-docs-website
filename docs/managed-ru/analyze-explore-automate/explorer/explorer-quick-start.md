# Быстрый старт Data Explorer

# Быстрый старт Data Explorer

* Чтение: 6 мин
* Опубликовано 28 октября 2021 г.

Вы попали по адресу, если хотите увидеть возможности Data Explorer и немного попрактиковаться в написании запросов.

Есть два варианта:

* [Начать с шаблона](#templates) — лёгкое введение
* [Начать с нуля](#from-scratch) — немного сложнее

## Начать с шаблона

Начиная с версии Dynatrace 1.251, Data Explorer открывается с разделом **Start with a template**. Это самый быстрый и простой способ начать работу с Data Explorer.

Не бойтесь экспериментировать. Ничего не сломается.

Если нужно начать заново, перейдите на другую страницу и вернитесь в Data Explorer, чтобы снова увидеть раздел **Start with a template**.

Чтобы начать работу с шаблоном

1. Перейдите в **Data Explorer**.
2. В разделе **Start with a template** выберите шаблон.

   При выборе шаблона Dynatrace автоматически заполняет определение запроса, настраивает параметры визуализации и выполняет запрос.
3. Наведите курсор на элементы визуализации, чтобы увидеть подсказки.
4. Нажмите на элементы визуализации для перехода к детализированным представлениям.
5. Поэкспериментируйте с определением запроса. После внесения изменений нажмите **Run query**, чтобы увидеть результат.
6. Поэкспериментируйте с панелью **Settings** справа. Измените параметры и посмотрите, что происходит.
7. Необязательно [Dashboards Classic](/managed/analyze-explore-automate/dashboards-classic "Узнайте, как создавать дашборды Dynatrace Classic, управлять ими и использовать их."): когда вам понравится результат, нажмите **Pin to dashboard**, чтобы добавить запрос на дашборд. Список дашбордов поддерживает поиск: нажмите на список и начните вводить текст для фильтрации, затем выберите дашборд из отфильтрованного списка.

   * Если у вас нет дашборда, просто нажмите **Create new dashboard** при появлении запроса **Where do you want to pin to?**.
   * Можно закрепить несколько версий своей работы на дашборде для сравнения.

![Data Explorer — шаблоны](https://dt-cdn.net/images/sample-templates-section-1285-7b25cb25e5.png)

Data Explorer — шаблоны

Если готовы исследовать больше, попробуйте начать с нуля.

## Начать с нуля

Используйте эту процедуру для практического знакомства с Data Explorer.

В этом руководстве вы:

* Создадите простую визуализацию метрики с нуля
* Используете визуализацию непосредственно в Data Explorer
* Закрепите визуализацию на дашборде как плитку

### Создание визуализации с нуля

1. Перейдите в **Data Explorer**.
2. На вкладке **Build** добавьте метрику в запрос.

   * Выберите метрику, например `CPU usage %` (`builtin:host.cpu.usage`).

     Как выбрать метрику

     При просмотре списка метрик

     + Можно ввести или вставить имя метрики непосредственно в поле, чтобы найти все совпадения. В этом примере несколько совпадений. Выбираем метрику в категории Host.

       ![Data Explorer: выбор метрики: ввод и выбор](https://dt-cdn.net/images/metric-selector-metric-type-471-10a8a83a2e.png)

       Data Explorer: выбор метрики: ввод и выбор

     + Если вы добавили метрики в избранное в [браузере метрик](/managed/analyze-explore-automate/dashboards-classic/metrics-browser "Просматривайте метрики с помощью браузера метрик Dynatrace."), они отображаются вверху списка.

       ![Data Explorer: выбор метрики: избранное](https://dt-cdn.net/images/metric-selector-favorites-475-665c98b195.png)

       Data Explorer: выбор метрики: избранное
     + Можно выбрать категорию метрик для сужения списка.

       ![Data Explorer: выбор метрики: категории](https://dt-cdn.net/images/metric-selector-categories-476-5cbd27551a.png)

       Data Explorer: выбор метрики: категории

     + При наведении курсора на любую метрику в списке боковая панель отображает подробные сведения о ней.

       ![Data Explorer: выбор метрики: подробности метрики](https://dt-cdn.net/images/metric-selector-metric-details-964-cd2f59a371.png)

       Data Explorer: выбор метрики: подробности метрики

       Нажмите **View all metric information**, чтобы открыть [браузер метрик](/managed/analyze-explore-automate/dashboards-classic/metrics-browser "Просматривайте метрики с помощью браузера метрик Dynatrace.") в новой вкладке с подробными сведениями о выбранной метрике.
   * Выберите агрегацию (например, `Average`, `Minimum` или `Maximum`)
   * Выберите измерения для **Split by**. В этом примере (`CPU usage %`) разделяем по хосту для просмотра использования CPU по хостам.
   * Укажите критерии **Filter by** по необходимости. В этом примере оставляем пустым.

   Подробности о построении запроса см. в разделах [Компоненты и концепции запроса](/managed/analyze-explore-automate/explorer#query-components-and-concepts "Выполняйте запросы по метрикам и преобразовывайте результаты для получения нужной информации.") и [Примеры](/managed/analyze-explore-automate/explorer#examples "Выполняйте запросы по метрикам и преобразовывайте результаты для получения нужной информации.").
3. Необязательно Чтобы просмотреть или отредактировать код запроса, включите **Advanced mode** — [расширенный редактор запросов](/managed/analyze-explore-automate/explorer/explorer-advanced-query-editor "Создавайте сложные запросы с помощью Advanced mode Data Explorer.").
4. Добавляйте и удаляйте метрики по необходимости.

   * В запросе может быть до 10 строк.
   * Чтобы добавить новую пустую строку, нажмите **Add metric** и повторите предыдущий шаг для определения этой строки.
   * Чтобы изменить порядок метрик, выберите метрику и перетащите её на новую позицию.

     ![Перетащите метрику для изменения порядка](https://dt-cdn.net/images/data-explorer-drag-metric-69-84144c5cd2.png)

     Перетащите метрику для изменения порядка

     + Метрики запроса отображаются сверху вниз, поэтому последняя располагается поверх остальных.
     + Порядок метрик также обновляется в разделе **Settings** на боковой панели.
     + Запустите запрос повторно, чтобы увидеть изменения.
   * Чтобы скопировать уже добавленную метрику, нажмите значок «…» > **Duplicate** и отредактируйте копию по необходимости.

     ![Меню «…» метрики](https://dt-cdn.net/images/data-explorer-metric-more-menu-74-1b3f17af8b.png)

     Меню «…» метрики
   * Чтобы включить или отключить метрику, нажмите кнопку с глазом.

     ![Data Explorer: включение и отключение метрики](https://dt-cdn.net/images/data-explorer-metric-enable-disable-80-d4980f418d.png)

     Data Explorer: включение и отключение метрики
   * Чтобы удалить метрику, нажмите значок «…» > **Delete**.
5. Необязательно Чтобы добавить или удалить трансформации метрики для строки, нажмите кнопку трансформаций (**+**) и установите или снимите флажки по необходимости.

   ![Кнопка «+» метрики](https://dt-cdn.net/images/data-explorer-metric-plus-button-46-3104fd992d.png)

   Кнопка «+» метрики
6. Нажмите **Run query** для первого просмотра визуализации. Кнопка **Run query** показывает статус отображаемых результатов:

   ![Data Explorer: кнопка Run query: время последнего выполнения](https://dt-cdn.net/images/data-explorer-run-button-last-run-293-a37ea8a71f.png)

   Data Explorer: кнопка Run query: время последнего выполнения

   ![Data Explorer: кнопка Run query: неприменённые изменения](https://dt-cdn.net/images/data-explorer-run-button-unapplied-472-7f10360ebb.png)

   Data Explorer: кнопка Run query: неприменённые изменения
7. Используйте панель **Settings** для настройки визуализации.

   * Визуализация обновляется при каждом изменении.
   * Выбирайте из нескольких типов визуализации:

     + [График](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/visualization-graph "Настройте и используйте визуализацию «График» в Data Explorer.")
     + [Накопленная столбчатая диаграмма](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/visualization-stacked-column "Настройте и используйте визуализацию «Накопленная столбчатая диаграмма» в Data Explorer.")
     + [Накопленная диаграмма с областями](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/visualization-stacked-area "Настройте и используйте визуализацию «Накопленная диаграмма с областями» в Data Explorer.")
     + [Круговая диаграмма](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/visualization-pie "Настройте и используйте визуализацию «Круговая диаграмма» в Data Explorer.")
     + [Одиночное значение](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/visualization-single-value "Настройте и используйте визуализацию «Одиночное значение» в Data Explorer.")
     + [Таблица](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/visualization-table "Настройте и используйте табличную визуализацию в Data Explorer.")
     + [Список лидеров](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/visualization-top-list "Настройте визуализацию «Список лидеров» в Data Explorer.")
     + [Тепловая карта](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/visualization-heatmap "Настройте и используйте визуализацию «Тепловая карта» в Data Explorer.")
     + [Соты](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/visualization-honeycomb "Настройте и используйте визуализацию «Соты» в Data Explorer.")

Когда визуализация готова, её можно использовать непосредственно в Data Explorer или закрепить на дашборде для дальнейшего использования.

### Использование визуализации в Data Explorer

Элементы визуализации активны. Например:

* Наведите курсор на элементы визуализации, чтобы увидеть подсказки
* Нажмите проблемный (красный) элемент для детализации
* Нажмите соответствующую метку в легенде визуализации, чтобы скрыть или показать элемент

Когда вы закончили работу с визуализацией (хотя бы на время), нужно решить, удалить её или сохранить.

* Чтобы удалить визуализацию (и сгенерировавший её запрос), просто перейдите на другую страницу.
* Чтобы сохранить визуализацию и запрос для дальнейшего использования, нужно закрепить визуализацию на дашборде. См. ниже.

### Закрепление визуализации на дашборде

Чтобы сохранить визуализацию как плитку дашборда, нажмите **Pin to dashboard**.

Чтобы вернуться с дашборда в Data Explorer с открытой визуализацией для просмотра и редактирования, откройте меню в правом верхнем углу плитки и выберите **Configure tile in Data Explorer**. В разделе **Results** Data Explorer появятся две кнопки:

* **Save changes to dashboard** — сохраняет визуализацию на той же плитке и дашборде, с которого был открыт Data Explorer. При наличии изменений они обновят плитку на дашборде.
* **Pin to dashboard** — сохраняет визуализацию как плитку на другом дашборде. Может потребоваться для закрепления одной и той же визуализации (возможно, с разными фильтрами) на нескольких дашбордах.

Подробности о закреплении плиток на дашбордах см. в разделе [Закрепление плиток на дашборде](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/pin-tiles-to-your-dashboard "Узнайте, как закреплять плитки на дашбордах.").
