## Конкретные вопросы

### Сколько стоит Log Monitoring?

Ценообразование Log Monitoring основано на модели Davis data units (DDU). Ознакомьтесь с разделом [DDU для Log Monitoring Classic](/managed/license/monitoring-consumption-classic/davis-data-units/log-monitoring-consumption "Узнайте, как рассчитывается объём потребления DDU для Dynatrace Log Monitoring Classic.") для подробностей о расчёте потребления DDU.

### Кто может увеличить лимит приёма на тенант?

Обратитесь к специалисту по продуктам Dynatrace через чат в реальном времени в вашем окружении.
