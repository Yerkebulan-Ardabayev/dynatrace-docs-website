# Предупреждения при приёме журналов (Logs Classic)

* Обзор
* Чтение: 2 мин
* Опубликовано 10 октября 2023 г.

Log Monitoring Classic

Если принятые журналы выглядят не так, как ожидалось, можно проверить, содержит ли конкретная запись журнала предупреждения о проблемах, возникших в пайплайне приёма и обработки журналов. Найдите атрибут `dt.ingest.warnings` в просмотрщике журналов. Он перечисляет предупреждения о проблемах, затронувших конкретную запись журнала.

Примеры возможных предупреждений:

| Предупреждение | Описание |
| --- | --- |
| content\_trimmed | Содержимое было обрезано после получения API, так как превысило максимальный размер содержимого события. |
| content\_trimmed\_pipe | Содержимое было обрезано после применения правил обработки, так как превысило максимальный размер содержимого события. |
| attr\_count\_trimmed | Количество атрибутов было обрезано после получения API, так как превысило максимальное количество атрибутов в одном событии. |
| attr\_count\_trimmed\_pipe | Количество атрибутов было обрезано после применения правил обработки, так как превысило максимальное количество атрибутов в одном событии. |
| attr\_key\_trimmed | Хотя бы один ключ атрибута был обрезан, так как превысил максимальный размер ключа. |
| attr\_val\_count\_trimmed | Хотя бы у одного многозначного атрибута количество значений было обрезано после получения API. |
| attr\_val\_count\_trimmed\_pipe | После применения правил обработки хотя бы у одного многозначного атрибута количество значений было обрезано. |
| attr\_val\_size\_trimmed | Хотя бы одно значение атрибута было обрезано после получения API, так как превысило максимальный размер значения. |
| attr\_val\_size\_trimmed\_pipe | Хотя бы одно значение атрибута было обрезано после применения правил обработки, так как превысило максимальный размер значения. |
| timestamp\_corrected | Временная метка была слишком далеко в будущем и была скорректирована до текущего времени. |
| common\_attr\_corrected | Хотя бы один из следующих атрибутов был скорректирован: `status`, `loglevel` или `event.type`. |
| processing\_batch\_timeout | Превышение тайм-аута пакета при выполнении правил обработки журналов. |
| processing\_transformer\_timeout | Превышение тайм-аута выполнения в одном из трансформеров обработки при выполнении правил. |
| processing\_transformer\_error | Ошибка выполнения в одном из трансформеров обработки при выполнении правил. |
| processing\_transformer\_throttled | Выполнение ограничено в одном из трансформеров обработки при выполнении правил. |
| processing\_output\_record\_conversion\_error | Ошибка конвертации вывода для некоторых записей при выполнении правил обработки журналов. |
| processing\_prepare\_input\_error | Ошибка подготовки входных данных в одном из включённых правил обработки журналов. |
