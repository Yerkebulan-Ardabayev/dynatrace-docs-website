# Дашборды

* Справочник
* Чтение: 7 мин
* Опубликовано 25 сентября 2018 г.

[Dashboards Classic](/managed/analyze-explore-automate/dashboards-classic "Узнайте, как создавать дашборды Dynatrace Classic, управлять ими и использовать их.")

Каждый дашборд состоит из плиток и визуализаций, которые можно выбирать, настраивать и размещать в соответствии с вашими потребностями. Dynatrace предоставляет множество предварительно настроенных плиток, а также ряд настраиваемых плиток, которые можно кастомизировать для отображения наиболее важных для вас метрик. Выберите любую плитку дашборда, отображающую данные по отслеживаемым объектам среды (хостам, процессам или сервисам), чтобы увидеть список этих объектов.

Пример дашборда

Этот простой пример содержит четыре стандартные плитки.

* Всё в порядке: всё зелёное, ни одного красного элемента, никаких проблем.
* Если бы проблемы были, вы бы увидели предупреждение и могли бы, например, перейти от проблемного (красного) хоста к детальному анализу.

Для более развёрнутых примеров

1. Перейдите в **Dashboards**.
2. Отфильтруйте таблицу по `Owner: Dynatrace`.
3. Откройте несколько готовых предустановленных дашбордов.

![Общий пример дашборда](https://dt-cdn.net/images/dashboard-general-example3-778-3ecb2bdf91.png)

Общий пример дашборда

Вы можете создавать различные дашборды для конкретных людей, ролей и команд, чтобы каждый мог сосредоточиться на том, что важно именно ему.

Основная навигация

1. Перейдите в **Dashboards**.
2. Выберите любой дашборд из списка, чтобы открыть его.
3. Открытый дашборд становится вашим *текущим дашбордом*.

* На плитке работоспособности (например, **Host health**, **Service health** и **Application health**), где отображаются зелёные и красные элементы (хосты, сервисы или приложения), наведите курсор на любой проблемный (красный) элемент, чтобы увидеть его идентификатор.
* Чтобы перейти к проблемному (красному) элементу, выберите красный шестиугольник и нажмите кнопку **View…**. На примере плитки **Synthetic monitor health** выбор красного элемента активирует кнопку перехода **View test** для этого элемента.

  ![Переход к проблемному объекту с плитки работоспособности](https://dt-cdn.net/images/tile-health-bad-drilldown-276-6265617519.png)

  Переход к проблемному объекту с плитки работоспособности
* С здорового (зелёного) элемента переход невозможен. Однако с любой плитки можно выбрать меню в правом верхнем углу плитки и нажать **View details**, чтобы перейти на соответствующую страницу Dynatrace. Например, **View details** с плитки **Synthetic monitor health** откроет таблицу **Synthetic monitors**.

  ![Выбор меню плитки](https://dt-cdn.net/images/tile-health-menu-open-272-17a936feef.png)

  Выбор меню плитки

  Выбор «View details» из меню плитки

* При переходе с плитки вы всегда можете нажать кнопку дашборда

  ![Кнопка дашборда](https://dt-cdn.net/images/home-icon-26-dd12d3ab04.png)

  Кнопка дашборда

  в левом верхнем углу веб-интерфейса Dynatrace, чтобы вернуться к текущему дашборду. Если вы нажмёте кнопку дашборда, находясь уже на дашборде, откроется страница **Dashboards**.
* Вы можете одновременно открывать несколько дашбордов в разных вкладках браузера — по одному дашборду на вкладку.

Частота обновления плиток дашборда

Каждая плитка дашборда автоматически обновляется для актуализации данных. В зависимости от содержимого плитки и применённого временного диапазона интервал обновления может варьироваться.

| Временной диапазон дашборда | Интервал обновления |
| --- | --- |
| Последние 2 часа и менее | 1 минута |
| От 2 до 6 часов | 5 минут |
| От 6 до 24 часов | 15 минут |
| От 24 до 72 часов | 30 минут |
| Более 72 часов | 1 час |

* В средах с высокой нагрузкой Dynatrace может снизить частоту обновления, чтобы предотвратить перегрузку сервера.
* В отличие от других типов плиток, плитка с изображением не обновляется автоматически. Для обновления плиток с изображениями необходимо вручную обновить дашборд.

## Процедуры

Используйте следующие процедуры для создания дашбордов и управления ими.

### Дашборды

Дашборды состоят из плиток, для каждой из которых можно задать собственные настройки временного диапазона, зоны управления и доступа.

Просмотр и отображение дашбордов

1. Перейдите в **Dashboards**.  
   Отобразятся все дашборды, которые вы можете просматривать или редактировать.
2. Чтобы отобразить только ваши дашборды, установите фильтр `Ownership: Mine`.
3. Чтобы открыть дашборд, выберите его имя в списке.

   * Чтобы открыть несколько копий одного дашборда, щёлкните правой кнопкой мыши по заголовку дашборда на странице **Dashboards** и откройте ссылку в новой вкладке или окне. (Либо скопируйте адрес страницы дашборда и вставьте его в адресную строку другой вкладки или окна браузера.)

Подробнее об использовании таблицы **Dashboards** см. в разделе [Организация дашбордов](/managed/analyze-explore-automate/dashboards-classic/dashboards/organize-dashboards "Узнайте, как организовывать дашборды Dynatrace.").

Сортировка дашбордов по популярности

1. Перейдите в **Dashboards**.  
   Отобразятся все дашборды, которые вы можете просматривать или редактировать.
2. Отсортируйте таблицу по столбцу **Popularity**.

Список моих дашбордов

Чтобы отобразить только ваши дашборды

1. Перейдите в **Dashboards**.
2. Установите фильтр одним из следующих способов:

   * В левом столбце в разделе **Ownership** выберите `Mine`
   * В строке **Filter by** выберите `Ownership: Mine`

Возврат к текущему дашборду

![Кнопка дашборда](https://dt-cdn.net/images/dashboardbutton-28-e3cbad6cfe.png)

Кнопка дашборда

Текущий дашборд — это последний посещённый вами дашборд.

* При переходе с дашборда нажмите кнопку дашборда в левом верхнем углу Dynatrace, чтобы вернуться к текущему дашборду.
* Если вы уже находитесь на дашборде, кнопка дашборда откроет обозреватель **Dashboards**.

Создание нового (пустого) дашборда

1. Перейдите в **Dashboards**.
2. Нажмите **Create Dashboard**.
3. Введите название дашборда и нажмите **Create**. Новый дашборд откроется в режиме редактирования.
4. Необязательно Чтобы добавить плитку, перетащите её из панели **Tiles** на дашборд или [закрепите плитку на дашборде](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/pin-tiles-to-your-dashboard "Узнайте, как закреплять плитки на дашбордах.").
5. Необязательно Чтобы настроить параметры дашборда, выберите плитку **Settings**.
6. Нажмите **Done**.  
   Новый дашборд отобразится в том виде, в каком его увидите вы и люди, которым вы [предоставите доступ](/managed/analyze-explore-automate/dashboards-classic/dashboards/share-dashboards "Узнайте, как делиться дашбордами Dynatrace с другими."), однако кнопку **Edit** увидите только вы, если не предоставите другим право на редактирование.

Подробнее о создании дашбордов см. в разделе [Создание и редактирование дашбордов Dynatrace](/managed/analyze-explore-automate/dashboards-classic/dashboards/create-dashboards "Узнайте, как создавать и редактировать дашборды Dynatrace.")

Клонирование существующего дашборда

1. Перейдите в **Dashboards**.
2. В таблице дашбордов для нужного дашборда выберите **More** (**…**) > **Clone**.

   * Копия откроется в режиме редактирования.
   * Исходный дашборд останется без изменений.

Подробнее о создании дашбордов см. в разделе [Создание и редактирование дашбордов Dynatrace](/managed/analyze-explore-automate/dashboards-classic/dashboards/create-dashboards "Узнайте, как создавать и редактировать дашборды Dynatrace.")

Редактирование существующего дашборда

1. Перейдите в **Dashboards**.
2. Выберите имя дашборда, чтобы открыть его.
3. Нажмите **Edit** в правом верхнем углу дашборда. Дашборд откроется в режиме редактирования.

   * Если кнопка **Edit** недоступна, у вас нет прав на редактирование этого дашборда.

Скрытие и отображение дашбордов

Чтобы скрыть дашборд

1. Перейдите в **Dashboards**.
2. В таблице дашбордов для нужного дашборда выберите **More** (**…**) > **Hide**.

Чтобы отобразить скрытый дашборд

1. Перейдите в **Dashboards**.
2. Отфильтруйте таблицу по `Hidden: Yes`, чтобы отобразить все скрытые дашборды.
3. В таблице дашбордов для нужного дашборда выберите **More** (**…**) > **Unhide**.

Удаление дашборда

1. Перейдите в **Dashboards**.
2. В таблице дашбордов для нужного дашборда выберите **More** (**…**) > **Delete**.

   * Если опция **Delete** недоступна, у вас нет прав на удаление этого дашборда.
3. Подтвердите удаление дашборда.

Добавление в избранное

Чтобы добавить текущий дашборд в избранное, выберите звёздочку в правом верхнем углу дашборда. Звёздочка переключает статус избранного.

![Добавление текущего дашборда в избранное](https://dt-cdn.net/images/favorite-current-dashboard-169-caf101b927.png)

Добавление текущего дашборда в избранное

Чтобы добавить несколько дашбордов в избранное

1. Перейдите в **Dashboards**.  
   По умолчанию таблица сортируется по столбцу **Favorite**, поэтому избранные дашборды должны быть в верхней части.
2. В столбце **Favorite** выберите звёздочку для каждого дашборда, который хотите добавить в избранное.

Подробнее об организации дашбордов см. в разделе [Организация дашбордов](/managed/analyze-explore-automate/dashboards-classic/dashboards/organize-dashboards "Узнайте, как организовывать дашборды Dynatrace.").

Удаление из избранного

Чтобы убрать текущий дашборд из избранного, выберите звёздочку в правом верхнем углу дашборда. Звёздочка переключает статус избранного.

![Удаление текущего дашборда из избранного](https://dt-cdn.net/images/unfavorite-current-dashboard-169-97f319d05f.png)

Удаление текущего дашборда из избранного

Чтобы убрать несколько дашбордов из избранного

1. Перейдите в **Dashboards**.  
   По умолчанию таблица сортируется по столбцу **Favorite**, поэтому избранные дашборды должны быть в верхней части.
2. Необязательно Чтобы отобразить только избранные, отфильтруйте таблицу по `Favorite: Yes`.
3. В столбце **Favorite** выберите звёздочку для каждого дашборда, который хотите убрать из избранного.

Подробнее об организации дашбордов см. в разделе [Организация дашбордов](/managed/analyze-explore-automate/dashboards-classic/dashboards/organize-dashboards "Узнайте, как организовывать дашборды Dynatrace.").

Список моих избранных дашбордов

1. Перейдите в **Dashboards**.  
   По умолчанию таблица сортируется по столбцу **Favorite**, поэтому избранные дашборды будут вверху, за ними — остальные дашборды, к которым у вас есть доступ.
2. Необязательно Чтобы отобразить только избранные, установите фильтр одним из следующих способов:

   * В левом столбце в разделе **Favorite** выберите `Yes`
   * В строке **Filter by** выберите `Favorite: Yes`

Подробнее об организации дашбордов см. в разделе [Организация дашбордов](/managed/analyze-explore-automate/dashboards-classic/dashboards/organize-dashboards "Узнайте, как организовывать дашборды Dynatrace.").

Добавление тега к дашборду

1. Откройте дашборд.
2. Нажмите **Edit**.  
   Если кнопка **Edit** недоступна, у вас нет прав на редактирование этого дашборда.
3. Выберите **Add tag** под именем дашборда, введите тег и нажмите **Add**.  
   При необходимости повторите этот шаг для добавления дополнительных тегов.
4. Нажмите **Done**.

Подробнее о тегировании дашбордов см. в разделе [Организация дашбордов](/managed/analyze-explore-automate/dashboards-classic/dashboards/organize-dashboards "Узнайте, как организовывать дашборды Dynatrace.").

Предоставление доступа к дашборду

1. Откройте дашборд, которым хотите поделиться.

   * Чтобы открыть дашборд, перейдите в **Dashboards** и выберите его имя.
2. Выберите **More** (**…**) > **Share** в правом верхнем углу дашборда.
3. Нажмите **Advanced settings**, чтобы открыть **Dashboard settings** на вкладке **Manage access**.

Подробнее о параметрах общего доступа к дашборду см. в разделе [Общий доступ к дашбордам](/managed/analyze-explore-automate/dashboards-classic/dashboards/share-dashboards "Узнайте, как делиться дашбордами Dynatrace с другими.").

Подписка на отчёты по дашборду

1. Откройте дашборд.
2. Выберите **More** (**…**) > **Subscribe** в правом верхнем углу дашборда.

   * Если опция **Subscribe** недоступна, владельцы дашборда не включили отчёты для него.
   * Если отображается кнопка **Enable reports**, у вас есть права на редактирование этого дашборда и необходимо включить отчёты для получения рассылки.
3. Выберите частоту получения отчётов по дашборду по электронной почте (включите **Weekly** или **Monthly**, либо оба варианта).

Подробнее о подписке на отчёты по дашборду см. в разделе [Подписка на отчёты по дашборду](/managed/analyze-explore-automate/dashboards-classic/dashboards/subscribe-to-dashboard-reports "Узнайте, как подписаться на отчёты по дашбордам Dynatrace.").

Установка временного диапазона для дашборда

1. Перейдите на вкладку **Settings**.
2. Включите **Default timeframe**.
3. Выберите временной диапазон, который будет использоваться по умолчанию для этого дашборда.

Обзор временных диапазонов и зон управления на дашбордах см. в разделе [Параметры временного диапазона и зоны управления дашборда Dynatrace](/managed/analyze-explore-automate/dashboards-classic/dashboards/dashboard-timeframe "Узнайте о параметрах временного диапазона и зоны управления дашборда Dynatrace.").

Установка зоны управления для дашборда

1. Перейдите на вкладку **Settings**.
2. Включите **Default management zone**.
3. Выберите зону управления, которая будет использоваться по умолчанию для этого дашборда.

* Обзор временных диапазонов и зон управления на дашбордах см. в разделе [Параметры временного диапазона и зоны управления дашборда Dynatrace](/managed/analyze-explore-automate/dashboards-classic/dashboards/dashboard-timeframe "Узнайте о параметрах временного диапазона и зоны управления дашборда Dynatrace.")
* Подробнее об использовании зон управления для организации дашбордов см. в разделе [Организация дашбордов Dynatrace](/managed/analyze-explore-automate/dashboards-classic/dashboards/organize-dashboards "Узнайте, как организовывать дашборды Dynatrace.")

Настройка дополнительных параметров

1. Откройте **Dashboard settings**.

   * На странице **Dashboards** найдите дашборд и выберите **More** (**…**) > **Configure**
   * На странице дашборда нажмите **Edit**, перейдите на вкладку **Settings** и нажмите **Configure more**.

Печать дашборда в PDF-файл

Печать дашборда в PDF поддерживается в браузерах на основе Chromium (Chrome, Edge, Opera) и Safari. Печать из Firefox пока не поддерживается.

1. Откройте дашборд, который нужно напечатать в PDF.

   * Чтобы открыть дашборд, перейдите в **Dashboards** и выберите его имя.
2. Выберите  > **Print PDF** в правом верхнем углу дашборда.
3. Настройте параметры по необходимости.

   * **Destination:** Выберите **Save as PDF**.
   * **Options:** Выберите **Background graphics**.
   * **Pages:** Если генерируется пустая страница, установите **Pages** в `Custom` и введите `1` для диапазона страниц.
4. Нажмите **Save**.
5. Выберите имя и место сохранения PDF-файла.

Экспорт и импорт дашбордов

Чтобы экспортировать дашборд

1. Перейдите в **Dashboards**.
2. В таблице дашбордов для нужного дашборда выберите **More** (**…**) > **Export**.  
   Определение дашборда будет экспортировано в виде JSON-файла на ваш компьютер.

Чтобы импортировать дашборд

1. Перейдите в **Dashboards**.
2. Нажмите **Import dashboard**.
3. Выберите JSON-файл дашборда, который нужно импортировать.  
   Импортированный дашборд откроется в режиме редактирования.

Подробнее см. в разделе [Редактирование JSON дашборда Dynatrace](/managed/analyze-explore-automate/dashboards-classic/dashboards/dashboard-json "Узнайте, как экспортировать, редактировать и импортировать JSON дашборда Dynatrace.").

### Плитки

Плитки — это компоненты дашбордов.

Список доступных плиток

1. Перейдите в **Dashboards**.
2. Выберите имя дашборда, чтобы открыть его.
3. Нажмите **Edit** в правом верхнем углу дашборда. Дашборд откроется в режиме редактирования.

   * Если кнопка **Edit** недоступна, у вас нет прав на редактирование этого дашборда.

Прокрутите панель **Tiles**, чтобы увидеть все доступные типы плиток.

Подробнее о доступных плитках см. в разделе [Доступные плитки](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/available-tiles "Узнайте, как настроить дашборд для отслеживания бизнес-критичных действий пользователей и целей конверсии.").

Создание плитки с визуализацией метрики

1. Перейдите в **Data Explorer**.
2. Выберите метрики.
3. Нажмите **Run query**, чтобы проверить запрос и просмотреть параметры отображения.
4. Выберите настройки диаграммы.
5. Нажмите **Pin to dashboard**.
6. Выберите дашборд, для которого у вас есть права на редактирование.
7. Нажмите **Pin**.

Добавление URL-адреса изображения в список разрешённых

Чтобы добавить изображение на дашборд по URL, сначала необходимо добавить URL изображения в список разрешённых.

1. Перейдите в **Settings** и выберите **Dashboards** > **Allowed URL pattern rules**.
2. Нажмите **Add item**.
3. Задайте **Rule** — указывает, как обрабатывать эту запись списка разрешённых.

   * **Starts with** — разрешить любое изображение, URL которого начинается с содержимого поля **Pattern**.
   * **Exact** — разрешить конкретное изображение, URL которого точно совпадает с содержимым поля **Pattern**.
4. Задайте **Pattern**:

   * Чтобы указать начало URL, введите достаточно символов URL, чтобы гарантировать соответствие подходящих изображений.  
     Пример: введите `https://example.com/images/`, чтобы разрешить любое изображение, URL которого начинается с `https://example.com/images/`, например  
     `https://example.com/images/image-x.jpg` и
     `https://example.com/images/my-picture.svg`
   * Чтобы указать точный URL, введите полный URL изображения.  
     Пример: введите `https://example.com/images/my-image-file-name.jpg`, чтобы разрешить только это изображение
5. Нажмите **Save changes**, чтобы добавить указанное правило в список разрешённых.

## API

Управлять дашбордами также можно через [Dashboards API](/managed/dynatrace-api/configuration-api/dashboards-api "Узнайте, как управлять конфигурацией дашборда через классический API конфигурации Dynatrace."):

* [GET all dashboards](/managed/dynatrace-api/configuration-api/dashboards-api/get-all "Просмотр всех дашбордов среды через классический API Dynatrace."): Получить список всех дашбордов вашей среды Dynatrace.
* [GET a dashboard](/managed/dynatrace-api/configuration-api/dashboards-api/get-dashboard "Просмотр дашборда через классический API Dynatrace."): Получить параметры указанного дашборда.
* [POST a dashboard](/managed/dynatrace-api/configuration-api/dashboards-api/post-dashboard "Создание дашборда через классический API Dynatrace."): Создать новый дашборд.
* [PUT a dashboard](/managed/dynatrace-api/configuration-api/dashboards-api/put-dashboard "Редактирование дашборда через классический API Dynatrace."): Обновить указанный дашборд.
* [DELETE a dashboard](/managed/dynatrace-api/configuration-api/dashboards-api/del-dashboard "Удаление дашборда через классический API Dynatrace."): Удалить указанный дашборд.
* [GET sharing configuration](/managed/dynatrace-api/configuration-api/dashboards-api/get-sharing-config "Просмотр конфигурации общего доступа к дашборду через классический API Dynatrace."): Получить конфигурацию общего доступа к указанному дашборду.
* [PUT sharing configuration](/managed/dynatrace-api/configuration-api/dashboards-api/put-sharing-config "Обновление конфигурации общего доступа к дашборду через классический API Dynatrace."): Обновить конфигурацию общего доступа к указанному дашборду.
* [Tile JSON models](/managed/dynatrace-api/configuration-api/dashboards-api/dashboards-api-tile-models "Изучите варианты JSON-моделей плиток в API Dynatrace Dashboards Classic."): JSON-модели плиток дашборда существенно различаются в зависимости от типа плитки. Ознакомьтесь с этой страницей, чтобы увидеть JSON-модели для каждого типа плитки.
