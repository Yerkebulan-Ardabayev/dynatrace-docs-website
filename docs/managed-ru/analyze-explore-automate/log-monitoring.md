# Мониторинг логов

* Чтение: 2 мин
* Обновлено 20 ноября 2025 г.

Благодаря мониторингу логов в составе платформы Dynatrace вы получаете прямой доступ к содержимому логов всех критически важных приложений, инфраструктуры и облачных платформ. Вы можете создавать пользовательские метрики логов для более быстрого и умного устранения неполадок, а также анализировать данные логов в контексте всего стека, включая влияние на реальных пользователей.

Определяйте шаблоны, события и пользовательские метрики логов для получения проактивных уведомлений.](/managed/analyze-explore-automate/log-monitoring/alert-log-data "Создавайте оповещения на основе данных логов в Dynatrace Log Monitoring.")[### API

Используйте Dynatrace API для отправки данных логов в Dynatrace, а также для быстрого поиска, агрегации и экспорта содержимого логов.](/managed/dynatrace-api/environment-api/log-monitoring-v2 "Узнайте о возможностях Log Monitoring API v2.")
