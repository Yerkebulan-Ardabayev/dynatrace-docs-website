# Предустановленные дашборды Dynatrace

* Практическое руководство
* Чтение: 4 мин
* Опубликовано 4 февраля 2021 г.

[Dashboards Classic](/managed/analyze-explore-automate/dashboards-classic "Узнайте, как создавать дашборды Dynatrace Classic, управлять ими и использовать их.")

Предустановленные дашборды по умолчанию видны всем пользователям.

* На странице **Dashboards** после имени каждого предустановленного дашборда указан тег `Preset`.
* Чтобы отображать только предустановленные дашборды, установите фильтр `Preset` в значение `Yes`.

Dynatrace предлагает несколько готовых предустановленных дашбордов для различных предметных областей. Используйте их как источник вдохновения для собственных дашбордов и клонируйте для создания пользовательских версий.

* Вы можете создавать собственные предустановленные дашборды.
* Вы можете назначить любой предустановленный дашборд домашним для группы пользователей.

## Управление глобальными параметрами

Предустановленные дашборды по умолчанию видны всем пользователям. С помощью глобальных параметров их можно полностью отключить или ограничить видимость для определённых групп пользователей.

### Включение предустановленных дашбордов

Используйте **Enable presets**, чтобы глобально включить или отключить предустановленные дашборды. При отключении дашборды, помеченные как предустановленные, перестанут отображаться в таблице **Dashboard** для всех пользователей.

### Ограничение видимости предустановленных дашбордов

Используйте правила предустановок, чтобы обеспечить видимость конкретных предустановленных дашбордов только для определённых групп пользователей, а не для всех пользователей среды.

1. Перейдите в **Settings** и выберите **Dashboards** > **Preset settings**.
2. В разделе **Limit preset visibility** нажмите **Add item**.

   * Установите **Preset dashboard** — предустановленный дашборд, для которого нужно управлять доступом группы.
   * Установите **User group** — группу пользователей, которая должна иметь доступ к выбранному предустановленному дашборду.
3. Нажмите **Save changes**.

## Просмотр всех предустановленных дашбордов

Чтобы отобразить все предустановленные дашборды

1. Перейдите в **Dashboards**.
2. Установите фильтр одним из следующих способов:

   * В левом столбце под **Preset** выберите `Yes`
   * На строке **Filter by** выберите `Preset: Yes`
3. Выберите имя любого предустановленного дашборда, чтобы открыть его.

## Готовые предустановленные дашборды

Готовые предустановленные дашборды нельзя редактировать, но их можно клонировать и редактировать клоны.

### Клонирование готового предустановленного дашборда

Используйте **Clone**, чтобы создать личную редактируемую копию дашборда — к имени дашборда будет добавлен суффикс `-cloned`.

1. Перейдите в **Dashboards**.
2. Найдите готовый предустановленный дашборд, который нужно клонировать.
3. Используйте один из следующих способов:

   * Нажмите **More** (**…**) > **Clone** для нужного дашборда.
   * Откройте дашборд, нажмите **More** (**…**) в правом верхнем углу и выберите **Clone**.

Для клонирования текущего дашборда нажмите **More** (**…**) > **Clone**.

### Просмотр и редактирование клона

Получив копию дашборда, вы можете экспериментировать с ней.

1. Нажмите **Edit**, чтобы просмотреть и отредактировать компоненты дашборда.

   * **Name** — нажмите значок редактирования рядом с именем дашборда, чтобы изменить имя клона.
   * **Tags** — под именем дашборда добавляйте и удаляйте теги по необходимости.
   * Плитки **Markdown** и **Header** — выберите их, чтобы узнать, как использовать текст и заголовки для разметки и пояснений на дашборде.
   * Остальные плитки — выберите плитку, чтобы просмотреть параметры в специальной панели справа. Помощь по конкретным плиткам см. в разделе [Доступные плитки](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/available-tiles "Узнайте, как настроить дашборд для отслеживания критически важных действий пользователей и целей конверсии.").
2. Нажмите **Done**, чтобы сохранить изменения и отобразить рабочий дашборд.
3. Используйте детализацию из плиток, чтобы изучить, куда они ведут.

   * Откройте меню плитки, чтобы просмотреть доступные пункты.
   * Нажмите на элементы плитки, чтобы увидеть доступные действия.

## Создание предустановленного дашборда

Вы можете создавать и изменять собственные предустановленные дашборды.

* Предустановленные дашборды автоматически предоставляются всем пользователям
* Предустановленные дашборды отображаются в таблице **Dashboards** для всех пользователей

Необходимо одно из следующих разрешений:

* Общесредовое разрешение **Change monitoring settings**
* Роль `ALLOW environment:roles:manage-settings`

Чтобы обозначить дашборд как предустановленный

1. [Создайте дашборд](/managed/analyze-explore-automate/dashboards-classic/dashboards/create-dashboards "Узнайте, как создавать и редактировать дашборды Dynatrace.") или откройте существующий дашборд, на редактирование которого у вас есть права.
2. Нажмите **Edit**.
3. Перейдите на вкладку **Settings** и нажмите **Configure more**.
4. На вкладке **Manage access** включите **Publish as preset**.
5. Нажмите **Save changes**.

Чтобы проверить изменение

1. Перейдите в **Dashboards**.
2. Отфильтруйте по `Preset: Yes`.
3. Убедитесь, что дашборд отображается в таблице.  
   Кроме того, у каждого предустановленного дашборда в таблице после имени указан тег `Preset`.

## Назначение домашнего дашборда

При наличии прав администратора можно назначить предустановленный дашборд домашним для группы пользователей. Выбранный дашборд станет стартовой страницей по умолчанию для этой группы.

1. Перейдите в **Settings** > **Dashboards** > **General settings**.
2. Нажмите **Configure home dashboard**.
3. Установите **User group** — группу, для которой нужно задать домашний дашборд.
4. Установите **Home dashboard** — один из предустановленных дашбордов из списка.  
   Если ваш дашборд отсутствует в списке, убедитесь, что он помечен как предустановленный.
5. Нажмите **Save changes**.
