# Редактирование JSON дашборда Dynatrace

* Практическое руководство
* Чтение: 3 мин
* Опубликовано 24 сентября 2020 г.

[Dashboards Classic](/managed/analyze-explore-automate/dashboards-classic "Узнайте, как создавать дашборды Dynatrace Classic, управлять ими и использовать их.")

Можно редактировать JSON-определение дашборда в автономном режиме или, для небольших изменений, непосредственно в Dynatrace.

Альтернатива через API

Для управления JSON дашборда в масштабе используйте [Dashboards API](/managed/dynatrace-api/configuration-api/dashboards-api "Узнайте, как управлять конфигурацией дашборда через классический API конфигурации Dynatrace.").

## Редактирование в автономном режиме

Используйте эту процедуру для загрузки JSON-определения дашборда, редактирования в автономном режиме и повторной загрузки в Dynatrace.

При загрузке дашборда по этой процедуре вы перезапишете дашборд, чьё определение изначально скачали. Для загрузки определения в новый дашборд см. раздел [Импорт дашборда](#import-dashboard) ниже.

1. Откройте дашборд.
2. Нажмите **Edit**.
3. Перейдите на вкладку **Settings** и нажмите **Configure more**.
4. Выберите **Dashboard JSON**.
5. На странице **Dashboard JSON** нажмите **Download**.  
   JSON-файл с именем дашборда будет загружен на ваш компьютер.
6. Отредактируйте JSON в предпочтительной среде разработки.  
   Подробности синтаксиса JSON см. в документации [Dashboards API](/managed/dynatrace-api/configuration-api/dashboards-api "Узнайте, как управлять конфигурацией дашборда через классический API конфигурации Dynatrace.").
7. На странице **Dashboard JSON** нажмите **Upload**, найдите отредактированный файл и загрузите его в Dynatrace.  
   Загруженный JSON отобразится на странице **Dashboard JSON**.
8. Нажмите **Save changes**, чтобы заменить старый JSON отредактированным.
9. Откройте дашборд для проверки изменений.

## Редактирование в Dynatrace

1. Откройте дашборд.
2. Нажмите **Edit**.
3. Перейдите на вкладку **Settings** и нажмите **Configure more**.
4. Выберите **Dashboard JSON**.
5. На странице **Dashboard JSON** JSON дашборда отображается в окне редактирования.
6. Нажмите в отображаемом JSON и начните редактирование.

   * Можно работать непосредственно в окне редактирования или копировать и вставлять текст из другого редактора.
   * Сообщение **You have unsaved changes** в нижнем левом углу страницы напомнит о незавершённой работе. Обязательно сохраните изменения перед переходом на другую страницу.
   * Синтаксис проверяется при каждом сохранении. Можно работать постепенно и использовать **Save changes** для проверки корректности.
   * Подробности синтаксиса JSON см. в документации [Dashboards API](/managed/dynatrace-api/configuration-api/dashboards-api "Узнайте, как управлять конфигурацией дашборда через классический API конфигурации Dynatrace.").
7. По завершении откройте дашборд для проверки изменений.

При импорте дашборда по этой процедуре в Dynatrace добавляется новый дашборд. Для перезаписи существующего дашборда см. раздел [Редактирование в автономном режиме](#edit-offline) выше.

### Шаг 1: Экспорт дашборда

Если нужно начать с существующего определения дашборда

1. Перейдите в **Dashboards**.
2. В таблице дашбордов для нужного дашборда выберите **More** (**…**) > **Export**.  
   Определение дашборда будет экспортировано в виде JSON-файла на ваш компьютер.

### Шаг 2: Редактирование JSON-файла дашборда

Отредактируйте JSON дашборда в предпочтительной среде разработки. Подробности синтаксиса JSON см. в документации [Dashboards API](/managed/dynatrace-api/configuration-api/dashboards-api "Узнайте, как управлять конфигурацией дашборда через классический API конфигурации Dynatrace.").

### Шаг 3: Импорт дашборда

1. Перейдите в **Dashboards**.
2. Нажмите **Import dashboard**.
3. Выберите JSON-файл дашборда, который нужно импортировать.  
   Импортированный дашборд откроется в режиме редактирования.
