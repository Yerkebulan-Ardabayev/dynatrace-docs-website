# Создание и редактирование дашбордов Dynatrace

* Практическое руководство
* Чтение: 2 мин
* Опубликовано 19 июля 2017 г.

[Dashboards Classic](/managed/analyze-explore-automate/dashboards-classic "Узнайте, как создавать дашборды Dynatrace Classic, управлять ими и использовать их.")

Для создания, редактирования или удаления классического дашборда перейдите в **Dashboards**.

## Создание дашборда

Для создания дашборда можно начать с нового (пустого) дашборда или клонировать существующий и затем отредактировать клон.

### Создание пустого дашборда

1. Перейдите в **Dashboards**.
2. Нажмите **Create Dashboard**.
3. Введите название дашборда и нажмите **Create**. Новый дашборд откроется в режиме редактирования.
4. Необязательно Чтобы добавить плитку, перетащите её из панели **Tiles** на дашборд или [закрепите плитку на дашборде](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/pin-tiles-to-your-dashboard "Узнайте, как закреплять плитки на дашбордах.").
5. Необязательно Чтобы настроить параметры дашборда, выберите плитку **Settings**.
6. Нажмите **Done**.  
   Новый дашборд отобразится в том виде, в каком его увидите вы и люди, которым вы [предоставите доступ](/managed/analyze-explore-automate/dashboards-classic/dashboards/share-dashboards "Узнайте, как делиться дашбордами Dynatrace с другими."), однако кнопку **Edit** другие пользователи не увидят, если вы не предоставите им право на редактирование.

Теперь можно [редактировать дашборд](#add-edit-delete-tiles): добавлять, редактировать и удалять плитки.

### Клонирование существующего дашборда

Дашборды с правами только на просмотр нельзя настраивать или [предоставлять к ним доступ](/managed/analyze-explore-automate/dashboards-classic/dashboards/share-dashboards "Узнайте, как делиться дашбордами Dynatrace с другими."), но их можно клонировать, а затем изменять и делиться клонами.

Клонируйте чужой дашборд, чтобы сэкономить время.

Клонируйте предустановленный дашборд Dynatrace для начала работы с более детальным примером.

1. Перейдите в **Dashboards**.
2. В таблице дашбордов для нужного дашборда выберите **More** (**…**) > **Clone**.

   * Копия откроется в режиме редактирования.
   * Исходный дашборд останется без изменений.

Отредактируйте клонированный дашборд.

## Редактирование дашборда

Чтобы отредактировать дашборд

1. Перейдите в **Dashboards**.
2. Выберите имя дашборда, чтобы открыть его.
3. Нажмите **Edit** в правом верхнем углу дашборда. Дашборд откроется в режиме редактирования.

   * Если кнопка **Edit** недоступна, у вас нет прав на редактирование этого дашборда.

* Используйте вкладку **Tiles** для изменения содержимого дашборда. Можно добавлять, редактировать и удалять плитки. При редактировании плитки изменения конфигурации мгновенно отражаются на плитке.
* Используйте вкладку **Settings** для изменения временного диапазона или зоны управления дашборда, а также настройки [подписки на отчёты](/managed/analyze-explore-automate/dashboards-classic/dashboards/subscribe-to-dashboard-reports "Узнайте, как подписаться на отчёты по дашбордам Dynatrace.").

## Добавление плитки

1. Перейдите в **Dashboards**.
2. Выберите имя дашборда, чтобы открыть его.
3. Нажмите **Edit** в правом верхнем углу дашборда. Дашборд откроется в режиме редактирования.

   * Если кнопка **Edit** недоступна, у вас нет прав на редактирование этого дашборда.
4. Перетащите тип плитки из панели **Tiles** на дашборд.  
   Чтобы просмотреть описание плитки, перетащите её на дашборд и прочитайте описание на панели настройки.

## Редактирование плитки

1. Откройте дашборд.
2. Наведите курсор на плитку, выберите меню плитки в правом верхнем углу и нажмите **Edit tile**.  
   Отобразятся параметры настройки плитки.
3. Измените настройки по необходимости.  
   Чтобы убедиться в корректности изменений, следите за плиткой. Изменения конфигурации отображаются в реальном времени.
4. По завершении нажмите **Done**.

## Удаление плитки

1. Перейдите в **Dashboards**.
2. Выберите имя дашборда, чтобы открыть его.
3. Нажмите **Edit** в правом верхнем углу дашборда. Дашборд откроется в режиме редактирования.

   * Если кнопка **Edit** недоступна, у вас нет прав на редактирование этого дашборда.
4. Выберите ненужную плитку.
5. Нажмите **Delete** и подтвердите действие.

   * Чтобы удалить несколько плиток, обведите их прямоугольником выделения, нажмите **Delete x tiles** и подтвердите действие.

## Настройка параметров дашборда

Для настройки основных параметров дашборда перейдите в режим редактирования и на панели **Edit dashboard** выберите вкладку **Settings**.

### Фильтры по умолчанию

Можно задать временной диапазон и зону управления по умолчанию для дашборда. Подробнее см. в разделе [Параметры временного диапазона и зоны управления дашборда Dynatrace](/managed/analyze-explore-automate/dashboards-classic/dashboards/dashboard-timeframe "Узнайте о параметрах временного диапазона и зоны управления дашборда Dynatrace.").

### Отчёты

При включении отчётов дашборда подписчики будут получать ссылку для просмотра дашборда без входа в систему. Подробнее см. в разделе [Подписка на отчёты по дашборду Dynatrace](/managed/analyze-explore-automate/dashboards-classic/dashboards/subscribe-to-dashboard-reports "Узнайте, как подписаться на отчёты по дашбордам Dynatrace.").

### Размер заголовка

Можно задать размер заголовка (маленький, средний или большой) для всех плиток дашборда.

### Согласованные цвета объектов

Используйте переключатель **Consistent entity colors**, чтобы определить, будет ли один и тот же объект на двух или более плитках дашборда отображаться одним цветом. Обратите внимание, что это применяется только к плиткам, созданным с помощью Data Explorer.

В примере ниже хосты A и B представлены на двух разных диаграммах. **Consistent entity colors** отключён, и цвета не согласованы от диаграммы к диаграмме.

### Дополнительная настройка

Для доступа к подробным параметрам дашборда нажмите **Configure more**.

## Удаление дашборда

Чтобы удалить весь дашборд (а не только плитку)

1. Перейдите в **Dashboards**.
2. В таблице дашбордов для нужного дашборда выберите **More** (**…**) > **Delete**.

   * Если опция **Delete** недоступна, у вас нет прав на удаление этого дашборда.
3. Подтвердите удаление дашборда.
