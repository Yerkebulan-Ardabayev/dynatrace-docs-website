# Подписка на отчёты по дашбордам Dynatrace

* Практическое руководство
* Чтение: 3 мин
* Опубликовано 29 октября 2019 г.

[Dashboards Classic](/managed/analyze-explore-automate/dashboards-classic "Узнайте, как создавать дашборды Dynatrace Classic, управлять ими и использовать их.")

Пользователи с доступом к классическому дашборду могут подписаться на email-отчёты (еженедельные, ежемесячные или оба варианта) по этому дашборду.

## Доступ к отчётам

Отчёты можно просматривать без ввода учётных данных Dynatrace. Это позволяет пересылать письма с отчётами внешним получателям, не имеющим доступа к Dynatrace.

Чтобы разрешить пользователям предоставлять анонимный доступ к дашбордам

1. Перейдите в **Settings** и выберите **Dashboards** > **General settings**.
2. Включите **Allow anonymous access**.

   Этот глобальный параметр (уровня учётной записи) определяет, можно ли открывать дашборды публично. При его отключении попытка анонимного доступа к отчёту дашборда вернёт ошибку 403.
3. Нажмите **Save changes**.

## Содержание письма

Письмо не содержит содержимого отчёта.

Каждое письмо с отчётом содержит публичную ссылку (учётные данные не требуются) на отчёт. Ссылка включает временной диапазон, соответствующий выбранной частоте отчётов (еженедельно или ежемесячно).

Тема письма основана на имени дашборда. Если имя дашборда содержит специальные символы, они экранируются (например, `&` становится `&amp`). Это сделано из соображений безопасности и не может быть изменено.

## Создание и доставка отчётов

Цель — отправить ссылку на утренний отчёт в понедельник: еженедельно (каждый понедельник утром) или ежемесячно (в первый понедельник каждого месяца утром).

* Отчёт формируется в какой-то момент после полуночи, предшествующей этому понедельному утру (00:00). Полночь вычисляется в соответствии с часовым поясом тенанта.
* Письмо со ссылкой на отчёт отправляется в течение последующих шести часов. При сбое отправки выполняется повторная попытка через определённое время.

## Включение отчётов дашборда

Отчёты дашборда отключены по умолчанию. При включении для дашборда пользователи с доступом к нему могут подписаться на отчёты и просматривать их без входа в Dynatrace.

1. Откройте дашборд.
2. Нажмите **Edit**.
3. Выберите вкладку **Settings**.
4. Включите **Enable reports**.
5. Нажмите **Done**.

## Отключение отчётов дашборда

Чтобы прекратить отправку отчётов всем подписчикам дашборда

1. Откройте дашборд.
2. Нажмите **Edit**.
3. Выберите вкладку **Settings**.
4. Отключите **Enable reports**.
5. Нажмите **Done**.

## Подписка на отчёт дашборда

Чтобы подписаться на отчёт дашборда

1. Откройте дашборд.
2. Нажмите **More** (**…**) > **Subscribe**.

   Если отчёты для дашборда отключены, подписаться на него нельзя.

   * Если у вас есть право на редактирование, Dynatrace отобразит кнопку **Enable reports** при попытке подписаться. Нажмите её, чтобы включить отчёты для этого дашборда, и продолжите.
   * Если у вас нет права на редактирование, попросите пользователя с таким правом включить отчёты, либо клонируйте дашборд и подпишитесь на клон.
3. Выберите частоту получения отчётов по email (`Weekly`, `Monthly` или оба варианта).

### Подписка других пользователей на дашборд

Чтобы подписать других людей на дашборд, даже если у них нет доступа к Dynatrace, используйте [Reports API](/managed/dynatrace-api/configuration-api/reports-api "Управление отчётами через API конфигурации Dynatrace.") для указания любого действительного email-адреса в качестве получателя отчётов дашборда.

## Отписка от отчёта дашборда

Чтобы отписаться от отчёта дашборда, нажмите ссылку **unsubscribe**, которая включена в каждое письмо с отчётом.
