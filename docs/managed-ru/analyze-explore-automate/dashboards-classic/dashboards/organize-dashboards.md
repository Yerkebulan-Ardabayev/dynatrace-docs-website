# Организация дашбордов Dynatrace

* Практическое руководство
* Чтение: 3 мин
* Опубликовано 18 января 2019 г.

[Dashboards Classic](/managed/analyze-explore-automate/dashboards-classic "Узнайте, как создавать дашборды Dynatrace Classic, управлять ими и использовать их.")

При большом количестве дашбордов необходимо ими управлять.

## Фильтры в таблице дашбордов

В зависимости от числа пользователей Dynatrace и созданных ими дашбордов страница **Dashboards** (перейдите в **Dashboards**) может содержать длинную таблицу. Необходимо знать, как её фильтровать.

Есть два способа установить фильтры в таблице **Dashboards**:

* Сворачиваемая панель слева от таблицы содержит такие фильтры, как **Ownership** и **Favorite**. Любой выбор добавляет соответствующий фильтр в строку фильтров над таблицей и применяет его.
* Фильтры можно добавлять непосредственно в строку фильтров над таблицей: выберите фильтр в строке фильтров, затем введите или выберите значение. Этот способ необходимо использовать для значений фильтров, требующих ввода текста.

#### Просмотр только своих дашбордов

Чтобы отобразить только свои дашборды

1. Перейдите в **Dashboards**.
2. Установите фильтр одним из следующих способов:

   * В левом столбце под **Ownership** выберите `Mine`
   * На строке **Filter by** выберите `Ownership: Mine`

#### Все фильтры таблицы дашбордов

К таблице **Dashboards** можно применять следующие фильтры:

* **Name**: введите любую часть имени дашборда и нажмите Enter
* **Ownership**: фильтрация по владельцу дашборда

  + `Mine`: все дашборды, которые вы создали
  + `Shared with me`: все дашборды, созданные другими пользователями, которые предоставили вам права на просмотр или редактирование
* **Favorite**: отображает только [избранные](#favorite) дашборды
* **Owner**: фильтрация по имени или идентификатору пользователя владельца
* **Tag**: фильтрация по [тегу дашборда](#tag)
* **Hidden**: установите значение `Yes`, чтобы отобразить скрытые дашборды
* **Preset**: отображать только [предустановленные дашборды](/managed/analyze-explore-automate/dashboards-classic/dashboards/dashboards-preset "Узнайте о готовых дашбордах Dynatrace и о том, как создавать собственные предустановленные дашборды.")

Если задано несколько фильтров, отображаются только дашборды, соответствующие всем условиям.

## Избранное

Избранные дашборды по умолчанию сортируются в начало таблицы **Dashboards**, за которыми следуют остальные.

### Добавление дашборда в избранное

Чтобы добавить в избранное текущий дашборд, нажмите звёздочку в правом верхнем углу дашборда. Звёздочка переключает состояние «в избранном».

![Добавление текущего дашборда в избранное](https://dt-cdn.net/images/favorite-current-dashboard-169-caf101b927.png)

Добавление текущего дашборда в избранное

Чтобы добавить в избранное несколько дашбордов

1. Перейдите в **Dashboards**.  
   По умолчанию таблица сортируется по столбцу **Favorite**, поэтому избранные дашборды должны быть в начале.
2. В столбце **Favorite** нажмите звёздочку для каждого дашборда, который нужно добавить в избранное.

### Удаление дашборда из избранного

Чтобы удалить из избранного текущий дашборд, нажмите звёздочку в правом верхнем углу дашборда. Звёздочка переключает состояние «в избранном».

![Удаление текущего дашборда из избранного](https://dt-cdn.net/images/unfavorite-current-dashboard-169-97f319d05f.png)

Удаление текущего дашборда из избранного

Чтобы удалить из избранного несколько дашбордов

1. Перейдите в **Dashboards**.  
   По умолчанию таблица сортируется по столбцу **Favorite**, поэтому избранные дашборды должны быть в начале.
2. Необязательно Чтобы отобразить только избранные, отфильтруйте таблицу по `Favorite: Yes`.
3. В столбце **Favorite** нажмите звёздочку для каждого дашборда, который нужно удалить из избранного.

### Просмотр избранных дашбордов

1. Перейдите в **Dashboards**.  
   По умолчанию таблица сортируется по столбцу **Favorite**, поэтому избранные дашборды должны быть в начале, за которыми следуют остальные доступные вам.
2. Необязательно Чтобы отобразить только избранные, установите фильтр одним из следующих способов:

   * В левом столбце под **Favorite** выберите `Yes`
   * На строке **Filter by** выберите `Favorite: Yes`

## Популярность

Столбец **Popularity** в таблице **Dashboards** показывает, какие дашборды просматривались чаще всего за последние 30 дней. Каждому дашборду присваивается оценка популярности от 1 (наименее популярный) до 10 (наиболее популярный).

![Популярность дашборда](https://dt-cdn.net/images/relnotes-232-dashboard-popularity-347-ec0f08449a.png)

Популярность дашборда

* Пользователи Dynatrace могут сортировать по **Popularity**, чтобы найти популярные дашборды, которые могут быть им полезны.
* Администраторы Dynatrace могут сортировать по **Popularity**, чтобы выявить редко используемые дашборды — кандидаты на удаление.

## Теги

Теги позволяют организовывать дашборды в группы.

Чтобы добавить теги к дашборду

1. Откройте дашборд.
2. Нажмите **Edit**.  
   Если кнопка **Edit** недоступна, у вас нет прав на редактирование этого дашборда.
3. Нажмите **Add tag** под именем дашборда, введите и выберите тег, затем нажмите **Add**.  
   Повторите этот шаг для добавления дополнительных тегов.
4. Нажмите **Done**, чтобы сохранить изменения.

Список **Dashboards** можно фильтровать по **Tag**.

### Фильтр по универсальному тегу

1. Перейдите на страницу **Dynamic filters** в настройках дашборда.

   Как открыть

   1. Нажмите **Edit** в правом верхнем углу дашборда.

      Если кнопка **Edit** недоступна, у вас нет прав на редактирование этого дашборда.
   2. Выберите вкладку **Settings**.
   3. Нажмите **Configure more**.
   4. Выберите **Dynamic filters**.

      ![Вкладка «Dynamic filters» дашборда](https://dt-cdn.net/images/empty-dynamic-filters-tab-1072-a5cc3d3c6e.png)

      Вкладка «Dynamic filters» дашборда
2. Нажмите **Add filter**.
3. Установите **Filter criteria** в значение `Generic tag key`.
4. Установите **Display name** — уникальное отображаемое имя, идентифицирующее этот универсальный фильтр (например, `Application name`).
5. Установите **Tag key** (например, `appName`).
6. Установите **Get suggestions from** — источник предложений для тегов (например, `Host`).
7. В разделе **Entity types affected by tag** нажмите **Add entity type**, чтобы добавить тип объекта, на который влияет этот тег.

   Этот шаг можно повторять несколько раз.
8. Нажмите **Save changes**.
9. Откройте дашборд.
10. При установке фильтра на дашборде можно применять один и тот же тип фильтра несколько раз в рамках одного универсального тега.

## Зоны управления

[Зоны управления](/managed/manage/identity-access-management/permission-management/management-zones "Узнайте о концепции зон управления, о том, как их определять и как максимально эффективно использовать.") используются для разделения данных мониторинга по принадлежности и ответственности команд. Содержимое дашборда автоматически фильтруется при выборе зоны управления.

Для каждого дашборда можно задать временной диапазон и зону управления по умолчанию — они применяются при каждом открытии дашборда. Они также используются при [предоставлении доступа](/managed/analyze-explore-automate/dashboards-classic/dashboards/share-dashboards "Узнайте, как делиться дашбордами Dynatrace с другими.") к дашборду.

Чтобы настроить временной диапазон и зону управления по умолчанию, перейдите в режим редактирования дашборда, выберите вкладку **Settings** и настройте один или оба параметра:

* **Default timeframe**: включите и выберите временной диапазон для дашборда.
* **Default management zone**: включите и выберите зону управления для дашборда.

Участники команды, просматривающие дашборд зоны без необходимых разрешений, увидят дашборд без содержимого. Они должны выбрать зону управления, к которой у них есть доступ, чтобы данные отобразились.

Подробнее о дашбордах и зонах управления см. в разделе [Параметры временного диапазона и зоны управления дашборда Dynatrace](/managed/analyze-explore-automate/dashboards-classic/dashboards/dashboard-timeframe "Узнайте о параметрах временного диапазона и зоны управления дашборда Dynatrace.").

## Дополнительные операции с дашбордами

Создание нового (пустого) дашборда

1. Перейдите в **Dashboards**.
2. Нажмите **Create Dashboard**.
3. Введите имя дашборда и нажмите **Create**. Новый дашборд откроется в режиме редактирования.
4. Необязательно Чтобы добавить плитку, перетащите её из панели **Tiles** на дашборд или [закрепите плитку на дашборде](/managed/analyze-explore-automate/dashboards-classic/charts-and-tiles/pin-tiles-to-your-dashboard "Узнайте, как закреплять плитки на дашбордах.").
5. Необязательно Чтобы настроить параметры дашборда, выберите плитку **Settings**.
6. Нажмите **Done**.  
   Новый дашборд отобразится в том виде, в каком его увидите вы и люди, которым вы [предоставите доступ](/managed/analyze-explore-automate/dashboards-classic/dashboards/share-dashboards "Узнайте, как делиться дашбордами Dynatrace с другими."), однако кнопку **Edit** другие пользователи не увидят, если вы не предоставите им право на редактирование.

Подробности создания дашбордов см. в разделе [Создание и редактирование дашбордов Dynatrace](/managed/analyze-explore-automate/dashboards-classic/dashboards/create-dashboards "Узнайте, как создавать и редактировать дашборды Dynatrace.").

Клонирование существующего дашборда

1. Перейдите в **Dashboards**.
2. В таблице дашбордов нажмите **More** (**…**) > **Clone** для нужного дашборда.

   * Копия откроется в режиме редактирования.
   * Исходный дашборд останется без изменений.

Подробности создания дашбордов см. в разделе [Создание и редактирование дашбордов Dynatrace](/managed/analyze-explore-automate/dashboards-classic/dashboards/create-dashboards "Узнайте, как создавать и редактировать дашборды Dynatrace.").

Редактирование существующего дашборда

1. Перейдите в **Dashboards**.
2. Выберите имя дашборда, чтобы открыть его.
3. Нажмите **Edit** в правом верхнем углу дашборда. Дашборд откроется в режиме редактирования.

   * Если кнопка **Edit** недоступна, у вас нет прав на редактирование этого дашборда.

Скрытие дашборда

1. Перейдите в **Dashboards**.
2. В таблице дашбордов нажмите **More** (**…**) > **Hide** для нужного дашборда.

Отображение скрытого дашборда

1. Перейдите в **Dashboards**.
2. Отфильтруйте таблицу по `Hidden: Yes`, чтобы отобразить все скрытые дашборды.
3. В таблице дашбордов нажмите **More** (**…**) > **Unhide** для нужного дашборда.

Удаление дашборда

1. Перейдите в **Dashboards**.
2. В таблице дашбордов нажмите **More** (**…**) > **Delete** для нужного дашборда.

   * Если пункт **Delete** недоступен, у вас нет прав на удаление этого дашборда.
3. Подтвердите удаление дашборда.
