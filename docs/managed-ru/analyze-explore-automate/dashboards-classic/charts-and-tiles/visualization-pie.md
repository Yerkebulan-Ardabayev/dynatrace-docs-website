## Смена визуализации

При переключении между визуализациями учтите, что некоторые параметры настройки характерны для конкретной визуализации.

* Если выбрать визуализацию, настроить для неё **Settings**, а затем переключиться на другую визуализацию, часть настроек первой может быть проигнорирована. Значок информации предупредит об этой возможности.

  ![Data Explorer: предупреждение о смене визуализации](https://dt-cdn.net/images/change-visualization-warning-1277-9438aaceea.png)

  Data Explorer: предупреждение о смене визуализации
* При возврате к исходной визуализации может потребоваться повторная настройка некоторых параметров.

## Изменение набора метрик

По умолчанию эта визуализация отображает первую метрику из запроса с несколькими метриками.

Чтобы выбрать другую метрику, выберите букву рядом с нужной метрикой или нажмите значок глаза.

## Настройки

Раздел **Settings** — один из разворачиваемых разделов на правой панели Data Explorer.

![Data Explorer: раздел Settings](https://dt-cdn.net/images/data-explorer-settings-settings-322-7b365c52e5.png)

Data Explorer: раздел Settings

### Показать легенду

Определяет, отображать ли легенду под визуализацией.

Обратите внимание, что легенда интерактивна: можно выбрать элемент легенды, чтобы включить или отключить отображение соответствующего элемента.

### Трансформация свёртки

Трансформация свёртки объединяет список точек данных (временной ряд) в одну точку данных.

* Трансформация свёртки по умолчанию — `Auto`, которая автоматически выбирает наиболее подходящую временну́ю агрегацию.
* Можно вручную переопределить `Auto`: `Last value`, `Average`, `Count`, `Maximum`, `Minimum`, `Sum`, `Median`, `Value`, `Percentile 10th`, `Percentile 75th`, `Percentile 90th`.
* Для просмотра последнего зафиксированного значения метрики выберите `Last value`.

Трансформация свёртки и разрешение

Параметр **Fold transformation** влияет на разрешение.

* Если **Fold transformation** установлен в `Auto` для визуализации `Table`, `Single value`, `Top list` или `Honeycomb`, используется разрешение `Inf` (бесконечность) для обеспечения обратной совместимости.
* Если **Fold transformation** задан в значении, отличном от `Auto`, используется `fold`.

Поскольку все селекторы метрик запрашиваются с использованием одного и того же механизма (`fold` или `Inf`), добавление нового селектора, требующего `fold`, может изменить результаты других.

Чтобы проверить фактический запрос Data Explorer, перейдите в раздел **Result** и выберите > **Copy request**.

## Настройки по метрике

### Переименование метрики

1. В разделе **Settings** выберите для метрики, которую хотите переименовать.
2. Отредактируйте имя и выберите галочку для сохранения.

### Изменение цветовой палитры

Чтобы изменить цветовую палитру для метрики, в разделе **Settings** выберите новую палитру из списка.

![Data Explorer: список цветовых палитр метрик](https://dt-cdn.net/images/metric-color-palette-list-67-fd8f7395d1.png)

Data Explorer: список цветовых палитр метрик

### Единица измерения и формат

Используйте параметры **Unit** и **Format** для определения способа отображения данных.

#### Единица измерения

* `None` = Единица измерения не отображается
* `Auto` = Dynatrace выбирает подходящую единицу
* Другие варианты задают конкретную единицу.
* В **Advanced mode** можно использовать `:setUnit(<unit>)`.

| Обозначение | Множитель | Значение |
| --- | --- | --- |
| k | 10^3 | кило, тысяча |
| M | 10^6 | мега, миллион |
| G | 10^9 | гига, миллиард |
| T | 10^12 | тера, триллион |

#### Формат

* `None` = Без форматирования.
* `Auto` = Dynatrace выбирает подходящий формат.
* Другие варианты: `0`, `0.0`, `0.00`, `0.000`

### Добавление переопределения цвета

Чтобы принудительно задать другой цвет для конкретного ряда

1. В разделе **Settings** нажмите **Add color override**
2. Выберите ряд из списка
3. Выберите цвет переопределения для этого ряда
