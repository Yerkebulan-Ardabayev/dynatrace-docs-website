# Фильтрация плиток

* Практическое руководство
* Чтение: 2 мин
* Опубликовано 19 июля 2017 г.

[Dashboards Classic](/managed/analyze-explore-automate/dashboards-classic "Узнайте, как создавать дашборды Dynatrace Classic, управлять ими и использовать их.")

Dynatrace предоставляет мощные возможности фильтрации, позволяющие создавать дашборды, соответствующие уникальным потребностям мониторинга каждой команды в вашей организации. В этой теме описывается, как создать дашборд и добавить на него отфильтрованную плитку.

## Создание дашборда

1. Перейдите в **Dashboards**.
2. Нажмите **Create dashboard**, введите **Dashboard name** и нажмите **Create**.  
   Дашборд будет создан и открыт в режиме редактирования.
3. Нажмите **Done**.  
   Теперь у вас есть пустой дашборд для работы.

## Создание отфильтрованной плитки

1. Перейдите в **Hosts**.
2. Задайте фильтр для таблицы. В этом примере задайте `Data center` = `Gdańsk, Poland`.

   Существует два способа задать фильтр:

   * Для распространённых фильтров можно выбрать элемент в столбце **Quick filters**. Это подмножество доступных фильтров.
   * В строке **Filtered by** можно задать любой доступный фильтр.

     Как задать фильтр

     Начните вводить текст для выбора фильтра

     ![Выбор фильтра](https://dt-cdn.net/images/filter-01-325-2fe63fb62a.png)

     Выбор фильтра

     затем выберите или введите значение

     ![Выбор значения](https://dt-cdn.net/images/filter-02-404-32504bda2e.png)

     Выбор значения

     чтобы добавить фильтр в строку фильтров

     ![Результирующий фильтр](https://dt-cdn.net/images/filter-03-379-52198a4b6b.png)

     Результирующий фильтр
3. Нажмите **Pin to dashboard**, выберите целевой дашборд и нажмите **Pin**, чтобы создать плитку и закрепить её на выбранном дашборде.
4. Нажмите **Open dashboard**, чтобы открыть дашборд в режиме редактирования с выбранной новой плиткой.
5. Необязательно Измените настройки плитки по необходимости:

   * Показать или скрыть диаграмму плитки
   * Выбрать временной диапазон для плитки
   * Выбрать зону управления для плитки
   * Выбрать среду для плитки
6. Нажмите **Done**.

Теперь у вас есть плитка, отображающая хосты из дата-центра в Гданьске, Польша.

Каждый шестиугольник на плитке представляет хост. Зелёный шестиугольник — исправный хост, красный — хост с проблемой.

* Если хосты добавляются или удаляются из этого дата-центра, плитка обновляется автоматически.
* При изменении состояния хоста меняется цвет соответствующего шестиугольника.
* Чтобы увидеть, какой хост имеет проблему, наведите курсор на красный шестиугольник.
* Чтобы перейти к деталям проблемного хоста, выберите красный шестиугольник и нажмите **View host**.
* Чтобы просмотреть отфильтрованную страницу **Hosts**, из которой была создана плитка, откройте меню в правом верхнем углу плитки и выберите **View details**.
