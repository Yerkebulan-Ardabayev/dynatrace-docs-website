# Закрепление плиток на дашборде

* Практическое руководство
* Чтение: 5 мин
* Updated on May 16, 2022

[Dashboards Classic](/managed/analyze-explore-automate/dashboards-classic "Узнайте, как создавать дашборды Dynatrace Classic, управлять ими и использовать их.")

Новую плитку дашборда можно автоматически создать с любой страницы Dynatrace, на которой есть кнопка **Pin to dashboard**. Для этого достаточно иметь [права на редактирование](/managed/analyze-explore-automate/dashboards-classic/dashboards/share-dashboards "Узнайте, как делиться дашбордами Dynatrace с другими.") дашборда. В этом примере создаётся новый дашборд, поэтому права на редактирование гарантированы.

## Создание дашборда

Начнём с создания пустого дашборда.

1. Перейдите в **Dashboards**.
2. Нажмите **Create dashboard**, введите **Dashboard name** (в этом примере используйте `Filtered Dashboard`) и нажмите **Create**.  
   Дашборд будет создан и открыт в режиме редактирования.
3. Нажмите **Done**.  
   Теперь у вас есть пустой дашборд с именем `Filtered Dashboard`.

## Создание отфильтрованной плитки

Теперь создадим плитку на основе отфильтрованного представления страницы **Hosts** и закрепим её на пустом дашборде.

1. Перейдите в **Hosts**.
2. Задайте фильтр для списка. В этом примере задайте `Data center: Gdańsk, Poland`.

   ![Результирующий фильтр](https://dt-cdn.net/images/filter-03-379-52198a4b6b.png)

   Результирующий фильтр

   Как задать фильтр

   1. В строке **Filtered by** начните вводить текст для выбора фильтра

      ![Выбор фильтра](https://dt-cdn.net/images/filter-01-325-2fe63fb62a.png)

      Выбор фильтра
   2. Выберите или введите значение

      ![Выбор значения](https://dt-cdn.net/images/filter-02-404-32504bda2e.png)

      Выбор значения

   Для наиболее распространённых фильтров можно выбрать фильтр непосредственно из столбца **Quick filters** на странице **Hosts**.
3. Нажмите **Pin to dashboard**.

   Dynatrace отобразит предварительный просмотр плитки и предложит выбрать дашборд для закрепления. Список дашбордов поддерживает поиск: выберите список и начните вводить текст для фильтрации, затем выберите нужный дашборд.

   ![Фильтрация плиток: закрепление](https://dt-cdn.net/images/filtered-tile-01-382-0dbefb8cc5.png)

   Фильтрация плиток: закрепление
4. Выберите дашборд (в этом примере `Filtered Dashboard`) и нажмите **Pin**. Плитка будет создана и закреплена на этом дашборде.

   ![Фильтрация плиток: открытие дашборда](https://dt-cdn.net/images/filtered-tile-02-380-4c83bb3eb7.png)

   Фильтрация плиток: открытие дашборда
5. Нажмите **Open dashboard**.  
   Ранее пустой дашборд откроется в режиме редактирования с выбранной новой плиткой. Перед сохранением можно настроить дополнительные параметры плитки. Для плитки **Host health** доступны следующие параметры:

   * Необязательно Выбрать, показывать ли визуализацию на плитке
   * Необязательно Задать пользовательский временной диапазон
   * Необязательно Задать пользовательскую зону управления
   * Необязательно Выбрать среду

   ![Фильтрация плиток: режим редактирования плитки](https://dt-cdn.net/images/filtered-tile-03-1187-acc59a36d4.png)

   Фильтрация плиток: режим редактирования плитки
6. Нажмите **Done**.  
   Дашборд отобразится с новой плиткой, представляющей хосты в Гданьске, Польша.

   ![Фильтрация плиток: режим просмотра](https://dt-cdn.net/images/filtered-tile-05-1186-285827111d.png)

   Фильтрация плиток: режим просмотра

Теперь у вас есть плитка, отображающая хосты из дата-центра в Гданьске, Польша.

* Фильтр полностью динамический. При добавлении или удалении хостов, а также при изменении их состояния плитка обновляется автоматически.
* Чтобы просмотреть отфильтрованную страницу **Hosts**, из которой была создана плитка, откройте меню в правом верхнем углу плитки и выберите **View details**.

## Добавление заголовка к плитке

Необязательно

Чтобы добавить понятную подпись к плитке

1. Нажмите **Edit** для перехода в режим редактирования дашборда.
2. Перетащите плитку **Header** из панели **Edit dashboard** на дашборд.  
   Это будет заголовок вашей плитки.
3. Отредактируйте текст заголовка. В этом примере используйте **Gdansk hosts**.
4. Перетащите плитку **Header** так, чтобы она располагалась над плиткой **Hosts**.  
   В зависимости от положения плитки **Hosts** на дашборде может потребоваться переместить обе плитки, чтобы разместить **Header** над **Hosts**.

   ![Плитка заголовка — редактирование дашборда](https://dt-cdn.net/images/filtered-tile-08-399-4d36db7eb2.png)

   Плитка заголовка — редактирование дашборда
5. Нажмите **Done**.

   ![Плитка заголовка](https://dt-cdn.net/images/filtered-tile-07-401-f9f22ea2e9.png)

   Плитка заголовка

## Обновление отфильтрованной плитки

Чтобы изменить фильтры плитки

1. На дашборде выберите плитку, чтобы открыть таблицу **Hosts** с применёнными фильтрами.
2. Добавьте или удалите фильтры.

   В этом примере оставьте `Data center: Gdańsk, Poland` и добавьте `Operating system: Linux`. Поскольку фильтры изменились, кнопка **Update dashboard tile** станет активной для сохранения изменений.

   ![Фильтрация плиток: дополнительный фильтр](https://dt-cdn.net/images/filtered-tile-06-1431-274639b9a2.png)

   Фильтрация плиток: дополнительный фильтр
3. Нажмите **Update dashboard tile**.  
   Новые настройки фильтра сохранятся в плитке. В этом примере плитка на дашборде будет отображать состояние хостов, соответствующих `Data center: Gdańsk, Poland` и `Operating system: Linux`, и открывать страницу **Hosts** с этими фильтрами.
4. Чтобы вернуться на дашборд, нажмите кнопку дашборда в левом верхнем углу Dynatrace.

   ![Кнопка дашборда](https://dt-cdn.net/images/dashboardbutton-28-e3cbad6cfe.png)

   Кнопка дашборда

## Закрепление в виде новой плитки

Вместо обновления плитки может потребоваться сохранить её новую копию. Например, чтобы иметь несколько похожих плиток, каждая из которых отфильтрована по другой операционной системе.

1. На том же примере дашборда откройте меню плитки (в правом верхнем углу) и выберите **View details**.  
   Страница **Hosts** откроется с теми же заданными фильтрами.
2. Выберите **More** (**…**) > **Pin as new tile**.

   * Выберите другой дашборд, чтобы разместить копии одной плитки на двух разных дашбордах.
   * Выберите тот же дашборд, чтобы разместить две копии плитки на одном дашборде, что может показаться излишним. Однако:

     + Каждую копию можно настроить на другую зону управления, временной диапазон и среду
     + Для каждой копии можно задать разные фильтры

## Клонирование плиток

Клонирование плиток позволяет сэкономить время. Например, если нужно, чтобы на дашборде отображались три одинаковые визуализации, отличающиеся только запрашиваемым хостом. Проще всего создать первую визуализацию, настроить её параметры и дважды клонировать. После этого останется лишь отредактировать клоны для запроса двух других хостов.

### Клонирование плитки на текущий дашборд

Чтобы закрепить клонированные плитки на том же дашборде

1. Перейдите в **Dashboards**.
2. Выберите имя дашборда, чтобы открыть его.
3. Нажмите **Edit** в правом верхнем углу дашборда. Дашборд откроется в режиме редактирования.

   * Если кнопка **Edit** недоступна, у вас нет прав на редактирование этого дашборда.
4. Выберите плитку для клонирования (добавления копии на текущий дашборд) и нажмите **Clone**.

   * Чтобы клонировать несколько плиток, обведите нужные *x* плиток прямоугольником выделения и нажмите **Clone x tiles**.
5. При необходимости отредактируйте клонированные плитки.

### Клонирование плитки на другой дашборд

Чтобы закрепить клонированные плитки на другом существующем дашборде или создать новый дашборд и закрепить клоны на нём

1. Перейдите в **Dashboards**.
2. Выберите имя дашборда, чтобы открыть его.
3. Нажмите **Edit** в правом верхнем углу дашборда. Дашборд откроется в режиме редактирования.

   * Если кнопка **Edit** недоступна, у вас нет прав на редактирование этого дашборда.
4. Выберите плитку для клонирования на другой дашборд и нажмите **Clone to**.

   * Чтобы клонировать несколько плиток на другой дашборд, обведите нужные *x* плиток прямоугольником выделения и нажмите **Clone x to**.
5. Во всплывающем окне **Where do you want to pin to?** выберите существующий дашборд или нажмите **Create new dashboard**.
6. Нажмите **Pin**, чтобы добавить выбранные плитки на дашборд.
7. При необходимости отредактируйте клонированные плитки.
