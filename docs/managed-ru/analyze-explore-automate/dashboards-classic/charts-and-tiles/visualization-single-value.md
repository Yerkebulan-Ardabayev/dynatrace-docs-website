# Настройка и использование визуализации «Одиночное значение» в Dynatrace

[Dashboards Classic](/managed/analyze-explore-automate/dashboards-classic "Узнайте, как создавать дашборды Dynatrace Classic, управлять ими и использовать их.")

Чтобы визуализировать результаты запроса в виде диаграммы или плитки с одиночным значением, выберите `Single value` из списка над определением запроса в левом верхнем углу страницы.

* Эта визуализация отображает одну метрику в виде диаграммы или плитки с одиночным значением
* Она объединяет все измерения в одну агрегацию.

## Смена визуализации

При переключении между визуализациями учтите, что некоторые параметры характерны для конкретной визуализации.

* Если выбрать визуализацию, настроить для неё **Settings**, а затем переключиться на другую, часть настроек может быть проигнорирована. Значок информации предупредит об этом.

  ![Data Explorer: предупреждение о смене визуализации](https://dt-cdn.net/images/change-visualization-warning-1277-9438aaceea.png)

  Data Explorer: предупреждение о смене визуализации
* При возврате к исходной визуализации может потребоваться повторная настройка некоторых параметров.

## Изменение набора метрик

По умолчанию эта визуализация отображает первую метрику из запроса с несколькими метриками.

Чтобы выбрать другую метрику, выберите букву рядом с нужной метрикой или нажмите значок глаза.

## Настройки

Раздел **Settings** — один из разворачиваемых разделов на правой панели Data Explorer.

![Data Explorer: раздел Settings](https://dt-cdn.net/images/data-explorer-settings-settings-322-7b365c52e5.png)

### Показать тренд

Определяет, отображать ли тренд на плитке (на основе первой и последней точек данных в выбранном временном диапазоне). В этом примере показаны две идентичные плитки: одна без **Show trend**, другая с включённым **Show trend** (тренд снижения).

![Тренд выключен (слева) и включён (справа)](https://dt-cdn.net/images/single-value-trend-off-on-609-98c831bf73.png)

### Показать спарклайн

Определяет, отображать ли спарклайн на плитке. В этом примере показаны две идентичные плитки: одна без **Show sparkline**, другая с включённым **Show sparkline**.

![Спарклайн выключен (слева) и включён (справа)](https://dt-cdn.net/images/single-value-sparkline-off-on-608-c9f1a6cb61.png)

Определяет, отражается ли цвет порога в цвете фона плитки. В этом примере показаны две идентичные плитки: одна без **Threshold background**, другая с включённым **Threshold background**.

### Трансформация свёртки

Трансформация свёртки объединяет список точек данных (временной ряд) в одну точку данных.

* Трансформация по умолчанию — `Auto`, которая автоматически выбирает наиболее подходящую временну́ю агрегацию.
* Можно вручную задать: `Last value`, `Average`, `Count`, `Maximum`, `Minimum`, `Sum`, `Median`, `Value`, `Percentile 10th`, `Percentile 75th`, `Percentile 90th`.
* Для просмотра последнего зафиксированного значения выберите `Last value`.

Трансформация свёртки и разрешение

Параметр **Fold transformation** влияет на разрешение.

* При `Auto` для `Table`, `Single value`, `Top list` или `Honeycomb` используется разрешение `Inf` для обратной совместимости.
* При значении, отличном от `Auto`, используется `fold`.

Чтобы проверить фактический запрос Data Explorer, перейдите в раздел **Result** и выберите > **Copy request**.

## Настройки по метрике

### Переименование метрики

1. В разделе **Settings** выберите для метрики, которую хотите переименовать.
2. Отредактируйте имя и выберите галочку для сохранения.

### Единица измерения и формат

Используйте параметры **Unit** и **Format** для определения способа отображения данных.

#### Единица измерения

* `None` = Единица измерения не отображается
* `Auto` = Dynatrace выбирает подходящую единицу
* Другие варианты задают конкретную единицу.
* В **Advanced mode** можно использовать `:setUnit(<unit>)`.

| Обозначение | Множитель | Значение |
| --- | --- | --- |
| k | 10^3 | кило, тысяча |
| M | 10^6 | мега, миллион |
| G | 10^9 | гига, миллиард |
| T | 10^12 | тера, триллион |

#### Формат

* `None` = Без форматирования.
* `Auto` = Dynatrace выбирает подходящий формат.
* Другие варианты: `0`, `0.0`, `0.00`, `0.000`
