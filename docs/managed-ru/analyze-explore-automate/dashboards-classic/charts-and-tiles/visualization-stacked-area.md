# Настройка и использование визуализации «Накопленная диаграмма с областями» в Dynatrace

Чтобы визуализировать результаты запроса в виде накопленной диаграммы с областями, выберите `Stacked area` из списка над определением запроса в левом верхнем углу страницы.

* Используйте набор метрик для управления тем, какие метрики накапливаются.
* Обратите внимание, что можно накапливать только метрики с одинаковой единицей измерения.

## Смена визуализации

При переключении между визуализациями учтите, что некоторые параметры характерны для конкретной визуализации.

* При переключении на другую визуализацию часть настроек может быть проигнорирована. Значок информации предупредит об этом.

  ![Data Explorer: предупреждение о смене визуализации](https://dt-cdn.net/images/change-visualization-warning-1277-9438aaceea.png)

  Data Explorer: предупреждение о смене визуализации
* При возврате к исходной визуализации может потребоваться повторная настройка некоторых параметров.

## Изменение набора метрик

Визуализация может отображать любой набор метрик в запросе с несколькими метриками.

Чтобы включать или отключать метрики, выберите букву рядом с нужной метрикой или нажмите значок глаза.

![Data Explorer: переключение метрики](https://dt-cdn.net/images/eye-toggle-metric-1238-d343b9cc0e.png)

## Настройки

Раздел **Settings** — один из разворачиваемых разделов на правой панели Data Explorer.

![Data Explorer: раздел Settings](https://dt-cdn.net/images/data-explorer-settings-settings-322-7b365c52e5.png)

Data Explorer: раздел Settings

### Разрешение

Разрешение — это детализация оси X (времени) визуализации.

* Установите **Resolution** в `Auto` для автоматического выбора разрешения.
* Чтобы задать конкретное разрешение, выберите его из списка.

Умное разрешение на плитках дашборда: максимальное количество точек данных — 4 000. При превышении Dynatrace автоматически повышает разрешение.

### Показать легенду

Определяет, отображать ли легенду под визуализацией. Легенда интерактивна.

### Заполнение пробелов

Чтобы заполнить пробелы на диаграмме, в разделе **Settings** включите **Connect gaps**.

## Настройки по метрике

### Переименование метрики

1. В разделе **Settings** выберите для метрики, которую хотите переименовать.
2. Отредактируйте имя и выберите галочку для сохранения.

### Изменение цветовой палитры

Чтобы изменить цветовую палитру, выберите новую палитру из списка.

![Data Explorer: список цветовых палитр метрик](https://dt-cdn.net/images/metric-color-palette-list-67-fd8f7395d1.png)

Data Explorer: список цветовых палитр метрик

### Единица измерения и формат

* **Unit**: `None`, `Auto`, или конкретная единица. В **Advanced mode** — `:setUnit(<unit>)`.
* **Format**: `None`, `Auto`, `0`, `0.0`, `0.00`, `0.000`.

| Обозначение | Множитель | Значение |
| --- | --- | --- |
| k | 10^3 | кило, тысяча |
| M | 10^6 | мега, миллион |
| G | 10^9 | гига, миллиард |
| T | 10^12 | тера, триллион |

### Добавление переопределения цвета

1. В разделе **Settings** нажмите **Add color override**
2. Выберите ряд из списка
3. Выберите цвет переопределения

## Оси

Раздел **Axes** позволяет управлять отображением оси X и осей Y.

![Data Explorer: раздел Axes](https://dt-cdn.net/images/data-explorer-settings-axes-324-238501d94f.png)

Data Explorer: раздел Axes

Пошаговое руководство по настройке осей

1. Перейдите в **Data Explorer**.
2. Добавьте метрику `builtin:host.mem.used` с разбивкой по `Host`, нажмите **Run query**.

   * Ось X — текущий временной диапазон (внизу графика).
   * Ось Y — слева (по умолчанию). Первая автоматически получает метку `Y axis 1`.

   ![Одна ось Y](https://dt-cdn.net/images/data-explorer-axes-1-metric-1614-4bce69f051.png)

   Одна ось Y
3. Нажмите **Add metric**, добавьте `builtin:host.cpu.usage` с разбивкой по `Host`, нажмите **Run query**.

   * Ось Y для второй метрики (`CPU usage %`) — справа.
   * В разделе **Axes** появится `Y axis 2`.

     ![Две оси Y](https://dt-cdn.net/images/data-explorer-axes-2-metrics-1613-5161f95bcd.png)

     Две оси Y

Дополнительные оси Y можно добавить вручную через **Add Y axis** в разделе **Axes**.

![Три оси Y](https://dt-cdn.net/images/data-explorer-axes-3-metrics-1658-3b518c3781.png)

Три оси Y

### Именование, скрытие, положение и диапазон оси

* **Имя**: выберите рядом с именем оси, введите новое имя, нажмите галочку.
* **Скрытие**: нажмите для скрытия/отображения.
* **Положение**: установите **Position** в `Left` или `Right`.
* **Диапазон**: измените **Min, Max** с `Auto, Auto` на числовые значения.
