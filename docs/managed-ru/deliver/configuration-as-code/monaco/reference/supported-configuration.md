# Configuration types and access permissions

# Configuration types and access permissions

* Reference
* 7-min read
* Updated on Feb 26, 2025

Dynatrace Configuration as Code через Monaco (Dynatrace Monaco CLI) поддерживает следующие типы конфигурации:

* [Settings 2.0](/managed/deliver/configuration-as-code/monaco/reference/supported-configuration#settings "Configuration types and access permissions for Dynatrace Configuration as Code via Monaco")
* [Configuration APIs](/managed/deliver/configuration-as-code/monaco/reference/supported-configuration#configs "Configuration types and access permissions for Dynatrace Configuration as Code via Monaco")

Подробности об определении типов конфигурации см. в [YAML-конфигурации Monaco](/managed/deliver/configuration-as-code/monaco/configuration/yaml-configuration#type "Learn how to set up your Monaco YAML configuration.").

## Settings 2.0

Settings 2.0 требует [токена доступа](/managed/dynatrace-api/basics/dynatrace-api-authentication "Find out how to get authenticated to use the Dynatrace API.").

Dynatrace Monaco CLI поддерживает любую схему [Settings 2.0](/managed/manage/settings/settings-20 "Introduction to the Settings 2.0 framework"), доступную в вашем окружении.

* Подробности о схемах Settings 2.0 см. в [Settings 2.0 — доступные схемы](/managed/dynatrace-api/environment-api/settings/schemas "View the entire settings schemas table of your monitoring environment via the Dynatrace API.").
* Для настройки Settings 2.0 ваш [токен доступа](/managed/dynatrace-api/basics/dynatrace-api-authentication "Find out how to get authenticated to use the Dynatrace API.") должен иметь следующие разрешения:

  + **Read settings** (`settings.read`)
  + **Write settings** (`settings.write`)

## Поддерживаемые типы Configuration API

Этот тип конфигурации доступен для последней платформы Dynatrace и предыдущих окружений Dynatrace.

Обратите внимание, что большинство Configuration API устарели в пользу [Settings 2.0](/managed/deliver/configuration-as-code/monaco/reference/supported-configuration#settings "Configuration types and access permissions for Dynatrace Configuration as Code via Monaco").

Конфигурация требует [токена доступа](/managed/dynatrace-api/basics/dynatrace-api-authentication "Find out how to get authenticated to use the Dynatrace API.").
