# Специальные возможности

Dynatrace разрабатывает продукты и услуги, способствующие разнообразию, равенству и инклюзивности. Высокие стандарты доступности необходимы для обеспечения удобства использования платформы людьми с ограниченными возможностями.

* Практики разработки и критерии доступности Dynatrace соответствуют **Web Content Accessibility Guidelines** (WCAG), международно признанному стандарту, разработанному **World Wide Web Consortium** (W3C).
* **Voluntary Product Accessibility Template** (VPAT), это стандартизированная модель для документирования того, насколько продукт соответствует набору стандартов доступности.
* VPAT формирует **Accessibility Conformance Report** (ACR), в котором подробно описывается, как функции продукта соответствуют этому набору критериев.

Dynatrace сотрудничает со сторонними поставщиками и инструментами для оценки состояния доступности по стандарту WCAG, в настоящее время это [WCAG 2.1 level AA﻿](https://www.w3.org/TR/WCAG21/).

Работа над доступностью продуктов в Dynatrace ведётся непрерывно, с последовательными улучшениями в направлении соответствия WCAG 2.1 level AA. Отчёт ACR по ссылке ниже отражает статус доступности на момент публикации, поэтому стоит проверять эту страницу на предмет ежегодных обновлений отчёта.

Отказ от ответственности

Любой ACR, размещённый ниже, в любом другом месте на сайте Dynatrace или иным образом находящийся в открытом доступе, не должен толковаться как включённый в какой-либо договор, явно или подразумеваемо, полностью или частично, при отсутствии прямого подтверждения Dynatrace формулировкой в самом договоре о том, что такой отчёт или его содержательный эквивалент включён в договор.
