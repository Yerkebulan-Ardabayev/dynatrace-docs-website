# Навигация с помощью веб-интерфейса

* Практическое руководство
* 4 минуты на чтение
* Обновлено 15 мая 2026 г.

Для навигации по Dynatrace Managed используй меню слева для доступа к страницам и меню пользователя для управления учётной записью.

## Меню Dynatrace

Используй меню для навигации по Dynatrace.

### Фильтрация меню

Используй поле фильтра в верхней части меню, чтобы найти и выбрать страницы Dynatrace.

1. Введи часть строки поиска.

   ![Фильтрация меню](https://dt-cdn.net/images/filter-265-fc8901c349.png)

   Фильтрация меню
2. Выбери наиболее подходящий вариант.

   ![Выбор пункта меню из результатов фильтрации](https://dt-cdn.net/images/filter-and-select-266-ac5a5c8210.png)

   Выбор пункта меню из результатов фильтрации

### Сохранение избранного

* Чтобы добавить страницу Dynatrace в избранное, наведи курсор на название страницы и выбери звёздочку.

  ![Добавление пункта меню в избранное](https://dt-cdn.net/images/favorite-add-266-7262cf1304.png)

  Добавление пункта меню в избранное
* Чтобы перейти к избранному пункту, разверни раздел **Favorites** и выбери страницу.

  ![Выбор пункта меню из избранного](https://dt-cdn.net/images/favorite-select-266-ca63c732de.png)

  Выбор пункта меню из избранного
* Чтобы удалить страницу из избранного, снова выбери звёздочку.

  ![Удаление пункта меню из избранного](https://dt-cdn.net/images/favorite-remove-264-6eb91e3ff5.png)

  Удаление пункта меню из избранного

### Разворачивание и сворачивание разделов меню

* Чтобы показать содержимое раздела меню, выбери кнопку разворачивания.

  ![Разворачивание раздела меню](https://dt-cdn.net/images/section-expand-261-70573d8144.png)

  Разворачивание раздела меню
* Чтобы скрыть содержимое раздела меню, выбери кнопку сворачивания.

  ![Сворачивание раздела меню](https://dt-cdn.net/images/section-collapse-261-d51061df78.png)

  Сворачивание раздела меню

### Скрытие и отображение меню

* Чтобы скрыть меню, выбери кнопку со стрелкой в верхней части меню.

  ![Скрытие меню](https://dt-cdn.net/images/menu-hide-2-271-588d75e5e9.png)

  Скрытие меню
* Чтобы отобразить меню, выбери кнопку меню в верхнем левом углу.

  ![Отображение скрытого меню](https://dt-cdn.net/images/menu-display-235-ba2b8a386f.png)

  Отображение скрытого меню

## Меню пользователя

![Отображение меню пользователя](https://dt-cdn.net/images/user-menu-86-9ab2cd2961.png)

Отображение меню пользователя

Содержимое меню пользователя зависит от твоих разрешений.

* Перейди в меню пользователя, чтобы посмотреть информацию о своей учётной записи, выйти из системы, переключиться между окружениями и узнать последние новости Dynatrace.
* Если у тебя есть права администрирования, выбери **Account settings** в меню пользователя для управления настройками пользователей, групп и учётной записи.
