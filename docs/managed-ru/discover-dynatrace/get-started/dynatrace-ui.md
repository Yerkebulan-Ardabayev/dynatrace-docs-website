## Меню Dynatrace

Используйте меню для навигации по Dynatrace.

### Фильтрация меню

Используйте поле фильтра над меню для поиска и выбора страниц Dynatrace.

1. Введите часть строки поиска.

   ![Фильтрация меню](https://dt-cdn.net/images/filter-265-fc8901c349.png)

   Фильтрация меню
2. Выберите наиболее подходящий результат.

   ![Выбор пункта меню из результатов фильтра](https://dt-cdn.net/images/filter-and-select-266-ac5a5c8210.png)

   Выбор пункта меню из результатов фильтра

### Сохранение избранного

* Чтобы добавить страницу Dynatrace в избранное, наведите курсор на название страницы и нажмите на звёздочку.

  ![Добавление пункта меню в избранное](https://dt-cdn.net/images/favorite-add-266-7262cf1304.png)

  Добавление пункта меню в избранное
* Чтобы открыть избранную страницу, разверните раздел **Favorites** и выберите нужную страницу.

  ![Выбор пункта меню из избранного](https://dt-cdn.net/images/favorite-select-266-ca63c732de.png)

  Выбор пункта меню из избранного
* Чтобы удалить страницу из избранного, нажмите на звёздочку ещё раз.

  ![Удаление пункта меню из избранного](https://dt-cdn.net/images/favorite-remove-264-6eb91e3ff5.png)

  Удаление пункта меню из избранного

### Разворачивание и сворачивание разделов меню

* Чтобы отобразить содержимое раздела меню, нажмите кнопку разворачивания.

  ![Развернуть раздел меню](https://dt-cdn.net/images/section-expand-261-70573d8144.png)

  Развернуть раздел меню
* Чтобы скрыть содержимое раздела меню, нажмите кнопку сворачивания.

  ![Свернуть раздел меню](https://dt-cdn.net/images/section-collapse-261-d51061df78.png)

  Свернуть раздел меню

### Скрытие и отображение меню

* Чтобы скрыть меню, нажмите кнопку со стрелкой над меню.

  ![Скрыть меню](https://dt-cdn.net/images/menu-hide-2-271-588d75e5e9.png)

  Скрыть меню
* Чтобы отобразить меню, нажмите кнопку меню в верхнем левом углу.

  ![Отобразить скрытое меню](https://dt-cdn.net/images/menu-display-235-ba2b8a386f.png)

  Отобразить скрытое меню

## Меню пользователя

![Открыть меню пользователя](https://dt-cdn.net/images/user-menu-86-9ab2cd2961.png)

Открыть меню пользователя

Содержимое меню пользователя зависит от ваших прав доступа.

* Через меню пользователя можно просмотреть информацию о своей учётной записи, выйти из системы, переключаться между средами и получать последние новости Dynatrace.
* Если у вас есть права администратора, выберите **Account settings** в меню пользователя для управления настройками пользователей, групп и учётной записи.
