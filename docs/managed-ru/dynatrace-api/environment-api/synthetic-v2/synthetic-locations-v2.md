# Synthetic locations API v2

* Reference
* Published Sep 25, 2018

Эндпоинты **locations** в **Synthetic API v2** позволяют получать информацию о синтетических локациях.

[### Получить список

Получить обзор всех синтетических локаций, настроенных в вашем окружении.](/managed/dynatrace-api/environment-api/synthetic-v2/synthetic-locations-v2/get-all-locations "Получение списка всех синтетических локаций через Synthetic v2 API.")[### Получить детали

Получить параметры синтетической локации по её ID.](/managed/dynatrace-api/environment-api/synthetic-v2/synthetic-locations-v2/get-a-location "Просмотр синтетической локации через Synthetic v2 API.")

[### Создать локацию

Создать новую приватную локацию с нужными параметрами.](/managed/dynatrace-api/environment-api/synthetic-v2/synthetic-locations-v2/post-a-location "Создание приватной синтетической локации через Synthetic v2 API.")[### Изменить локацию

Обновить существующую конфигурацию приватной синтетической локации.](/managed/dynatrace-api/environment-api/synthetic-v2/synthetic-locations-v2/put-a-location "Обновление приватной синтетической локации через Synthetic v2 API.")[### Удалить локацию

Удалить приватную локацию, которая больше не нужна.](/managed/dynatrace-api/environment-api/synthetic-v2/synthetic-locations-v2/delete-a-location "Удаление синтетической локации через Synthetic v2 API.")

[### Просмотр статуса

Проверить статус публичных синтетических локаций.](/managed/dynatrace-api/environment-api/synthetic-v2/synthetic-locations-v2/get-location-status "Проверка статуса публичных синтетических локаций через Synthetic v2 API.")[### Изменить статус

Изменить статус публичных синтетических локаций.](/managed/dynatrace-api/environment-api/synthetic-v2/synthetic-locations-v2/put-location-status "Изменение статуса публичных синтетических локаций через Synthetic v2 API.")
