# Synthetic locations API v1

* Справочник
* Опубликовано 25 сентября 2018 г.

Доступна новая версия этого API, [Synthetic API v2](/managed/dynatrace-api/environment-api/synthetic-v2 "Узнайте, что предлагает Dynatrace Synthetic v2 API."). Попробуйте её!

Эндпоинты **locations** в **Synthetic API** позволяют получать информацию о синтетических локациях.

[### Получить список

Получить обзор всех синтетических локаций в вашем окружении.](/managed/dynatrace-api/environment-api/synthetic/synthetic-locations/get-all-locations "Получение списка всех синтетических локаций через Synthetic v1 API.")[### Получить детали

Получить параметры синтетической локации по её ID.](/managed/dynatrace-api/environment-api/synthetic/synthetic-locations/get-a-location "Просмотр синтетической локации через Synthetic v1 API.")

[### Создать локацию

Создать новую приватную локацию с нужными параметрами.](/managed/dynatrace-api/environment-api/synthetic/synthetic-locations/post-a-location "Создание приватной синтетической локации через Synthetic v1 API.")[### Изменить локацию

Обновить существующую конфигурацию приватной синтетической локации.](/managed/dynatrace-api/environment-api/synthetic/synthetic-locations/put-a-location "Обновление приватной синтетической локации через Synthetic v1 API.")[### Удалить локацию

Удалить приватную локацию, которая больше не нужна.](/managed/dynatrace-api/environment-api/synthetic/synthetic-locations/delete-a-location "Удаление синтетической локации через Synthetic v1 API.")

[### Просмотр статуса

Проверить статус публичных синтетических локаций.](/managed/dynatrace-api/environment-api/synthetic/synthetic-locations/get-location-status "Проверка статуса публичных синтетических локаций через Synthetic v1 API.")[### Изменить статус

Изменить статус публичных синтетических локаций.](/managed/dynatrace-api/environment-api/synthetic/synthetic-locations/put-location-status "Изменение статуса публичных синтетических локаций через Synthetic v1 API.")
