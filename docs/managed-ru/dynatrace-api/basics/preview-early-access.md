# Preview- и Early Access-релизы Dynatrace API

* Reference
* Published Feb 07, 2020

Некоторые API-запросы доступны как Early Access или как часть Preview. Такие запросы помечены отметками «Early Access» и «Preview» на соответствующих страницах документации Dynatrace и в API Explorer (см. примеры ниже).

## Early Access

Запросы Early Access-релиза доступны по opt-in. Они не считаются полностью завершёнными (feature-complete), но публично доступны, полностью поддерживаются и готовы к использованию в production.

Несовместимые изменения в Early Access API-запросах маловероятны, но полную обратную совместимость GA-версии с Early Access-релизом гарантировать невозможно.

## Preview

Preview даёт доступ к новым функциям, чтобы мы могли получить обратную связь и в итоге выпустить функции, точно отвечающие вашим потребностям. Preview-запросы не предназначены для использования в production-окружениях.

Preview-запросы публично недоступны: нужно записаться в соответствующий Preview. Подробнее об условиях и записи смотрите [Preview and Early Access releases](/managed/whats-new/preview-releases "Preview-релизы и участие в них.").
