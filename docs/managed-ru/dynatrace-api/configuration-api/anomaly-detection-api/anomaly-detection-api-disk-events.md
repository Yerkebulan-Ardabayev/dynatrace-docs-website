# Anomaly detection API - Disk events

* Reference
* Published Oct 30, 2018

API **Anomaly detection-Disk events** позволяет управлять конфигурацией кастомных disk events. Той же конфигурацией можно [управлять](/managed/dynatrace-intelligence/anomaly-detection/adjust-sensitivity-anomaly-detection/adjust-sensitivity-infastructure "Настройка чувствительности обнаружения проблем для инфраструктуры.") в веб-интерфейсе Dynatrace в **Settings > Anomaly detection > Hosts** (прокрутите вниз до раздела **Custom disk-detection rule**).

[### Список всех disk events

Получить обзор всех кастомных disk events, настроенных в вашем Dynatrace-окружении.](/managed/dynatrace-api/configuration-api/anomaly-detection-api/anomaly-detection-api-disk-events/get-all "Просмотр всех правил disk event вашего окружения через Dynatrace API.")[### Просмотр disk event

Получить параметры disk event по ID события.](/managed/dynatrace-api/configuration-api/anomaly-detection-api/anomaly-detection-api-disk-events/get-event "Просмотр правила disk event через Dynatrace API.")

[### Создание disk event

Создать новое событие с нужными вам параметрами.](/managed/dynatrace-api/configuration-api/anomaly-detection-api/anomaly-detection-api-disk-events/post-event "Создание правила disk event через Dynatrace API.")[### Редактирование disk event

Обновить существующую конфигурацию события.](/managed/dynatrace-api/configuration-api/anomaly-detection-api/anomaly-detection-api-disk-events/put-event "Редактирование правила disk event через Dynatrace API.")[### Удаление disk event

Удалить ненужную конфигурацию события.](/managed/dynatrace-api/configuration-api/anomaly-detection-api/anomaly-detection-api-disk-events/del-event "Удаление правила disk event через Dynatrace API.")
