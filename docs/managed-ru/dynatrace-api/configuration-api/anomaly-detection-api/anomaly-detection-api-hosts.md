# Anomaly detection API - Hosts

* Reference
* Published Oct 30, 2018

API **Anomaly detection-Hosts** позволяет управлять конфигурацией обнаружения аномалий для хостов. Той же конфигурацией можно также [управлять](/managed/dynatrace-intelligence/anomaly-detection/adjust-sensitivity-anomaly-detection/adjust-sensitivity-infastructure "Настройка чувствительности обнаружения проблем для инфраструктуры.") в веб-интерфейсе Dynatrace на странице **Settings > Anomaly detection > Infrastructure > Hosts**.

Получить обзор конфигурации обнаружения аномалий для хостов.](/managed/dynatrace-api/configuration-api/anomaly-detection-api/anomaly-detection-api-hosts/get-config "Просмотр конфигурации обнаружения аномалий для хостов через Dynatrace API.")[### Обновление конфигурации
