# Anomaly detection API - AWS

* Reference
* Published Oct 30, 2018

API **Anomaly detection-AWS** позволяет управлять конфигурацией обнаружения аномалий для инфраструктуры Amazon Web Services. Той же конфигурацией можно управлять в веб-интерфейсе Dynatrace на странице **Settings > Anomaly detection > Infrastructure > Amazon Web Services**.
