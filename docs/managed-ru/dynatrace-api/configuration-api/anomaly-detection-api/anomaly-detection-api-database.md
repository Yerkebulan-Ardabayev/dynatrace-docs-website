# Anomaly detection API - Database services

* Reference
* Published Oct 30, 2018

API **Anomaly detection-Database services** позволяет управлять конфигурацией обнаружения аномалий для сервисов баз данных. Той же конфигурацией можно управлять в веб-интерфейсе Dynatrace на странице **Settings** > **Anomaly detection** > **Database services**.
