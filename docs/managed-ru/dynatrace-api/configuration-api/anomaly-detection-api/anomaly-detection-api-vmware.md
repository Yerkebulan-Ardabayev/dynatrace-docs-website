# Anomaly detection API - VMware

# Anomaly detection API - VMware

* Reference
* Published Oct 30, 2018

API **Anomaly detection-VMware** позволяет управлять конфигурацией обнаружения аномалий для виртуальной инфраструктуры. Той же конфигурацией можно также [управлять](/managed/dynatrace-intelligence/anomaly-detection/adjust-sensitivity-anomaly-detection/adjust-sensitivity-infastructure "Настройка чувствительности обнаружения проблем для инфраструктуры.") в веб-интерфейсе Dynatrace на странице **Settings > Anomaly detection > Infrastructure > VMware**.

Получить обзор конфигурации обнаружения аномалий для VMware.](/managed/dynatrace-api/configuration-api/anomaly-detection-api/anomaly-detection-api-vmware/get-config "Просмотр конфигурации обнаружения аномалий для VMware через Dynatrace API.")[### Обновление конфигурации
