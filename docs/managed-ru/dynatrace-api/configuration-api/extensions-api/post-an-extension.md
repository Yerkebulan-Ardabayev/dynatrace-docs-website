# Extensions API - POST an extension .zip file

* Reference
* Published Mar 06, 2020

Загружает .zip-файл расширения в ваше окружение Dynatrace.

Запрос принимает payload `multipart/form-data` и возвращает payload `application/json`.

|  |  |  |
| --- | --- | --- |
| POST | ManagedDynatrace for Government | `https://{your-domain}/e/{your-environment-id}/api/config/v1/extensions` |
| POST | Environment ActiveGate | `https://{your-activegate-domain}/e/{your-environment-id}/api/config/v1/extensions` |

## Аутентификация

Для выполнения этого запроса нужен access token со scope `WriteConfig`.

Как его получить и использовать, смотрите [Tokens and authentication](/managed/discover-dynatrace/references/dynatrace-api/basics/dynatrace-api-authentication).

## Параметры

| Параметр | Тип | Описание | Где | Обязательный |
| --- | --- | --- | --- | --- |
| file | .zip file | Загружаемый .zip-файл расширения.  Имя файла должно совпадать с полем **name** в файле `plugin.json`.  Например, для расширения, у которого **name** равно `custom.remote.python.demo`, имя файла расширения должно быть `custom.remote.python.demo.zip`. | body | Required |
| overrideAlerts | Boolean | Использовать пороги, заданные расширением, для оповещений (`true`) или пороги, заданные пользователем (`false`).  Пороги, заданные расширением, хранятся в файле `plugin.json`.  Если не задано, используются пороги, заданные пользователем. | query | Optional |

## Ответ

### Коды ответа

| Код | Тип | Описание |
| --- | --- | --- |
| **201** | [EntityShortRepresentation](#openapi-definition-EntityShortRepresentation) | Успех. Расширение загружено. Тело ответа содержит ID расширения. |
| **400** | [ErrorEnvelope](#openapi-definition-ErrorEnvelope) | Сбой. Невалидный ввод |

| id | string | ID сущности Dynatrace. |
| name | string | Имя сущности Dynatrace. |

#### Объект `ErrorEnvelope`

| Элемент | Тип | Описание |

| message | string | Сообщение об ошибке |

Список нарушений ограничений

## Validate payload

Рекомендуется валидировать payload перед отправкой реального запроса. Код ответа **204** означает валидный payload.

Запрос принимает payload `application/json`.

|  |  |  |
| --- | --- | --- |
| POST | ManagedDynatrace for Government | `https://{your-domain}/e/{your-environment-id}/api/config/v1/extensions/validator` |
| POST | Environment ActiveGate | `https://{your-activegate-domain}/e/{your-environment-id}/api/config/v1/extensions/validator` |

### Аутентификация
