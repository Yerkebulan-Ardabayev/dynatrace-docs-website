# Alerting profiles API

* Reference
* Published Aug 08, 2019

Этот API устарел. Используйте [Settings API](/managed/dynatrace-api/environment-api/settings "Узнайте, что предлагает Dynatrace Settings API."). Ищите schema **Problem alerting profiles** (`builtin:alerting.profile`).

[### Список всех профилей оповещений

Получить обзор всех профилей оповещений, доступных в вашем Dynatrace-окружении.](/managed/dynatrace-api/configuration-api/alerting-profiles-api/get-all "Просмотр всех профилей оповещений вашего окружения через Dynatrace API.")[### Просмотр профиля оповещений

Получить параметры профиля оповещений по его ID.](/managed/dynatrace-api/configuration-api/alerting-profiles-api/get-profile "Просмотр профиля оповещений через Dynatrace API.")

[### Создание профиля оповещений

Создать профиль оповещений именно с теми параметрами, которые вам нужны.](/managed/dynatrace-api/configuration-api/alerting-profiles-api/post-profile "Создание профиля оповещений через Dynatrace API.")[### Редактирование профиля оповещений

Обновить существующую конфигурацию профиля оповещений.](/managed/dynatrace-api/configuration-api/alerting-profiles-api/put-profile "Редактирование профиля оповещений через Dynatrace API.")[### Удаление профиля оповещений

Удалить профили оповещений, которые больше не нужны.](/managed/dynatrace-api/configuration-api/alerting-profiles-api/del-profile "Удаление профиля оповещений через Dynatrace API.")
