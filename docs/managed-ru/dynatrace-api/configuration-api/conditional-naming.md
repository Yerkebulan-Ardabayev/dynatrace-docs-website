# Conditional naming API

* Reference
* Published Apr 23, 2020

API **Conditional naming** позволяет управлять конфигурацией правил именования для process groups.

[### Список всех правил

Получить обзор всех правил условного именования, заданных в вашем окружении.](/managed/dynatrace-api/configuration-api/conditional-naming/get-all "Просмотр всех правил условного именования вашего окружения через Dynatrace API.")[### Просмотр правила

Получить параметры правила условного именования по его ID.](/managed/dynatrace-api/configuration-api/conditional-naming/get-rule "Просмотр правила условного именования через Dynatrace API.")

[### Создать правило

Создать новое правило условного именования с нужными параметрами.](/managed/dynatrace-api/configuration-api/conditional-naming/post-rule "Создание правила условного именования через Dynatrace API.")[### Редактировать правило

Обновить правило условного именования. Также можно создать новое правило с указанным ID.](/managed/dynatrace-api/configuration-api/conditional-naming/put-rule "Обновление правила условного именования через Dynatrace API.")[### Удалить правило

Удалить ненужное правило условного именования.](/managed/dynatrace-api/configuration-api/conditional-naming/del-rule "Удаление правила условного именования через Dynatrace API.")
