# Custom services API

# Custom services API

* Reference
* Published Oct 22, 2018

API **Custom services** позволяет управлять правилами обнаружения пользовательских сервисов.

[### List all

Обзор всех правил пользовательских сервисов.](/managed/dynatrace-api/configuration-api/service-api/custom-services-api/get-all "Просмотр всех правил пользовательских сервисов вашего окружения через Dynatrace API.")[### View a rule

Получить параметры правила пользовательского сервиса по его ID.](/managed/dynatrace-api/configuration-api/service-api/custom-services-api/get-rule "Просмотр правила пользовательского сервиса через Dynatrace API.")[### Reorder rules

Правила пользовательских сервисов выполняются одно за другим. Применяется первое совпавшее правило, дальнейшая обработка прекращается.

Измените порядок правил пользовательских сервисов, чтобы получить нужный порядок выполнения.](/managed/dynatrace-api/configuration-api/service-api/custom-services-api/reorder-rules "Изменение порядка правил пользовательских сервисов через Dynatrace API.")[### Create a rule

Создать новое правило с нужными вам параметрами.](/managed/dynatrace-api/configuration-api/service-api/custom-services-api/post-rule "Создание правила пользовательского сервиса через Dynatrace API.")[### Edit a rule

Обновить существующее правило пользовательского сервиса или создать новое правило с указанным ID.](/managed/dynatrace-api/configuration-api/service-api/custom-services-api/put-rule "Редактирование правила пользовательского сервиса через Dynatrace API.")[### Delete a rule

Удалить правило, которое вам больше не нужно.](/managed/dynatrace-api/configuration-api/service-api/custom-services-api/del-rule "Удаление правила пользовательского сервиса через Dynatrace API.")
