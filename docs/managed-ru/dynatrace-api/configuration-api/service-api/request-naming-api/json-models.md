# Request naming API - JSON models

# Request naming API - JSON models

* Reference
* Published Dec 16, 2019

Некоторые JSON-модели API **Request naming** различаются в зависимости от поля **type** некоторых объектов. JSON-модели для каждой вариации перечислены ниже.

## Вариации объекта `ComparisonInfo`

### BOOLEAN

BooleanComparisonInfo

Parameters

JSON model

#### Объект `BooleanComparisonInfo`

Сравнение для атрибутов `BOOLEAN`.

| Элемент | Тип | Описание |
| --- | --- | --- |
| comparison | string | Оператор сравнения. Его можно инвертировать, установив **negate** в `true`. Возможные значения: * `EQUALS` * `EQUALS_ANY_OF` * `EXISTS` |
| value | string | Значение для сравнения. |
| values | object[] | Значения для сравнения. |

```
{



"comparison": "EQUALS",



"value": true,



"negate": false,



"type": "BOOLEAN"



}
```
