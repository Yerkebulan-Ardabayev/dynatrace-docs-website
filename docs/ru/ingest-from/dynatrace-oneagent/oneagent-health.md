# OneAgent health overview

* Последняя версия Dynatrace
* Чтение: 4 мин
* Обновлено 15 января 2026 г.

Обзор работоспособности OneAgent позволяет в масштабе обнаруживать все развёрнутые [модули OneAgent](../technology-support/oneagent-platform-and-capability-support-matrix.md "Узнайте, какие возможности поддерживает OneAgent на разных операционных системах и платформах.") и выявлять аномалии до того, как они становятся проблемами. Например, с помощью обзора работоспособности OneAgent вы можете обнаружить:

* Устаревшие модули OneAgent, требующие обновления для продолжения полной поддержки Dynatrace.
* Проблемы связности между модулями OneAgent и вашим кластером Dynatrace.
* Несовместимые модули OneAgent, требующие корректирующих действий.
* Процессы, требующие перезапуска для продолжения мониторинга.
* Модули OneAgent, в которых отключено автоматическое обновление.

## Начало работы

Чтобы начать работу с обзором работоспособности OneAgent, перейдите в раздел **OneAgent Health**.

![OneAgent Health](https://dt-cdn.net/images/oneagent-health-1744-afe02d9e3b.png)

**Обзор работоспособности OneAgent** включает:

* Диаграмму с областями, ориентированную на [метрику состояния работоспособности](#health-state-metric).

  Диаграмма с областями содержит данные о модулях OneAgent за выбранный период времени. Диаграмму можно разбить по состоянию работоспособности, состоянию мониторинга, типу модуля, версии модуля, типу процесса и типу операционной системы.
* Таблицу данных, представляющую различные атрибуты участвующих модулей OneAgent.

  Таблица данных содержит данные в реальном времени о модулях OneAgent, обнаруженных в вашей среде за последние 5 минут. Её можно фильтровать по конкретным атрибутам. Для отображения дополнительных атрибутов нажмите **Configure columns**.

  Над таблицей данных отображаются самые низкие и самые высокие совместимые версии OneAgent для данного кластера Dynatrace.

  Для визуализации данных в этой таблице необходимо разрешение среды [Install OneAgent](../../manage/identity-access-management/permission-management/role-based-permissions.md#environment "Разрешения на основе ролей").

Выберите конкретную точку данных на диаграмме с областями, чтобы увидеть участвующие модули OneAgent в таблице данных.

## Метрика состояния работоспособности

Метрика **Health state** (`dsfm:cluster.oneagent.agent_modules`) указывает на состояние работоспособности развёрнутых модулей OneAgent. Она разделена на четыре класса: Critical (Критический), Warning (Предупреждение), Info (Информационный) и Healthy (Здоровый). Каждый класс имеет различные последствия.

### Ограничения

* Только метрика **Health state** включает модуль z/OS Java, а также интеграции AWS Lambda, Azure и Google Cloud.
* В таблице данных отображаются только модули OneAgent, обнаруженные за последние 5 минут.
* Фильтрация по зоне управления доступна только в таблице данных.
