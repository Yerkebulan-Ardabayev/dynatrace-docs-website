# Устранение проблем с глубоким мониторингом OneAgent

* Latest Dynatrace
* 3 мин. чтения
* Опубликовано 16 июл. 2018 г.

Dynatrace OneAgent поддерживается на многих [платформах](../../technology-support/oneagent-platform-and-capability-support-matrix.md "Узнайте, какие возможности поддерживает OneAgent на различных операционных системах и платформах.") и предлагает глубокий мониторинг для многих [технологий и фреймворков](../../technology-support.md "Найдите технические сведения о поддержке Dynatrace для конкретных платформ и фреймворков разработки."). Команда Dynatrace тестирует все возможные сценарии глубокого мониторинга, но для упрощения поддержки в ситуациях, когда глубокий мониторинг может не работать должным образом, можно воспользоваться страницей [функций OneAgent](../oneagent-features.md "Управляйте функциями OneAgent глобально и для отдельных групп процессов.").

## Сценарии использования

Параметры на странице **OneAgent features** позволяют точечно отключать определённые функции OneAgent для упрощения устранения неполадок. Это может быть полезно для выявления первопричины проблемы вплоть до конкретной функции.

Эти параметры также позволяют временно отключить определённые функции, которые могут вызывать проблемы, с сохранением всех остальных функций. При этом команда поддержки Dynatrace может работать над постоянным исправлением.

Перед деактивацией параметров OneAgent

Мы не рекомендуем деактивировать параметры функций OneAgent без предварительной консультации со службой поддержки Dynatrace. При отключении конкретной функции OneAgent в вашей среде OneAgent перестаёт собирать данные наблюдаемости для этой функции, что может привести к потере важной информации.

## Параметры

Каждую функцию на странице **OneAgent features** можно отключить для всей среды, конкретной группы процессов или отдельного процесса.

### Параметры среды

Чтобы включить или отключить функцию для всей среды

1. Перейдите в **Settings**.
2. Выберите **Preferences** > **OneAgent features**.
3. Найдите нужную функцию. Можно ввести строку фильтра над таблицей.
4. Разверните запись, чтобы:

   * Необязательно Просмотреть описание функции
   * Необязательно Просмотреть метаданные
   * Необязательно Перейти по ссылке к соответствующим примечаниям к выпуску
5. При необходимости измените параметр **Enabled**.

Все функции инструментирования можно включить или отключить с помощью основного переключателя, а для некоторых из них предусмотрены два дополнительных параметра:

* **Activate this feature also in OneAgents only fulfilling the minimum Opt-In version**
  Этот параметр активирует функцию с минимальной версией для подключения при первом выпуске. Обычно функцию можно активировать, если она соответствует минимальной версии OneAgent, но некоторые из них сначала выпускаются как opt-in функции с минимальной версией для подключения. Это полезно, когда общедоступная (GA) версия функции OneAgent выше минимальной версии для подключения. Если вы уже включили функцию в период opt-in, вы можете продолжать использовать OneAgent без обновления до новой GA-версии.
* **Instrumentation enabled**
  Этот параметр вступает в силу немедленно. Однако при перезапуске или запуске процесса функция не будет добавлена в ваше приложение. Это означает, что функция не повлияет на ваше приложение. Недостатком является то, что при повторном включении функции вам потребуется перезапустить затронутые процессы.

### Параметры для конкретных процессов

Если конкретная функция вызывает проблемы только в контексте определённой группы процессов, можно отключить её именно для этой группы, не затрагивая остальную среду.

Чтобы включить или отключить функцию для конкретного процесса или группы процессов

1. Перейдите в **Smartscape Topology**.
2. Выберите **Processes**.
3. На карте топологии наведите указатель мыши на процесс и выберите значок ссылки для перехода на страницу сведений о процессе. Например:

   ![Ссылка из топологии Smartscape на страницу сведений о процессе](https://dt-cdn.net/images/link-to-process-page-377-da0282e435.png)
4. На странице сведений о процессе выберите **More** (**...**) > **Settings**, затем перейдите на вкладку **OneAgent features**.
5. Выберите **Add override**, чтобы добавить параметр для конкретного процесса, переопределяющий параметр среды.
6. Введите строку поиска в поле **Feature**, чтобы найти и выбрать функцию.
7. При необходимости настройте переключатели для переопределения и выберите **Save changes**.
