Вы можете установить OneAgent на Windows с помощью установщика, поставляемого в виде самораспаковывающегося EXE-файла — для установки на одном сервере — а также в виде пакета MSI для развёртывания через групповые политики. Для получения аналитической информации о поддерживаемых возможностях OneAgent для Windows см. матрицу поддержки платформ и возможностей OneAgent. Для получения информации о поддерживаемых версиях Windows ознакомьтесь с поддерживаемыми технологиями и версиями OneAgent.
