Dynatrace поддерживает Oracle Solaris (x86 и SPARC) для Java, Apache HTTP Server и IBM HTTP Server. Если ваша инфраструктура основана на этих технологиях, вы можете использовать Dynatrace OneAgent для эффективного мониторинга хостов, служб и пользовательского опыта. Аналитическую информацию о поддерживаемых возможностях OneAgent для Solaris см. в матрице поддержки платформ и возможностей OneAgent.
