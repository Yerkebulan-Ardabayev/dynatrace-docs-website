# Поддерживаемые версии Go

* Latest Dynatrace
* Чтение: 7 мин.
* Обновлено 2 марта 2026 г.

При выпуске каждой новой мажорной версии Go Dynatrace добавляет поддержку этой версии. Поддержка каждой минорной версии также добавляется -- подробнее см. в [Матрице версий](#go-version-matrix).

[Политика выпуска Go](https://dt-url.net/uos3rmi) поддерживает последние две мажорные версии Go. Однако Dynatrace решил поддерживать каждую версию Go как минимум на полгода дольше, чтобы у вас было достаточно времени для обновления.

## Внешние метаданные

OneAgent может поддерживать более новые версии Go, запрашивая внешние метаданные из кластера Dynatrace. После получения внешних метаданных OneAgent может инструментировать приложение, основанное на версии Go, для которой у OneAgent нет встроенной поддержки. Требуется перезапуск процесса Go.

Поэтому в [матрице версий](#go-version-matrix) есть две разные верхние границы в каждом диапазоне версий. Встроенные версии Go всегда поддерживаются, а верхняя граница в скобках может поддерживаться, если OneAgent может получить необходимые внешние метаданные из кластера Dynatrace.

Внешние метаданные публикуются независимо от цикла выпуска OneAgent, и OneAgent всегда получает последние доступные внешние метаданные из кластера Dynatrace. Это также означает, что в поэтапной среде приложения могут быть инструментированы на более позднем этапе, хотя они не были инструментированы на более раннем этапе, поскольку за это время стали доступны новые метаданные. Если такое поведение нежелательно, отключите функцию OneAgent **Go external metadata**, чтобы внешние метаданные не использовались. Функции OneAgent можно найти в **Settings** > **Preferences** > **OneAgent features**.
