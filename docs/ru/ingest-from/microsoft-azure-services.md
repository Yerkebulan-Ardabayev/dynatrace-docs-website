# Set up Dynatrace on Microsoft Azure


* Последняя версия Dynatrace
* Обзор
* Чтение: 1 мин
* Обновлено 20 февраля 2026 г.

Dynatrace автоматически обнаруживает, определяет базовые показатели и интеллектуально отслеживает облачные среды Microsoft Azure, обеспечивая изоляцию доменов сбоев, а также видимость инфраструктуры и действий конечных пользователей в режиме реального времени.

Дополнительные причины для мониторинга Azure с помощью Dynatrace см. на странице [Microsoft Azure](https://www.dynatrace.com/hub/detail/microsoft-azure/) в Dynatrace Hub.
