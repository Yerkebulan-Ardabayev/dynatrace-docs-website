# Extend Dynatrace with domain-specific web UI


* Classic
* Обзор
* 1 мин. чтения
* Обновлено 16 мая 2022 г.

Фреймворк Dynatrace Extensions 2.0 позволяет адаптировать веб-интерфейс Dynatrace к конкретным потребностям данных, принимаемых вашим расширением. Вы можете задавать настраиваемые дашборды, создавать специализированные страницы унифицированного анализа и расширять встроенные страницы унифицированного анализа данными, предоставляемыми вашим расширением.

Расширить веб-интерфейс Dynatrace с помощью страниц унифицированного анализа, адаптированных к сущностям.](extend-ui/unified-analysis.md "Расширить веб-интерфейс Dynatrace с помощью страниц унифицированного анализа, адаптированных к сущностям.")[### Расширение встроенных страниц унифицированного анализа
