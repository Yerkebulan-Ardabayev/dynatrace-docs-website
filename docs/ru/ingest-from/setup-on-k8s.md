# Настройка Dynatrace на Kubernetes


* Latest Dynatrace
* 2-min read

Dynatrace помогает отслеживать и управлять работоспособностью, производительностью и безопасностью ваших кластеров Kubernetes и рабочих нагрузок, консолидируя метрики, трассировки, журналы и события в одном месте.

### Быстрый старт

Разверните Dynatrace на Kubernetes за одну минуту.### Развёртывание

Определите рекомендуемый вариант развёртывания для вашей конфигурации с учётом требований.### Как это работает

Ознакомьтесь с компонентами Dynatrace, развёртываемыми в вашем кластере Kubernetes.### Руководства

Узнайте, как настроить Dynatrace Operator для поддержки конкретных сценариев использования.### Интеграции

Используйте компоненты с открытым исходным кодом для передачи дополнительных сигналов наблюдаемости в Dynatrace.### Справочник

Справочник по API и параметры конфигурации для всех компонентов Dynatrace в вашем кластере Kubernetes.### Потоковая передача журналов Kubernetes

Собирайте журналы из систем оркестрации контейнеров Kubernetes с помощью Dynatrace Log Module.### Примечания к выпускам Dynatrace Operator

Ознакомьтесь с примечаниями к выпускам Dynatrace Operator.[![cost-allocation](https://cdn.bfldr.com/B686QPH3/at/9vhmq6x4kh88sq7hq4qbwg9t/DT0634.svg?auto=webp&width=72&height=72 "cost-allocation")
