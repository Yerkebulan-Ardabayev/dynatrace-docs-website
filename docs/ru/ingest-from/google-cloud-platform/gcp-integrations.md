Dynatrace OneAgent обеспечивает полноценный мониторинг стека на Google Compute Engine и Google Kubernetes Engine, а также мониторинг на уровне контейнеров и приложений. Dynatrace поддерживает Google App Engine в режиме мониторинга только на уровне приложений.

## Интеграции

Мониторинг Google Kubernetes Engine (GKE)

Мониторинг Google Cloud Functions

Мониторинг управляемого Google Cloud Run

Мониторинг Google App Engine

Мониторинг Google Compute Engine

Комплексное руководство по мониторингу сервисов Google Cloud с интеграцией Operations Suite
