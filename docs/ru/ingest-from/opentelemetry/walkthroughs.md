# Инструментирование вашего приложения

* Latest Dynatrace
* How-to guide
* 1-min read
* Updated on Nov 15, 2023

Следующие пошаговые руководства представляют собой структурированные, поэтапные инструкции для различных языков программирования, поддерживаемых OpenTelemetry. В них содержатся примеры кода, демонстрирующие, как интегрировать библиотеки OpenTelemetry в ваше приложение, инициализировать OpenTelemetry, создавать различные сигналы (трассировки, метрики и логи) и экспортировать данные в бэкенд Dynatrace.
