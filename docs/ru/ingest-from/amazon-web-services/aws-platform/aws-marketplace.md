# Частное предложение AWS Marketplace


* Classic
* How-to guide
* 3-min read
* Published May 26, 2025

Платформенный сервис Dynatrace можно лицензировать через [AWS Marketplace](https://aws.amazon.com/marketplace/pp/prodview-si2angoettdnc?applicationId=AWSMPContessa) в рамках частного предложения.

На этой странице описывается, как связать частное предложение со средой Dynatrace.
Вы можете связать существующую среду Dynatrace, созданную в AWS, или создать и связать новую среду.

Если у вас еще нет частного предложения, обратитесь в отдел продаж Dynatrace по адресу [sales@dynatrace.com](mailto:sales@dynatrace.com).

## Ключевые термины

Частное предложение
:   Индивидуальное лицензионное предложение, подготовленное Dynatrace для конкретного клиента. Оно включает пользовательскую [тарифную карту](https://www.dynatrace.com/pricing/rate-card) с согласованными возможностями и ценами.

Публичный облачный сервис
:   Вычислительные сервисы, предоставляемые сторонним провайдером. Dynatrace может быть лицензирован через публичный облачный сервис.

Среда Dynatrace
:   [Среда Dynatrace](../../../discover-dynatrace/get-started/monitoring-environment.md "Understand and learn how to work with monitoring environments.") — это место, где выполняется весь анализ производительности Dynatrace. Среды имеют уникальный идентификатор в формате `abc12345`.

## Предварительные требования

* У вас уже есть индивидуальное частное предложение для Dynatrace в AWS Marketplace.
* Если вы связываете существующую учетную запись Dynatrace, ваш пользователь Dynatrace должен иметь разрешение **View and manage account billing information** для этой учетной записи.

## Шаги

Следуйте приведенным ниже шагам, чтобы связать частное предложение с учетной записью Dynatrace.

Связать существующую учетную запись Dynatrace

Создать новую учетную запись Dynatrace для привязки

Если у вас уже есть существующая учетная запись и среда Dynatrace в AWS, следуйте приведенным ниже шагам, чтобы связать учетную запись с частным предложением AWS Marketplace.
Это может быть пробная или рабочая среда, при условии, что она находится в активном состоянии.

1. В AWS Marketplace откройте ваше частное предложение Dynatrace.
2. Примите частное предложение Dynatrace.
3. Выберите **Set up your account**.
4. На странице **Welcome to your new Dynatrace Platform Subscription** выберите **Link existing account**.
5. Войдите как пользователь Dynatrace, назначенный учетной записи, которую вы хотите связать.

   1. Введите адрес электронной почты, затем выберите **Next**.
   2. Введите пароль учетной записи, затем выберите **Next**.
6. В зависимости от количества учетных записей, назначенных пользователю, выполните одно из следующих действий:

   * Одна учетная запись: учетная запись распознается автоматически, и вы можете перейти к следующему шагу.
   * Несколько учетных записей: в выпадающем меню **Account to link** выберите конкретную учетную запись, которую вы хотите связать с этим частным предложением.
     Затем перейдите к следующему шагу.
   * Более 20 учетных записей: в поле **Account UUID** укажите UUID учетной записи, которую вы хотите связать с этим частным предложением.
     Затем перейдите к следующему шагу.

     Чтобы получить UUID учетной записи, перейдите в **Account Management** и скопируйте UUID из URL-адреса в адресной строке браузера.
7. После входа с выбранной учетной записью информация о вашей учетной записи отображается в разделах **User details**, **Account details** и **Environments**.
   Это учетная запись, которая будет связана с вашим частным предложением, поэтому проверьте правильность данных.
8. Установите флажок рядом с **I agree to link this Dynatrace account with my marketplace subscription.**, затем выберите **Next**.

   Это действие нельзя отменить.

Когда появится окно **Congratulations!**, ваше частное предложение будет связано с выбранной учетной записью Dynatrace.

Информация о подписке в **Account Management** отразит полностью оплаченную новую лицензию в течение трех дней.
В это время вы можете начать использовать среду Dynatrace в соответствии с существующей лицензией учетной записи (если вы связали существующую учетную запись) или с пробной лицензией (если вы создали новую учетную запись).

Выберите **Open Dynatrace**, чтобы начать использовать Dynatrace и [настроить Dynatrace в Amazon Web Services](../../amazon-web-services.md "Set up and configure monitoring for Amazon Web Services.").

Если у вас еще нет учетной записи Dynatrace, следуйте приведенным ниже шагам, чтобы создать учетную запись и связать её с частным предложением AWS Marketplace.

1. В AWS Marketplace откройте ваше частное предложение Dynatrace.
2. Примите частное предложение Dynatrace.
3. Выберите **Set up your account**.
4. На странице **Welcome to your new Dynatrace Platform Subscription** выберите **Create a new account**.
5. Введите необходимую информацию, как описано на странице:

   1. **User details**
   2. **Account details**
   3. **Environment details**
6. Выберите **Create account**.

Появится окно **You're almost done!**.
Dynatrace создает новую учетную запись и среду в соответствии с введенной вами информацией.
Этот шаг может занять до 5 минут.

Когда учетная запись будет успешно создана, появится окно **Congratulations!**.
Ваше частное предложение теперь связано с только что созданной учетной записью Dynatrace.

Информация о подписке в **Account Management** отразит полностью оплаченную новую лицензию в течение трех дней.
В это время вы можете начать использовать среду Dynatrace в соответствии с существующей лицензией учетной записи (если вы связали существующую учетную запись) или с пробной лицензией (если вы создали новую учетную запись).

Выберите **Open Dynatrace**, чтобы начать использовать Dynatrace и [настроить Dynatrace в Amazon Web Services](../../amazon-web-services.md "Set up and configure monitoring for Amazon Web Services.").
