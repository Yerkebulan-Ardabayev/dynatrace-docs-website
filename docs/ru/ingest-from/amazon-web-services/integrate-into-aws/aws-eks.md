Dynatrace OneAgent обеспечивает расширенный мониторинг подов, узлов и кластеров Elastic Kubernetes Service. Процесс развёртывания OneAgent аналогичен другим дистрибутивам. Подробнее см. [Что такое Amazon EKS?](https://dt-url.net/zn0388b).

Для Amazon Elastic Kubernetes Service потребление мониторинга рассчитывается на основе единиц хостов. Подробнее см. Мониторинг приложений и инфраструктуры (единицы хостов).
