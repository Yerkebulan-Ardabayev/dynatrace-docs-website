Dynatrace обеспечивает комплексную поддержку мониторинга сервисов Azure посредством интеграции как с OneAgent, так и с Azure Monitor.

## Интеграция с вычислительными и бессерверными сервисами Azure

Мониторинг Azure App Service

Серверы с поддержкой Microsoft Azure Arc

Мониторинг Azure Functions с помощью Azure App Service (встроенный)

Мониторинг Azure Spring Apps

Azure Kubernetes Service (AKS)

Мониторинг Azure Service Fabric

Мониторинг виртуальных машин Azure

Мониторинг Azure Virtual Machine Scale Set (VMSS)

## Интеграция с Azure Monitor

Помимо мониторинга рабочих нагрузок Azure с помощью OneAgent, вы можете интегрировать Dynatrace с Azure Monitor для мониторинга инфраструктуры и получения аналитических данных даже в сценариях с бессерверными приложениями.

Дополнительную информацию см. в разделе Мониторинг сервисов Azure с помощью метрик Azure Monitor
