## Интеграция с Azure Monitor

Помимо мониторинга рабочих нагрузок Azure с помощью OneAgent, вы можете интегрировать Dynatrace с Azure Monitor для мониторинга инфраструктуры и получения аналитических данных даже в сценариях с бессерверными приложениями.

Дополнительную информацию см. в разделе [Мониторинг сервисов Azure с помощью метрик Azure Monitor](/docs/ingest-from/microsoft-azure-services/azure-integrations/azure-monitoring-guide "Настройка и конфигурация мониторинга Azure в Dynatrace.")
