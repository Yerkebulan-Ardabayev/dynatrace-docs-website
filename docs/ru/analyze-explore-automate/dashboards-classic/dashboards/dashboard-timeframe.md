Глобальные селекторы для временного диапазона и зоны управления доступны для использования на всех страницах и представлениях. Вы найдёте их в верхнем правом углу.

![Селектор временного диапазона: панель меню](https://dt-cdn.net/images/timeframe-selector-menu-bar-264-8193110c8c.png)

* Нажмите кнопку фильтра, чтобы выбрать новую зону управления
* Нажмите на временной диапазон, чтобы выбрать новый временной диапазон
  Элементы управления селектором временного диапазона

  Глобальный селектор временного диапазона служит временным фильтром, который в большинстве случаев позволяет выбрать конкретный временной диапазон анализа, сохраняющийся на всех страницах и представлениях продукта при навигации.

  ![Селектор временного диапазона: предустановки](https://dt-cdn.net/images/timeframe-selector-basic-355-f0a835da1e.png)

  + Вкладка **Предустановки** перечисляет все доступные стандартные временные диапазоны. Выберите один, чтобы изменить временной диапазон на эту предустановку.
  + Вкладка **Пользовательский** отображает календарь. Щёлкните начальный день, затем конечный день, а потом нажмите **Применить**, чтобы выбрать этот диапазон дат в качестве временного диапазона.

    - Выбранные интервалы календаря заканчиваются в начале следующего дня (время установлено на `00:00`). Например, если вы выберете 3 сентября — 4 сентября в календаре, временной диапазон начнётся 3 сентября в `00:00` и закончится **5** сентября в `00:00`, чтобы вы не пропустили последнюю минуту временного диапазона. Вы можете редактировать отображаемое время.
  + Вкладка **Недавние** отображает недавно использованные временные диапазоны. Выберите один, чтобы вернуться к этому временному диапазону.
  + Элементы управления **<** и **>** сдвигают временной диапазон вперёд или назад во времени. Шаг равен длине исходного временного диапазона. Например, если текущий временной диапазон — `Последние 2 часа` (двухчасовой диапазон, заканчивающийся сейчас), нажмите **<**, чтобы сдвинуть временной диапазон на два часа назад, до `-4ч до -2ч` (двухчасовой диапазон, заканчивающийся два часа назад).
  + Наведите курсор на временной диапазон, чтобы увидеть время начала, продолжительность и время окончания.

    ![Селектор временного диапазона: наведение](https://dt-cdn.net/images/timeframe-selector-hover-168-cfb13dc777.png)

  Выражения селектора временного диапазона

  Если вы выберете текущий временной диапазон в панели меню, отобразится редактируемое выражение временного диапазона.

  + Читая слева направо, выражение временного диапазона содержит время начала, оператор `to` и время окончания.
  + Если явное время окончания не указано, подразумеваются `to` и `now`. Например, `-2h` равносильно `-2h to now`.
  + Поддерживаемые единицы: `s`, `m`, `h`, `d`, `w`, `M`, `q`, `y` (также можно использовать полные слова, такие как `minutes` и `quarter`)

## Сохранение временного диапазона и зоны управления

Выбор временного диапазона и зоны управления является «липким» — он распространяется на все посещаемые страницы. Например, после изменения временного диапазона и зоны управления на панели, выбор сохраняется при переходе от плитки **Приложения** к страницам отдельных приложений.

Исключение — открытие новой панели или возврат к текущей панели при нажатии **Dashboard** в верхнем левом углу страницы. В этом случае временной диапазон и зона управления сбрасываются до [значений по умолчанию](#default) панели.

Селектор временного диапазона запоминает до 10 недавно использованных временных диапазонов.

## Временной диапазон и зона управления по умолчанию

Вы можете установить временной диапазон и зону управления по умолчанию для каждой панели; они выбираются каждый раз при открытии панели. Они также используются при совместном использовании панели.

Вы также можете указать временной диапазон и зону управления по умолчанию для каждой плитки.

### Временной диапазон панели

Чтобы установить временной диапазон панели, переопределяющий глобальный временной диапазон

1. Перейдите на вкладку **Настройки**.
2. Включите **Временной диапазон по умолчанию**.
3. Выберите временной диапазон, который должен быть значением по умолчанию для этой панели.

### Зона управления панели

Чтобы установить зону управления панели, переопределяющую глобальную зону управления

1. Перейдите на вкладку **Настройки**.
2. Включите **Зона управления по умолчанию**.
3. Выберите зону управления, которая должна быть значением по умолчанию для этой панели.

### Временной диапазон плитки

Чтобы установить временной диапазон плитки, переопределяющий временной диапазон панели

1. Перейдите в ![Dashboards Classic](https://dt-cdn.net/images/dashboards-classic-512-15764940e8.png "Dashboards Classic") **Dashboards Classic**.
2. Выберите имя панели для её отображения.
3. Нажмите **Редактировать** в верхнем правом углу панели. Панель откроется в режиме редактирования.

   * Если вы не видите опцию **Редактировать**, у вас нет разрешения на редактирование этой панели.
4. Выберите плитку, которую хотите настроить.
5. Включите **Пользовательский временной диапазон**.
6. Выберите временной диапазон, который должен быть значением по умолчанию для этой плитки.
7. **Готово**.  
   При установке временного диапазона плитки фильтр отображается в верхнем правом углу плитки. Наведите курсор на фильтр, чтобы увидеть настройку.

### Зона управления плитки

Чтобы установить зону управления плитки, переопределяющую зону управления панели

1. Перейдите в ![Dashboards Classic](https://dt-cdn.net/images/dashboards-classic-512-15764940e8.png "Dashboards Classic") **Dashboards Classic**.
2. Выберите имя панели для её отображения.
3. Нажмите **Редактировать** в верхнем правом углу панели. Панель откроется в режиме редактирования.

   * Если вы не видите опцию **Редактировать**, у вас нет разрешения на редактирование этой панели.
4. Выберите плитку, которую хотите настроить.
5. Включите **Пользовательская зона управления**.
6. Выберите зону управления, которая должна быть значением по умолчанию для этой плитки.
7. **Готово**.  
   При установке зоны управления плитки фильтр отображается в верхнем правом углу плитки. Наведите курсор на фильтр, чтобы увидеть настройку.
