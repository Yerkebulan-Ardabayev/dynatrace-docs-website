# Предустановленные дашборды Dynatrace


Dashboards Classic

Эта страница относится к классическим дашбордам, созданным с помощью функциональности ![Dashboards Classic](https://dt-cdn.net/images/dashboards-classic-512-15764940e8.png "Dashboards Classic") **Dashboards Classic**, интегрированной с Dynatrace Classic.

* Если вы уже используете приложение ![Dashboards](https://dt-cdn.net/images/dashboards-512-b1f1e9690b.png "Dashboards") **Dashboards** в последней версии Dynatrace, см. Dashboards для соответствующей документации.
* Если вы всё ещё используете классические дашборды, мы рекомендуем обновить ваши дашборды и воспользоваться всеми новейшими возможностями, предоставляемыми приложением ![Dashboards](https://dt-cdn.net/images/dashboards-512-b1f1e9690b.png "Dashboards") **Dashboards** в последней версии Dynatrace.

Предустановленные дашборды по умолчанию видны всем пользователям.

* На странице **Dashboards** после названия каждого предустановленного дашборда отображается тег `Preset`.
* Чтобы отобразить только предустановленные дашборды, установите фильтр `Preset` в значение `Yes`.

Dynatrace предлагает несколько готовых предустановленных дашбордов для различных предметных областей. Используйте их как источник идей для своих собственных дашбордов и клонируйте их для создания своих настроенных версий.

* Вы можете создавать собственные предустановленные дашборды.
* Вы можете назначить любой предустановленный дашборд домашним дашбордом для группы пользователей.

## Управление глобальными настройками

Предустановленные дашборды по умолчанию видны всем пользователям. Вы можете использовать глобальные настройки, чтобы полностью отключить их или ограничить видимость определёнными группами пользователей.

### Включение предустановленных дашбордов

Используйте **Enable presets**, чтобы включить или отключить предустановленные дашборды глобально. Если вы отключите эту опцию, дашборды, отмеченные как предустановленные, больше не будут отображаться в таблицах **Dashboard** для всех пользователей.

### Ограничение видимости предустановленных дашбордов

Используйте правила предустановленных дашбордов, чтобы обеспечить видимость определённых предустановленных дашбордов только для определённых групп пользователей, а не для всех пользователей среды.

1. Перейдите в **Settings** и выберите **Dashboards** > **Preset settings**.
2. В разделе **Limit preset visibility** нажмите **Add item**.

   * Установите **Preset dashboard** на предустановленный дашборд, для которого вы хотите управлять групповым доступом.
   * Установите **User group** на группу пользователей, которая должна иметь доступ к выбранному предустановленному дашборду.
3. Нажмите **Save changes**.

## Список всех предустановленных дашбордов

Чтобы вывести список всех предустановленных дашбордов

1. Перейдите в ![Dashboards Classic](https://dt-cdn.net/images/dashboards-classic-512-15764940e8.png "Dashboards Classic") **Dashboards Classic**.
2. Установите фильтр одним из следующих способов:

   * В разделе **Preset** в левом столбце выберите `Yes`
   * В строке **Filter by** выберите `Preset: Yes`
3. Выберите название любого предустановленного дашборда, чтобы отобразить его.

## Готовые предустановленные дашборды

Вы не можете редактировать готовые предустановленные дашборды, но можете клонировать их и редактировать клоны.

### Клонирование готового предустановленного дашборда

Используйте **Clone**, чтобы создать вашу личную, редактируемую копию дашборда с суффиксом `-cloned`, добавленным к названию дашборда.

1. Перейдите в ![Dashboards Classic](https://dt-cdn.net/images/dashboards-classic-512-15764940e8.png "Dashboards Classic") **Dashboards Classic**.
2. Найдите готовый предустановленный дашборд, который вы хотите клонировать.
3. Используйте один из следующих способов для клонирования дашборда:

   * Нажмите **More** (**...**) > **Clone** для дашборда, который вы хотите клонировать.
   * Откройте дашборд, нажмите **More** (**...**) в правом верхнем углу дашборда и выберите **Clone**.

Чтобы клонировать дашборд, который вы сейчас просматриваете, выберите **More** (**...**) > **Clone**.

### Просмотр и редактирование клона

Теперь, когда у вас есть копия дашборда, вы можете экспериментировать с ней.

1. Нажмите **Edit**, чтобы просмотреть и отредактировать компоненты дашборда.

   * **Name** -- нажмите на иконку после названия дашборда, чтобы персонализировать имя вашего клона.
   * **Tags** -- под названием дашборда добавляйте и удаляйте теги по необходимости.
   * Плитки **Markdown** и **Header** -- выберите их, чтобы увидеть, как использовать текст и заголовки на вашем дашборде для обозначения и пояснения элементов дашборда.
   * Другие плитки -- выберите плитки, чтобы увидеть настройки в панели справа, специфичной для данной плитки. См. Доступные плитки для справки по конкретным плиткам.
2. Нажмите **Done**, чтобы сохранить изменения и отобразить рабочий дашборд.
3. Попробуйте переходы (drilldown) из плиток, чтобы увидеть, куда они ведут.

   * Откройте меню плитки, чтобы увидеть доступные опции.
   * Выберите элементы плитки, чтобы увидеть доступные действия.

## Создание предустановленного дашборда

Вы можете создавать и изменять собственные предустановленные дашборды.

* Предустановленные дашборды автоматически становятся доступными всем пользователям
* Предустановленные дашборды отображаются в таблице **Dashboards** для всех пользователей

Вам необходимо одно из следующих разрешений:

* Разрешение на уровне среды **Change monitoring settings**
* Роль `ALLOW environment:roles:manage-settings`

Чтобы назначить дашборд предустановленным

1. Создайте дашборд или откройте существующий дашборд, для которого у вас есть права на редактирование.
2. Нажмите **Edit**.
3. Перейдите на вкладку **Settings** и затем нажмите **Configure more**.
4. На вкладке **Manage access** включите **Publish as preset**.
5. Нажмите **Save changes**.

Чтобы проверить изменение

1. Перейдите в ![Dashboards Classic](https://dt-cdn.net/images/dashboards-classic-512-15764940e8.png "Dashboards Classic") **Dashboards Classic**.
2. Отфильтруйте по `Preset: Yes`.
3. Убедитесь, что дашборд отображается в таблице.
   Кроме того, каждый предустановленный дашборд имеет тег `Preset` после его названия в таблице.

## Назначение домашнего дашборда

Если у вас есть права администратора, вы можете назначить предустановленный дашборд домашним дашбордом для группы пользователей. Выбранный дашборд станет страницей по умолчанию для этой группы.

1. Перейдите в **Settings** > **Dashboards** > **General settings**.
2. Нажмите **Configure home dashboard**.
3. Установите **User group** на группу, для которой вы хотите задать домашний дашборд.
4. Установите **Home dashboard** на один из предустановленных дашбордов в списке.
   Если вашего дашборда нет в списке, убедитесь, что он помечен как предустановленный дашборд.
5. Нажмите **Save changes**.
