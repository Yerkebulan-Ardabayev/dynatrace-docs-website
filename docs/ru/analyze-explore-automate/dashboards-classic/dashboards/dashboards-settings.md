## Общие настройки

### Разрешить анонимный доступ

При публичном предоставлении общего доступа даже пользователи, не имеющие доступа к соответствующему окружению Dynatrace или вообще не имеющие доступа к Dynatrace, могут просматривать (но не изменять) ваш дашборд.

1. Перейдите в **Settings** и выберите **Dashboards** > **General settings**.
2. Включите или отключите **Allow anonymous access**, чтобы определить на глобальном (аккаунтном) уровне, могут ли дашборды предоставляться в публичный доступ.
3. Выберите **Save changes**.

### Домашние дашборды

Вы можете назначить предустановленный дашборд в качестве домашнего дашборда для любой группы пользователей. Выбранный дашборд станет стартовой страницей по умолчанию для этой группы.

1. Перейдите в **Settings** и выберите **Dashboards** > **General settings**.
2. Выберите **Configure home dashboard**.
3. Выберите **User group**.
4. Выберите предустановленный дашборд из списка **Home dashboard**.
   Если ваш дашборд не отображается в списке, убедитесь, что он является [предустановленным дашбордом](dashboards-preset.md "Узнайте о встроенных дашбордах Dynatrace и о том, как создавать собственные предустановленные дашборды.").
5. Выберите **Save changes**.

## Настройки предустановленных дашбордов

Используйте эту страницу для настройки параметров предустановленных дашбордов на глобальном уровне.

Предустановленные дашборды по умолчанию видны всем пользователям. Вы можете использовать глобальные настройки, чтобы полностью отключить их или ограничить видимость определёнными группами пользователей.

### Включение предустановленных дашбордов

Используйте **Enable preset**, чтобы глобально включить или отключить предустановленные дашборды. Если вы отключите эту настройку, все дашборды, помеченные как предустановленные, перестанут отображаться в таблицах **Dashboard** других пользователей.

### Ограничение видимости предустановленных дашбордов

Используйте правила для предустановленных дашбордов, чтобы обеспечить их видимость только для определённых групп пользователей, а не для всех пользователей окружения.

1. Перейдите в **Settings** и выберите **Dashboards** > **Preset settings**.
2. В разделе **Limit preset visibility** нажмите **Add item**.

   * Установите **Preset dashboard** в предустановленный дашборд, для которого вы хотите управлять доступом групп.
   * Установите **User group** в группу пользователей, которая должна иметь доступ к выбранному предустановленному дашборду.
3. Выберите **Save changes**.

Подробнее о предустановленных дашбордах см. в разделе [Предустановленные дашборды](dashboards-preset.md "Узнайте о встроенных дашбордах Dynatrace и о том, как создавать собственные предустановленные дашборды.").

## Правила разрешённых URL-шаблонов

Чтобы добавить изображение на дашборд по URL-адресу, сначала необходимо добавить URL-правило в список разрешённых.

1. Перейдите в **Settings** и выберите **Dashboards** > **Allowed URL pattern rules**.
2. Выберите **Add item**.
3. Установите **Rule**, определяющее способ обработки этой записи списка разрешённых.

   * **Starts with** — разрешить любое изображение, URL которого начинается с содержимого поля **Pattern**.
   * **Exact** — разрешить конкретное изображение, URL которого точно совпадает с содержимым поля **Pattern**.
4. Установите **Pattern**:

   * Чтобы указать начало URL-адреса, введите достаточную часть URL, чтобы убедиться, что все соответствующие URL-адреса изображений будут подходящими для ваших дашбордов.
     Пример: введите `https://example.com/images/`, чтобы разрешить любое изображение с URL, начинающимся с `https://example.com/images/`, например
     `https://example.com/images/image-x.jpg` и
     `https://example.com/images/my-picture.svg`
   * Чтобы указать точный URL, введите полный URL изображения, которое вы хотите разрешить.
     Пример: введите `https://example.com/images/my-image-file-name.jpg`, чтобы разрешить только это изображение
5. Выберите **Save changes**, чтобы добавить указанное правило в список разрешённых.

После этого вы можете добавить изображение на дашборд по URL-адресу.

1. Перейдите в ![Dashboards Classic](https://dt-cdn.net/images/dashboards-classic-512-15764940e8.png "Dashboards Classic") **Dashboards Classic**.
2. Выберите имя дашборда, чтобы отобразить его.
3. Выберите **Edit** в правом верхнем углу дашборда. Дашборд откроется в режиме редактирования.

   * Если вы не видите опцию **Edit**, у вас нет прав на редактирование этого дашборда.
4. Перетащите плитку **Image** в нужное место.
5. На панели **Image** выберите вкладку **Add image URL**.
6. Введите URL файла изображения, который вы хотите отобразить в плитке. Он должен соответствовать одному из правил в списке разрешённых.
7. При необходимости измените размер и положение плитки.
