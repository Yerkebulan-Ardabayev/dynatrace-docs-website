# Smartscape views


![Smartscape](https://dt-cdn.net/images/smartscapes-256-eb41c7cddc.png "Smartscape") Представления **Smartscape** позволяют визуализировать и анализировать топологические данные с помощью набора готовых настраиваемых перспектив. Каждое представление использует комплексные топологические и реляционные данные Dynatrace для формирования практических выводов о сложности современных цифровых систем, обеспечивая их актуальность для конкретных сценариев использования.

Независимо от того, изучаете ли вы связи между сервисами, понимаете зависимости инфраструктуры или визуализируете влияние проблем, ![Smartscape](https://dt-cdn.net/images/smartscapes-256-eb41c7cddc.png "Smartscape") **Smartscape** гарантирует, что у вас есть нужная перспектива для решения конкретных задач.
