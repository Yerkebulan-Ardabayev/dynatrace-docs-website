# Visual Resolution Path


* Latest Dynatrace
* Explanation
* 2-min read
* Published Jan 28, 2026

Visual Resolution Path (VRP) -- это модальное окно ![Smartscape](https://dt-cdn.net/images/smartscapes-256-eb41c7cddc.png "Smartscape") **Smartscape**, которое позволяет вам получить доступ к пути разрешения проблемы непосредственно из любого приложения на платформе Dynatrace. В представлении Visual Resolution Path вы можете увидеть путь разрешения проблемы и получить немедленное представление о проблеме, не покидая текущий контекст и не переходя в ![Problems app - new](https://dt-cdn.net/images/dynatrace-davis-new-256-340162f8c6.webp "Problems app - new") **Problems**.

Из Visual Resolution Path вы можете запускать дополнительные действия и углублённый анализ для более детального исследования. Наряду с визуализированными сущностями Visual Resolution Path, он включает временную шкалу базовых событий проблемы. Это обеспечивает всестороннее контекстное представление проблемы и её пути разрешения.

Для получения дополнительной информации о проблемах и их разрешении см. [Анализ первопричин](../../../dynatrace-intelligence/root-cause-analysis.md "Как Dynatrace анализирует проблемы для определения их первопричины.").

## Поддержка Visual Resolution Path в приложениях Dynatrace

Из деталей проблемы в любом приложении Dynatrace вы можете перейти к Visual Resolution Path.

У некоторых проблем нет обнаруженной первопричины. Если первопричина проблемы не была обнаружена, Visual Resolution Path не будет отображаться.

Ниже приведены примеры вызова Visual Resolution Path:

* [![Smartscape](https://dt-cdn.net/images/smartscapes-256-eb41c7cddc.png "Smartscape") **Smartscape**](../../smartscape.md "Визуализируйте структуру ваших сред и понимайте взаимосвязи и зависимости между сущностями ваших сервисов."): вы можете вызвать Visual Resolution Path непосредственно из узла проблемы в [представлении Problem graph](../smartscape-views/problem-graph.md "Используйте Problem graph для визуализации и быстрого выявления проблем и узлов, требующих немедленного внимания.").
* [![Problems app - new](https://dt-cdn.net/images/dynatrace-davis-new-256-340162f8c6.webp "Problems app - new") **Problems**](../../../dynatrace-intelligence/davis-problems-app.md "Используйте приложение Problems для быстрого определения первопричины инцидентов в вашей среде."): вы можете вызвать Visual Resolution Path, выбрав **Maximize** в разделе **Visual resolution path** на вкладке обзора.

## Обзор

Представление Visual Resolution Path состоит из выбранной сущности проблемы, представленной в виде группы на графе, затронутых сущностей, представленных в виде узлов, и связей между узлами, представленных в виде рёбер.

### Группа

Проблема в Visual Resolution Path представлена как группа на графе. Цвет группы отражает текущее состояние проблемы. Вы также можете просмотреть дополнительные сведения о самой проблеме во всплывающей подсказке, наведя курсор на заголовок группы проблемы.

### Узлы

В Visual Resolution Path сущности, вовлечённые в проблему, расположены внутри группы проблемы и отображаются как узлы. Следующие сущности могут быть найдены внутри группы проблемы:

* Фронтенды
* Сервисы

### Рёбра

Рёбра в графе Visual Resolution Path представляют дерево зависимостей и связи между сущностями. Эти визуальные соединения помогают понять, как различные компоненты и сервисы связаны между собой.

Для получения дополнительной информации см. [Анализ первопричин](../../../dynatrace-intelligence/root-cause-analysis.md "Как Dynatrace анализирует проблемы для определения их первопричины.").

### Временная шкала событий

В дополнение к графовому представлению узлов, модальное окно Visual Resolution Path содержит временную шкалу событий. Свёрнутое представление временной шкалы предлагает общий обзор проблемы, включая время начала всех связанных событий и полный временной диапазон проблемы.

Развернув временную шкалу, вы можете просмотреть индивидуальные временные шкалы событий для каждой из вовлечённых сущностей. Это предоставляет детальную информацию о том, как события развивались во времени, и помогает лучше понять прогрессию проблемы.

## Сценарии использования

Вы можете использовать ![Smartscape](https://dt-cdn.net/images/smartscapes-256-eb41c7cddc.png "Smartscape") **Smartscape** Visual Resolution Path для:

* Просмотра ключевых сведений о проблеме, включая её статус, категорию и продолжительность.
* Просмотра временной шкалы событий для лучшего понимания прогрессии проблемы во времени.
* Выявления каскадных сбоев путём анализа того, как проблемы в одном сервисе распространяются на другие связанные сервисы или приложения.

## Рекомендации по эффективному использованию

Чтобы максимизировать эффективность при использовании ![Smartscape](https://dt-cdn.net/images/smartscapes-256-eb41c7cddc.png "Smartscape") **Smartscape** Visual Resolution Path, вы можете:

* Навести курсор на группу проблемы для просмотра ключевых сведений о проблеме.
* Использовать всплывающую подсказку проблемы для:

  + Объяснения проблемы с помощью Dynatrace Intelligence.
  + Открытия проблемы в специализированном приложении ![Problems app - new](https://dt-cdn.net/images/dynatrace-davis-new-256-340162f8c6.webp "Problems app - new") **Problems** для выполнения расширенного анализа и устранения неполадок.
* Углубления в конкретные сущности непосредственно из Visual Resolution Path для лучшего понимания узких мест производительности, ошибок или сбоев. Это упрощает анализ первопричин, направляя вас к затронутым системам и их зависимостям.
