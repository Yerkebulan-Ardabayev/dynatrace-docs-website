# Отправка уведомлений Dynatrace в Trello

* Classic
* 2-min read
* Updated on Oct 10, 2022

Atlassian Trello предлагает удобный визуальный способ организации проектов, независимо от того, какой agile-процесс вы применяете. Всё в Trello организовано в карточках. Карточки содержат подробную информацию о задачах, историях или ошибках, которые должна решить ваша команда.

Dynatrace предлагает удобную интеграцию с Trello, позволяющую визуально организовывать все автоматически обнаруженные инциденты прямо на досках Trello. Подключите среду мониторинга Dynatrace к своей доске Trello и напрямую передавайте обнаруженные проблемы Dynatrace в указанный список.

Чтобы настроить интеграцию уведомлений о проблемах с Trello

1. Перейдите в **Settings** > **Integration** > **Problem notifications**.
2. Выберите **Add notification**.
3. Выберите **Trello** из доступных типов уведомлений.
4. Настройте интеграцию:

   * **Display name** — произвольное название этой интеграции, которое будет отображаться в Dynatrace в разделе **Settings** > **Integration** > **Problem notifications** после завершения настройки.
   * Введите ключ приложения Trello.

     Ключ приложения Trello

     Войдите в Trello и позвольте Trello автоматически сгенерировать [ключ приложения](https://trello.com/app-key), который можно использовать здесь.
   * Введите токен авторизации Trello.
   * Укажите доску Trello, которой должны быть назначены карточки проблем.
   * Укажите список Trello, в который должны быть добавлены новые карточки проблем.
   * Укажите список Trello, в который должны быть перемещены карточки решённых проблем.
   * Введите текст карточки и заполнители проблем, которые должны отображаться на новых карточках проблем.

     Заполнители

     Раздел **Available placeholders** страницы настройки содержит список заполнителей, которые можно использовать для этой интеграции. Заполнители автоматически заменяются фактическими значениями в тексте карточки.
   * Введите описание карточки.
   * Назначьте [профиль оповещения](../alerting-profiles.md "Узнайте, как создавать профили оповещений и управлять ими.").
5. Выберите **Send test notification**, чтобы убедиться в работоспособности интеграции с Trello. В Trello должно поступить тестовое уведомление.
6. Если тест прошёл успешно, нажмите **Save changes**.
