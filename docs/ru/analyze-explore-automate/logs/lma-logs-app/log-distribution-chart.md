# Spot trends with the log distribution chart

# Выявление тенденций с помощью графика распределения логов

* Latest Dynatrace
* How-to guide
* 2-min read
* Published Jul 02, 2025

Используйте график распределения логов, доступный в ![Logs](https://dt-cdn.net/images/logs-256-ae0a9ca67f.png "Logs") **Logs**, чтобы выявлять тенденции в ваших логах. Вы можете:

* Быстро обнаруживать тенденции и аномалии. Например, внезапное увеличение логов `ERROR` может свидетельствовать об инциденте или регрессии.
* Детализировать конкретные статусы логов или временные диапазоны.
* Выполнять целевые запросы, не покидая визуализацию. Взаимодействуя с графиком, вы можете выбрать статус лога и более точный временной диапазон для следующего запроса.

![Logs app showing the log distribution chart](https://dt-cdn.net/images/logs-app-chart-3840-33eaf4738e.png)

## Обзор графика

График распределения логов предоставляет визуальный обзор записей логов за определённый период времени.

Записи логов группируются по статусу. Каждый статус представлен отдельным цветом для удобства различия. Доступны следующие статусы:

* `INFO`
* `WARN`
* `ERROR`
* `NONE`

Эти статусы соответствуют атрибуту `status`, который создаётся в процессе приёма логов. Подробности см. в информации об автоматическом обогащении логов для логов, принятых через [OneAgent](../lma-log-ingestion/lma-log-ingestion-via-oa/lma-log-data-transformation-oa.md#transform-all-types-of-logs "Generic log ingestion automatically transforms log data into output values for the loglevel attribute.") или [API](../lma-log-ingestion/lma-log-ingestion-via-api/lma-log-data-transformation.md#transform-all-types-of-logs "Log ingestion API automatically transforms log data into output values for the loglevel attribute.").

## Взаимодействие с графиком распределения логов

Вы можете взаимодействовать с графиком распределения логов для уточнения анализа.

* **Выделение конкретных статусов логов**.

  + Выберите статус на графике или в легенде, чтобы сосредоточиться только на этом статусе.
  + Для отображения нескольких статусов используйте легенду графика.
* **Увеличение временного диапазона**. Выделите фрагмент графика, затем выберите **Zoom to selection**.
* **Использование панели инструментов графика**. Наведите курсор на график, чтобы отобразить панель инструментов.

  + Выберите (**Change mode**), чтобы переключаться между режимами **Explore mode** (выделение отдельных статусов), **Zoom mode** и **Pan mode** (горизонтальное перемещение по графику).
  + Увеличьте или уменьшите масштаб графика.
  + Выберите (**Reset**), чтобы вернуть график в исходное состояние.

Для более быстрого и удобного взаимодействия с графиком распределения логов используйте сочетания клавиш. Они отображаются при наведении курсора на значки панели инструментов.

## Влияние на тарификацию

Загрузка и взаимодействие с графиком распределения логов не потребляет лицензию на запросы.

График распределения логов может быть основан на выборочных данных, что означает: отображаемые данные являются репрезентативными, и не каждая запись лога отображается.
