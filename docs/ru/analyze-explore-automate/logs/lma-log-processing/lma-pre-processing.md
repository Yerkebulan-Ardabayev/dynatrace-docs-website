Предварительная обработка логов с помощью OpenPipeline с готовыми технологическими наборами позволяет обогащать и нормализовать данные, обеспечивая более быстрый и масштабируемый анализ логов для популярных технологий.

## Преимущества использования готовых наборов

* Автоматическое обогащение логов из поддерживаемых технологий без ручной настройки.
  Это добавляет контекст для упрощения диагностики и позволяет группировать, фильтровать и выполнять поиск по значимым атрибутам, а не только по необработанному тексту.
* Улучшенная структура логов и метаданные для более эффективной фильтрации и запросов.
* Упрощённая настройка и обработка для новых тенантов со встроенной поддержкой стандартных технологических стеков.

## Ключевые преимущества

* Логи следуют единой структуре.
* Упрощённые запросы для более быстрых и точных результатов.
* Единообразная интерпретация на протяжении всего процесса анализа логов.

## Готовые наборы для популярных технологий

Существует список готовых наборов для популярных технологий, которые можно найти как **Источники приёма данных** (Ingest sources).

Преимущества наборов **Ingest sources** следующие:

* Обеспечивают централизованную предварительную обработку логов, позволяя масштабируемую и гибкую обработку.
* Автоматически применяют предопределённые технологические наборы к выбранным **Ingest sources** со встроенными правилами разбора и обогащения.

Технологические наборы применяются автоматически и не могут быть настроены.

### Список технологических наборов

Чтобы найти список готовой поддержки для популярных технологий в виде **Ingest sources**, перейдите в ![Settings](https://dt-cdn.net/images/settings-icon-256-38e1321b51.webp "Settings") **Settings** > **Process and contextualize** > **OpenPipeline** > **Logs**.
