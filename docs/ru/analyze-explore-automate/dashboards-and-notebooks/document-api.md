# API для панелей управления и ноутбуков

* Последняя Dynatrace
* Справочник
* 2-минутное чтение
* Опубликовано 23 апреля 2024 г.

Платформа Dynatrace предоставляет коллекцию [сервисов платформы](https://dt-url.net/sx23ug5), каждый из которых имеет определенную область ответственности. Вам нужен один из этих сервисов, [сервис документов](https://dt-url.net/x403ua9), чтобы управлять документами Dynatrace, такими как панели управления и ноутбуки, через API.

## Доступ к данным документов

Приложения ![Панели управления](https://dt-cdn.net/images/dashboards-512-b1f1e9690b.png "Панели управления") **Панели управления** и ![Ноутбуки](https://dt-cdn.net/images/notebooks-768-046137830a.webp "Ноутбуки") **Ноутбуки** используют API сервиса документов для сохранения данных в виде документов.

Документы являются независимыми от содержимого, но документы как ноутбуков, так и панелей управления имеют общие метаданные: уникальный идентификатор, имя и описание.

Чтобы различать документы ноутбуков и панелей управления и чтобы сделать их видимыми для соответствующих приложений, им необходим атрибут `type` документа: `dashboard` или `notebook`. Этот атрибут также можно использовать для запроса API, как показано в следующих примерах.

#### Список всех доступных панелей управления

```
https://environment/platform/document/v1/documents?filter=type='dashboard'
```

#### Список всех доступных ноутбуков

```
https://environment/platform/document/v1/documents?filter=type='notebook'
```

## Доступ к полной документации сервиса документов

Чтобы просмотреть полную документацию API для сервиса документов

1. Перейдите на страницу [Сервис документов](https://dt-url.net/x403ua9) сайта [Dynatrace Developer](https://developer.dynatrace.com/).
2. В разделе **Связанные ссылки** выберите ссылку **Swagger API**.

   Возможно, вам потребуется войти в свою среду Dynatrace.
