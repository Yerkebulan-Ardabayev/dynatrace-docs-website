# Готовые панели мониторинга

* Последнее Dynatrace
* Справка
* 9-минутное чтение

Готовые панели мониторинга Dynatrace предлагают предварительно настроенные визуализации данных и фильтры, разработанные для общих сценариев, таких как устранение неполадок и оптимизация.

* Используйте их сразу же
* Сохраните копию и настройте свою копию

Где найти готовые панели мониторинга

1. В Dynatrace, перейдите к ![Панели мониторинга](https://dt-cdn.net/images/dashboards-512-b1f1e9690b.png "Панели мониторинга") **Панели мониторинга**.
2. Выберите способ отображения всех готовых панелей мониторинга.

   ![Панели мониторинга: Выберите вкладку "Готовые"](https://dt-cdn.net/images/dashboards-select-ready-made-tab-469-5224ee5ca2.png)

   ![Выберите "Готовые панели мониторинга" в списке недавних панелей мониторинга](https://dt-cdn.net/images/ready-made-dashboards-button-under-recent-dashboards-263-a29f7fe076.png)
3. Выберите готовую панель мониторинга, которую вы хотите использовать.  
   Попробуйте ссылки **Explore in Playground** ниже, чтобы увидеть их в действии.

Использование панелей мониторинга только для чтения

Когда вы открываете документ (панель мониторинга или тетрадь), для которого у вас нет разрешения на запись, вы все равно можете редактировать документ во время сеанса. После завершения у вас есть два варианта:

* Сохранить изменения в новый документ
* Отменить изменения

Пример:

1. Перейдите к ![Панели мониторинга](https://dt-cdn.net/images/dashboards-512-b1f1e9690b.png "Панели мониторинга") **Панели мониторинга**, отобразите готовые панели мониторинга и выберите панель мониторинга **Начало работы**.

   В верхнем левом углу, рядом с именем документа, написано **Готовая**.
2. Выберите плитку круговой диаграммы и затем выберите **Редактировать**.
3. Измените визуализацию с круговой диаграммы на пончик.
   
   Теперь у вас есть две кнопки: **Сохранить как новый** и **Отменить изменения**.
4. Используйте обновленную панель мониторинга по мере необходимости. У вас есть полный доступ к редактированию для этой сессии.
5. Когда вы закончите, выберите, что делать с вашими изменениями:

   * **Сохранить как новый** — сохраняет ваши изменения в новой копии отредактированной панели мониторинга.
   * **Отменить изменения** — отменяет ваши изменения и возвращает вас к неизменной панели мониторинга только для чтения.
