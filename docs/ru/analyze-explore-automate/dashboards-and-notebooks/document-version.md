Версии документов (notebook или dashboard) сохраняются автоматически. Доступно до 50 последних версий, каждая хранится 30 дней.

## Меню History

1. Откройте документ.
2. Нажмите ![History](https://dt-cdn.net/images/history-icon-dc67cc1e2c.svg "History") в правом верхнем углу.

Для каждой версии доступны действия:

| Действие | Описание |
| --- | --- |
| **Preview** | Предварительный просмотр версии (изменения не сохраняются) |
| **Restore** | Переключение документа на выбранную версию |
| **Make a copy** | Создание нового документа из версии (оригинал не меняется) |
| **Download** | Сохранение JSON-файла версии локально |
| **Preview in new tab** | Просмотр версии в новой вкладке |
| **Delete this version** | Удаление версии |

Выберите **Show all** внизу меню для отображения таблицы **Version history** со всеми версиями (до 50). Таблица содержит столбцы: Version (ID), Updated on (дата), Updated by (автор), Actions.
