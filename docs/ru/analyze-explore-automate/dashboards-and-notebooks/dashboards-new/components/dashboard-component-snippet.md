# Добавьте сниппет на панель мониторинга

* Последнее Dynatrace
* Руководство по действию
* 10-минутное чтение

Несколько сниппетов данных и кода доступны сразу. Используйте наши предопределенные DQL или сниппеты кода, чтобы быстро начать ваше путешествие по анализу данных.

## Выберите сниппет

Чтобы начать работу на основе сниппета

1. В правом верхнем углу панели мониторинга выберите **Добавить**, чтобы открыть меню **Добавить**.

   Горячая клавиша: **Ctrl**/**Cmd**+**Shift**+**Enter**

   ![Панели мониторинга: Кнопка добавления плитки (Плюс)](https://dt-cdn.net/images/updated-dashboards-add-tile-button-481-c21ba8f200.png)
2. Прокрутите вниз до раздела **Начать со сниппета** и выберите один из сниппетов. В этом примере мы выбираем сниппет **Получить журналы**, который отображается в панели предварительного просмотра.

   ![Пример: выберите сниппет "Получить журналы".](https://dt-cdn.net/images/add-snippet-menu-example-fetch-logs-754-d4435c09eb.png)
3. После выбора сниппета панель редактирования отображает добавленный сниппет.

   ![Пример: сниппет "Получить журналы" добавлен.](https://dt-cdn.net/images/add-snippet-menu-example-fetch-logs-result-1436-77cc9fa8f1.png)
4. Отредактируйте запрос или код (в зависимости от типа сниппета, который вы выбрали) и настройки визуализации по мере необходимости.
5. Выберите **Выполнить**, чтобы увидеть результаты.
6. Закройте боковую панель, когда закончите.

## Примеры

### Получить журналы

1. В ![Панели мониторинга](https://dt-cdn.net/images/dashboards-512-b1f1e9690b.png "Панели мониторинга") **Панели мониторинга** выберите **Панель мониторинга**, чтобы создать новую панель мониторинга.
2. В правом верхнем углу панели мониторинга выберите **Добавить**, чтобы открыть меню добавления.

   ![Панели мониторинга: Кнопка добавления плитки (Плюс)](https://dt-cdn.net/images/updated-dashboards-add-tile-button-481-c21ba8f200.png)
3. В меню прокрутите вниз до раздела **Начать со сниппета** меню и выберите **Получить журналы**.

Всё. Вы создали панель мониторинга с плиткой DQL, которая получает 100 последних журналов и отображает их, начиная с наиболее недавних.

![Пример сниппета: Получить журналы](https://dt-cdn.net/images/example-snippet-fetch-logs-1335-7f23a8343c.png)

DQL за вашей плиткой является сниппетом, который вы выбрали с **Получить журналы**:

```
fetch logs


| sort timestamp desc


| limit 100
```

Команды DQL выполняются в порядке:

1. `fetch logs` получает данные журнала
2. `sort timestamp desc` сортирует данные по метке времени в порядке убывания
3. `limit 100` ограничивает результаты до 100.

### Диаграмма (Топ 5 событий по типу)

1. В ![Панели мониторинга](https://dt-cdn.net/images/dashboards-512-b1f1e9690b.png "Панели мониторинга") **Панели мониторинга** выберите **Панель мониторинга**, чтобы создать новую панель мониторинга.
2. В правом верхнем углу панели мониторинга выберите **Добавить**, чтобы открыть меню добавления.

   ![Панели мониторинга: Кнопка добавления плитки (Плюс)](https://dt-cdn.net/images/updated-dashboards-add-tile-button-481-c21ba8f200.png)
3. В меню прокрутите вниз до раздела **Начать со сниппета** меню и выберите **Диаграмма (Топ 5 событий по типу)**.

Это создает плитку на основе следующего DQL:

```
fetch events


| summarize `event count`=count(), by:{event.kind}


| limit 5
```

Результаты отображаются в виде круговой диаграммы.

![Пример сниппета: Диаграмма (Топ 5 событий по типу)](https://dt-cdn.net/images/example-snippet-pie-top-five-events-1004-a42b6a2b21.png)
