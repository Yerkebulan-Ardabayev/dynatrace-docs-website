# Добавление Markdown в панель управления

* Последнее Dynatrace

Плитка Markdown может быть чем угодно, от незначительной заметки о чем-то на панели управления до целой страницы отформатированной информации с ссылками и изображениями.

## Добавление плитки Markdown

Чтобы добавить плитку Markdown в панель управления

1. В правом верхнем углу панели управления откройте меню **Добавить** и выберите **Добавить Markdown**.

   Клавишная комбинация: **Shift**+**M**

   ![Панели управления: Кнопка добавления плитки (Плюс)](https://dt-cdn.net/images/updated-dashboards-add-tile-button-481-c21ba8f200.png)

   Открывается боковая панель **Параметры** справа.
2. В боковой панели **Параметры** введите свой текст.

   * Используйте Markdown для форматирования текста и добавления ссылок и изображений.
   * Плитка обновляется по мере редактирования.
   * При редактировании текста нажмите Ctrl+Space, чтобы увидеть варианты.
3. Закройте боковую панель **Параметры**, когда закончите.

## Синтаксис

Используйте Markdown для форматирования аннотаций.

При наборе текста в поле редактирования ваш Markdown отображается в плитке.

Нажмите Ctrl+Space на пустой строке, чтобы появился список доступных элементов Markdown.

![Как редактировать плитку Markdown](https://dt-cdn.net/images/markdown-editing-1105-1fc4ba5612.png)

### Курсив

Оберните текст в одинарные звездочки (`*так*`) чтобы получить курсив *так*.

### Жирный

Оберните текст в двойные звездочки (`**так**`) чтобы получить жирный текст **так**.

### Зачеркнутый

Оберните текст в двойные тильды (~~так~~) чтобы получить зачеркнутый (перечеркнутый) текст ~~так~~.

### Цитата

Начните строку с `>` чтобы получить цитаты, где все до нажатия Enter цитируется.

### Код

* Оберните текст в обратные кавычки (`так`) чтобы получить жирный текст `так`.
* Оберните текст в тройные обратные кавычки (```так```) чтобы показать блок кода (несколько строк `текста кода`).

### Заголовки

Чтобы добавить заголовок, начните строку с символа `#` как это: `# Это ваш текст заголовка`.

### Горизонтальная линия

Чтобы визуально разделить разделы вашей аннотации, добавьте горизонтальную линию с тремя дефисами (`---`):

### Списки

Каждая строка неупорядоченного (маркированного) списка начинается с звездочки (`*`):

```
* Строка 1



* Строка 2
```

Альтернативно, можно использовать дефис (`-`):

```
- Строка 1



- Строка 2
```

Упорядоченный (нумерованный) список начинается с номера и точки (`1.`) за которой следует пробел и затем ваш текст:

```
1. Первая строка моей процедуры.



2. Вторая строка моей процедуры.



3. Третья строка моей процедуры.
```

Если вы используете `1.` для каждого номера строки, строки будут перенумерованы автоматически при отображении панели управления.

### Таблицы

Чтобы добавить таблицу, определите заголовки, строку форматирования столбцов, и затем строки данных, которые вы хотите отобразить

```
| Заголовок 1 | Заголовок2



--- | ---



содержимое2 | содержимое2
```

| Заголовок 1 | Заголовок2 |
| --- | --- |
| содержимое2 | содержимое2 |

### Ссылки

Вы можете ссылаться на другие места в вашей панели управления и в других местах.

Ссылка в Markdown представляет собой метку и ссылку в форме `[метка](адрес)`, где:

* `метка` — свободный текст для отображения на ссылке в вашей плитке Markdown или разделе
* `адрес` указывает, что открыть при выборе ссылки, например, веб-сайт или приложение Dynatrace
### Изображения

Чтобы ссылаться на картинку, используйте следующий формат:

`![альтернативный текст](https://www.example.com/имя_файла.jpg)`

```
Вот некоторые из людей, которые начали [Dynatrace](https://www.dynatrace.com).



![основатели Dynatrace](https://dt-cdn.net/images/original-dynatrace-team-1500-7334dbe9a8.jpg)
```

### Переменные

Чтобы сослаться на [переменную](dashboard-component-variable.md "Добавить переменные в ваши панели управления Dynatrace.") в аннотации, используйте имя переменной, предваряемое символом `$`. Например:

```
Статус: $Status.
```

Если у панели управления есть переменная с именем `Status` и текущее значение `Status` равно `Good`, приведенный выше пример будет отображаться в вашей плитке как "Статус: Good." и будет обновляться при изменении значения `Status`.

Для получения подробной информации о переменных см. [Добавление переменной в панель управления](dashboard-component-variable.md "Добавить переменные в ваши панели управления Dynatrace.").
