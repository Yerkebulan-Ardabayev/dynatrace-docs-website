# Действия рабочих процессов


* Последняя Dynatrace
* Обзор
* 1 мин. чтения
* Опубликовано 2 апреля 2024 г.

Действие — это универсальная, многократно используемая функция, настраиваемая и запускаемая [задачей](../workflows.md#concept-task "Автоматизируйте ИТ-процессы с помощью Dynatrace Workflows — реагируйте на события, планируйте задачи и подключайте сервисы."). Например, действие может быть **Запустить JavaScript**, а задача будет выполнять действие с определённой конфигурацией для получения деталей проблемы. Действия — это инструменты, которые собираются в ваш рабочий процесс для настройки под ваш пользовательский процесс. Действия предоставляются Dynatrace «из коробки» и могут быть установлены через Dynatrace Hub.
