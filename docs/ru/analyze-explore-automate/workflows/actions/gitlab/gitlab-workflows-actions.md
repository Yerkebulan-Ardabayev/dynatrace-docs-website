## Создание файла

Создаёт новый файл в новой ветке на основе указанной базовой ветки.

Действие может работать только с одним файлом и приводит к созданию новой ветки. API GitLab не позволяет повторно использовать существующие ветки. Любая попытка сделать это приведёт к ошибке API GitLab.

### Входные данные

| Поле | Описание | Обязательное |
| --- | --- | --- |
| **Start branch** | Начальная ветка, на основе которой создаётся новая ветка. | Обязательное |
| **Branch** | Имя новой ветки. | Обязательное |
| **File path** | Относительный путь к создаваемому файлу. | Обязательное |
| **File content** | Содержимое нового файла. | Обязательное |
| **Commit message** | Сообщение коммита для нового файла. | Обязательное |
| **Author name** | Имя автора результирующего коммита. | Необязательное |

### Выходные данные

Возвращает поля, описанные в документации GitLab [Repository files API - Create new file in repository](https://dt-url.net/7o43w88).

## Получение файла

Получает файл из репозитория.

Это действие следует использовать только для получения содержимого текстового файла.

### Входные данные

| Поле | Описание | Обязательное |
| --- | --- | --- |
| **File path** | Относительный путь к файлу. | Обязательное |
| **Ref** | Ссылка, из которой извлекается файл. Может быть именем ветки или хешем коммита. | Обязательное |

### Выходные данные

Возвращает поля, описанные в документации GitLab [Repository files API - Get file from repository](https://dt-url.net/l863wwz).

## Обновление файла

Обновляет файл в вашем репозитории GitLab. Это действие создаёт новую ветку на основе указанной базовой ветки.

Действие может работать только с одним файлом и приводит к созданию новой ветки. API GitLab не позволяет повторно использовать существующие ветки. Любая попытка сделать это приведёт к ошибке API GitLab.

### Входные данные

| Поле | Описание | Обязательное |
| --- | --- | --- |
| **Start branch** | Начальная ветка, на основе которой создаётся новая ветка. | Необязательное |
| **Branch** | Имя новой ветки. | Обязательное |
| **File path** | Относительный путь к обновляемому файлу. | Обязательное |
| **File content** | Обновлённое содержимое файла. | Обязательное |
| **Commit message** | Сообщение коммита для редактирования. | Обязательное |
| **Author name** | Имя автора результирующего коммита. | Необязательное |

### Выходные данные

Возвращает поля, описанные в документации GitLab [Repository files API - Update existing file in repository](https://dt-url.net/6o83wg0).

## Получение мерж-реквеста

Получает отдельный мерж-реквест в GitLab.

### Входные данные

| Поле | Описание | Обязательное |
| --- | --- | --- |
| **Internal ID** | Внутренний идентификатор мерж-реквеста в GitLab. | Обязательное |
| **Commits behind the target branch** | Включите переключатель для отображения коммитов, отстающих от целевой ветки. | Необязательное |
| **Rebase operation in progress** | Включите переключатель для отображения текущей операции rebase. | Необязательное |

### Выходные данные

Возвращает поля, описанные в документации GitLab [Merge requests API - Get single MR](https://dt-url.net/u8e3wkb).

## Слияние мерж-реквеста

Выполняет слияние мерж-реквеста в вашем репозитории GitLab.

### Входные данные

| Поле | Описание | Обязательное |
| --- | --- | --- |
| **Merge request** | Мерж-реквест для слияния. | Обязательное |
| **SHA** | Хеш предполагаемого последнего коммита в мерж-реквесте. Если фактический последний коммит отличается, действие завершится ошибкой. | Необязательное |
| **Enable squash merge** | Включите переключатель для использования стратегии squash merge или выключите для стратегии merge commit. | Необязательное |

### Выходные данные

Возвращает поля, описанные в документации GitLab [Merge requests API - Merge a merge request](https://dt-url.net/myi3wx9).

| --- | --- | --- |
| **Title** | Заголовок новой задачи. | Обязательное |
| **Labels** | Список меток через запятую для добавления к задаче. | Необязательное |

## Список задач

Получает список задач из вашего репозитория GitLab.

### Входные данные

| Поле | Описание | Обязательное |
| --- | --- | --- |
| **Issue Ids** | Показать только задачи с указанными идентификаторами, разделёнными запятыми. | Необязательное |
| **Title or Description** | Показать только задачи, содержащие указанный текст в заголовке или описании. | Необязательное |
| **Issue type** | Показать только задачи указанного типа. | Необязательное |
| **Labels** | Показать только задачи, которым назначены все указанные через запятую метки. | Необязательное |
| **State** | Укажите, хотите ли вы получить список открытых или закрытых задач. | Необязательное |

### Выходные данные

Возвращает поля, описанные в документации GitLab [Issues API - List issues](https://dt-url.net/kqq3wj6).
