# Действие DQL-запроса для Workflows

* Latest Dynatrace
* Reference
* 1-min read
* Published Apr 02, 2024

Действие **Execute DQL Query** для Workflows позволяет использовать DQL-запросы в рабочем процессе. В поле **DQL query** введите ваш [DQL-запрос](/docs/platform/grail/dynatrace-query-language "Как использовать Dynatrace Query Language."). В поле **Segment** можно дополнительно выбрать сегмент фильтрации.

Результаты можно найти на вкладке **Result** в разделе **Execution** — их можно использовать в последующих задачах.
