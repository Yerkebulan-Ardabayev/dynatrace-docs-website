# Коннекторы Workflows

* Latest Dynatrace
* Обзор
* Чтение: 3 мин
* Обновлено 17 февраля 2026 г.

Коннекторы Workflows можно выбрать из каталога Dynatrace Platform. Коннекторы Workflows предоставляют различные действия, которые можно использовать в задачах рабочего процесса. Например, можно использовать действия для создания задач в Jira, отправки уведомлений в Slack и маршрутизации на основе извлечённой информации о владельцах сущностей.

Действия Ownership позволяют извлекать контактную информацию о владельцах сущностей и использовать её в рабочих процессах для уведомлений и назначения задач.](../../deliver/ownership-app.md "Предоставляет пользовательские действия для определения рабочих процессов, интегрирующих владельцев сущностей и их контактную информацию.")[![Site Reliability Guardian](https://dt-cdn.net/images/site-reliability-guardian-ec19b393a6.svg "Site Reliability Guardian")
