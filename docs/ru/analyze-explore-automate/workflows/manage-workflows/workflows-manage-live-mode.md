# Управление активными или черновыми рабочими процессами


* Latest Dynatrace
* How-to guide
* 1-min read

Чтобы получить доступ к различным командам управления рабочими процессами в режиме **Live** или **Draft**:

1. Перейдите в ![Workflows](https://dt-cdn.net/images/workflows-1024-b5708f3cf9.webp "Workflows") **Workflows**.
2. Найдите нужный рабочий процесс.
3. Выберите **Edit in draft** или **Switch to live**.
4. Рядом с кнопкой **Switch to live** откройте меню.
5. Выберите команду.

   * **Workflow** — см. Загрузить рабочий процесс или шаблон
   * **Template** — см. Загрузить рабочий процесс или шаблон
   * **Compare draft changes** — сравнить **Draft** с текущей версией **Live** рабочего процесса.
   * **Undeploy workflow** — рабочий процесс переводится только в режим черновика; существующий **Draft** будет заменён. Автоматический запуск прекращается.
   * **Delete draft** — удаляет только **Draft** рабочего процесса.
