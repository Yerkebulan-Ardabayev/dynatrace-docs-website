# Доступ к функциям управления рабочими процессами


* Latest Dynatrace
* How-to guide
* 1-min read

Для доступа к различным командам управления рабочими процессами:

1. Найдите нужный рабочий процесс и нажмите на соответствующий элемент управления.
2. Выберите команду. Список доступных вам команд зависит от ваших разрешений.

   * **Edit workflow** — см. Create workflows in Dynatrace Workflows
   * **Run workflow** — см. Run and monitor workflows created in Dynatrace Workflows
   * **View execution history** — отображает список выполнений рабочего процесса
   * **Duplicate workflow** — создаёт копию выбранного рабочего процесса
   * **Make public** и **Make private** — включают и выключают общий доступ. Любой пользователь с соответствующими разрешениями может получить доступ к публичному рабочему процессу.
   * **Transfer ownership** — см. Change ownership of a workflow
   * **Version history** — см. Compare and restore versions of a workflow
   * **Workflow** — см. Download a workflow or template
   * **Template** — см. Download a workflow or template
   * **Delete workflow** — удаляет выбранный рабочий процесс и все доступные **черновики**.
   * **Delete draft** — см. удаление только черновика рабочего процесса.
