* 3 мин. чтения

Начиная с 8 декабря 2020 года Apple обязывает вас предоставлять информацию о практиках конфиденциальности вашего приложения, включая практики сторонних партнёров, таких как Dynatrace.

На этой странице вы найдёте информацию о том, какие данные Dynatrace OneAgent для iOS собирает по умолчанию. Категории и типы данных отражают анкету Apple, однако обратите внимание, что ответы соответствуют стандартному состоянию «из коробки». Подробное описание отдельных типов данных см. в разделе [App privacy details on the App Store](https://developer.apple.com/app-store/app-privacy-details/) на портале для разработчиков Apple.

OneAgent может собирать дополнительные данные через вашу ручную инструментацию. Если вы инструментируете приложение для сбора дополнительных данных, обязательно отразите это в анкете о конфиденциальности приложения.
