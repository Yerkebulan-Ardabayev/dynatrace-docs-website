# Конфиденциальность пользователей для iOS


* Последняя версия Dynatrace
* 3 мин. чтения
* Обновлено 21 ноября 2024 г.

Начиная с 8 декабря 2020 года Apple обязывает вас предоставлять информацию о практиках конфиденциальности вашего приложения, включая практики сторонних партнёров, таких как Dynatrace.

На этой странице вы найдёте информацию о том, какие данные [Dynatrace OneAgent для iOS](../../../observe/digital-experience/mobile-applications/instrument-ios-app.md "Инструментирование мониторинга мобильных приложений для iOS, настройка автоматической инструментации и сбор дополнительных данных через ручную инструментацию.") собирает по умолчанию. Категории и типы данных отражают анкету Apple, однако обратите внимание, что ответы соответствуют стандартному состоянию «из коробки». Подробное описание отдельных типов данных см. в разделе [App privacy details on the App Store](https://developer.apple.com/app-store/app-privacy-details/) на портале для разработчиков Apple.

OneAgent может собирать дополнительные данные через вашу [ручную инструментацию](../../../observe/digital-experience/mobile-applications/instrument-ios-app/customization/oneagent-sdk-for-ios.md "Расширение мониторинга пользовательского опыта мобильных приложений с помощью OneAgent SDK для iOS."). Если вы инструментируете приложение для сбора дополнительных данных, обязательно отразите это в анкете о конфиденциальности приложения.
