### Шаг 1. Применение сегмента для фильтрации данных на дашборде

Сегменты обеспечивают быстрый доступ к предопределённым логическим фильтрам. Как и **Dashboards**, многие приложения в Dynatrace имеют селектор сегментов, предоставляющий доступ к списку доступных сегментов. Вы можете искать сегменты по имени в селекторе сегментов.

![Выпадающий список фильтрации по сегментам](https://dt-cdn.net/images/segments-1-2466-b26bdc327d.png)

1. Перейдите в **Dashboards** и выберите дашборд
2. Откройте селектор сегментов и в разделе **Filter by segments** выберите сегмент по имени
3. Выберите одно или несколько значений переменной в выбранном сегменте
4. Нажмите **Apply**, чтобы применить выбор и отфильтровать данные на дашборде

После выбора сегментов приложения либо автоматически обновят данные в соответствии с выбранными сегментами, либо потребуют действий пользователя для обновления.

### Шаг 2 (необязательно). Выбор нескольких значений для более гибкой фильтрации

В то время как простые сегменты можно выбрать одним кликом, более продвинутые сегменты предложат вторичный выбор значения измерения, как в этом примере — выбор из динамического списка отдельных развёртываний. Этот вторичный выбор значений динамического сегмента также поддерживает множественный выбор.

![Выпадающий список фильтрации по сегментам](https://dt-cdn.net/images/segments-2-2466-e23d7b3f6b.png)

1. Откройте селектор сегментов
2. Откройте список значений для изменения вашего выбора
3. Выберите несколько значений для просмотра данных, соответствующих любому из этих значений
4. Нажмите **Apply**, чтобы применить выбор и отфильтровать данные на дашборде

Если выбранные сегменты или их комбинация с другими параметрами (такими как временной интервал, фильтры или разрешения) приводят к пустым результатам запроса, приложение не отобразит никаких данных для выбранного контекста.

### Шаг 3 (необязательно). Закрепление недавних выборок для быстрого доступа

Селектор сегментов показывает две последние выборки и отображает до 10, если нажать **Show more**. Это позволяет быстро применять недавние выборки без необходимости просматривать полный список доступных сегментов.

1. Перейдите в ![Notebooks](https://dt-cdn.net/images/notebooks-768-046137830a.webp "Notebooks") **Notebooks** и выберите **Notebook**
2. Выберите **Logs**
3. Откройте селектор сегментов и найдите недавно использованные сегменты из предыдущих шагов
4. Нажмите **Pin in Notebooks**, чтобы закрепить конкретную выборку для быстрого доступа
5. Нажмите, чтобы применить закреплённую запись

![Закрепление недавнего сегмента](https://dt-cdn.net/images/segments-recent-pin-2878-da6346e2b3.png)

Закреплённые выборки сегментов уникальны для каждого приложения, что обеспечивает полный контроль, поскольку большинство сегментов не являются универсально применимыми.

## Заключение

Вы научились фильтровать данные на дашборде с помощью сегментов и узнали, как быстро применять недавние и закреплённые выборки, повышая эффективность работы. Попробуйте сделать то же самое в других приложениях, чтобы узнать, как они используют сегменты для своих конкретных задач.
