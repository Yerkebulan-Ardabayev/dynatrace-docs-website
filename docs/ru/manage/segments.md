Используйте сегменты для логической структуризации и удобной фильтрации данных наблюдаемости в приложениях на платформе Dynatrace.

## Варианты использования

Сегменты, как правило, моделируются для обеспечения фильтрации отслеживаемых сущностей, журналов, метрик, событий и других типов данных по следующим измерениям:

* Бизнес-приложения, эксплуатируемые клиентом
* Организационные единицы — для понимания ответственности и принадлежности
* Инфраструктурные компоненты — для понимания и планирования распределения ресурсов
