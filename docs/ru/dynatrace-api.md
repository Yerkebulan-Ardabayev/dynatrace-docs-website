## API Explorer

Environment | Конфигурация | Управление учетными записями

### Доступ к API Explorer

- **Последняя Dynatrace:** **Access Tokens** > **Dynatrace API Explorer**
- **Предыдущая Dynatrace:** [Меню пользователя](discover-dynatrace/get-started/dynatrace-ui.md#user-menu-previous-dynatrace) > **Dynatrace API**
- **Управление учетными записями:** [myaccount.dynatrace.com](https://myaccount.dynatrace.com/) > **Управление идентификаторами и доступом** > **Клиенты OAuth** > **Управление учетными записями API**
- **Прямая ссылка:** `https://{your-environment-id}.live.dynatrace.com/rest-api-doc/`

### Аутентификация

Нажмите значок замка рядом с конечной точкой для просмотра требуемых OAuth 2.0-токенов. Кнопка **Authorize** разблокирует все конечные точки через глобальный диалог **Available authorizations**.

### Выполнение вызовов

После авторизации нажмите **Try it out** на любой конечной точке, задайте параметры и нажмите **Execute**.
