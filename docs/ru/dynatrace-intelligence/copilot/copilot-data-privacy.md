## Данные промптов

Промпты отправляются в LLM через корпоративных поставщиков (Microsoft Azure AI, AWS Bedrock). Поставщики:
* Не хранят данные и ответы.
* Не используют промпты для обучения моделей.

Передача через SSL-шифрование. Региональная обработка: EMEA -- ЕС, остальные регионы -- США.

Dynatrace может сохранять промпты и ответы для анализа сценариев использования и улучшения продукта.

[Архитектура и поток данных](copilot-overview.md#copilot-data-flow)

## Маскирование PII

Dynatrace версии 1.305+

Маскируемые данные:
* Email, телефон, IBAN, кредитные карты, IP-адреса
* Номера банковских счетов, SSN, маршрутные номера ABA (США)
* Параметры URL (> 2 символов)
* Канадский SIN

Значения заменяются фиктивными данными в журналах и вызовах LLM.

**Агентный Dynatrace Assist** не маскирует PII -- вместо этого блокирует запросы с обнаруженными PII.
