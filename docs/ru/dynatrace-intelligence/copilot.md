Агентный и генеративный ИИ Dynatrace Intelligence позволяет исследовать данные на естественном языке, преобразуя запросы в DQL с возможностью автоматического выполнения.

## Сценарии использования

* Быстрое освоение DQL через запросы на естественном языке.
* Генерация и выполнение DQL-запросов.
* Ответы на вопросы без обращения к документации.
* Суммирование и пояснение существующего DQL.
* Контекстные аналитические данные через Dynatrace Assist (сводки проблем, пояснения сигналов Kubernetes, планы выполнения баз данных).
* Поиск руководств по устранению неполадок.
