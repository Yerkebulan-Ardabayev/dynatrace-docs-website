Каузальный ИИ Dynatrace Intelligence автоматически оценивает всю захваченную информацию и определяет первопричину, используя контекстную информацию (топология на уровне кода, зависимости).

## Сценарии использования

* Включение всех значимых аномалий и ранжирование факторов первопричин.
* Снижение нагрузки оповещений -- объединение связанных аномалий в одну проблему.
* Повышение точности через использование всех доступных данных.
