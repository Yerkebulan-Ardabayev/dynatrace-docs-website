Интеллект Dynatrace с помощью искусственного интеллекта для анализа коренной причины автоматически оценивает всю захваченную и ингестируемую информацию и выделяет сущности внутри топологии причинно-следственных связей, определенные как коренная причина сложной ситуации. Он предлагает контекстно-зависимый подход, который использует доступную контекстную информацию, такую как топология информации на уровне кода, для определения точной причины проблемы.

## Сценарии использования

* Экономьте время, включая все соответствующие аномалии и ранжируя вкладчиков коренной причины для определения основной причины.
* Снижайте нагрузку оповещений, объединяя несколько связанных аномалий в одну проблему.
* Улучшайте точность анализа коренной причины, используя все доступные данные из нескольких источников.
