# Анализ коренной причины

* Последнее Dynatrace
* Объяснение
* 1-минутное чтение
* Обновлено 28 января 2026 г.

Интеллект Dynatrace с помощью причинно-следственного ИИ автоматически анализирует всю захваченную и обработанную информацию и выделяет сущности внутри причинной топологии, определенные как коренная причина сложной ситуации. Он предлагает контекстно-зависимый подход, использующий доступную контекстную информацию, такую как топология кода, для определения точной причины проблемы.

## Сценарии использования

* Экономия времени за счет включения всех релевантных аномалий и ранжирования вкладчиков коренной причины для определения основной причины.
* Снижение нагрузки оповещений за счет объединения нескольких связанных аномалий в одну проблему.
* Повышение точности анализа коренной причины с использованием всех доступных данных из нескольких источников.
