# Анализ первопричин

* Последняя версия Dynatrace
* Пояснение
* Чтение: 1 мин
* Обновлено 28 янв. 2026

Каузальный ИИ-анализ первопричин Dynatrace Intelligence автоматически оценивает всю захваченную и принятую информацию и выделяет сущности в каузальной топологии, определённые как первопричина сложной ситуации. Он предлагает контекстно-зависимый подход, который использует доступную контекстную информацию, такую как топология на уровне кода, для определения точной причины проблемы.

## Сценарии использования

* Экономьте время, включая все значимые аномалии и ранжируя факторы первопричин для определения основной причины.
* Снижайте нагрузку оповещений, объединяя несколько связанных аномалий в одну проблему.
* Повышайте точность анализа первопричин, используя все доступные данные из множества источников.
