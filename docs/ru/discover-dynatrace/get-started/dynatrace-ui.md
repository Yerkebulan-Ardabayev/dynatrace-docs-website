## Док

Док — это ваша отправная точка. Это раскрываемая/сворачиваемая панель в левой части экрана Dynatrace. Это ваша панель инструментов, которую вы можете оснастить ссылками на приложения, необходимые вам ежедневно.

Используйте Док, чтобы:

* Быстро переключаться между закреплёнными или недавно использованными приложениями.
* Переходить к Лаунчеру, глобальному поиску по платформе, всем приложениям в вашей среде, дополнительным ресурсам поддержки и вашим пользовательским настройкам.

![Док: верхняя часть](https://dt-cdn.net/images/dock-all-4-top-280-e15e9030df.png)

Верхняя часть Дока включает:

* [**Dynatrace**](#dynatrace) — ваша домашняя страница.
* [**Поиск**](#search) позволяет выполнять поиск по платформе.
* [**Приложения**](#apps)
* [Закреплённые и активные приложения](#pinned) в зависимости от вашей конфигурации и активности. В этом примере перечислены приложения ![Notebooks](https://dt-cdn.net/images/notebooks-768-046137830a.webp "Notebooks") Notebooks, ![Dashboards](https://dt-cdn.net/images/dashboards-512-b1f1e9690b.png "Dashboards") Dashboards и ![Segments](https://dt-cdn.net/images/segments-256-8e66310720.webp "Segments") Segments.

![Док: нижняя часть](https://dt-cdn.net/images/dock-all-4-bottom-280-1767392d75.png)

Нижняя часть Дока включает:

* и  для [скрытия или отображения Дока](#collapse-and-expand)
* для доступа к [ссылкам поддержки](#support)
* **[ваши инициалы] [ваше имя]** для доступа к [пользовательским ссылкам](#user)

## Dynatrace

![Док: Dynatrace](https://dt-cdn.net/images/dock-all-4-top-dynatrace-280-f021988a23.png)

Нажмите  **Dynatrace** в верхнем левом углу, чтобы перейти на домашнюю страницу Dynatrace.

* Стандартная страница **Начало работы с Dynatrace** даёт быстрый обзор основных функций для начала работы с платформой. Она ориентирована на знакомство с ключевыми аспектами платформы и сокращение кривой обучения.
* Вы можете создавать дополнительные стартовые страницы (пользовательские домашние страницы) с подобранными списками приложений, документов, ссылок и информационного текста. Подробнее см. Стартовые страницы.

## Поиск

![Док: Поиск](https://dt-cdn.net/images/dock-all-4-top-search-280-ab9aaa4258.png)

Опция  **Поиск** всегда доступна в Доке.

* Если вы свернёте Док, вы увидите только значок .
* Сочетание клавиш для **Поиска** — **CTRL**+**K** или **CMD**+**K**.

Поиск по платформе Dynatrace предоставляет настраиваемые опции и функции для фильтрации результатов поиска. Подробнее о расширенных функциях поиска см. Поиск по платформе.

## Приложения

![Док: Приложения](https://dt-cdn.net/images/dock-all-4-top-apps-280-4e7b6d0d7c.png)

Нажмите  **Приложения** для просмотра и запуска установленных приложений. (Если вы свернёте Док, вы увидите только значок .)

* Просматривайте приложения по категориям.
* Используйте строку фильтра для поиска по имени приложения.
* Нажмите ![Hub](https://dt-cdn.net/images/hub-512-82db3c583e.png "Hub") **Узнать больше**, чтобы найти дополнительные приложения в Dynatrace Hub.

## Закреплённые и активные приложения

![Док: закреплённые и активные приложения](https://dt-cdn.net/images/dock-all-4-top-pinned-280-39e97c223b.png)

Средняя часть Дока содержит приложения по умолчанию и любые другие приложения, которые вы закрепили в Доке или недавно использовали. Закрепите избранные приложения здесь для быстрого доступа.

Чтобы закрепить приложение в Доке

1. Используйте  [Поиск](#search) или  [Приложения](#apps), чтобы найти и выбрать приложение, которое вы хотите закрепить. Значок и название приложения отобразятся в Доке.
2. В Доке наведите курсор на приложение и нажмите  **Закрепить в Доке**.
   Закреплённое приложение останется в Доке, пока вы его не удалите (наведите курсор на приложение и нажмите  **Открепить из Дока**).

## Свернуть и развернуть

![Док: Свернуть](https://dt-cdn.net/images/dock-all-4-bottom-collapse-280-0a6e0c1306.png)

* **Свернуть** скрывает подписи и отображает только узкий столбец значков.

  + Это освобождает экранное пространство для отображения ваших данных.
  + Значков может быть достаточно для часто используемых приложений и функций.
* **Развернуть** снова отображает полный Док, чтобы вы могли читать подписи.

## Поддержка

![Док: Поддержка](https://dt-cdn.net/images/dock-all-4-bottom-support-280-b11b4ae9b9.png)

Если вам нужна дополнительная информация, вы всегда можете нажать  **Поддержка** в нижнем левом углу, чтобы отобразить ссылки на ресурсы поддержки и разработки, описанные ниже.

### Ресурсы поддержки

* **Документация** обеспечивает лёгкий доступ к документации и даёт обзор всех новинок, документации по приложениям и всего остального, что вам нужно знать.
* **Примечания к выпуску** ведут непосредственно к примечаниям к выпуску.
* **Сообщество** перенаправляет вас в [Сообщество Dynatrace](https://community.dynatrace.com), где вы можете задавать вопросы, оставлять отзывы и делиться идеями. Учитесь у лучших пользователей Dynatrace.
* **Университет** перенаправляет вас в [Университет Dynatrace](https://university.dynatrace.com), где вы можете развивать свои навыки и продвигаться в карьере с помощью онлайн-обучения, живых тренингов и сертификаций от экспертов Dynatrace.
* **Живой чат** связывает вас с экспертом по продуктам Dynatrace (реальным человеком!) для получения помощи в реальном времени.

### Ресурсы для разработчиков

* **Dynatrace Developer** перенаправляет вас на [Dynatrace Developer](https://dt-url.net/developers) — отличный ресурс с руководствами, практическими инструкциями и техническими справочниками для начала разработки приложений.
* **Dynatrace API** перенаправляет вас к браузеру API. Там выберите определение API из списка **Select a definition**, чтобы найти нужный API Dynatrace. Если вы знакомы со Swagger, вам будет привычно.
* **Форум разработчиков** перенаправляет вас на [Форум разработчиков Dynatrace](https://dt-url.net/rf23801) в Сообществе Dynatrace, где вы можете делиться знаниями, задавать вопросы, находить лучшие ответы, общаться с другими и изучать лучшие практики создания приложений на платформе Dynatrace.

## Меню пользователя

![Док: Меню пользователя](https://dt-cdn.net/images/dock-all-4-bottom-user-280-46267f092c.png)

Чтобы открыть меню пользователя, нажмите на своё имя в нижнем левом углу, внизу [Дока](#dock).

* **[ваше имя]** ![Настраиваемый](https://dt-cdn.net/images/configurable-490-8b015913d4.svg "Настраиваемый") отображает **Настройки пользователя**, где вы можете изменить такие параметры, как язык интерфейса, часовой пояс и светлый/тёмный режим.
* **Среды** содержит ссылки на ваши доступные среды.
* **Оформление** предоставляет быстрое переключение между светлым, тёмным режимом или режимом браузера.
* **Управление учётной записью** открывает Управление учётной записью в новой вкладке.
* **Latest Dynatrace** включает и выключает последнюю версию Dynatrace.
* **Выход** выполняет выход из Dynatrace.

## Предыдущая версия Dynatrace

Если вы ещё не обновились до последней версии Dynatrace, следующие разделы описывают базовую навигацию для предыдущей версии Dynatrace.

Фильтрация меню

Используйте поле фильтра над меню для поиска и выбора страниц Dynatrace.

1. Введите часть строки поиска.

   ![Фильтрация меню](https://dt-cdn.net/images/filter-265-fc8901c349.png)
2. Выберите лучшее совпадение.

   ![Выбор пункта меню из результатов фильтра](https://dt-cdn.net/images/filter-and-select-266-ac5a5c8210.png)

Сохранение избранного

* Чтобы добавить страницу Dynatrace в избранное, наведите курсор на название страницы и нажмите звёздочку.

  ![Добавление пункта меню в Избранное](https://dt-cdn.net/images/favorite-add-266-7262cf1304.png)
* Чтобы открыть избранное, разверните раздел **Избранное** и выберите страницу.

  ![Выбор пункта меню из Избранного](https://dt-cdn.net/images/favorite-select-266-ca63c732de.png)
* Чтобы удалить страницу из избранного, нажмите звёздочку снова.

  ![Удаление пункта меню из Избранного](https://dt-cdn.net/images/favorite-remove-264-6eb91e3ff5.png)

Развёртывание и сворачивание разделов меню

* Чтобы показать содержимое раздела меню, нажмите кнопку развёртывания.

  ![Развернуть раздел меню](https://dt-cdn.net/images/section-expand-261-70573d8144.png)
* Чтобы скрыть содержимое раздела меню, нажмите кнопку сворачивания.

  ![Свернуть раздел меню](https://dt-cdn.net/images/section-collapse-261-d51061df78.png)

Скрытие или отображение меню

* Чтобы скрыть меню, нажмите кнопку со стрелкой над меню.

  ![Скрыть меню](https://dt-cdn.net/images/menu-hide-2-271-588d75e5e9.png)
* Чтобы отобразить меню, нажмите кнопку меню в верхнем левом углу.

  ![Отобразить скрытое меню](https://dt-cdn.net/images/menu-display-235-ba2b8a386f.png)

Меню пользователя

![Отобразить меню пользователя](https://dt-cdn.net/images/user-menu-86-9ab2cd2961.png)

Содержимое этого меню пользователя зависит от типа развёртывания, конфигурации и разрешений.

* Перейдите в меню пользователя для просмотра информации о вашей учётной записи, выхода из системы, переключения между средами и получения последних новостей Dynatrace.
* Если у вас есть права администратора, выберите **Настройки учётной записи** в меню пользователя для управления настройками пользователей, групп и учётной записи.
