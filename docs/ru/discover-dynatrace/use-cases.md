# Учебные руководства Dynatrace


Ознакомьтесь с подборкой учебных руководств Dynatrace, доступных командам для того, чтобы сделать свою работу наблюдаемой и более удобной. Вы можете использовать эти руководства для предоставления возможностей автоматизации и самообслуживания вашим пользователям — командам разработчиков.
