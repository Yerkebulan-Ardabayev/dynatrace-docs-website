Dynatrace — это платформа мониторинга на основе программного интеллекта, которая упрощает управление сложными корпоративными облачными средами и ускоряет цифровую трансформацию. Подробнее см. в разделе Что такое Dynatrace.

## Приступим!

Начните работу с Dynatrace всего за три шага

### Шаг 1. Зарегистрируйтесь и начните пробный период

* Получите 15-дневный [бесплатный аккаунт Dynatrace](https://www.dynatrace.com/signup/).

### Шаг 2. Получите данные

* Установите OneAgent на хосте для автоматического мониторинга и анализа производительности и работоспособности всей вашей ИТ-инфраструктуры,
* или [загрузите данные из OpenTelemetry](https://www.dynatrace.com/news/blog/send-opentelemetry-data-to-dynatrace/#configure-the-otel-collector-for-dynatrace) для более гибридной инфраструктуры.

### Шаг 3. Убедитесь в получении данных

* Изучите обнаруженные сервисы с помощью приложения Services ![Services](https://dt-cdn.net/hub/logos/services.png "Services").
* Просмотрите **обнаруженную топологию** в Smartscape.

### Шаг 4. Визуализируйте данные

* **Визуализируйте данные с помощью** готовых дашбордов или создайте собственные с помощью Dashboards ![Dashboards](https://dt-cdn.net/images/dashboards-512-b1f1e9690b.png "Dashboards").

### Шаг 5. Настройте уведомления

* Настройте первые уведомления.

## Изучите Playground

Не готовы к установке или загрузке данных? Изучите наши возможности с примерами данных в Playground — нашей [публичной песочнице](https://www.dynatrace.com/signup/playground/).

## Готовы к большему?

* Проактивно исследуйте и выявляйте корневые причины проблем в контексте с помощью нашего приложения Logs ![Logs](https://dt-cdn.net/images/logs-256-ae0a9ca67f.png "Logs").
  [Изучите приложение Logs в нашей песочнице](https://wkf10640.apps.dynatrace.com/ui/apps/dynatrace.logs/).
* Оптимизируйте работоспособность и производительность ваших мультиоблачных сред Kubernetes с помощью нашего приложения Kubernetes ![Kubernetes](https://dt-cdn.net/images/kubernetes-512-90e7075764.png "Kubernetes").
  [Изучите приложение Kubernetes в нашей песочнице](https://wkf10640.apps.dynatrace.com/ui/apps/dynatrace.kubernetes/).
* Сортируйте, исследуйте и устраняйте инциденты автоматически и в контексте с помощью нашего приложения Problems ![Problems](https://dt-cdn.net/images/problems-512-34e46d913e.png "Problems").
  [Изучите приложение Problems в нашей песочнице](https://wkf10640.apps.dynatrace.com/ui/apps/dynatrace.davis.problems/).
* Анализируйте и фильтруйте данные трассировок с помощью приложения Distributed Tracing![Distributed Tracing](https://dt-cdn.net/images/distributed-tracing-4ed13d1274.svg "Distributed Tracing").

## Продолжайте изучение

* Присоединяйтесь к [**сообществу Dynatrace**](https://community.dynatrace.com/t5/Start-with-Dynatrace/bd-p/GetStarted), чтобы обмениваться опытом с экспертами в области наблюдаемости, получать ответы на вопросы и узнавать, с чего начать.
* Следите за обновлениями — новыми и обновлёнными материалами, а также [скоро ожидаемым функционалом](https://www.dynatrace.com/news/category/product-news/).
