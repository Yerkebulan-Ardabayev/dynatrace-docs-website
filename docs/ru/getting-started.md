# Начало работы с Dynatrace

Dynatrace -- платформа мониторинга на основе ИИ: APM, инфраструктура, цифровой опыт, AIOps.

## Быстрый старт

### 1. Развертывание

Dynatrace Managed -- локальное развертывание с полным контролем данных и соответствием требованиям.

### 2. Установка OneAgent

```bash
wget -O Dynatrace-OneAgent.sh "<download-url>"
sudo /bin/sh Dynatrace-OneAgent.sh
```

[Полное руководство по установке](#)

### 3. Мониторинг

После установки OneAgent автоматически обнаруживает приложения, сервисы, процессы, сеть и пользовательский опыт.
