# Начало работы с Dynatrace

Добро пожаловать в Dynatrace! Этот гид поможет вам начать мониторинг ваших приложений и инфраструктуры.

## Что такое Dynatrace?

Dynatrace — это платформа мониторинга на основе ИИ, которая обеспечивает:

- **Мониторинг производительности приложений (APM)**
- **Мониторинг инфраструктуры**
- **Мониторинг цифрового опыта**
- **AIOps и автоматизация**

## Быстрые шаги для начала работы

### 1. Выберите способ развертывания

Dynatrace предлагает два варианта развертывания:

=== "SaaS"

    **Рекомендуется для большинства пользователей**
    
    - Полностью управляется Dynatrace
    - Автоматические обновления
    - Не требуется управление инфраструктурой
    
    [Узнайте больше о SaaS →](#)

=== "Managed"

    **Для развертывания на месте**
    
    - Самостоятельное размещение в вашем центре данных
    - Полный контроль над вашими данными
    - Соответствует конкретным требованиям по соответствию
    
    [Узнайте больше о Managed →](managed/index.md)

### 2. Установите OneAgent

OneAgent — это единый агент мониторинга:

```bash
# Пример установки на Linux
wget -O Dynatrace-OneAgent.sh "<download-url>"
sudo /bin/sh Dynatrace-OneAgent.sh
```

[Полное руководство по установке →](#)

### 3. Начните мониторинг

После установки OneAgent Dynatrace автоматически обнаруживает и мониторит:

- ✅ Приложения и сервисы  
- ✅ Процессы и инфраструктуру
- ✅ Сетевые соединения
- ✅ Пользовательский опыт
