Dynatrace Query Language (DQL) -- инструмент для исследования данных в Grail: поиск закономерностей, аномалий, статистическое моделирование. Не требует предварительного описания схемы (в отличие от SQL).

### Разделы документации

* **Как использовать DQL** -- ключевые концепции и начало работы
* **Справочник по языку** -- синтаксис DQL
* **Изучение DQL** -- интерактивные руководства
* **Переход на DQL** -- сравнение с другими языками запросов
* **Примеры DQL:** логи, данные безопасности, бизнес-события, проблемы и события Davis, метрики

## Связанные темы

* Использование запросов DQL
* Сравнение DQL с SQL и другими языками
* Справочник по языку DQL
* Команды, функции, операторы и типы данных DQL
