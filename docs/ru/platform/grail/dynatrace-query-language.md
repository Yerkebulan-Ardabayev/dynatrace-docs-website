# Язык запросов Dynatrace

* Последняя версия Dynatrace
* Ссылка
* Обновлено 28 января 2026 г.

Язык запросов Dynatrace (DQL) — это мощный инструмент для изучения ваших данных и обнаружения закономерностей, выявления аномалий и выбросов, создания статистического моделирования и многого другого на основе данных, хранящихся в [Система хранения данных Dynatrace Grail](/docs/platform/grail/dynatrace-grail "Grail is the Dynatrace data lakehouse that's designed explicitly for observability and security data and acts as single unified storage for logs, metrics, traces, events, and more.").

DQL обеспечивает максимальную гибкость, поскольку он создан для обработки произвольных данных о событиях и не требует предварительного описания схемы входных данных, в отличие от реляционных баз данных, таких как таблицы SQL.

Прочтите [как использовать запросы DQL](/docs/platform/grail/dynatrace-query-language/dql-guide "Find out how DQL works and what are DQL key concepts."), чтобы начать или изучить язык.
