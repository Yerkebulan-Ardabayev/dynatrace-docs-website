# Dynatrace Query Language

# Dynatrace Query Language

* Latest Dynatrace
* Справочник
* Обновлено 28 января 2026 г.

Dynatrace Query Language (DQL) -- это мощный инструмент для исследования данных, обнаружения закономерностей, выявления аномалий и выбросов, построения статистических моделей и многого другого на основе данных, хранящихся в [хранилище Dynatrace Grail](dynatrace-grail.md "Grail -- это хранилище данных Dynatrace типа data lakehouse, специально разработанное для данных наблюдаемости и безопасности, выступающее единым унифицированным хранилищем для логов, метрик, трассировок, событий и многого другого.").

DQL обеспечивает максимальную гибкость, поскольку создан для обработки произвольных данных о событиях и не требует предварительного описания схемы входных данных, в отличие от реляционных баз данных, таких как таблицы SQL.

Прочитайте [как использовать запросы DQL](dynatrace-query-language/dql-guide.md "Узнайте, как работает DQL и каковы ключевые концепции DQL."), чтобы начать работу или изучить язык.
