* Latest Dynatrace

Dynatrace Query Language (DQL) -- это мощный инструмент для исследования данных, обнаружения закономерностей, выявления аномалий и выбросов, построения статистических моделей и многого другого на основе данных, хранящихся в хранилище Dynatrace Grail.

DQL обеспечивает максимальную гибкость, поскольку создан для обработки произвольных данных о событиях и не требует предварительного описания схемы входных данных, в отличие от реляционных баз данных, таких как таблицы SQL.

Прочитайте как использовать запросы DQL, чтобы начать работу или изучить язык.
