# Функции DQL

# Функции DQL

* Последняя версия Dynatrace
* Справочник
* Обновлено 02 февр. 2026 г.

Функции DQL, сгруппированные по категориям. Для получения подробной информации о конкретной функции выберите её название.

## [Функции агрегации](/docs/platform/grail/dynatrace-query-language/functions/aggregation-functions "Список функций агрегации DQL.")

Функции агрегации вычисляют результаты из списка записей.

## [Строковые функции](/docs/platform/grail/dynatrace-query-language/functions/string-functions "Список строковых функций DQL.")

Строковые функции позволяют создавать выражения, которые различными способами обрабатывают текстовые строки.

## [Функции преобразования и приведения типов](/docs/platform/grail/dynatrace-query-language/functions/conversion-and-casting-functions "Список функций преобразования и приведения типов DQL.")

Функции преобразования и приведения типов преобразуют выражение или значение из одного типа данных в другой.

## [Условные функции](/docs/platform/grail/dynatrace-query-language/functions/conditional-functions "Список условных функций DQL.")

Функции, возвращающие условный результат.

## [Булевы функции](/docs/platform/grail/dynatrace-query-language/functions/boolean-functions "Список булевых функций DQL.")

Функции, вычисляющие булевы выражения и проверяющие наличие значений.

## [Функции времени](/docs/platform/grail/dynatrace-query-language/functions/time-functions "Список функций времени DQL.")

Функции времени возвращают десятичное число для конкретного значения времени, вычисляют количество единиц времени (дней, месяцев, лет) между двумя датами, а также позволяют определять временные метки и временные диапазоны и многое другое.

## [Функции массивов](/docs/platform/grail/dynatrace-query-language/functions/array-functions "Список функций массивов DQL.")

Функции для работы с коллекцией элементов одного типа данных, хранящихся в смежных ячейках памяти.

## [Функции векторного расстояния](/docs/platform/grail/dynatrace-query-language/functions/vector-distance-functions "Список функций векторного расстояния DQL.")

Функции, вычисляющие расстояние между выражениями числовых массивов.

## [Сетевые функции](/docs/platform/grail/dynatrace-query-language/functions/network-functions "Список функций массивов DQL.")

Функции для работы с IP-адресами.

## [Хеш-функции](/docs/platform/grail/dynatrace-query-language/functions/hash-functions "Список хеш-функций DQL.")

Функции, связанные с хешированием.

## [Побитовые функции](/docs/platform/grail/dynatrace-query-language/functions/bitwise-functions "Список побитовых функций DQL.")

Побитовые операции над выражениями типа long.

## [Математические функции](/docs/platform/grail/dynatrace-query-language/functions/mathematical-functions "Список математических функций DQL.")

Функции для выполнения математических вычислений.

## [Функции объединения](/docs/platform/grail/dynatrace-query-language/functions/join-functions "Список функций объединения DQL.")

Функции, объединяющие записи из подзапросов.

## [Общие функции](/docs/platform/grail/dynatrace-query-language/functions/general-functions "Список общих функций DQL.")

Функции общего назначения.
