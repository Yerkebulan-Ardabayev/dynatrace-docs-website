### API конфигураций

Заменён на Settings API

* API OpenPipeline — GET всех конфигураций
* API OpenPipeline — GET конфигурации
* API OpenPipeline — PUT конфигурации

### API предварительного просмотра

* API OpenPipeline — POST предварительного просмотра процессора

### API сопоставления

* API OpenPipeline — POST проверки сопоставителя
* API OpenPipeline — POST подсказок автодополнения для сопоставителя
* API OpenPipeline — POST преобразования LQL-сопоставителя в DQL-сопоставитель

### API процессора DQL

* API OpenPipeline — POST проверки процессора DQL
* API OpenPipeline — POST подсказок автодополнения для процессора DQL

### API технологий

* API OpenPipeline — GET всех технологических пакетов
* API OpenPipeline — GET процессоров технологии
