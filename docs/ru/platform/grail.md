# Grail

# Grail

* Latest Dynatrace
* Overview
* 1-min read
* Published Feb 07, 2023

Grail — это хранилище данных типа data lakehouse от Dynatrace, разработанное специально для данных наблюдаемости. Оно служит единым унифицированным решением для хранения логов, метрик, трассировок, событий и многого другого. Все данные, хранящиеся в Grail, взаимосвязаны в рамках модели реального времени, отражающей топологию и зависимости внутри отслеживаемой среды.
