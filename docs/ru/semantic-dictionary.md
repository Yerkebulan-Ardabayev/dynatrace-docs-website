Семантический Словарь определяет стандартизированные имена полей для всех типов данных (журналы, события, спаны, метрики, сущности) и доменные модели данных. Обеспечивает стабильность приложений и автоматизаций.

В Grail данные организованы в записи из полей двух типов:

* **Глобальные поля** -- единая семантика во всем Grail. См. справочник глобальных полей.
* **Доменно-специфические поля** -- определяются в справочнике конкретной модели данных.

Модели данных -- группы полей для различных доменов. Полный список в разделе Модели данных.

### Уровни стабильности

* **Экспериментальный** -- в стадии тестирования, может измениться или быть удалён.
* **Стабильный** -- подтверждённый, надёжный статус.
* **Устаревший** -- не рекомендуется, может быть исключён в будущем.
