# Ownership

## Предварительные требования

### Разрешения

* `settings:objects:read` — чтение settings 2.0
* `settings:objects:write` — запись settings 2.0
* `app-engine:apps:run` — просмотр и запуск приложений
* `app-engine:functions:run` — использование исполнителя функций

### Установка

Убедитесь, что приложение [установлено в вашей среде](../manage/hub.md#install).

## Возможности

**Ownership** предоставляет действия для рабочих процессов: запрос команды-владельца сущности и контактной информации. Позволяет интегрировать данные о владении с Slack, Jira, MS Teams, Email.

Поддерживает импорт команд через Workflows из:
* Microsoft Entra ID
* ServiceNow
* Любой JSON-источник

## Сценарии использования

* Целевые уведомления при инцидентах через Workflows
* Прозрачность команд и зависимостей
* Сокращение MTTR за счёт автоматического информирования нужных команд
* Автоматический импорт и синхронизация данных о владельцах
