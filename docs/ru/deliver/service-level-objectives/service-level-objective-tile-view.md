# Просмотр сведений о плитке целевого уровня обслуживания (SLO) на дашборде


Вы можете просмотреть сведения о плитке SLO на вашем дашборде.

## Шаги

1. Перейдите в ![Dashboards](https://dt-cdn.net/images/dashboards-512-b1f1e9690b.png "Dashboards") **Dashboards**.
2. Выберите дашборд с вашим SLO.
3. На дашборде выберите плитку SLO, чтобы отобразить действия с плиткой SLO.

   ![Service-level objective tile screenshot shows the open with option](https://dt-cdn.net/images/chrome-kntow83har-947-3eec6f8cb7.png)
4. Выберите  >  **Open with**.
5. Выберите **Dashboards**, чтобы добавить SLO на дашборд.
6. Или выберите **Service-level Objectives**, чтобы просмотреть сведения о SLO.
