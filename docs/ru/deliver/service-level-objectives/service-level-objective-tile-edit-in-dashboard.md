# Редактирование плитки целевого показателя уровня сервиса (SLO) в дашборде

* Актуальная версия Dynatrace
* Практическое руководство
* Чтение: 1 мин
* Опубликовано 24 ноя. 2024 г.

Вы можете настраивать и редактировать плитки целевых показателей уровня сервиса (SLO) непосредственно в разделе **Дашборды** ![Dashboards](https://dt-cdn.net/images/dashboards-512-b1f1e9690b.png "Dashboards"), чтобы визуализировать их именно так, как вам нужно.

## Шаги

### Редактирование плитки

1. Перейдите в ![Dashboards](https://dt-cdn.net/images/dashboards-512-b1f1e9690b.png "Dashboards") **Dashboards**.
2. Выберите дашборд с вашим SLO.
3. На дашборде наведите курсор на SLO и щёлкните по нему, чтобы отобразить действия с плиткой SLO.

   ![Screenshot of Edit a service-level objective tile](https://dt-cdn.net/images/empty-slo-tile-2400-b7b459694a.png)
4. Нажмите **Edit**.
5. Отредактируйте плитку на панели редактирования справа.

   При изменении выбранного SLO заголовок изменится на имя выбранного SLO.

   ![Screenshot of  Service-level objective panel with the Data tab link to the SLO list](https://dt-cdn.net/images/new-slo-in-dashboard-2907-a5f32f2b0c.png)

   1. На вкладке **Visual** вы можете выбрать или отменить выбор **столбцов** **Data mapping** для отображения вашего SLO: **Status**, **Error budget**, **Target** и **Evaluation period**.
   2. На вкладке **Visual** в разделе **Service-level objective options** вы можете переключить параметр **Show labels** для их отображения. Вы также можете переключить **Apply status threshold color to** на значение SLO **Value** или **Background**.

      ![Screenshot of Service-level objective panel with the Visual tab](https://dt-cdn.net/images/wtp95711-dev-apps-dynatracelabs-com-ui-apps-dynatrace-dashboards-dashboard-f8f0ee42-e4a3-4b63-a3af-ca753060d7f7-3-2400-a36ac46dc8.png)

### Дублирование плитки

Вы можете дублировать одну плитку за раз или одновременно дублировать несколько плиток.

Чтобы дублировать одну плитку:

1. Выберите плитку, чтобы отобразить команды, специфичные для плитки.
2. Нажмите **Duplicate**.

   Копия плитки создаётся на текущем дашборде.

Чтобы одновременно дублировать несколько плиток:

1. Щёлкните по одной плитке, чтобы выбрать её.
2. Удерживая Ctrl, щёлкните по дополнительным плиткам, которые хотите дублировать.

   * Каждая выбранная плитка выделяется.
   * Общее количество выбранных плиток отображается на панели команд плитки.
3. На панели команд плитки нажмите **Duplicate**.

   Выбранные плитки дублируются на текущем дашборде.

### Удаление плитки

Вы можете удалить одну плитку за раз или выбрать и удалить несколько плиток одновременно.

Чтобы удалить одну плитку:

1. Выберите плитку, чтобы отобразить команды, специфичные для плитки.
2. На панели команд плитки нажмите **More actions** > **Delete**.
3. В сообщении **Are you sure you want to delete this tile?** нажмите **Cancel** или **Delete**.

Чтобы одновременно удалить несколько плиток:

1. Щёлкните по одной плитке, чтобы выбрать её.
2. Удерживая Ctrl, щёлкните по дополнительным плиткам, которые хотите удалить.

   * Каждая выбранная плитка выделяется.
   * Общее количество выбранных плиток отображается на панели команд плитки.
3. На панели команд плитки нажмите **Delete**.
4. В сообщении **Are you sure you want to delete this tile?** нажмите **Cancel** или **Delete**.

### Копирование и вставка плитки

Вы можете копировать и вставлять плитки на тот же дашборд или другой дашборд.

Чтобы скопировать и вставить одну плитку:

1. Выберите плитку, чтобы отобразить команды, специфичные для плитки.
2. На панели команд плитки нажмите **Copy to clipboard**.

   Выбранная плитка копируется в буфер обмена.
3. Вставьте плитку (Ctrl-V).

   * Вы можете вставить скопированную плитку на текущий дашборд — в этом случае это равносильно действию **Duplicate**.
   * Вы можете перейти на другой дашборд и вставить скопированную плитку там.

Чтобы одновременно скопировать и вставить несколько плиток:

1. Щёлкните по одной плитке, чтобы выбрать её.
2. Удерживая Ctrl, щёлкните по дополнительным плиткам, которые хотите скопировать и вставить.

   * Каждая выбранная плитка выделяется.
   * Общее количество выбранных плиток отображается на панели команд плитки.
3. На панели команд плитки нажмите **Copy to clipboard**.

   Выбранные плитки копируются в буфер обмена.
4. Вставьте плитки (Ctrl-V) на тот же дашборд или перейдите на другой дашборд и вставьте их там.
