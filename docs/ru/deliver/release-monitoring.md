# Release monitoring Classic

Dynatrace предоставляет встроенный анализ релизов:

* Развёрнутые версии по стадиям и средам
* Стадии релиза, changelog, известные ошибки
* Риски конкретных версий
* Нагрузка на версии (например, канареечные развёртывания)
* Сравнение поведения версий
* Статистика проблем

## Настройка

* Настройте переменные окружения для определения версий.
* При необходимости интегрируйте системы отслеживания задач и настройте динамические запросы.

## Анализ

После настройки анализируйте жизненный цикл релизов вашего ПО.
