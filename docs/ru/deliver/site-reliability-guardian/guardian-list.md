# Список гардианов и работа с ними

* Latest Dynatrace
* How-to guide
* 1-min read
* Published Jul 09, 2025

Представление проверки, которое называется страницей «All guardians», даёт общее представление о всех ваших гардианах.
Вы можете сравнивать или массово удалять гардианов в режиме **Table**.

### Список

Чтобы просмотреть список всех ваших гардианов:

1. Найдите ![Site Reliability Guardian](https://dt-cdn.net/images/site-reliability-guardian-ec19b393a6.svg "Site Reliability Guardian") Site Reliability Guardian.
2. Выберите ![Site Reliability Guardian](https://dt-cdn.net/images/site-reliability-guardian-ec19b393a6.svg "Site Reliability Guardian") Site Reliability Guardian.

   Откроется страница **All guardians** со списком всех ваших гардианов.
   По умолчанию гардианы отображаются в режиме **Grid**.

   ![Screenshot is showing a list of guardians in a grid view.](https://dt-cdn.net/images/gmg80500-dev-apps-dynatracelabs-com-ui-apps-dynatrace-site-reliability-guardian-2-3434-8a52040e6d.png)

### Работа с гардианами

Чтобы работать с вашими гардианами:

1. Перейдите в правый верхний угол окна, прямо под кнопкой **New guardian**.
2. Если вы находитесь в режиме **Grid**, переключитесь в режим **Table**.

   В этом режиме можно быстро сравнивать и удалять гардианов.
   Если вы в последний раз использовали режим **Table**, он будет отображаться по умолчанию.

   ![Screenshot is showing a list of guardians in a table view.](https://dt-cdn.net/images/gmg80500-dev-apps-dynatracelabs-com-ui-apps-dynatrace-site-reliability-guardian-1-3414-1e523a8817.png)

### Просмотр более ранних результатов

Вы можете просмотреть более ранние результаты, открыв гардиана и выбрав другой временной диапазон.
