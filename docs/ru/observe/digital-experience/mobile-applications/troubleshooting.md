## Log sharing for troubleshooting

For iOS mobile applications, Dynatrace provides the `shareLogsFile` API, which enables easy sharing of locally stored log files via an iOS sharing sheet (`UIActivityViewController`). For more information, see [Log sharing](/docs/manage/data-privacy-and-security/data-privacy/personal-data-captured-by-dynatrace#log-sharing "Find out what types of end-user data may be captured during Dynatrace monitoring and the methods that are available for masking personal end-user data.").
