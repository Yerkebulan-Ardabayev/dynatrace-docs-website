# Создание пользовательских имён действий для мобильных приложений

* Classic
* How-to guide
* 1-min read
* Published Jan 27, 2023

Dynatrace автоматически создаёт имена для большинства действий пользователей. Вы можете определить правила именования действий пользователей, чтобы заменить автоматически сгенерированные имена на предопределённые.

Чтобы создать правило именования действий пользователей

1. Перейдите в раздел **Frontend**.
2. Выберите приложение, которое хотите настроить.
3. Нажмите **More** (**…**) > **Edit** в правом верхнем углу плитки с именем вашего приложения.
4. В настройках приложения выберите **Naming Rules**.
5. Разверните раздел **x action naming rules** и нажмите **Add naming rule**.
6. Введите **Resulting user action name** и укажите условия правила именования.
