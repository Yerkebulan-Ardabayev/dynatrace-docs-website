# World map view


* 1 мин. чтения

Представление **Карта мира** позволяет легко переключаться между различными измерениями мониторинга, выбирая вкладки в верхней части представления.

![Worldmap](https://dt-cdn.net/images/map-2313-3f7868e353.png)

Чтобы открыть представление **Карта мира**

1. Перейдите в раздел **Web**.
2. Выберите нужное приложение.
3. Выберите плитку **Просмотр разбивки по геолокации** в инфографике.
4. В разделе **Разбивка по геолокации**, который появится слева под инфографикой, выберите **Показать полную карту мира**.

Вы можете закрепить представление карты мира на классическом дашборде по своему выбору, нажав **Закрепить на дашборде** и выбрав конкретный дашборд. Подробнее см. в разделе Закрепление плиток на дашборде.

Если данные не отображаются на карте мира, возможно, потребуется сопоставить внутренние IP-адреса с местоположениями.
