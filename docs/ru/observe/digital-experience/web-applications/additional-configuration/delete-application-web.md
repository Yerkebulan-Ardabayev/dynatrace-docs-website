# Удаление веб-приложения


* Classic
* Практическое руководство
* 1 минута чтения
* Опубликовано 9 августа 2022 г.

Если приложение больше не нужно, его можно удалить через веб-интерфейс Dynatrace или через API.

## Удаление приложения через интерфейс

Чтобы удалить приложение через веб-интерфейс Dynatrace:

1. Перейдите в **Settings** > **Monitoring** > **Monitoring overview**.
2. Откройте вкладку **Applications** и найдите приложение, которое нужно удалить.
3. Рядом с именем приложения выберите **Expand** ![Expand row](https://dt-cdn.net/images/expand-row-icon-9c4950fc2e.svg "Expand row"), затем выберите **Remove** > **Yes**.

## Удаление приложения через API

Приложения также можно удалять через Configuration API в Dynatrace. Подробности см. в разделе [Удаление веб-приложения](../../../../dynatrace-api/configuration-api/rum/web-application-configuration-api/web-application/del-web-application.md "Удаление веб-приложения через Dynatrace API.").
