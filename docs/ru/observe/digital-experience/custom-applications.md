# Пользовательские приложения

* Classic
* Обзор
* Чтение: 1 мин
* Опубликовано 6 февраля 2023

Пользовательские приложения охватывают все цифровые точки взаимодействия в вашей среде — от традиционных клиентских приложений до умных IoT-приложений и даже навыков Alexa. Такие приложения поддерживаются с помощью [Dynatrace OpenKit](/docs/ingest-from/extend-dynatrace/openkit "Узнайте, как инструментировать ваше приложение с помощью OpenKit, как использовать методы API Dynatrace OpenKit и многое другое.").
