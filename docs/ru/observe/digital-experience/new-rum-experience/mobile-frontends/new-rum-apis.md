# Новые RUM API для мобильных фронтендов


New RUM Experience включает мощный новый API, позволяющий настраивать и расширять мониторинг мобильных фронтендов для ваших приложений. Этот API подкреплён исчерпывающей документацией, включая практические примеры использования, которые помогут вам быстро начать работу.
