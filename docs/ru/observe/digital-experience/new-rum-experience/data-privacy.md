# Конфиденциальность данных в New RUM Experience


* Latest Dynatrace
* How-to guide
* Published Jan 15, 2026

New RUM Experience предоставляет комплексные настройки конфиденциальности данных, разработанные для того, чтобы помочь вашей организации защитить персональные данные конечных пользователей и обеспечить соответствие требованиям законодательства о защите данных. Хотя Dynatrace предоставляет эти инструменты, ответственность за их эффективное применение для защиты информации ваших пользователей лежит на вас.

## Рекомендации

Для защиты данных пользователей мы рекомендуем следующие практики.

1. **Изучите нормативные требования:** убедитесь, что ваша конфигурация конфиденциальности соответствует GDPR, CCPA и другим применимым нормам.
2. **Найдите баланс между конфиденциальностью и наблюдаемостью:** выбирайте настройки, которые защищают пользователей, сохраняя при этом эффективность мониторинга.
3. **Документируйте свои решения:** ведите чёткие записи с обоснованием включения или отключения конкретных параметров конфиденциальности.
4. **Регулярные аудиты:** периодически проверяйте конфигурации конфиденциальности для поддержания соответствия требованиям.
5. **Прозрачность для пользователей:** информируйте о практиках сбора данных через политики конфиденциальности и механизмы получения согласия.

## Конфигурация

Настройки конфиденциальности данных, доступные в RUM Classic, также поддерживаются в New RUM Experience. При включении New RUM Experience для приложения, уже инструментированного с помощью RUM Classic, ваши существующие настройки конфиденциальности данных применяются автоматически.
