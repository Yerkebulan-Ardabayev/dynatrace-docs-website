# Сегментация сессий

* Classic
* Overview
* 1-min read
* Published Oct 19, 2021

Эффективная сегментация пользовательских сессий является ключом к пониманию того, как пользователи с общими характеристиками взаимодействуют с вашим приложением.

[Пользовательская сессия](/docs/observe/digital-experience/rum-concepts/user-session "Learn how a user session is defined, when a user session starts or ends, how user session duration is calculated, and more."), также называемая «визитом», «путешествием» или «кликпутём», — это любое взаимодействие устройства отдельного пользователя с вашим приложением. По сути, это последовательность действий пользователя, выполненных одним пользователем в вашем приложении за ограниченный период времени. Каждая пользовательская сессия включает как минимум одно [пользовательское действие](/docs/observe/digital-experience/rum-concepts/user-actions "Learn what user actions are and how they help you understand what users do with your application.").
