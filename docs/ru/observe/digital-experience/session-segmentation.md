# Сегментация сессий


* 1-min read

Эффективная сегментация пользовательских сессий является ключом к пониманию того, как пользователи с общими характеристиками взаимодействуют с вашим приложением.

[Пользовательская сессия](rum-concepts/user-session.md "Learn how a user session is defined, when a user session starts or ends, how user session duration is calculated, and more."), также называемая «визитом», «путешествием» или «кликпутём», — это любое взаимодействие устройства отдельного пользователя с вашим приложением. По сути, это последовательность действий пользователя, выполненных одним пользователем в вашем приложении за ограниченный период времени. Каждая пользовательская сессия включает как минимум одно [пользовательское действие](rum-concepts/user-actions.md "Learn what user actions are and how they help you understand what users do with your application.").
