# Distributed Traces Classic

* Classic
* Overview
* 1-min read
* Updated on May 17, 2024

Распределённая трассировка — это незаменимый инструмент мониторинга, отладки и оптимизации для распределённых программных архитектур. Визуализируя связи между сервисами и то, как запросы проходят через различные уровни приложения, она помогает командам быстро оценивать производительность каждого микросервиса.
