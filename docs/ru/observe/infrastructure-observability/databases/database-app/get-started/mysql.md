# Мониторинг базы данных MySQL

* Последняя версия Dynatrace
* Практическое руководство
* Опубликовано 20 января 2026 г.

Используйте фреймворк расширений Dynatrace, чтобы расширить наблюдаемость приложения данными, получаемыми непосредственно из слоя базы данных MySQL, и отслеживать влияние задач сервера базы данных на ваше приложение.

## Предварительные требования

Убедитесь, что ваша система соответствует [требованиям](../../extensions/mysql-remote-monitoring-v2.md#requirements "Удалённый мониторинг экземпляров MySQL, сбор ключевых KPI и деталей медленных запросов.") для полной поддержки функций.

## Настройка расширения MySQL для мониторинга

Чтобы настроить и активировать расширение, выполните [подробные шаги активации](../../extensions/mysql-remote-monitoring-v2.md#activation-and-setup "Удалённый мониторинг экземпляров MySQL, сбор ключевых KPI и деталей медленных запросов.").
