# Мониторинг базы данных MariaDB

* Latest Dynatrace
* How-to guide
* Published Jan 20, 2026

Используйте фреймворк расширений Dynatrace, чтобы расширить наблюдаемость вашего приложения за счёт данных, получаемых непосредственно с уровня базы данных MariaDB, и отслеживать влияние задач сервера базы данных на ваше приложение.

## Предварительные требования

Убедитесь, что ваша система соответствует этим [требованиям](/docs/observe/infrastructure-observability/databases/extensions/mariadb-1#requirements "Удалённый мониторинг экземпляров MariaDB, сбор ключевых KPI и деталей медленных запросов") для полной поддержки функций.

## Настройка расширения MariaDB для мониторинга

Чтобы настроить и активировать расширение, выполните следующие [подробные шаги активации](/docs/observe/infrastructure-observability/databases/extensions/mariadb-1#activation-and-setup "Удалённый мониторинг экземпляров MariaDB, сбор ключевых KPI и деталей медленных запросов").

1. Добавьте экземпляр базы данных.
2. Выберите тип размещения.
3. Выберите группу ActiveGate.
4. Создайте подключение к базе данных.
5. Установите расширение.
