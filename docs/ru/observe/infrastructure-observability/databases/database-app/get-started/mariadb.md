Используйте фреймворк расширений Dynatrace, чтобы расширить наблюдаемость вашего приложения за счёт данных, получаемых непосредственно с уровня базы данных MariaDB, и отслеживать влияние задач сервера базы данных на ваше приложение.

## Предварительные требования

Убедитесь, что ваша система соответствует этим требованиям для полной поддержки функций.

## Настройка расширения MariaDB для мониторинга

Чтобы настроить и активировать расширение, выполните следующие подробные шаги активации.

1. Добавьте экземпляр базы данных.
2. Выберите тип размещения.
3. Выберите группу ActiveGate.
4. Создайте подключение к базе данных.
5. Установите расширение.
