# Мониторинг базы данных Oracle


* Последняя версия Dynatrace
* Практическое руководство
* Опубликовано 20 января 2026 г.

Используйте фреймворк расширений Dynatrace для расширения наблюдаемости приложений за счёт данных, получаемых непосредственно из уровня базы данных Oracle, и мониторинга влияния задач сервера баз данных на ваше приложение.

## Предварительные условия

Убедитесь, что ваша система соответствует [требованиям](../../extensions/oracle-database.md#requirements "Наблюдайте, анализируйте и оптимизируйте использование, работоспособность и производительность вашей базы данных.") для полной поддержки функций.

## Настройка расширения Oracle для мониторинга

Для настройки и активации расширения следуйте [подробным инструкциям по активации](../../extensions/oracle-database.md#activation-and-setup "Наблюдайте, анализируйте и оптимизируйте использование, работоспособность и производительность вашей базы данных.").

## Часто задаваемые вопросы и устранение неполадок

Обратитесь к разделам [FAQ](../../extensions/oracle-database.md#faq "Наблюдайте, анализируйте и оптимизируйте использование, работоспособность и производительность вашей базы данных.") и [устранение неполадок](../../extensions/oracle-database.md#troubleshooting "Наблюдайте, анализируйте и оптимизируйте использование, работоспособность и производительность вашей базы данных.") расширения для базы данных Oracle.

## Наборы функций

[Наборы функций](../../extensions/oracle-database.md#feature-sets "Наблюдайте, анализируйте и оптимизируйте использование, работоспособность и производительность вашей базы данных.") ограничивают набор собираемых метрик при активации расширения.

## Варианты использования

Мониторинг базы данных Oracle помогает понять взаимодействие приложения с базой данных, узкие места производительности и влияние на ресурсы посредством мониторинга выполняемых операторов, шаблонов запросов и работоспособности сервера. Дополнительные сведения см. в разделе [Варианты использования мониторинга базы данных Oracle](../../extensions/oracle-database.md#use-cases "Наблюдайте, анализируйте и оптимизируйте использование, работоспособность и производительность вашей базы данных.").
