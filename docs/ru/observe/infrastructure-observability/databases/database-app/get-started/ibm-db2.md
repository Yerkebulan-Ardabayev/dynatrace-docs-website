# Мониторинг базы данных IBM DB2

* Latest Dynatrace
* How-to guide
* Опубликовано 20 января 2026 г.

Используйте фреймворк расширений Dynatrace, чтобы расширить наблюдаемость вашего приложения за счёт данных, полученных непосредственно с уровня базы данных IBM, и отслеживайте влияние задач сервера базы данных на ваше приложение. Подробнее см. в разделе [Начало работы с расширением для базы данных IBM DB2](/docs/observe/infrastructure-observability/databases/extensions/ibm-db2-for-luw-remote-monitoring#get-started "Удалённо собирайте метрики мониторинга из баз данных DB2.").

## Предварительные требования

Убедитесь, что ваша система соответствует этим [требованиям](/docs/observe/infrastructure-observability/databases/extensions/ibm-db2-for-luw-remote-monitoring#requirements "Удалённо собирайте метрики мониторинга из баз данных DB2.") для полной поддержки функций.

## Настройка расширения IBM DB2 для мониторинга

Чтобы настроить и активировать расширение, следуйте этим [подробным шагам активации](/docs/observe/infrastructure-observability/databases/extensions/ibm-db2-for-luw-remote-monitoring#activation-and-setup "Удалённо собирайте метрики мониторинга из баз данных DB2.").

1. Установите расширение IBM DB2.
2. Определите конечные точки.
3. Выберите группу ActiveGate.
4. Активируйте расширение.
