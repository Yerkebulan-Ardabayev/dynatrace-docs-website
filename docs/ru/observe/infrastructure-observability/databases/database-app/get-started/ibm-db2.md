Используйте фреймворк расширений Dynatrace, чтобы расширить наблюдаемость вашего приложения за счёт данных, полученных непосредственно с уровня базы данных IBM, и отслеживайте влияние задач сервера базы данных на ваше приложение. Подробнее см. в разделе Начало работы с расширением для базы данных IBM DB2.

## Предварительные требования

Убедитесь, что ваша система соответствует этим требованиям для полной поддержки функций.

## Настройка расширения IBM DB2 для мониторинга

Чтобы настроить и активировать расширение, следуйте этим подробным шагам активации.

1. Установите расширение IBM DB2.
2. Определите конечные точки.
3. Выберите группу ActiveGate.
4. Активируйте расширение.
