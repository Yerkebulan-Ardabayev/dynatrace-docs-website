Используйте фреймворк расширений Dynatrace, чтобы расширить наблюдаемость приложения на данные, получаемые непосредственно из уровня базы данных SAP HANA, и отслеживать влияние задач сервера базы данных на ваше приложение.

## Предварительные требования

Убедитесь, что ваша система соответствует требованиям для полной поддержки функций.

## Настройка расширения SAP HANA для мониторинга

Чтобы настроить и активировать расширение, следуйте подробным шагам активации.

1. Добавьте экземпляр базы данных.
2. Выберите тип хостинга.
3. Выберите группу ActiveGate.
4. Создайте подключение к базе данных.
5. Установите расширение.
