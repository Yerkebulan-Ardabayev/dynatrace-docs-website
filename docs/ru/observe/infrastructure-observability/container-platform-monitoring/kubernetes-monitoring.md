# Kubernetes Classic

Dynatrace обеспечивает полностековый мониторинг Kubernetes от приложений до инфраструктуры.

## Предварительные требования

1. Настройте интеграцию Dynatrace в Kubernetes.
2. Подключите кластер к Dynatrace.

Подробности: Настройка Dynatrace в Kubernetes.

## Функции мониторинга

Настройка по умолчанию: Глобальные настройки мониторинга для Kubernetes/OpenShift.

* Использование кластера Kubernetes/OpenShift
* События Kubernetes/OpenShift
* Рабочие нагрузки
