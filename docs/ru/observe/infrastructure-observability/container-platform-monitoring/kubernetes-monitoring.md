# Kubernetes Classic


* 1 мин. чтения

Dynatrace обеспечивает полностековый мониторинг Kubernetes, охватывая все уровни от приложений до инфраструктуры.

## Предварительные требования

1. Настройте и сконфигурируйте интеграцию Dynatrace в Kubernetes
2. Подключите кластер к Dynatrace

Дополнительные параметры и подробности см. в разделе Настройка Dynatrace в Kubernetes.
