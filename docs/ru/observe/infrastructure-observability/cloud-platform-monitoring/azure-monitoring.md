# Мониторинг Microsoft Azure

* Последняя версия Dynatrace
* Пояснение
* Чтение: 2 мин
* Обновлено 28 янв. 2026 г.

Dynatrace автоматически обнаруживает, устанавливает базовые показатели и интеллектуально отслеживает облачные окружения Microsoft Azure.

## Мониторинг и обнаружение проблем

Мониторинг Azure в Dynatrace выполняет основанный на AI автоматический анализ влияния на бизнес, обнаружение проблем и анализ корневых причин для всех сервисов Azure на основе всех метрик, публикуемых в Azure Monitor. Этот анализ может быть дополнительно расширен за счёт полного мониторинга стека, обеспечиваемого OneAgent, развёртываемым на вычислительных ресурсах, таких как Azure Virtual Machines, Azure App Service и Azure Kubernetes Service. Все сущности также могут быть подвергнуты комплексному анализу журналов.

Помимо вышесказанного, Dynatrace позволяет понять динамику рабочих нагрузок применяемых ресурсов Azure и окружения в разрезе регионов.

Мониторинг Azure доступен для всех клиентов Dynatrace независимо от того, является ли их кластер Dynatrace SaaS или Managed.

## Простота использования и удобная визуализация

Мониторинг Azure поставляется с согласованными готовыми метриками, дашбордами и оповещениями сразу после включения мониторинга.

Измерения и анализ основных сервисов Azure визуализируются на специально разработанном инфографическом дашборде (см. галерею). Удобные отчёты и дашборды также доступны для анализа влияния на бизнес и топологических карт.

## Гибкость

Пользователи Dynatrace могут выбирать, какие сервисы и экземпляры отслеживаются и как выполняется этот процесс. Такой выбор можно сделать как в рамках начальной настройки мониторинга, так и в любой более поздний момент. Это также распространяется на новые сервисы, включённые в облаке Azure или добавленные компанией Microsoft.

Для каждого из поддерживаемых сервисов можно выбрать метрики мониторинга. Точно так же все дашборды для мониторинга Azure можно клонировать и легко настраивать по мере необходимости.

## Гибридные окружения

Отслеживание сервисов и зависимостей не ограничено только Azure. Благодаря Dynatrace Intelligence и её способности обрабатывать данные из всех типов окружений, Dynatrace оптимизирован для мониторинга гибридных окружений, включая бизнес-приложения, охватывающие несколько облачных и виртуализационных платформ, таких как AWS, Google Cloud, VMware, Kubernetes, Openshift и локальная инфраструктура.
