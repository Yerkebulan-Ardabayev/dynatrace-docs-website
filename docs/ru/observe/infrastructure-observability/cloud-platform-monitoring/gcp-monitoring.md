Dynatrace автоматически обнаруживает, устанавливает базовые показатели и интеллектуально отслеживает динамические рабочие нагрузки Google Cloud.

## Мониторинг полного стека

Dynatrace OneAgent обеспечивает мониторинг полного стека для основных вычислительных ресурсов, таких как Google Kubernetes Engine (GKE), Google Compute Engine (GCE) и Google Application Engine (GAE). Это даёт глубокую видимость на уровне кода и сквозные трассировки для всего, что выполняется в вычислительных сервисах.

## Мониторинг сервисов Google Cloud

Все метрики, публикуемые в Operations API (ранее Stackdriver), могут автоматически поступать в Dynatrace для обеспечения данных в целях обнаружения проблем с помощью ИИ и автоматического анализа первопричин. Интеграция с Google Cloud помогает отслеживать динамику гибридного облака, предоставляя общий обзор сервисов Google Cloud в вашей учётной записи с разграничением работоспособных и неработоспособных сервисов.
