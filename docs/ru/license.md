## Обзор

Dynatrace Platform Subscription (DPS) — единая модель подписки, позволяющая использовать любую возможность Dynatrace в любом объеме в рамках одного соглашения. DPS является лицензионной основой для Grail, AppEngine, AutomationEngine и других инноваций.

Dynatrace измеряет использование по UTC+00:00.

**Предварительные условия:**
- Документация относится к текущей модели DPS (после апреля 2023). Для более ранних версий — см. соответствующую документацию.
- Некоторые функции требуют специальных разрешений DPS.

## Ключевые концепции

Ежегодная приверженность
:   Минимальная годовая сумма расходов в рамках DPS. Соглашение обычно на 1-3 года.

Возможность
:   Единица лицензирования (например, Full-Stack мониторинг, мониторинг инфраструктуры). Перечислены в тарифной карте.

Потребление
:   Использование возможностей, измеряемое в различных единицах (например, часы памяти-GiB).

Затраты
:   Использование x цена из тарифной карты.

Использование по требованию
:   После исчерпания годовой приверженности — те же тарифы, ежемесячная оплата, без штрафных надбавок.

Карта тарифов
:   Структурированный список цен возможностей, часть вашего заказа. Доступна в Account Management.

## FAQ

**Как управлять потреблением?** Через инструменты Account Management: прогнозирование, оповещения, детализация.

**Доступ ко всем возможностям?** Да, DPS дает доступ ко всем возможностям платформы.

**Штрафы за превышение?** Нет. Потребление сверх приверженности оплачивается по тем же тарифам ежемесячно.

**Использование менее часа?** Округление до ближайших 15 минут (1 мин = 15 мин; 1ч 3мин = 1.25ч).

**Управление доступом к данным потребления?** Через разрешения на трех уровнях.
