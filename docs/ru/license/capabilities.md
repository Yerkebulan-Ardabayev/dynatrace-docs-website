Соглашение DPS заключается на 1-3 года с минимальным годовым обязательством. Каждая возможность имеет стоимость в тарифной карте.

Использование формирует расходы в счет годового обязательства. Данные о потреблении доступны ежедневно в **Account Management** или через DPS API.

После исчерпания годового обязательства начисляется потребление по требованию по тем же тарифам, без надбавок. Ежемесячные счета за расходы по требованию.

Отчет: **Subscription** > **Billing report**.

## Категории возможностей

- **Наблюдаемость приложений и инфраструктуры** — Full-Stack, Infrastructure, Foundation & Discovery, Mainframe Monitoring
- **Наблюдаемость контейнеров** — Kubernetes Platform Monitoring, Code Monitoring
- **Безопасность приложений** — RAP, RVA, SPM
- **Мониторинг цифрового опыта** — RUM, RUM Property, RUM with Session Replay, Browser Monitor, HTTP Monitor, Third-Party Synthetic
- **Аналитика логов** — Ingest & Process, Retain, Query
- **Трассировки на базе Grail** — Ingest & Process, Retain, Query
- **События на базе Grail** — Ingest & Process, Retain, Query
- **Метрики на базе Grail** — Ingest & Process, Retain, Query
- **Автоматизация** — Automation
- **Функции AppEngine** — AppEngine Functions
- **Расширения платформы** — Custom Metrics Classic, Log Monitoring Classic, Custom Traces Classic, Custom Events Classic, Serverless Functions Classic
