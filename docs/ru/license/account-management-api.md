* Объяснение

API позволяет легко интегрировать управление подпиской DPS в операционный рабочий процесс.
Используйте Dynatrace Platform Subscription API для получения:

* Обзора и подробных сведений о подписке DPS.
* Сводных данных об использовании Dynatrace Platform Subscription.
* Сводных данных о затратах по Dynatrace Platform Subscription.
* Прогнозов и событий затрат.
* Информации о распределении затрат.

Полная документация по Dynatrace Platform Subscription API доступна в разделе Dynatrace Platform Subscription API.
