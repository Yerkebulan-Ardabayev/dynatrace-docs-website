* Пояснение

Ваш отчет о выставлении счетов DPS предоставляет все сведения о состоянии вашего бюджета DPS за каждый день текущего периода. Он показывает, какие расходы были зарегистрированы в тот или иной день относительно даты их учета. Существует разница между датой потребления и датой учета: потребление всегда происходит при использовании возможностей Dynatrace. Учет происходит, когда потребление фиксируется и вычитается из вашего годового обязательства DPS. Поэтому дата учета может отличаться от даты потребления.

Чтобы получить доступ к отчету о выставлении счетов в веб-интерфейсе Dynatrace, перейдите в **Subscription** > **Billing report**.

![DPS billing reports](https://dt-cdn.net/images/billingreport-1913-1256f12b00.png)

Отчет о выставлении счетов состоит из следующих диаграмм:

1. **Учтенные расходы на текущую дату:** Эта диаграмма отображает накопленные расходы по дням учета. Она также показывает минимальное годовое обязательство вашей подписки.
2. **Учтенные расходы за день:** Эта диаграмма показывает расходы, учтенные за каждый отдельный день. Она не содержит накопленного состояния бюджета. Эта диаграмма помогает выявить колебания в ежедневных расходах.

API для Dynatrace Platform Subscription предоставляет доступ к данным, лежащим в основе этого представления. Для получения дополнительных сведений ознакомьтесь с документацией по API.
