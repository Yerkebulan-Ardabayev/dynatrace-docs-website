Отчет DPS показывает состояние бюджета за каждый день текущего периода по дате учета.

**Важно:** дата учета может отличаться от даты потребления — учет происходит при фиксации потребления и вычете из годового обязательства.

Доступ: **Subscription** > **Billing report**.

## Диаграммы

1. **Учтенные расходы на текущую дату** — накопленные расходы по дням учета с отображением годового обязательства.
2. **Учтенные расходы за день** — ежедневные расходы без накопления, для выявления колебаний.

Данные также доступны через Dynatrace Platform Subscription API.
