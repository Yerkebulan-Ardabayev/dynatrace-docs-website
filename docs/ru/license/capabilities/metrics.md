| Понятие | Пояснение |
| --- | --- |
| Выполнение DQL-запроса | DQL-запрос сканирует и извлекает данные, хранящиеся в Grail. Запросы к метрикам с помощью команды `timeseries` всегда включены. |
| Использование приложений | DQL-запросы могут выполняться: - Приложениями, такими как Notebooks **Notebooks**, Dashboards **Dashboards**, Workflows **Workflows** и Anomaly Detection - new **Anomaly Detection**. - Плитками дашбордов на основе метрик, запускающими выполнение DQL-запросов при обновлении. - Пользовательскими приложениями. - Dynatrace API. |
