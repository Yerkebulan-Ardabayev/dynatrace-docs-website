# Automation Workflow capability overview (DPS)


* Latest Dynatrace
* Explanation
* 5-min read

Dynatrace Automation позволяет вам осуществить цифровую трансформацию с помощью облачной автоматизации, основанной на интеллектуальных выводах из данных наблюдаемости, безопасности и бизнес-данных.

В Dynatrace существует три типа рабочих процессов автоматизации.

* Стандартные рабочие процессы (Standard workflows): рабочие процессы автоматизации, запускающие две или более задач.
* Простые рабочие процессы (Simple workflows): однозадачные рабочие процессы с ограниченной функциональностью.
* Черновые рабочие процессы (Draft-only workflows): рабочие процессы, которые ещё не развёрнуты или были отменены.
