Application Security помогает обеспечить безопасность облачных и локальных приложений во время выполнения с помощью интеллектуальной автоматизации.
Это включает визуализацию, анализ и мониторинг уязвимостей безопасности в вашей среде, а также блокировку обнаруженных атак на ваши приложения.

## Runtime Vulnerability Analytics

Dynatrace Runtime Vulnerability Analytics позволяет обнаруживать, визуализировать, анализировать и контролировать устранение сторонних уязвимостей и уязвимостей на уровне кода в вашей среде.

К таким уязвимостям относятся:

* Сторонние уязвимости, которые могут возникать, когда ваше приложение использует определённую библиотеку или языковую среду выполнения, содержащую уязвимости.
* Уязвимости на уровне кода, которые могут возникать, когда Dynatrace Application Security обнаруживает уязвимость в вашем коде, анализируя запросы, проходящие через ваши приложения.

## Runtime Application Protection (RAP)

Dynatrace Runtime Application Protection использует аналитику на уровне кода и анализ транзакций для автоматического обнаружения и блокировки атак на ваши приложения в реальном времени.

## Security Posture Management (SPM)

Security Posture Management (SPM) обеспечивает непрерывный мониторинг и автоматизированную оценку для сред Kubernetes.
Благодаря аналитике конфигурации, проблем соответствия требованиям и управления рисками клиенты могут поддерживать надёжный уровень безопасности.

Это включает Kubernetes Security Posture Management (KSPM), который направлен на выявление и снижение рисков безопасности в кластерах Kubernetes.
