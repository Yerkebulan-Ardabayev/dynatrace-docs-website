# EdgeConnect release notes version 1.416.14


* Актуальная версия Dynatrace
* Примечания к выпуску
* Опубликовано 13 мая 2025 г.

Дата выпуска: 29 апр. 2025 г.

## Новые функции и улучшения

* EdgeConnect теперь записывает предупреждение и указывает на документацию, если настроенный ресурс OAuth не соответствует формату `urn:dtenvironment:<tenant>`.

* EdgeConnect теперь предоставляет более подробную информацию об ошибках HTTP, что помогает определить первопричины неудачных HTTP-запросов.

* После потери соединения с платформой Dynatrace EdgeConnect завершает работу через 1 минуту в случае нескольких неудачных попыток повторного подключения. Такое поведение при завершении работы поможет вам быстро выявлять проблемы с подключением к платформе Dynatrace.

## Изменения в этой версии

EdgeConnect больше не следует HTTP-перенаправлениям автоматически. Перенаправленные HTTP(S)-запросы теперь корректно обрабатываются в среде выполнения JavaScript.

Это изменение позволяет вам:

* Управлять поведением запросов с помощью флагов перенаправления `follow`, `error` и `manual` в вашем JavaScript-коде, используя API `fetch`.
* Предотвращать перенаправление к запрещённым HTTP-конечным точкам согласно конфигурации EdgeConnect.
* Направлять каждое перенаправление в выделенный экземпляр EdgeConnect, как настроено в приложении **EdgeConnect Management**.

Если вы используете `node:http(s)` в своём JavaScript-коде, учтите, что обработка перенаправлений теперь должна выполняться вручную.
