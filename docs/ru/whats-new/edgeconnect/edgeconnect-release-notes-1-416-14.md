**Дата выпуска:** 29 апреля 2025

## Новое

- Предупреждение при несоответствии OAuth-ресурса формату `urn:dtenvironment:<tenant>`
- Более подробная информация об ошибках HTTP
- Завершение работы через 1 минуту при потере соединения с платформой (после нескольких неудачных попыток)

## Изменения

EdgeConnect больше не следует HTTP-перенаправлениям автоматически. Перенаправления обрабатываются в среде выполнения JavaScript.

- Управление перенаправлениями через `fetch` API (флаги `follow`, `error`, `manual`)
- Предотвращение перенаправления к запрещенным endpoint
- Маршрутизация перенаправлений через выделенные экземпляры EdgeConnect

При использовании `node:http(s)` обработку перенаправлений нужно выполнять вручную.
