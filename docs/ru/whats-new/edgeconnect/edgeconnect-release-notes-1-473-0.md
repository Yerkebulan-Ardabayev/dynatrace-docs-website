# Заметки о выпуске EdgeConnect версии 1.473.0


* Заметки о выпуске

Дата выпуска: 18 июня 2025 г.

## Новые функции и улучшения

* EdgeConnect теперь включает `Software Bill of Materials (SBOM)` для обеспечения прозрачности цепочки поставок. Это помогает вам определить источник потенциальных уязвимостей безопасности в образе EdgeConnect.
