# Примечания к выпуску OneAgent

Dynatrace OneAgent выпускается каждый второй спринт. Процесс поэтапного развёртывания обычно занимает до четырёх недель. Ниже приведены последние версии с описанием новых функций, изменений и исправлений.

---

## Версия 1.327

**Начало развёртывания:** 18 ноября 2025
**Обновлено:** 3 февраля 2026

### Новые возможности

- **Специализированный сенсор для AWSSDK.DynamoDBv2** — реализован новый сенсор для мониторинга вызовов к базам данных DynamoDB через AWS SDK v2. Обеспечивает детальное отслеживание операций чтения и записи.

- **Поддержка Go 1.24.9 и 1.25.3** — OneAgent теперь поддерживает мониторинг приложений, написанных на Go версий 1.24.9 и 1.25.3.

- **Автоматический парсинг JSON-логов** — модуль логирования OneAgent (Log Module) теперь по умолчанию распознаёт формат JSON для принимаемых логов, включая определение границ записей, извлечение временных меток и определение уровня логирования.

- **Переименование Java RUM v2** — функция Java Real User Monitoring v2 переименована в **Java Real User Monitoring (RUM) Troubleshooting Mode** и понижена с уровня opt-in до DevOps-only. Теперь доступна только командам DevOps для целей отладки.

---

## Версия 1.325

**Начало развёртывания:** 21 октября 2025

### Новые возможности

- **Событие взаимодействия при нажатии Space** — RUM Web Agent теперь отправляет событие пользовательского взаимодействия при нажатии отдельной клавиши Space.

- **Поддержка MongoDB Reactive Streams и sync 5.x** — добавлена поддержка MongoDB Reactive Streams и новых синхронных версий MongoDB-драйвера (5.x).

- **Трассировка Kotlin Coroutine** — добавлена поддержка трассировки Kotlin Coroutine (отключена по умолчанию). Для включения требуется настройка через параметры OneAgent.

- **Session Replay для мобильных приложений** — позволяет просматривать записи взаимодействия пользователей с нативными мобильными приложениями: нажатия, жесты, переходы — в формате видеоподобного воспроизведения.

---

## Версия 1.323

**Начало развёртывания:** сентябрь 2025

### Новые возможности

- **Улучшенное определение бинарных источников логов** — начиная с версии 1.323, OneAgent индивидуально проверяет каждый файл, назначенный источнику логов, на наличие бинарного содержимого. Если хотя бы один файл содержит бинарные данные, весь источник логов помечается как бинарный.

- **Новые AWS Lambda Layers для Python** — добавлены новые Lambda Layers для автоматической инструментации AWS Lambda на Python. Поддерживаются AWS SDK (SQS, SNS), базы данных (DynamoDB, Redis), EventBridge и Lambda Triggers.

- **Инжекция Node.js через --require** — OneAgent для Node.js теперь инжектируется с помощью опции `--require`, что решает известную проблему в Next.js 15+ при использовании команд `next dev` или `next build`.

## Как обновить OneAgent

1. Перейдите в **Dynatrace → Manage → Deployment status**
2. Проверьте текущую установленную версию
3. Включите автоматическое обновление или выполните обновление вручную
4. После обновления проверьте статус агента в разделе **Host details**

!!! note "Рекомендация"
    Рекомендуется всегда обновлять OneAgent до последней поддерживаемой версии для получения исправлений безопасности и новых функций.
