# Примечания к выпуску OneAgent

Выпускается каждый второй спринт. Развертывание занимает до 4 недель.

---

## Версия 1.327

**Развертывание:** 18 ноября 2025 | **Обновлено:** 3 февраля 2026

- Новый сенсор для AWSSDK.DynamoDBv2
- Поддержка Go 1.24.9 и 1.25.3
- Автоматический парсинг JSON-логов в Log Module
- Java RUM v2 переименован в Java RUM Troubleshooting Mode (DevOps-only)

---

## Версия 1.325

**Развертывание:** 21 октября 2025

- RUM Web Agent: событие взаимодействия при нажатии Space
- Поддержка MongoDB Reactive Streams и sync 5.x
- Трассировка Kotlin Coroutine (отключена по умолчанию)
- Session Replay для мобильных приложений

---

## Версия 1.323

**Развертывание:** сентябрь 2025

- Улучшенное определение бинарных источников логов (проверка каждого файла)
- Новые AWS Lambda Layers для Python (SQS, SNS, DynamoDB, Redis, EventBridge)
- Инжекция Node.js через `--require` (исправление для Next.js 15+)

---

## Поддержка

| Платформа | Стандартная | Корпоративная |
| --- | --- | --- |
| OneAgent | 9 месяцев | 12 месяцев |

## Обновление

1. **Dynatrace -> Manage -> Deployment status** — проверьте версию
2. Включите автообновление или обновите вручную
3. Проверьте статус в **Host details**

Рекомендуется всегда обновлять до последней версии.
