# Примечания к выпуску ActiveGate

Dynatrace ActiveGate выпускается каждый второй спринт. Процесс поэтапного развёртывания обычно занимает до четырёх недель. Ниже приведены последние версии с описанием новых функций, изменений и исправлений.

---

## Версия 1.327

**Начало развёртывания:** 18 ноября 2025
**Обновлено:** 28 января 2026

### Новые возможности

- **Улучшенная производительность парсинга OpenMetrics** — значительно повышена производительность парсинга OpenMetrics в ActiveGate для мониторинга Kubernetes.

- **Освобождение простаивающих соединений** — ActiveGate теперь автоматически освобождает неактивные соединения в фоновом режиме, предотвращая сценарии насыщения, которые могли приводить к пробелам в мониторинге.

- **Ручная установка Chrome for Testing** — добавлена поддержка ручной установки Chrome for Testing на Amazon Linux 2023, Ubuntu 24.04 и Oracle Linux 9.5 для Synthetic-enabled ActiveGate.

- **Маскирование данных Kubernetes** — расширено маскирование данных для Kubernetes Enhanced Object Visibility в представлении YAML. Теперь маскируются чувствительные данные: значения переменных окружения workload и pod, а также массивы command и args.

- **Обогащение метрик Prometheus** — метрики kube-state-metrics, принимаемые через ActiveGate Kubernetes monitoring, теперь обогащаются измерением `k8s.cluster.uid`.

### Исправления

- Устранена проблема с приёмом данных для Kubernetes Enhanced Object Visibility Preview, затрагивавшая определённые пограничные случаи.

## Версия 1.325

**Начало развёртывания:** 21 октября 2025

### Новые возможности

- **Потоковый вывод support-archive** — команда `agctl create-support-archive` теперь поддерживает опцию `--stdout`, позволяющую передавать содержимое архива напрямую в stdout с перенаправлением логов в stderr. Это упрощает рабочие процессы, устраняет ограничения дискового пространства и обеспечивает интеграцию с инструментами Kubernetes.

### Важные изменения

!!! warning "Прекращение поддержки платформ"
    ActiveGate версии 1.325 — последняя версия Synthetic-enabled ActiveGate, поддерживающая **Red Hat / Oracle Linux / Rocky Linux 8**. Начиная с версии 1.327, эти платформы больше не поддерживаются.

!!! note "Требование к Chromium"
    Начиная с Synthetic версии 1.325, установки больше не работают с версиями Chromium старше 126.

## Как обновить ActiveGate

1. Перейдите в **Dynatrace → Manage → Deployment status → ActiveGates**
2. Проверьте текущую версию ActiveGate
3. Скачайте установщик новой версии из Dynatrace или используйте автоматическое обновление
4. На Linux: выполните команду установки с правами root
5. На Windows: запустите установщик от имени администратора
6. Проверьте статус в разделе **ActiveGate details**

!!! note "Рекомендация"
    Перед обновлением ActiveGate в production-среде рекомендуется выполнить обновление в тестовой среде.
