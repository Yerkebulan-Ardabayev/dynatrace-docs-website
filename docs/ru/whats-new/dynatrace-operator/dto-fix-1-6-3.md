**Дата выпуска:** 20 сентября 2025

Патч-релиз. Новые функции см. в примечаниях к версии 1.6.

## Исправления

- Восстановлено событие `mark for termination` (удалено в 1.6.0) для обнаружения завершения узлов при автомасштабировании. Требуется область `DataExport` в API-токене.

## Известные проблемы

- При переключении CSI-драйвера на node image pull — убедитесь, что файловая система CSI не содержит модуля кода для данного DynaKube, иначе произойдет ошибка.

## Обновление с версии 1.5

Критических изменений нет.
