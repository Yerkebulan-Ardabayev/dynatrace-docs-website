# Примечания к выпуску Dynatrace Operator версии 0.10.1

# Примечания к выпуску Dynatrace Operator версии 0.10.1

* Latest Dynatrace
* Примечания к выпуску
* Опубликовано 15 дек. 2022

Дата выпуска: 15 дек. 2022

## Исправленные проблемы

* Исправлен вебхук преобразования в определении пользовательского ресурса: поле пространства имён теперь обновляется корректно.

* Поддержка неизменяемого OneAgent: информация о взаимодействии теперь хранится как в `Secret` (`TenantToken`), так и в `ConfigMap` (`TenantUUID`, `CommunicationEndpoints`). Это устраняет перезапуски при изменении конечных точек взаимодействия путём монтирования их из configmap вместо добавления содержимого configmap непосредственно в аргументы.

* Исправлено отсутствие меток Docker на образе выпуска.

* Компонент node-driver-registrar обновлён до версии 2.6.2.

* Исправлена ошибка в режиме `hostMonitoring` только для чтения: режим `hostMonitoring` только для чтения требует наличия драйвера CSI, однако драйвер CSI работал некорректно, что препятствовало запуску пода OneAgent.

* Исправлена ошибка, вызывавшая некорректное размонтирование подов из-за заблокированной базы данных: при одновременном запросе тома несколькими подами драйвер CSI не размонтировал поды корректно, если база данных была заблокирована в момент монтирования.
