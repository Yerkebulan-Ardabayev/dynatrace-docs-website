# EdgeConnect release notes


* Актуальная версия Dynatrace
* Примечания к выпуску
* Обновлено 26 января 2026 г.

Используйте [EdgeConnect](../ingest-from/edgeconnect.md "Используйте EdgeConnect для управления взаимодействием ваших приложений и рабочих процессов с внутренними системами."), чтобы обеспечить безопасное взаимодействие приложений и рабочих процессов с вашими системами.

В таблице ниже перечислены все выпуски EdgeConnect. Выберите номер версии, чтобы просмотреть примечания к соответствующему выпуску EdgeConnect.

* EdgeConnect имеет собственный цикл выпусков.
* Поддержка каждой версии EdgeConnect заканчивается через 12 месяцев после её выпуска.
