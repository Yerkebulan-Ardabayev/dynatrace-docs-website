## Dynatrace OneAgent

Dynatrace, как правило, поддерживает технологии и их версии на шесть месяцев дольше, чем вендор, чтобы дать вам достаточно времени для обновления вашей среды. [Среды выполнения](../../ingest-from/technology-support/application-software.md "Discover what runtimes can be monitored with Dynatrace.") могут иметь расширенный жизненный цикл поддержки.

Объявления об окончании поддержки публикуются за шесть месяцев до его наступления.

Пример типичного жизненного цикла поддержки технологии

1. Вендор объявил об окончании поддержки конкретной технологии с 15 января 2023 г.
2. Dynatrace также объявляет об окончании поддержки данной технологии, в результате чего дата окончания поддержки составляет 31 июля 2023 г. (последний день месяца).

Схема версий для окончания поддержки

Определение схемы версий для окончания поддержки с примерами:

* **Мажорная версия 5 завершила поддержку**

  + Мажорная версия 5 завершила поддержку, включая все её минорные версии, например 5.1 и 5.2
  + Другие мажорные версии по-прежнему поддерживаются, например 6 и 7
* **Минорная версия 5.1 завершила поддержку**

  + Минорная версия 5.1 завершила поддержку, включая все её патч-версии, например 5.1.1 и 5.1.2
  + Другие минорные версии по-прежнему поддерживаются, например 5.2 и 5.3
* **Патч-версия 5.1.1 завершила поддержку**

  + Патч-версия 5.1.1 завершила поддержку
  + Другие патч-версии по-прежнему поддерживаются, например 5.1.2 и 5.1.3
* **Version range 5.1 â 5.3 is end of support**

  + Минорные версии 5.1, 5.2 и 5.3 завершили поддержку, включая все их патч-версии, например 5.1.1, 5.2.1 и 5.3.1
  + Другие минорные версии по-прежнему поддерживаются, например 5.0 и 5.4
* **Минимально необходимая версия 5+ завершила поддержку**

  + Все мажорные, минорные и патч-версии, начиная с версии 5, завершили поддержку, например 5, 5.1, 5.1.1 и 6

## Поддержка Dynatrace ActiveGate

Список версий операционных систем, поддерживаемых Dynatrace ActiveGate, обновляется постоянно.

* Мы выпускаем новую версию Dynatrace ActiveGate каждый месяц.
* Список версий операционных систем, поддерживаемых Dynatrace ActiveGate, обновляется с той же периодичностью.
* ActiveGate поддерживает конкретные мажорные дистрибутивы Linux. До тех пор пока мажорная версия ОС поддерживается вендором, каждый последующий минорный выпуск будет поддерживаться ActiveGate, если только прекращение поддержки не будет явно объявлено.
* Мы объявляем дату окончания поддержки (EOS) для версии ОС, когда вендор объявляет о её конце жизни (EOL).
* Поддержка версии ОС прекращается не менее чем через 6 месяцев после её конца жизни (EOL), чтобы дать вам достаточно времени для обновления вашей среды.
* Когда версия ОС больше не поддерживается ActiveGate, Dynatrace не устраняет проблемы ActiveGate, связанные с операционной системой. Это также относится к ещё поддерживаемым версиям ActiveGate, выпущенным до окончания поддержки данной версии ОС со стороны ActiveGate.

### Окончание поддержки частных Synthetic-локаций на Red Hat/Oracle Linux/Rocky Linux 8

Для Red Hat/Oracle Linux/Rocky Linux 8 не выходят новые версии Chromium выше версии 133. По важным соображениям безопасности и стабильности мы приняли решение прекратить поддержку установки **Synthetic-enabled** ActiveGate на Red Hat/Oracle Linux/Rocky Linux 8 начиная с версии ActiveGate 1.325.

Для обеспечения непрерывности и безопасности синтетических мониторов рекомендуем перенести Synthetic-enabled ActiveGate на одну из [поддерживаемых операционных систем](../../observe/digital-experience/synthetic-monitoring/private-synthetic-locations/system-and-hardware-requirements-for-private-synthetic.md#linux-supported-os "Supported operating systems, Chromium versions, and hardware requirements for running synthetic monitors from private locations"), например Red Hat/Oracle Linux/Rocky Linux 9.

Версия ActiveGate 1.325 — **последняя версия Synthetic-enabled** ActiveGate, поддерживаемая на Red Hat/Oracle Linux/Rocky Linux 8.

Кроме того, начиная с Dynatrace версии 1.326, мы планируем ввести механизмы, предотвращающие обновление Synthetic-enabled ActiveGate на Red Hat/Oracle Linux/Rocky Linux 8 выше версии 1.325.
