# Metrics API v2


### Получить все

Получить обзор всех метрик, доступных в вашем окружении.### Получить описание

Получить полный дескриптор метрики.### Прочитать точки данных

Получить точки данных из нужных вам метрик.

Для каждой метрики можно получить либо одну агрегированную точку данных, либо список точек данных.

### Выбрать нужные данные

Metrics API — это гибкий инструмент для получения данных. Узнайте, как использовать преобразования селектора метрик для точной настройки области считываемых данных.### Изменить данные при чтении

Помимо преобразования метрики, вы можете выполнять простые арифметические операции непосредственно в запросе. Узнайте, как использовать выражения метрик для изменения считываемых данных.

### Добавить пользовательские метрики

Отправить пользовательские точки данных в Dynatrace.### Удалить метрику

Удалить метрику, которая больше не нужна.### Удалить метрики

Удалить метрики старше указанного количества дней.
