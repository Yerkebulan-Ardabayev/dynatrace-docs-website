# API тегов ручной вставки RUM


**API тегов ручной вставки RUM** позволяет получать JavaScript для RUM в двух различных сценариях ручной вставки:

* Мониторинг без агента и
* [Ручная вставка для страниц автоматически внедряемого приложения](../../../observe/digital-experience/web-applications/initial-setup/rum-injection.md#manual-insertion-using-oneagent "Configure automatic injection of the RUM JavaScript into the pages of your applications").

Доступны различные форматы фрагментов, а поддерживаемые атрибуты тегов можно контролировать через параметры API. Интегрируя этот API в свои скрипты сборки, вы можете автоматизировать вставку JavaScript для RUM и гарантировать, что ваше приложение всегда использует актуальную конфигурацию.

Для получения [фрагмента кода](../../../observe/digital-experience/web-applications/initial-setup/snippet-formats.md#code-snippet "Select a format for the RUM JavaScript snippet that best fits your specific use case") в формате фрагмента используйте API JavaScript для Real User Monitoring.
