**API хранилища учётных данных** позволяет управлять учётными данными для синтетических браузерных и HTTP- мониторов.

### Список всех учётных данных

Обзор всех учётных данных, настроенных в вашей среде Dynatrace.### Просмотр метаданных учётных данных

Получение параметров учётных данных по ID конфигурации.

### Создать учётные данные

Создание новой конфигурации учётных данных с точными параметрами, которые вам нужны.### Редактировать учётные данные

Обновление существующей конфигурации учётных данных.### Удалить учётные данные

Удаление ненужной конфигурации учётных данных.

## Связанные темы

* Настройка HTTP-мониторов
