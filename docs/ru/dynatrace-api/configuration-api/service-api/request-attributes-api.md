API **Request attributes** позволяет управлять конфигурацией атрибутов запросов.

### Просмотр всех атрибутов

Получите обзор всех атрибутов запросов.### Просмотр атрибута запроса

Получите параметры атрибута запроса по его идентификатору.

### Создание атрибута запроса

Создайте новый атрибут запроса с точными необходимыми параметрами.### Редактирование атрибута запроса

Обновите существующий атрибут запроса или создайте новый атрибут запроса с указанным идентификатором.### Удаление атрибута запроса

Удалите атрибут запроса, который больше не нужен.
