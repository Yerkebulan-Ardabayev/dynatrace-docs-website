# Request attributes API

* Reference
* Published Dec 05, 2018

API **Request attributes** позволяет управлять конфигурацией [атрибутов запросов](/docs/observe/application-observability/services/request-attributes "Understand what request attributes are and learn how to use them across all levels of all service-analysis views.").

[### Просмотр всех атрибутов

Получите обзор всех атрибутов запросов.](/docs/dynatrace-api/configuration-api/service-api/request-attributes-api/get-all "View all request attributes of your environment via the Dynatrace API.")[### Просмотр атрибута запроса

Получите параметры атрибута запроса по его идентификатору.](/docs/dynatrace-api/configuration-api/service-api/request-attributes-api/get-request-attribute "View a request attribute via the Dynatrace API.")

[### Создание атрибута запроса

Создайте новый атрибут запроса с точными необходимыми параметрами.](/docs/dynatrace-api/configuration-api/service-api/request-attributes-api/post-request-attribute "Create a request attribute via the Dynatrace API.")[### Редактирование атрибута запроса

Обновите существующий атрибут запроса или создайте новый атрибут запроса с указанным идентификатором.](/docs/dynatrace-api/configuration-api/service-api/request-attributes-api/put-request-attribute "Update a request attribute via the Dynatrace API.")[### Удаление атрибута запроса

Удалите атрибут запроса, который больше не нужен.](/docs/dynatrace-api/configuration-api/service-api/request-attributes-api/del-request-attribute "Delete a request attribute via the Dynatrace API.")
