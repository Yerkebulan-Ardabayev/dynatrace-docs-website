# API обнаружения сервисов


API **обнаружения сервисов на основе правил** позволяет управлять конфигурацией правил обнаружения сервисов.

Получить обзор всех правил обнаружения сервисов для:

* [Полных веб-сервисов](detection-rules/full-web-service/get-all.md "Просмотрите все правила обнаружения сервисов для полных веб-сервисов через Dynatrace API.")

### Изменить порядок правил

Правила обнаружения сервисов применяются последовательно одно за другим. Применяется первое подходящее правило, и дальнейшая обработка прекращается. Измените порядок правил обнаружения сервисов, чтобы добиться нужной последовательности применения.

* [Полные веб-запросы](detection-rules/full-web-request/reorder-rules.md "Измените порядок правил обнаружения сервисов для полных веб-запросов через Dynatrace API.")
* [Непрозрачные веб-запросы](detection-rules/opaque-web-request/reorder-rules.md "Измените порядок правил обнаружения сервисов для непрозрачных и внешних веб-запросов через Dynatrace API.")
* [Полные веб-сервисы](detection-rules/full-web-service/reorder-rules.md "Измените порядок правил обнаружения сервисов для полных веб-сервисов через Dynatrace API.")
* [Непрозрачные веб-сервисы](detection-rules/opaque-web-service/reorder-rules.md "Измените порядок правил обнаружения сервисов для внешних и непрозрачных веб-сервисов через Dynatrace API.")
