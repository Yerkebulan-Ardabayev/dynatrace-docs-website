# API обнаружения сервисов

* Справочник
* Опубликовано 19 июня 2019 г.

API **обнаружения сервисов на основе правил** позволяет управлять конфигурацией правил обнаружения сервисов.

### Получить все

Получить обзор всех правил обнаружения сервисов для:

* [Полных веб-запросов](/docs/dynatrace-api/configuration-api/service-api/detection-rules/full-web-request/get-all "Просмотрите все правила обнаружения сервисов для полных веб-запросов через Dynatrace API.")
* [Непрозрачных веб-запросов](/docs/dynatrace-api/configuration-api/service-api/detection-rules/opaque-web-request/get-all "Просмотрите все правила обнаружения сервисов для непрозрачных и внешних веб-запросов через Dynatrace API.")
* [Полных веб-сервисов](/docs/dynatrace-api/configuration-api/service-api/detection-rules/full-web-service/get-all "Просмотрите все правила обнаружения сервисов для полных веб-сервисов через Dynatrace API.")
* [Непрозрачных веб-сервисов](/docs/dynatrace-api/configuration-api/service-api/detection-rules/opaque-web-service/get-all "Просмотрите все правила обнаружения сервисов для внешних и непрозрачных веб-сервисов через Dynatrace API.")

### Изменить порядок правил

Правила обнаружения сервисов применяются последовательно одно за другим. Применяется первое подходящее правило, и дальнейшая обработка прекращается. Измените порядок правил обнаружения сервисов, чтобы добиться нужной последовательности применения.

* [Полные веб-запросы](/docs/dynatrace-api/configuration-api/service-api/detection-rules/full-web-request/reorder-rules "Измените порядок правил обнаружения сервисов для полных веб-запросов через Dynatrace API.")
* [Непрозрачные веб-запросы](/docs/dynatrace-api/configuration-api/service-api/detection-rules/opaque-web-request/reorder-rules "Измените порядок правил обнаружения сервисов для непрозрачных и внешних веб-запросов через Dynatrace API.")
* [Полные веб-сервисы](/docs/dynatrace-api/configuration-api/service-api/detection-rules/full-web-service/reorder-rules "Измените порядок правил обнаружения сервисов для полных веб-сервисов через Dynatrace API.")
* [Непрозрачные веб-сервисы](/docs/dynatrace-api/configuration-api/service-api/detection-rules/opaque-web-service/reorder-rules "Измените порядок правил обнаружения сервисов для внешних и непрозрачных веб-сервисов через Dynatrace API.")
