# API обнаружения сервисов


API **обнаружения сервисов на основе правил** позволяет управлять конфигурацией правил обнаружения сервисов.

### Получить все

Получить обзор всех правил обнаружения сервисов для:

* Полных веб-запросов
* Непрозрачных веб-запросов
* Полных веб-сервисов
* Непрозрачных веб-сервисов

### Просмотреть правило

Получить параметры правила обнаружения сервиса для:

* Полных веб-запросов
* Непрозрачных веб-запросов
* Полных веб-сервисов
* Непрозрачных веб-сервисов

### Изменить порядок правил

Правила обнаружения сервисов применяются последовательно одно за другим. Применяется первое подходящее правило, и дальнейшая обработка прекращается. Измените порядок правил обнаружения сервисов, чтобы добиться нужной последовательности применения.

* Полные веб-запросы
* Непрозрачные веб-запросы
* Полные веб-сервисы
* Непрозрачные веб-сервисы

### Создать правило

Создать новое правило обнаружения сервиса для:

* Полных веб-запросов
* Непрозрачных веб-запросов
* Полных веб-сервисов
* Непрозрачных веб-сервисов

### Редактировать правило

Обновить существующее правило обнаружения сервиса или создать новое правило с указанным идентификатором.

* Полные веб-запросы
* Непрозрачные веб-запросы
* Полные веб-сервисы
* Непрозрачные веб-сервисы

### Удалить правило

Удалить правило обнаружения сервиса, которое вам больше не нужно.

* Полные веб-запросы
* Непрозрачные веб-запросы
* Полные веб-сервисы
* Непрозрачные веб-сервисы
