# API конфигурации веб-приложений


**API конфигурации веб-приложений** позволяет управлять конфигурацией веб-приложений.

Этот API поддерживает только веб-приложения. Для мобильных и пользовательских приложений см. API для мобильных и пользовательских приложений.

### Список всех приложений

Получить обзор всех веб-приложений.### Просмотр веб-приложения

Получить параметры веб-приложения по его идентификатору.

### Создание веб-приложения

Создать новое веб-приложение с точными требуемыми параметрами.### Редактирование веб-приложения

Обновить существующее веб-приложение или создать новое приложение с указанным идентификатором.### Удаление веб-приложения

Удалить ненужное веб-приложение.

Приложение по умолчанию предварительно настроено в вашей среде Dynatrace. По умолчанию весь трафик направляется в это приложение. После настройки собственных приложений весь трафик, не соответствующий ни одному из ваших приложений, направляется в приложение по умолчанию.

### Просмотр конфигурации

Получить параметры веб-приложения по умолчанию.### Обновление конфигурации

Обновить параметры веб-приложения по умолчанию.

### Проверка конфиденциальности данных

Просмотр параметров конфиденциальности данных для

* Всех приложений
* Конкретного приложения
* Приложения по умолчанию

### Обновление конфиденциальности данных

Просмотр параметров конфиденциальности данных для

* Конкретного приложения
* Приложения по умолчанию

### Просмотр ключевых пользовательских действий

Получить список ключевых пользовательских действий в приложении.### Редактирование списка ключевых пользовательских действий

Отметить пользовательское действие как ключевое в приложении.### Удаление ключевого пользовательского действия

Удалить пользовательское действие из списка ключевых действий в приложении.

### Просмотр правил ошибок

Получить обзор конфигурации правил ошибок.### Обновление правил ошибок

Обновить конфигурацию правил ошибок.
