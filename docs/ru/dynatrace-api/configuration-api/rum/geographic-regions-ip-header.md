# Географические регионы - заголовки сопоставления IP API

* Ссылка
* Опубликовано 24 сентября 2020 г.

**Географические регионы - заголовки сопоставления IP** API позволяет вам управлять конфигурацией сопоставления IP-адресов с географическими местами. Вы также можете управлять той же конфигурацией в интерфейсе Dynatrace в **Настройки > Мониторинг веба и мобильных приложений > Определение IP**.

[### Просмотр конфигурации

Получите обзор сопоставления IP-адресов.](/docs/dynatrace-api/configuration-api/rum/geographic-regions-ip-header/get-configuration "Прочитайте конфигурацию заголовков сопоставления IP через Dynatrace API.")[### Обновление конфигурации

Обновите конфигурацию сопоставления IP-адресов.](/docs/dynatrace-api/configuration-api/rum/geographic-regions-ip-header/put-configuration "Обновите конфигурацию заголовков сопоставления IP через Dynatrace API.")
