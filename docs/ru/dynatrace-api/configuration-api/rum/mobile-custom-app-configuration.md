# API для мобильных и пользовательских приложений


### Список всех приложений

Получить обзор всех мобильных и пользовательских приложений.### Просмотр приложения

Получить параметры приложения по его идентификатору.

### Создание приложения

Создать новое мобильное или пользовательское приложение с точными необходимыми параметрами.### Редактирование приложения

Обновить существующее мобильное или пользовательское приложение либо создать новое приложение с указанным идентификатором.### Удаление приложения

Удалить приложение, которое больше не нужно.

### Просмотр ключевых действий пользователя

Получить список ключевых действий пользователя в указанном приложении.### Редактирование списка ключевых действий пользователя

Пометить действие пользователя как ключевое в указанном приложении.### Удаление ключевого действия пользователя

Удалить действие пользователя из списка ключевых действий в указанном приложении.

### Список всех свойств пользовательских сессий

Получить обзор всех свойств сессий приложения.### Просмотр свойства пользовательской сессии

Получить параметры свойства сессии по его идентификатору.

### Создание свойства пользовательской сессии

Создать новое свойство пользовательской сессии для мобильного или пользовательского приложения.### Редактирование свойства пользовательской сессии

Обновить существующее свойство пользовательской сессии для мобильного или пользовательского приложения.### Удаление свойства пользовательской сессии

Удалить свойство пользовательской сессии, которое больше не нужно.
