Dynatrace Managed exposes cluster-wide functionality via REST endpoints. This interactive documentation also acts as a REST client you can use to interact with Dynatrace Managed clusters.

To access the Cluster API

1. Open the Cluster Management Console.
2. Open the User menu by clicking the User icon in the upper-right corner.
3. Select **Cluster API v2**.

### Environments

List all existing environments

Create a new environment

List properties for specific environment

Update specific environment

Delete specific environment

### Synthetic locations and nodes

List all cluster private Synthetic locations

Create new private Synthetic locations

Get properties of specified cluster locations

Update specified private Synthetic cluster location

Delete specified private Synthetic cluster location

List all Synthetic cluster nodes

List properties of specified Synthetic cluster nodes

### Tokens

List available cluster tokens

Create new cluster token

Update cluster token

Delete cluster token

List cluster token metadata with request

List cluster token metadata with ID

### User management

Get cluster user sessions configuration

Update cluster user sessions configuration

Get cluster user sessions

Delete user sessions of a given user

### Remote access

Get all cluster access requests

Grant remote access permission

Get Cluster access request

Change state of access request

### License

Export license data

### Log Monitoring

Get Log Monitoring status

Update log events per cluster for Log Monitoring
