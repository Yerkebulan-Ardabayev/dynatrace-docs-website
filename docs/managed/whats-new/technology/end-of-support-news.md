## Dynatrace OneAgent

Dynatrace typically supports technologies and their versions six months longer than the vendor to give you enough time to upgrade your environment. [Runtimes](/managed/ingest-from/technology-support/application-software "Discover what runtimes can be monitored with Dynatrace.") may have an enhanced support lifecycle.

End of support announcements are provided six months in advance.

A typical technology support lifecycle example

1. The vendor has announced the end of support for a specific technology as of January 15, 2023.
2. Dynatrace is also announcing the end of support for this technology, resulting in an end-of-support date of July 31, 2023 (last day of the month).

End of support version schema

Definition of the end of support version schema with examples:

* **Major version 5 is end of support**

  + Major version 5 is end of support, including all of its minor versions like 5.1 and 5.2
  + Other major versions are still supported like 6 and 7
* **Minor version 5.1 is end of support**

  + Minor version 5.1 is end of support, including all of its patch versions like 5.1.1 and 5.1.2
  + Other minor versions are still supported like 5.2 and 5.3
* **Patch version 5.1.1 is end of support**

  + Patch version 5.1.1 is end of support
  + Other patch versions are still supported like 5.1.2 and 5.1.3
* **Version range 5.1 – 5.3 is end of support**

  + Minor versions 5.1, 5.2, and 5.3 are end of support, including all of their patch versions like 5.1.1, 5.2.1, and 5.3.1
  + Other minor versions are still supported like 5.0 and 5.4
* **The minimum required version 5+ is end of support**

  + All major, minor, and patch versions starting from version 5 are end of support like 5, 5.1, 5.1.1, and 6

## Dynatrace ActiveGate

The list of operating system versions supported by Dynatrace ActiveGate is updated continuously.

* We release a new version of Dynatrace ActiveGate every month.
* We update the list of operating system versions supported by Dynatrace ActiveGate at the same cadence.
* ActiveGate supports specific major Linux distributions. As long as the major OS version is supported by its vendor, each subsequent minor release will be supported by ActiveGate unless discontinuation is explicitly announced.
* We announce an end-of-support (EOS) date for an OS version when its end of life (EOL) is announced by the vendor.
* An OS version is de-supported at least 6 months after its end of life (EOL), to give you enough time to upgrade your environment.
* When an OS version is no longer supported by ActiveGate, Dynatrace doesn’t resolve any ActiveGate issues related to the operating system. This also applies to still-supported ActiveGate versions released before the end of support of the operating system version by ActiveGate.

## Dynatrace Managed

We typically support an operating system (OS) six months longer than the vendor, giving you enough time to update your Dynatrace Managed environment. When the vendor announces the regular end of support for a particular OS, we usually use this as the trigger to announce the end of support for it as well (with a 6-month transition period). Note that we can't consider extended paid support contracts with individual vendors.

The last compatible Dynatrace Managed version indicates when a new cluster installation on a particular OS is no longer possible. This includes adding a node to an existing cluster. However, you can still update your existing cluster to higher Dynatrace Managed versions.

For currently supported OS, see [Dynatrace Managed system requirements](/managed/managed-cluster/installation/operating-system-requirements "Review the operating system, host, and network requirements you need to meet before installing Dynatrace Managed on a Linux host.").
