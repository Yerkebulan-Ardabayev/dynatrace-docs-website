# Verify network zones deployment (new installation)

# Verify network zones deployment (new installation)

* 1-min read
* Published Jan 28, 2020

When the deployment is done, verify that connections work as intended, by checking the number of OneAgents and ActiveGates in each network zone . You can do it via Dynatrace API:

* Dynatrace SaaS Use Network zones calls from the [Environment API](/managed/dynatrace-api/environment-api/network-zones "Manage network zones via the Dynatrace API.").
* Dynatrace Managed Use Network zones calls from the [Cluster API](/managed/dynatrace-api/cluster-api/cluster-api-v2 "Find out about managing environments, network zones, Synthetic locations, nodes, and tokens in Dynatrace Managed using the Cluster API v2.").
