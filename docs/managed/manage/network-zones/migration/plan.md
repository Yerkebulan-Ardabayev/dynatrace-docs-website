# Plan network zones (migration)

* 1-min read
* Published Jan 28, 2020

When the [analysis](/managed/manage/network-zones/migration/analyze "Find out how to analyze your network in preparation for network zones deployment.") is done, move on to the next step—planning your network zones.

In the planning stage, you need to check how much traffic your OneAgents generate. This will help you to plan the number of required ActiveGates. See [ActiveGate hardware and system requirements](/managed/ingest-from/dynatrace-activegate/installation "Learn how to configure ActiveGate")
