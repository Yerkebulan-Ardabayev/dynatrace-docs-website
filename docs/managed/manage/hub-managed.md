# Dynatrace Hub

# Dynatrace Hub

* Overview
* 7-min read
* Published May 26, 2025

Unlock the full potential of Dynatrace by finding, activating, and running extensions that address your specific observability needs.

Dynatrace Hub ![Hub](https://dt-cdn.net/images/hub-512-82db3c583e.png "Hub") is the central place to explore and activate all capabilities of Dynatrace. You can find solutions to monitor your environment, analyze your observability data to get insights and automate problem resolution based on your insights. You stay up to date with recent additions to Dynatrace Managed. You can easily install extensions with one click via Dynatrace Hub and benefit from continuous updates. You can manage your extensions to stay in control.

* Browse all the capabilities of Dynatrace Managed
* Keep up to date with the recent additions to Dynatrace Managed
* Find the solution to your monitoring problem
* Easy one-click installation of extensions
* Get continuous updates to receive the latest features
* Manage your extensions

## Where to find Dynatrace Hub

To access Hub, in the menu, go to the **Manage** section and select **Dynatrace Hub**.

## What's listed in Dynatrace Hub

Generally, Dynatrace Hub lists any technology that Dynatrace officially supports, such as:

* OneAgent out-of-the-box supported technologies.
* Supported Open Observability Frameworks.
* Dynatrace built extensions.

## Extensions

With [Extensions](/managed/ingest-from/extensions/develop-your-extensions "Develop your own Extensions in Dynatrace.") you can declaratively bring metrics into Dynatrace that feed platform analytics and monitoring capabilities. Dynatrace links your data in a meaningful way so you can explore it, build instrumentation, and set up alerting.

## Discover

You can browse through all the Hub listings using the provided categories.

If you have a specific problem in mind, you can search the Hub listings and filter your results.

### Get information

Once you find an interesting listing, select the icon to open the overview page with the following information:

* **Product information**
* **Technical information**
* **Content**
* **Release notes**

### Filter

In the **Browse** tab in Dynatrace Hub, you can use the search bar to filter out the list of available extensions and technologies.
