# User and group management

# User and group management

* Published Jul 01, 2024

Dynatrace enables you to manage user permissions based on user account membership in user groups. You can manage these accounts and groups using the Dynatrace-provided user database, through LDAP, or through an IdP with the supported federation protocols SAML and SCIM.
