# Доставка программного обеспечения

Инструменты для DevOps-инженеров, SRE и команд разработки для более быстрой, качественной и безопасной поставки ПО.

## Основные возможности

- **[Мониторинг релизов](release-monitoring/)** — оценка производительности отдельных версий релизов
- **[Интеграция нагрузочного тестирования](test-automation/)** — остановка сломанных сборок на ранних этапах
- **[Владение сервисами](ownership/)** — привязка команд к мониторируемым объектам для совместной работы
- **[Конфигурация как код](configuration-as-code/)** — управление конфигурацией через Monaco или Terraform
- **[Backstage](backstage-integration/)** — интеграция Dynatrace в Backstage

## Dynatrace Classic

- **[SLO Classic](service-level-objectives-classic/)** — цели уровня обслуживания для ускорения SRE-практик
