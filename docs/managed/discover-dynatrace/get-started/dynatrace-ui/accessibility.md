# Accessibility

* Reference
* 1-min read
* Updated on May 15, 2026

Dynatrace develops products and services that promote diversity, equality, and inclusion. High accessibility standards are essential to support the usability of the platform for individuals with disabilities.

* Dynatrace development best practices and accessibility criteria align with **Web Content Accessibility Guidelines** (WCAG), the internationally recognized standard developed by the **World Wide Web Consortium** (W3C).
* The **Voluntary Product Accessibility Template** (VPAT) is a standardized model to document how a product conforms to a set of accessibility standards.
* The VPAT creates an **Accessibility Conformance Report** (ACR), which provides details of how a product’s features meet this set of criteria.

Dynatrace works with third-party vendors and tools to evaluate its accessibility posture against the WCAG standard, currently [WCAG 2.1 level AA﻿](https://www.w3.org/TR/WCAG21/).

The product accessibility journey at Dynatrace is continuous, with iterative improvements toward WCAG 2.1 level AA compliance. The ACR linked below represents accessibility status at the time of publishing, so check back for annual updates to this report.
