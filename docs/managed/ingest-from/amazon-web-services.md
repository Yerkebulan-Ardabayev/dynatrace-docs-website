Dynatrace offers both infrastructure monitoring of all AWS services and full-stack monitoring of services that can be instrumented with our OneAgent technology. On top of that, you can choose to monitor logs generated by the cloud services as well as your own services hosted within virtual machines or containers.

For more reasons to monitor AWS with Dynatrace, see [AWS](https://www.dynatrace.com/hub/detail/aws/?query=aws) in Dynatrace Hub.

## Cloud Platform Monitoring

Learn how to integrate Dynatrace with Amazon CloudWatch.

### End-to-end guide for monitoring AWS services

Set up Infrastructure Monitoring for AWS and ingest the data into Dynatrace remotely from CloudWatch.### Ingest Amazon CloudWatch Metric Streams

Stream CloudWatch metrics issued in a given AWS region through Amazon Data Firehose to Dynatrace.### Log Monitoring

Monitor logs generated by the AWS cloud services as well as your own services hosted within virtual machines or containers.### AWS cloud services

Monitor additional AWS cloud services and view their specific metrics in Dynatrace.
