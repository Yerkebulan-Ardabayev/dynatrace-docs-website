# Delete a custom application in RUM Classic

# Delete a custom application in RUM Classic

* How-to guide
* 1-min read
* Published Jan 30, 2023

When you no longer need your application, you can delete it either via the Dynatrace web UI or via API.

## Delete an application via UI

To delete an application via Dynatrace web UI

1. Go to **Settings** > **Monitoring** > **Monitoring overview**.
2. Switch to the **Applications** tab and find the application that you want to delete.
3. Next to the application name, select **Expand** ![Expand row](https://dt-cdn.net/images/expand-row-icon-9c4950fc2e.svg "Expand row"), and then select **Remove** > **Yes**.

## Delete an application via API

You can also delete your applications from Dynatrace via the Configuration API. For details, see [Delete a mobile or custom application](/managed/dynatrace-api/configuration-api/rum/mobile-custom-app-configuration/apps/delete-app "Delete a mobile or custom app via the Dynatrace API.").
