# Обновления Dynatrace Managed

Руководство по обновлению Dynatrace Managed.

## Проверка обновлений

1. Откройте CMC: `https://<cluster-ip>:8443/cmc`
2. Перейдите в **Cluster → Updates**
3. Проверьте доступные версии

## Перед обновлением

!!! warning "Важно"
    Всегда делайте полный бэкап перед обновлением!

### Чек-лист

- [ ] Бэкап конфигурации
- [ ] Бэкап данных Cassandra
- [ ] Проверка здоровья кластера
- [ ] Уведомление пользователей о maintenance

## Процесс обновления

### Rolling Update (без простоя)

```bash
# На каждом узле по очереди
sudo /opt/dynatrace-managed/updater/update.sh
```

### Полное обновление

1. Остановите все узлы
2. Обновите каждый узел
3. Запустите узлы по очереди

## После обновления

1. Проверьте здоровье кластера в CMC
2. Проверьте все OneAgent соединения
3. Протестируйте основные функции

## Откат

```bash
# Откат на предыдущую версию
sudo /opt/dynatrace-managed/rollback.sh
```
