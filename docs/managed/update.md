# Обновления Dynatrace Managed

## Проверка обновлений

CMC → **Cluster → Updates**

## Перед обновлением

!!! warning "Важно"
    Всегда делайте полный бэкап перед обновлением!

Чек-лист: бэкап конфигурации, бэкап Cassandra, проверка здоровья кластера, уведомление пользователей.

## Процесс обновления

### Rolling Update (без простоя)

```bash
sudo /opt/dynatrace-managed/updater/update.sh
```

### Полное обновление

Остановите все узлы, обновите каждый, запустите по очереди.

## После обновления

Проверьте здоровье кластера в CMC, соединения OneAgent и основные функции.

## Откат

```bash
sudo /opt/dynatrace-managed/rollback.sh
```
