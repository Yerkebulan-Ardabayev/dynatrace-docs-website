Управление кластером Dynatrace Managed — установка узлов, настройка высокой доступности и мониторинг состояния кластера.

## Основные разделы

- **Управление кластером** — полное руководство по настройке и обслуживанию кластера
- **Установка** — требования и процесс установки
- **Настройка** — конфигурация после установки
- **Обслуживание** — мониторинг и поддержка кластера
- **Обновления** — процесс обновления Dynatrace Managed
- **Бэкап** — резервное копирование данных
- **Безопасность** — настройки безопасности кластера

## Архитектура кластера

Кластер Dynatrace Managed состоит из:

| Компонент | Описание |
|-----------|----------|
| **Cluster Node** | Узел кластера с Server и Embedded ActiveGate |
| **Cassandra** | Распределённое хранилище данных |
| **Elasticsearch** | Поиск и аналитика |
| **NGINX** | Обратный прокси и балансировка нагрузки |
