# Лицензирование Dynatrace Managed

## Модели

- **По хостам** — лицензия на каждый мониторируемый хост, все функции включены. Для статичных инфраструктур.
- **По потреблению (DPS)** — оплата за Davis Data Units (DDU). Для динамичных окружений.

## Управление

Активация: **CMC → Settings → License** → ввести ключ.

Использование: **CMC → License → Usage**

## Лимиты

| Функция | Starter | Standard | Enterprise |
|---------|---------|----------|------------|
| Хосты | 10 | 100 | Unlimited |
| Retention | 7 дней | 35 дней | Custom |
| Support | Email | 24/7 | Premium |
