# Лицензирование Dynatrace Managed

Информация о лицензировании.

## Модели лицензирования

### По хостам

- Лицензия на каждый мониторируемый хост
- Включает все функции для хоста
- Подходит для статичных инфраструктур

### По потреблению (DPS)

- Davis Data Units (DDU)
- Оплата за реально потребленные данные
- Подходит для динамичных окружений

## Управление лицензией

### Активация

1. Откройте CMC
2. **Settings → License**
3. Введите лицензионный ключ

### Просмотр использования

```
CMC → License → Usage
```
