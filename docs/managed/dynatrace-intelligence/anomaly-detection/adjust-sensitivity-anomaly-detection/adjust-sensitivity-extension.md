# Adjust the sensitivity of anomaly detection for extension events

* How-to guide
* 1-min read
* Published Sep 06, 2021

[Extensions](/managed/ingest-from/extend-dynatrace "Learn what extension mechanisms are offered by Dynatrace.") that you have installed provide various event thresholds that are evaluated by anomaly detection. The particular list depends on the extension.

To configure performance-anomaly thresholds for infrastructure components that are monitored with extensions

1. Go to **Settings** > **Anomaly detection** > **Extension events**. Here you'll find a list of all the event types that your installed monitoring extensions provide.
2. Expand the required event to see and configure available thresholds.
