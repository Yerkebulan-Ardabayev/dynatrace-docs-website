# Request naming API

* Reference
* Published Mar 05, 2019

The **Request naming** API enables you to manage the configuration of request naming rules.

[### List all rules

Get an overview of all request naming rules defined in your environment.](/managed/dynatrace-api/configuration-api/service-api/request-naming-api/get-all "View all request naming rules of your environment via the Dynatrace API.")[### View a rule

Get parameters of a request naming rule by its ID.](/managed/dynatrace-api/configuration-api/service-api/request-naming-api/get-a-rule "View a request naming rule via the Dynatrace API.")

[### Create a rule

Create a new request naming rule with the exact parameters you need.](/managed/dynatrace-api/configuration-api/service-api/request-naming-api/post-new-rule "Create a request naming rule via the Dynatrace API.")[### Edit a rule

Update a request naming rule. You can also create a new rule with the specified ID.](/managed/dynatrace-api/configuration-api/service-api/request-naming-api/put-a-rule "Edit a request naming rule via the Dynatrace API.")[### Delete a rule

Delete a request naming rule you don't need anymore.](/managed/dynatrace-api/configuration-api/service-api/request-naming-api/delete-a-rule "Delete a request naming rule via the Dynatrace API.")
