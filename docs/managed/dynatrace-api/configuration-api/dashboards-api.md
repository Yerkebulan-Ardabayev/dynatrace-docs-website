The **Dashboards Classic** API enables you to manage the configurations of your dashboards.

### List all dashboards

Get an overview of all dashboards configured in your Dynatrace environment.### View a dashboard

Get dashboard parameters by dashboard ID.

### Create a dashboard

Create a dashboard with the exact parameters you need.### Edit a dashboard

Update the existing configuration of a dashboard.### Delete a dashboard

Delete the configuration of a dashboard you no longer need.

### View sharing settings

Get an overview of sharing configuration of a dashboard.### Update sharing settings

Update the sharing configuration of a dashboard.
