# Conditional naming API

* Reference
* Published Apr 23, 2020

The **Conditional naming** API enables you to manage the configuration of naming rules for process groups.

[### List all rules

Get an overview of all conditional naming rules defined in your environment.](/managed/dynatrace-api/configuration-api/conditional-naming/get-all "View all conditional naming rules of your environment via the Dynatrace API.")[### View a rule

Get parameters of a conditional naming rule by its ID.](/managed/dynatrace-api/configuration-api/conditional-naming/get-rule "View a conditional naming rule via the Dynatrace API.")

[### Create a rule

Create a new conditional naming rule with the exact parameters you need.](/managed/dynatrace-api/configuration-api/conditional-naming/post-rule "Create a conditional naming rule via the Dynatrace API.")[### Edit a rule

Update a conditional naming rule. You can also create a new rule with the specified ID.](/managed/dynatrace-api/configuration-api/conditional-naming/put-rule "Update a conditional naming rule via the Dynatrace API.")[### Delete a rule

Delete a conditional naming rule you don't need anymore.](/managed/dynatrace-api/configuration-api/conditional-naming/del-rule "Delete a conditional naming rule via the Dynatrace API.")
