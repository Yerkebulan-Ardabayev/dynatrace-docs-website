# Anomaly detection API - Disk events

# Anomaly detection API - Disk events

* Reference
* Published Oct 30, 2018

The **Anomaly detection—Disk events** API enables you to manage the configuration of custom disk events. You can also [manage](/managed/dynatrace-intelligence/anomaly-detection/adjust-sensitivity-anomaly-detection/adjust-sensitivity-infastructure "Adjust problem detection sensitivity for infrastructure.") the same configuration in the Dynatrace web UI at **Settings > Anomaly detection > Hosts** (scroll down to the **Custom disk-detection rule** section).

[### List all disk events

Get an overview of all custom disk events configured in your Dynatrace environment.](/managed/dynatrace-api/configuration-api/anomaly-detection-api/anomaly-detection-api-disk-events/get-all "View all disk event rules of your environment via the Dynatrace API.")[### View a disk event

Get disk event parameters by event ID.](/managed/dynatrace-api/configuration-api/anomaly-detection-api/anomaly-detection-api-disk-events/get-event "View a disk event rule via the Dynatrace API.")

[### Create a disk event

Create a new event with the exact parameters you need.](/managed/dynatrace-api/configuration-api/anomaly-detection-api/anomaly-detection-api-disk-events/post-event "Create a disk event rule via the Dynatrace API.")[### Edit a disk event

Update the existing configuration of an event.](/managed/dynatrace-api/configuration-api/anomaly-detection-api/anomaly-detection-api-disk-events/put-event "Edit a disk event rule via the Dynatrace API.")[### Delete a disk event

Delete the configuration of an event you no longer need.](/managed/dynatrace-api/configuration-api/anomaly-detection-api/anomaly-detection-api-disk-events/del-event "Delete a disk event rule via the Dynatrace API.")
