The **locations** endpoints of the **Synthetic API v2** enable you to retrieve information about synthetic locations.

### Fetch list

Get an overview of all synthetic locations configured in your environment.### Get details

Get synthetic location parameters by location ID.

### Create a location

Create a new private location with the exact parameters you need.### Edit a location

Update the existing configuration of a private Synthetic location.### Delete a location

Delete the private location you no longer need.

### View status

Check the status of public synthetic locations.### Change status

Change the status of public synthetic locations.
