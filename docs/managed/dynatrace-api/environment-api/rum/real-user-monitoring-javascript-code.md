The **Real User Monitoring JavaScript API** helps you to set up and maintain your manually injected applications.

The endpoints for retrieving manual insertion tags in formats other than code snippet have been deprecated. For these formats, use the newer RUM manual insertion tags API.

### List injected applications

Fetch detailed information about your existing manually injected applications.### List available versions

Fetch a list of all available versions of the RUM JavaScript.### Check configured versions

Check the configured latest stable, previous stable, and custom RUM JavaScript versions.

### Check latest version

Check the most recent version of the Real User Monitoring JavaScript.### Check your version

Check the most recent version of the Real User Monitoring JavaScript you're using.

### Get synchronous code snippet

Obtain the most recent Real User Monitoring JavaScript as a code snippet for the synchronous load.### Get asynchronous code snippet

Obtain the most recent Real User Monitoring JavaScript as a code snippet for the deferred load.
