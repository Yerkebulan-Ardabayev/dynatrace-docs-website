The **third-party** endpoints of the **Synthetic API** enable you to push third-party synthetic data to Dynatrace.

### Push data

Upload third-party synthetic data to Dynatrace.### Push events

Upload events to your third-party synthetic monitors.### Modify state

Modify the state of your third-party synthetic monitors.
