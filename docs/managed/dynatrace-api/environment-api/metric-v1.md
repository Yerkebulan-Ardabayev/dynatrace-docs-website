This API is deprecated. It will be removed in the future. Use Metrics API v2 instead.

The **Timeseries v1** API enables you to manage metrics that Dynatrace collects from the different monitored entities over time.

### Get all

Get an overview of all metrics available in your environment.### Get description

Get descriptions of one or several metrics.### Read data points - GET

Fetch data points from metrics you need.### Read data points - POST

Fetch data points from metrics you need. A POST request provides less data about the metric than a GET request.

### Delete a custom metric

Delete a custom metric you no longer need.
