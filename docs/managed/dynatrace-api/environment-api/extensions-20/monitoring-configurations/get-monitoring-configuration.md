# Extensions 2.0 API - GET a monitoring configuration

* Reference
* Published Apr 07, 2021

Gets the monitoring configuration of the specified Extensions 2.0 extension.

The request produces an `application/json` payload.

|  |  |  |
| --- | --- | --- |
| GET | ManagedDynatrace for Government | `https://{your-domain}/e/{your-environment-id}/api/v2/extensions/{extensionName}/monitoringConfigurations/{configurationId}` |
| GET | Environment and Cluster ActiveGate (default port 9999) | `https://{your-activegate-domain}:9999/e/{your-environment-id}/api/v2/extensions/{extensionName}/monitoringConfigurations/{configurationId}` |

## Authentication

To execute this request, you need an access token with `extensionConfigurations.read` scope.

To learn how to obtain and use it, see [Tokens and authentication](/managed/discover-dynatrace/references/dynatrace-api/basics/dynatrace-api-authentication).

## Parameters

| Parameter | Type | Description | In | Required |
| --- | --- | --- | --- | --- |
| extensionName | string | The name of the requested extension 2.0. | path | Required |
| configurationId | string | The ID of the requested monitoring configuration. | path | Required |

## Response

### Response codes

| Code | Type | Description |
| --- | --- | --- |
