# 🚀 Groq AI - Супер-Быстрый Чат

Получайте **мгновенные ответы** на вопросы о Dynatrace с помощью **Groq Llama 3.3 70B**!

## ⚡ Как использовать

1. **Нажмите кнопку "AI ⚡"** в левом нижнем углу любой страницы
2. **Задайте вопрос** на русском или английском
3. **Получите мгновенный ответ** (обычно за 1-2 секунды!)

## 🎯 Особенности

| Функция | Описание |
|---------|----------|
| ⚡ **Мгновенные ответы** | В 10 раз быстрее обычных AI |
| 🌐 **Двуязычный** | Понимает русский и английский |
| 📄 **Контекст страницы** | Читает текущую страницу документации |
| 💬 **История диалога** | Помнит предыдущие вопросы в сессии |
| 🆓 **Бесплатный** | Groq API полностью бесплатный |
| 🛡️ **Rate limiting** | 8-10 запросов в минуту |

## 🔧 Режимы работы

### Локальный сервер (рекомендуется)
```bash
# API ключ на сервере — безопасно
python local_server.py
# Открыть http://localhost:5000
```

### GitHub Pages (Direct API)
На GitHub Pages чат работает напрямую с Groq API.
Для настройки создайте файл `docs/assets/ai-config.json`:
```json
{
  "groq_api_key": "gsk_ваш_ключ"
}
```

## 🤖 О технологии

**Groq** — инновационная компания, создавшая специальные чипы (LPU) для AI.
Модель **Llama 3.3 70B** работает в **10-100 раз быстрее** обычных GPU.

- **Модель:** Meta Llama 3.3 70B Versatile
- **Скорость:** ~500 токенов/секунду
- **Стоимость:** Бесплатно (с rate limits)
- **Контекст:** 128K токенов

!!! tip "Совет"
    Кнопка AI ⚡ доступна на **каждой странице** документации!
