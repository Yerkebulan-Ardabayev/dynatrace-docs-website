# Dynatrace Platform Subscription API


* Latest Dynatrace
* Explanation
* 1-min read
* Published Sep 03, 2025

The API lets you seamlessly integrate the management of your DPS subscription into your an operational workflow.
Use the Dynatrace Platform Subscription API to get:

* DPS subscription overview and details.
* Aggregated usage data of your Dynatrace Platform Subscription.
* Aggregated cost data of your Dynatrace Platform Subscription.
* Forecasts and cost events.
* Cost Allocation information.

Complete documentation of the Dynatrace Platform Subscription API is available at [Dynatrace Platform Subscription API](../dynatrace-api/account-management-api/dynatrace-platform-subscription-api.md "Query the data about your Dynatrace Platform Subscription via the Account Management API.").
