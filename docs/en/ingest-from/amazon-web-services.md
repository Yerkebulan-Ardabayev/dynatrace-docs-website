# Set up Dynatrace on Amazon Web Services


* Latest Dynatrace
* How-to guide
* 2-min read
* Published Aug 12, 2021

Dynatrace offers both infrastructure monitoring of all AWS services and full-stack monitoring of services that can be instrumented with our OneAgent technology. On top of that, you can choose to monitor logs generated by the cloud services as well as your own services hosted within virtual machines or containers.
