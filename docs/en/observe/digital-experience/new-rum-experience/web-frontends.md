Prerequisites

* Ensure you have the permissions described in New RUM Experience permissions.
* The minimum supported RUM JavaScript version is RUM JavaScript version 1.329. Older versions starting with RUM JavaScript version 1.317 were part of the preview and do not provide the full set of features. We recommend to update the RUM JavaScript version automatically as described in [Configure the RUM JavaScript version for a frontend](web-frontends/additional-configuration/rum-javascript-version.md#configure-js-version "Learn how to control the RUM JavaScript version used to monitor your frontends in the New RUM Experience.").
* While the minimum required OneAgent version is OneAgent version 1.301+, we recommend using the latest available version, as new features are added regularly.
* Ensure that your infrastructure complies with the firewall constraints for RUM.
