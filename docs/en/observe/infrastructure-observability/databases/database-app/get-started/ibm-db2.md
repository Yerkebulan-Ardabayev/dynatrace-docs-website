# Monitor IBM DB2 database


* Latest Dynatrace
* How-to guide
* Published Jan 20, 2026

Use Dynatrace Extension framework to extend your application observability into data acquired directly from your IBM Database layer and monitor how database server tasks impact your app. To learn more, see Get started with IBM DB2 database extension.

## Prerequisites

Ensure your system meets these requirements for full feature support.

## Set up IBM DB2 extension for monitoring

To set up and activate the extension, follow these detailed activation steps.

1. Install the IBM DB2 extension.
2. Define endpoints.
3. Select an ActiveGate group.
4. Activate the extension.
