# Grail

# Grail

* Latest Dynatrace
* Overview
* 1-min read
* Published Feb 07, 2023

Grail is the Dynatrace data lakehouse designed explicitly for observability data. It acts as a single unified storage solution for logs, metrics, traces, events, and more. All data stored in Grail is interconnected within a real-time model that reflects the topology and dependencies within a monitored environment.
