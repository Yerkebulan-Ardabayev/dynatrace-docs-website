# Mobile and custom app API


* Reference
* Published Nov 05, 2020

### List all apps

Get an overview of all mobile and custom apps.### View an app

Get parameters of an app by its ID.

### Create an app

Create a new mobile or custom app with the exact parameters you need.### Edit an app

Update an existing mobile or custom application or create a new app with the specified ID.### Delete an app

Delete an app you don't need anymore.

### View key user actions

Get the list of key user actions in the specified application.### Edit key user actions list

Mark a user action as the key action in the specified application.### Delete a key user action

Remove a user action from the list of key actions in the specified application.

### List all user session properties

Get an overview of all session properties of an app.### View a user session property

Get parameters of a session property its ID.

### Create a user session property

Create a new user session property for your mobile or custom app.### Edit a user session property

Update an existing user session property for your mobile or custom app.### Delete a user session property

Delete a user session property you don't need anymore.
