# Request attributes API


* Reference
* Published Dec 05, 2018

The **Request attributes** API enables you to manage the configuration of request attributes.

### List all

Get an overview of all request attributes.### View a request attribute

Get parameters of a request attribute by its ID.

### Create a request attribute

Create a new request attribute with the exact parameters you need.### Edit a request attribute

Update an existing request attribute or create a new request attribute with the specified ID.### Delete a request attribute

Delete a request attribute you don't need anymore.
