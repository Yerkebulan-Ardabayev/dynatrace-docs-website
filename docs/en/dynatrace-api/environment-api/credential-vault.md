# Credential vault API


* Reference
* Published Oct 14, 2019

[### List all credentials

Get an overview of all credentials configured in your Dynatrace environment.](credential-vault/get-all.md "Fetch the list of stored credentials via the Dynatrace API.")[### View credentials metadata

Get credentials parameters by configuration ID.](credential-vault/get-credentials.md "View a stored credentials configuration via the Dynatrace API.")

[### Create credentials

Create a new credentials configuration with the exact parameters you need.](credential-vault/post-credentials.md "Create a credentials configuration via Dynatrace API.")[### Edit credentials

Update the existing configuration of credentials.](credential-vault/put-credentials.md "Update a stored credentials configuration via the Dynatrace API.")[### Delete credentials
