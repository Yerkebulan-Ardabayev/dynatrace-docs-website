# Identity and access management (IAM)

* Latest Dynatrace
* Overview
* 1-min read
* Published Feb 06, 2023

Dynatrace provides an Identity and Access Management (IAM) solution that supports the rising demand for security and compliance, and that is resilient and easy to maintain. We provide scalable, effortless, and flexible access management for the whole Dynatrace ecosystem, enabling you to handle users and permissions with ease.

Dynatrace supports most industry standards for authentication and authorization, such as OIDC and OAuth 2.0, as well as SAML and SCIM for user and group federation.
