    Модель поддержки Dynatrace для VMware Tanzu Application Service¶

    • Последняя версия Dynatrace
    • Чтение: 5 мин
    • Обновлено 10 мая 2024

    VMware поддерживает релизы N, N-1 и N-2 Tanzu Application Service. Подробности можно найти в политике поддержки VMware.

    Dynatrace придерживается той же модели поддержки, что и VMware, обеспечивая согласованный цикл обновлений. Однако поддержка OneAgent и Dynatrace Service Broker для Tanzu может отставать от релизов VMware, чтобы обеспечить достаточное тестирование.

    Подробнее см. в объявлениях о прекращении поддержки для OneAgent.

    Существуют две разновидности Dynatrace OneAgent BOSH Release. Полный BOSH-релиз OneAgent не содержит установщик OneAgent. Вместо этого установщик загружается из вашей среды Dynatrace во время развёртывания релиза. Мы рекомендуем использовать последний релиз дополнения Dynatrace OneAgent BOSH, поскольку он содержит последние улучшения. Тем не менее, минимально необходимые версии BOSH-релиза для каждой версии VMware Tanzu Application Service также указаны выше.

    1

    Все вышеперечисленные версии были проверены с несколькими версиями OneAgent.

    2

    До мая 2020 года (до версии 1.191) Dynatrace предоставляла трёхмесячный льготный период для обновления дистрибутивов Cloud Foundry из-за расхождения графиков поставки. Начиная с мая 2020 года Dynatrace ускорила цикл тестирования. Поэтому наша модель поддержки больше не требует трёхмесячного льготного периода. Для всех релизов OneAgent до мая 2020 года (ранее версии OneAgent 1.191) Dynatrace продлит поддержку на три месяца, чтобы учесть существующие установки OneAgent на основе предыдущей модели поддержки.

    3

    Существуют две схемы версионирования. Неизменяемый BOSH-релиз OneAgent версионируется так же, как и OneAgent. Неизменяемый BOSH-релиз OneAgent является рекомендованным способом развёртывания OneAgent. Предыдущие версии использовали облегчённый подход, доступный на GitHub, и используют схему версионирования GitHub.
