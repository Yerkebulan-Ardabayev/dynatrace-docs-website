    DQL запрос действий для Workflows

    • Последнее Dynatrace
    • Справка
    • 1-минутное чтение

    Действие Выполнить запрос DQL для Workflows позволяет использовать запросы DQL в рабочем процессе. В запросе DQL введите свой запрос DQL. В Сегменте вы можете необязательно выбрать фильтровый сегмент.

    Вы можете найти результаты во вкладке Результат выполнения, который можно использовать в последующих задачах.
