    Коннекторы Workflows

    • Latest Dynatrace

    Коннекторы Workflows можно выбрать из каталога Dynatrace Platform. Коннекторы Workflows предоставляют различные действия, которые можно использовать в задачах рабочего процесса. Например, можно использовать действия для создания задач в Jira, отправки уведомлений в Slack и маршрутизации на основе извлечённой информации о владельцах сущностей.

    Владение¶

    Действия рабочего процесса Ownership доступны непосредственно в Workflows после установки приложения Ownership. Действия Ownership позволяют извлекать контактную информацию о владельцах сущностей и использовать её в рабочих процессах для уведомлений и назначения задач.](../../deliver/ownership-app.md "Предоставляет пользовательские действия для определения рабочих процессов, интегрирующих владельцев сущностей и их контактную информацию.")[

    Site Reliability Guardian¶

    Действие рабочего процесса Site Reliability Guardian доступно непосредственно в Workflows после установки приложения Site Reliability Guardian. Оно позволяет выполнять guardian непосредственно в рабочих процессах.](../../deliver/site-reliability-guardian.md#automation "Автоматически проверяйте цели по производительности, доступности и ёмкости критически важных сервисов для принятия правильных решений о релизах.")### Dynatrace Intelligence (Preview)
