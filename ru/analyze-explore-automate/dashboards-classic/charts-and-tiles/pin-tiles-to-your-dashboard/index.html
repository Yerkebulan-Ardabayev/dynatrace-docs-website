    Закрепление плиток на дашборде¶

    Dashboards Classic

    Эта страница относится к классическим дашбордам, созданным с помощью функциональности Dashboards Classic, интегрированной с Dynatrace Classic.

    • Если вы уже используете приложение Dashboards в последней версии Dynatrace, см. Dashboards для соответствующей документации.
    • Если вы по-прежнему используете классические дашборды, мы рекомендуем обновить ваши дашборды и воспользоваться всеми новейшими возможностями, предоставляемыми приложением Dashboards в последней версии Dynatrace.

    Вы можете автоматически создать новую плитку дашборда с любой страницы Dynatrace, содержащей кнопку Pin to dashboard. Всё, что вам нужно, это разрешение на редактирование для дашборда. В этом примере мы создаём новый дашборд, поэтому вам гарантированно предоставлено разрешение на редактирование.

    Создание дашборда¶

    Начнём с создания пустого дашборда.

    1. Перейдите в Dashboards Classic.
    2. Нажмите Create dashboard, введите Dashboard name (в данном примере используйте Filtered Dashboard) и нажмите Create. Это создаст дашборд и откроет его в режиме редактирования.
    3. Нажмите Done. Теперь у нас есть пустой дашборд с именем Filtered Dashboard.

    Создание фильтрованной плитки¶

    Теперь мы создадим плитку на основе отфильтрованного представления страницы Hosts и закрепим плитку на пустом дашборде.

    1. Перейдите в Hosts Classic.
    2. Установите фильтр для списка. В этом примере мы устанавливаем Data center: Gdansk, Poland.

    Как установить фильтр

    1. В строке Filtered by начните вводить текст для выбора фильтра

      2. Выберите или введите значение

    Для наиболее распространённых фильтров можно выбрать фильтр непосредственно из столбца Quick filters на странице Hosts. 3. Нажмите Pin to dashboard.

    Dynatrace отобразит предварительный просмотр плитки и предложит вам выбрать дашборд, к которому вы хотите её закрепить. Список дашбордов поддерживает поиск: нажмите на список и начните вводить текст для фильтрации, затем выберите дашборд из отфильтрованного списка.

    4. Выберите дашборд (в данном примере Filtered Dashboard) и нажмите Pin. Это создаст плитку и закрепит её на выбранном дашборде.

    5. Нажмите Open dashboard. Ранее пустой дашборд теперь откроется в режиме редактирования с выбранной для редактирования новой плиткой. Вы можете выполнить дополнительные настройки этой плитки перед сохранением. Для плитки Host health доступны следующие параметры:

    • Необязательно: выберите, показывать ли визуализацию на плитке
    • Необязательно: выберите пользовательский временной диапазон
    • Необязательно: выберите пользовательскую зону управления
    • Необязательно: выберите окружение

    6. Нажмите Done. Дашборд отобразится с новой плиткой, представляющей хосты в Гданьске, Польша.

    Теперь у нас есть плитка, относящаяся к хостам в центре обработки данных Гданьска, Польша.

    • Фильтр полностью динамический. При добавлении или удалении хостов, а также при изменении их статусов плитка обновляется автоматически.
    • Чтобы просмотреть отфильтрованную страницу Hosts, с которой мы создали плитку, откройте меню в правом верхнем углу плитки и выберите View details.

    Добавление заголовка к плитке¶

    Необязательно

    Чтобы добавить информативную метку к вашей плитке

    1. Нажмите Edit для редактирования дашборда.
    2. Перетащите плитку Header из панели Edit dashboard на ваш дашборд. Это будет заголовок вашей плитки.
    3. Отредактируйте текст заголовка. В данном примере мы используем Gdansk hosts.
    4. Перетащите плитку Header в положение над плиткой Hosts. В зависимости от положения плитки Hosts на дашборде вам может потребоваться переместить обе плитки, чтобы разместить плитку Header над плиткой Hosts.

    5. Нажмите Done.

    Обновление фильтрованной плитки¶

    Чтобы изменить фильтры на вашей плитке

    1. На дашборде выберите плитку, чтобы отобразить таблицу Hosts с применённым фильтром.
    2. Добавьте или удалите фильтры.

    В данном примере мы сохраняем Data center: Gdansk, Poland и добавляем Operating system: Linux. Поскольку мы изменили фильтры, кнопка Update dashboard tile становится активной, чтобы вы могли сохранить изменения.

    3. Нажмите Update dashboard tile. Новые настройки фильтра сохраняются в плитке. В данном примере плитка на вашем дашборде отображает состояние хостов, соответствующих Data center: Gdansk, Poland и Operating system: Linux, и открывает страницу Hosts с этими фильтрами. 4. Чтобы вернуться к дашборду, нажмите кнопку дашборда в верхнем левом углу Dynatrace.

    Закрепление как новой плитки¶

    Вместо обновления плитки вы можете сохранить новую копию плитки. Например, вам могут понадобиться несколько похожих плиток, каждая из которых отфильтрована по разным операционным системам.

    1. На том же примере дашборда откройте меню плитки (в правом верхнем углу плитки) и выберите View details. Откроется страница Hosts с теми же установленными фильтрами.

    2. Выберите More (...) > Pin as new tile.

    3. Выберите другой дашборд, чтобы иметь копии одной и той же плитки на двух разных дашбордах.

    4. Выберите тот же дашборд, чтобы иметь две копии плитки на одном дашборде, что может показаться бессмысленным. Однако:

      • Вы можете редактировать каждую копию плитки, задавая разные зоны управления, временные диапазоны и окружения
      • Вы можете установить разные фильтры для каждой копии плитки

    Клонирование плиток¶

    Клонирование плиток может сэкономить вам много работы. Например, предположим, что вы хотите, чтобы ваш дашборд отображал три визуализации, идентичные за исключением того, что каждая запрашивает данные другого хоста. Самый простой способ -- создать первую визуализацию, настроить параметры визуализации до совершенства, а затем клонировать её дважды. После этого вам нужно лишь отредактировать клоны для запроса двух других хостов.

    Клонирование плитки на текущий дашборд¶

    Чтобы закрепить клонированные плитки на том же дашборде

    1. Перейдите в Dashboards Classic.
    2. Выберите имя дашборда, чтобы отобразить его.

    3. Нажмите Edit в правом верхнем углу дашборда. Дашборд откроется в режиме редактирования.

    4. Если вы не видите опцию Edit, у вас нет разрешения на редактирование этого дашборда.

    5. Выберите плитку, которую хотите клонировать (добавить копию плитки на текущий дашборд), и нажмите Clone.

    6. Чтобы клонировать несколько плиток на текущий дашборд, выделите прямоугольной областью x плиток, которые хотите клонировать, и нажмите Clone x tiles.

    7. Отредактируйте клонированные плитки по необходимости.

    Клонирование плитки на другой дашборд¶

    Чтобы закрепить клонированные плитки на другом существующем дашборде или создать новый дашборд и закрепить клоны на нём

    1. Перейдите в Dashboards Classic.
    2. Выберите имя дашборда, чтобы отобразить его.

    3. Нажмите Edit в правом верхнем углу дашборда. Дашборд откроется в режиме редактирования.

    4. Если вы не видите опцию Edit, у вас нет разрешения на редактирование этого дашборда.

    5. Выберите плитку, которую хотите клонировать на другой дашборд, и нажмите Clone to.

    6. Чтобы клонировать несколько плиток на другой дашборд, выделите прямоугольной областью x плиток, которые хотите клонировать, и нажмите Clone x to.

    7. Во всплывающем окне Where do you want to pin to? выберите существующий дашборд или Create new dashboard.
    8. Нажмите Pin, чтобы добавить выбранные плитки на дашборд.
    9. Отредактируйте клонированные плитки по необходимости.
