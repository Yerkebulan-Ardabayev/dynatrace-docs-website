    Готовые панели мониторинга¶

    • Последнее Dynatrace
    • Справка
    • 9-минутное чтение

    Готовые панели мониторинга Dynatrace предлагают предварительно настроенные визуализации данных и фильтры, разработанные для общих сценариев, таких как устранение неполадок и оптимизация.

    • Используйте их сразу же
    • Сохраните копию и настройте свою копию

    Где найти готовые панели мониторинга

    1. В Dynatrace, перейдите к Панели мониторинга.
    2. Выберите способ отображения всех готовых панелей мониторинга.

    3. Выберите готовую панель мониторинга, которую вы хотите использовать.
    Попробуйте ссылки Explore in Playground ниже, чтобы увидеть их в действии.

    Использование панелей мониторинга только для чтения

    Когда вы открываете документ (панель мониторинга или тетрадь), для которого у вас нет разрешения на запись, вы все равно можете редактировать документ во время сеанса. После завершения у вас есть два варианта:

    • Сохранить изменения в новый документ
    • Отменить изменения

    Пример:

    1. Перейдите к Панели мониторинга, отобразите готовые панели мониторинга и выберите панель мониторинга Начало работы.

    В верхнем левом углу, рядом с именем документа, написано Готовая. 2. Выберите плитку круговой диаграммы и затем выберите Редактировать. 3. Измените визуализацию с круговой диаграммы на пончик.

    Теперь у вас есть две кнопки: Сохранить как новый и Отменить изменения. 4. Используйте обновленную панель мониторинга по мере необходимости. У вас есть полный доступ к редактированию для этой сессии. 5. Когда вы закончите, выберите, что делать с вашими изменениями:

    • Сохранить как новый — сохраняет ваши изменения в новой копии отредактированной панели мониторинга.
    • Отменить изменения — отменяет ваши изменения и возвращает вас к неизменной панели мониторинга только для чтения.

    Обзор диагностики ActiveGate¶

    Предлагает фильтруемый обзор диагностики ActiveGate с разделами для:

    • Виталины хоста — метрики здоровья, связанные с хостом или контейнером, где запущен ActiveGate.
    • Процесс — метрики здоровья, связанные с процессом Java ActiveGate.
    • Сетевое взаимодействие — входящий и исходящий сетевой трафик.
    • REST.API — вызовы API, ошибки, размер запроса и размер ответа.

    Explore in Playground.

    Связанное приложение Dynatrace: Dynatrace ActiveGate
