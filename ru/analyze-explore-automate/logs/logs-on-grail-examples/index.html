    Log on Grail examples¶

    • Latest Dynatrace
    • Overview
    • 17-min read
    • Updated on Oct 15, 2025

    Log Management and Analytics powered by Grail enables you to pinpoint and retrieve any log data with the help of Dynatrace Query Language. After reviewing the fundamentals of DQL queries, use the examples on this page to start getting answers from your log data.

    To run DQL queries with logs on Grail, go to Logs & Events Classic > Advanced mode.

    • Example 1 - Get the distribution of HTTP status codes and counts per error type.
    • Example 2 - Define an average cart size based on logs.
