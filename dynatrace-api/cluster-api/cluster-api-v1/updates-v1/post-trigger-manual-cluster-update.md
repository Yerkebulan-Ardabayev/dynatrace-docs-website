Я понимаю процедуру, описанную в разделе **📋 Как перевести**:

2.  Использовать одну из следующих команд:
    *   `переведи observe/dashboards.md` (для конкретного файла)
    *   `переведи всё новое` (для пакетного перевода)
3.  Claude выполнит следующие шаги: прочитает файл, переведёт его, сохранит результат в директорию `docs/managed-ru/`, и обновит навигацию.
4.  После завершения процесса останется только выполнить `git push` для автоматического обновления сайта.
