    Configuration types and access permissions¶

    Configuration types and access permissions¶

    • Reference
    • 7-min read
    • Updated on Feb 26, 2025

    Dynatrace Configuration as Code через Monaco (Dynatrace Monaco CLI) поддерживает следующие типы конфигурации:

    • Settings 2.0
    • Configuration APIs

    Подробности об определении типов конфигурации см. в YAML-конфигурации Monaco.

    Settings 2.0¶

    Settings 2.0 требует токена доступа.

    Dynatrace Monaco CLI поддерживает любую схему Settings 2.0, доступную в вашем окружении.

    • Подробности о схемах Settings 2.0 см. в Settings 2.0 — доступные схемы.

    • Для настройки Settings 2.0 ваш токен доступа должен иметь следующие разрешения:

    • Read settings (settings.read)

    • Write settings (settings.write)

    Поддерживаемые типы Configuration API¶

    Этот тип конфигурации доступен для последней платформы Dynatrace и предыдущих окружений Dynatrace.

    Обратите внимание, что большинство Configuration API устарели в пользу Settings 2.0.

    Конфигурация требует токена доступа.
