    ZDTP001S¶

    • Полное сообщение - ZDTP001S Persistent Storage allocation failed.
    • Пояснение - Программа CICS PLT не смогла выделить необходимую область постоянной памяти.
    • Действие системы - Программа PLT завершается, и кодовый модуль CICS не включается.
    • Реакция пользователя - Выясните, по какой причине регион CICS не может предоставить память для транзакции DTAX.

    Если проблема не устраняется, обратитесь к специалисту по продуктам Dynatrace через чат в вашем окружении Dynatrace.

    • Пояснение - Команда EXEC CICS START, используемая для запуска транзакции DTAX ICE с интервалом 5 минут, завершилась ошибкой.

    • Реакция пользователя - Изучите подробное сообщение в соответствующем журнале задания CICS, чтобы получить точный код EIBRESP, указывающий причину сбоя команды EXEC CICS START.
