    Управление аккаунтом¶

    Управление аккаунтом¶

    • Пояснение
    • Чтение 5 мин
    • Обновлено 24 июн 2025 г.

    Управление аккаунтом Dynatrace предоставляет единое место для:

    • Управления лицензией и подписками Dynatrace.

    • Мониторинга использования платформы Dynatrace и работоспособности окружений.

    Доступ¶

    Веб-интерфейс¶

    Чтобы открыть Управление аккаунтом через веб-интерфейс, выберите значок пользователя в правом верхнем углу, затем выберите Account settings.

    Отображение меню пользователя

    Адресная строка браузера¶

    Чтобы перейти в Управление аккаунтом непосредственно из адресной строки браузера, перейдите по адресу https://myaccount.dynatrace.com/.

    Разрешения¶

    Для получения информации о разрешениях, необходимых для доступа к Управлению аккаунтом, см. Разрешения Управления аккаунтом.

    Навигация верхнего уровня¶

    На верхнем уровне Управления аккаунтом расположены три вкладки для управления информацией.

    My accounts¶

    Управление аккаунтом: My accounts

    Вкладка My accounts отображает ваши аккаунты.

    • Выберите аккаунт, которым хотите управлять.
    • После выбора нужного аккаунта вы можете добавить страницу аккаунта в закладки для прямого доступа в следующий раз.

    Чтобы вернуться на вкладку My accounts позже, откройте меню в правом верхнем углу и выберите My accounts.

    My profile¶

    Управление аккаунтом: My profile

    Используйте вкладку My profile для обновления профиля пользователя и настроек уведомлений.

    • My profile — информация о вас: имя, должность и местонахождение.

    Настройка часового пояса здесь носит информационный характер и не влияет на функциональность Dynatrace, кроме генерации отчётов для подписок на информационные панели в Dashboards Classic. * Environment notifications — список окружений, для которых можно включить email-уведомления об отключениях.

    Чтобы вернуться на вкладку My profile позже, откройте меню в правом верхнем углу и выберите имя своего аккаунта.

    My platform tokens¶

    Управление аккаунтом: My platform tokens

    Platform tokens позволяют взаимодействовать с платформой Dynatrace через API для внешних интеграций или скриптов. Подробнее см. Недоступно в Dynatrace Managed.

    Чтобы вернуться на вкладку My platform tokens позже, откройте меню в правом верхнем углу и выберите Platform tokens.

    Навигация на уровне аккаунта¶

    После выбора аккаунта на вкладке My accounts в Управлении аккаунтом отображается меню-панель на большом экране.

    • На маленьком экране панель меню сворачивается в компактное меню с теми же параметрами.
    • Доступные параметры меню зависят от типа развёртывания, прав пользователя и модели лицензирования.

    Примеры¶

    Пример меню-панели Управления аккаунтом: License

    Пример меню-панели Управления аккаунтом: Subscription

    Home¶

    Выберите Home из любого места в Application Management, чтобы вернуться на главную страницу выбранного аккаунта.

    License / Subscription¶

    • License отображается для классических контрактов
    • Subscription отображается для DPS

    Откройте меню License или Subscription, чтобы:

    • Просмотреть использование и историю потребления лицензии/подписки

    • Просмотреть прогноз будущего использования

    • Просмотреть все уведомления о квотах лицензий SaaS и Managed

    Обзор¶

    • License > Overview

    Потребление основано на различных типах единиц мониторинга, которые расходуются вашими окружениями Dynatrace при мониторинге приложений и связанных сервисов. * Subscription > Overview

    Dynatrace Platform Subscription: потребление основано на различных типах единиц мониторинга, которые расходуются вашими окружениями Dynatrace. Отображаемые значения представляют собой вычет из вашего годового обязательства.

    Identity & access management¶

    Для клиентов Dynatrace SaaS и развёртываний Dynatrace Platform Subscription, использующих модель SaaS, откройте меню Identity & access management (IAM) для управления:

    • Группами пользователей и разрешениями
    • Пользователями и группами с SAML в Dynatrace SaaS
    • Пользователями и группами с SCIM в Dynatrace SaaS
    • API OAuth tokens для Account Management API

    IAM для развёртываний Dynatrace Managed управляется внутри каждого кластера. Подробнее см. Группы пользователей, разрешения и политики.

    People¶

    Используйте Identity & access management > People для приглашения и назначения пользователей в группы с целью управления разрешениями и доступом в Dynatrace.

    Groups¶

    Используйте Identity & access management > Group management для создания групп пользователей с целью управления разрешениями и доступом.

    Policies¶

    Используйте Identity & access management > Policy management для управления политиками Dynatrace.

    Domain verification¶

    Используйте Identity & access management > Domain verification для проверки доменов аккаунта, используемых в конфигурациях SSO, таких как SAML или SCIM.

    SAML configuration¶

    Используйте Identity & access management > SAML configuration для настройки аутентификации пользователей для нескольких доменов. Если вы хотите использовать корпоративные учётные данные для аутентификации в Dynatrace SaaS, вы можете настроить SAML для делегирования аутентификации вашему поставщику удостоверений.

    SCIM configuration¶

    Используйте Identity & access management > SCIM configuration для управления удостоверениями пользователей в облачных приложениях и сервисах. SCIM используется для автоматизации обмена удостоверениями пользователей между разными доменами и системами.

    API OAuth clients¶

    Используйте Identity & access management > OAuth clients для настройки и управления клиентами OAuth API аккаунта. Account Management API помогает управлять аккаунтом и пользователями.

    Lens¶

    Функция Lens в Управлении аккаунтом предоставляет аналитику по использованию платформы и её работоспособности, помогая понять, как Dynatrace используется в вашей организации, и рекомендует области для оптимизации.

    Подробнее о Lens см. Lens.

    Adoption¶

    Страница Lens > Adoption даёт представление о принятии платформы и её работоспособности:

    • Активность пользователей
    • Какие технологии отслеживаются в Dynatrace
    • Способ развёртывания мониторируемых хостов (тип лицензии, тип платформы)
    • Количество запущенных синтетических тестов
    • Количество настроенных интеграций уведомлений о проблемах

    Подробнее см. Adoption.

    Environments¶

    Lens > Environments предлагает:

    • Обзор количества технологий, обнаруженных и отслеживаемых Dynatrace с течением времени.
    • Сводку по хостам окружений в вашем развёртывании Dynatrace.
    • Сводку о количестве мониторируемых хостов, развёрнутых в облаке.
    • Сводку о количестве публичных и частных синтетических тестов, запущенных в ваших окружениях.
    • Общую сводку о количестве определённых API tokens, интеграций уведомлений о проблемах, используемых PaaS-интеграций и экспортов сессий.

    Подробнее см. Environments.

    Settings¶

    Используйте Settings для указания общей информации об аккаунте.

    Подробнее см. Settings.

    Contact information¶

    Используйте Settings > Contact information для указания контактной, платёжной и адресной информации аккаунта.

    Environments¶

    Используйте Settings > Environments для просмотра списка окружений и, с помощью Action > Edit environment, изменения имени или часового пояса выбранного окружения.

    Настройка часового пояса здесь носит информационный характер и не влияет на функциональность Dynatrace, кроме генерации отчётов для подписок на информационные панели в Dashboards Classic.

    Back to Account Settings¶

    Выберите Back to Account Settings для приобретения или продления аккаунта Dynatrace.

    Уведомления¶

    Выберите Notifications для просмотра истории уведомлений о бюджете и лимитах кластера или окружений.

    Ресурсы поддержки¶

    Выберите Support для перехода к ресурсу поддержки:

    • Support Center
    • Документация
    • Сообщество
    • University
    • Тикеты поддержки

    Chat¶

    Выберите Chat для связи со специалистом по продукту.

    Profile¶

    Выберите Profile для проверки аккаунта, под которым вы вошли, просмотра своих аккаунтов, platform tokens или выхода из системы.
